Question #973
A company is designing a web application with an internet-facing Application Load Balancer (ALB).
The company needs the ALB to receive HTTPS web traffic from the public internet. The ALB must send only HTTPS traffic to the
web application servers hosted on the Amazon EC2 instances on port 443. The ALB must perform a health check of the web
application servers over HTTPS on port 8443.
Which combination of configurations of the security group that is associated with the ALB will meet these requirements? (Choose
three.)
一家公司正在设计一个具有面向互联网的 Application Load Balancer (ALB) 的网页应用程序。
该公司需要 ALB 从公共互联网接收 HTTPS 网络流量。ALB 必须仅将 HTTPS 流量发送到托管在 Amazon EC2 实例上的网页应用程序
服务器的 443 端口。ALB 必须通过 8443 端口上的 HTTPS 对网页应用程序服务器执行健康检查。
与 ALB 关联的安全组的哪些配置组合可以满足这些要求？（选择三个）

Correct Answer: ACE
正确答案：ACE
题目分析与解答
1. 考察的知识点
安全组配置：如何正确设置安全组规则以满足应用程序的需求。
HTTPS流量管理：理解HTTPS流量的入站和出站规则。
健康检查：如何配置安全组以支持ALB的健康检查功能。
2. 选项分析
A. Allow HTTPS inbound traffic from 0.0.0.0/0 for port 443.
正确：ALB需要接收来自互联网的HTTPS流量，因此需要允许从0.0.0.0/0到443端口的入站流量。
B. Allow all outbound traffic to 0.0.0.0/0 for port 443.
错误：ALB不需要向互联网发送HTTPS流量，出站流量应该限制到目标实例。
C. Allow HTTPS outbound traffic to the web application instances for port 443.
正确：ALB需要将HTTPS流量发送到托管Web应用程序的EC2实例，因此需要允许到443端口的出站流量。
D. Allow HTTPS inbound traffic from the web application instances for port 443.
错误：Web应用程序实例不会向ALB发送HTTPS流量，因此不需要此规则。
E. Allow HTTPS outbound traffic to the web application instances for the health check on port 8443.
正确：ALB需要对Web应用程序实例进行健康检查，健康检查使用HTTPS协议并通过8443端口，因此需要允许此出站流量。
F. Allow HTTPS inbound traffic from the web application instances for the health check on port 8443.
错误：健康检查是ALB向实例发起的请求，实例不会向ALB发送流量，因此不需要此规则。

A company has an internal application that runs on Amazon EC2 instances in an Auto Scaling group. The EC2 instances are
compute optimized and use Amazon Elastic Block Store (Amazon EBS) volumes.
The company wants to identify cost optimizations across the EC2 instances, the Auto Scaling group, and the EBS volumes.
Which solution will meet these requirements with the MOST operational efficiency?

该公司希望识别 EC2 实例、Auto Scaling 组和 EBS 卷的成本优化。
哪种解决方案能够以最高的运营效率满足这些需求？

题目分析与解答
1. 考察的知识点
本题主要考察AWS成本优化工具的使用，包括AWS Compute Optimizer和AWS Cost and Usage Report。
了解如何高效地获取成本优化建议以减少运营成本。
2. 选项分析
A. Create a new AWS Cost and Usage Report. Search the report for cost recommendations for the EC2 instances, the Auto
Scaling group, and the EBS volumes.
错误：AWS Cost and Usage Report主要用于提供详细的成本和使用数据，但它不会直接提供优化建议。
缺乏自动化和效率，用户需要手动分析数据。
B. Create new Amazon CloudWatch billing alerts. Check the alert statuses for cost recommendations for the EC2 instances,
the Auto Scaling group, and the EBS volumes.
错误：CloudWatch billing alerts用于监控成本阈值，但它不会提供具体的优化建议。
无法满足题目中“成本优化建议”的需求。
C. Configure AWS Compute Optimizer for cost recommendations for the EC2 instances, the Auto Scaling group and the
EBS volumes.
正确：AWS Compute Optimizer专门设计用于提供优化建议，包括EC2实例、Auto Scaling组和EBS卷。
它能够自动分析资源使用情况并提供具体的优化建议，符合题目要求的“最高运营效率”。
D. Configure AWS Compute Optimizer for cost recommendations for the EC2 instances. Create a new AWS Cost and Usage
Report. Search the report for cost recommendations for the Auto Scaling group and the EBS volumes.
错误：虽然部分使用了AWS Compute Optimizer，但仍然依赖AWS Cost and Usage Report进行手动分析。
与选项C相比，效率较低，不符合“最高运营效率”的要求。
3. 我的选择
答案：C

答案：C

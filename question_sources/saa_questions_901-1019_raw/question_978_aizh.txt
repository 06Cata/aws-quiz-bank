Question #978
A company is migrating its on-premises Oracle database to an Amazon RDS for Oracle database. The company needs to retain
data for 90 days to meet regulatory requirements. The company must also be able to restore the database to a specific point in
time for up to 14 days.
Which solution will meet these requirements with the LEAST operational overhead?
一家公司正在将其本地 Oracle 数据库迁移到 Amazon RDS for Oracle 数据库。 该公司需要保留数据 90 天以满足监管要求。 公司还
必须能够将数据库恢复到最多 14 天内的特定时间点。
哪种解决方案能够以最少的运营开销满足这些要求？
A. Create Amazon RDS automated backups. Set the retention period to 90 days.
创建 Amazon RDS 自动备份。将保留期设置为 90 天。
B. Create an Amazon RDS manual snapshot every day. Delete manual snapshots that are older than 90 days.
每天创建一个 Amazon RDS 手动快照。删除超过 90 天的手动快照。
C. Use the Amazon Aurora Clone feature for Oracle to create a point-in-time restore. Delete clones that are older than 90 days.
使用 Amazon Aurora Clone 功能为 Oracle 创建一个时间点恢复。删除超过 90 天的克隆。
D. Create a backup plan that has a retention period of 90 days by using AWS Backup for Amazon RDS. Most Voted
使用 AWS Backup 为 Amazon RDS 创建一个具有 90 天保留期的备份计划。
题目分析与解答
1. 考察的知识点
Amazon RDS备份与恢复机制，包括自动备份、手动快照和AWS Backup的使用。
Correct Answer: D
正确答案：D
Community vote distribution
社区投票分布
D (100%)
如何满足数据保留要求和点时间恢复需求。
选择最少操作开销的解决方案。
2. 选项分析
A. Create Amazon RDS automated backups. Set the retention period to 90 days.
错误：Amazon RDS自动备份的最大保留期为35天，无法满足90天的保留要求。
虽然自动备份支持点时间恢复，但无法满足90天的保留需求。
B. Create an Amazon RDS manual snapshot every day. Delete manual snapshots that are older than 90 days.
正确性：手动快照可以满足90天的保留要求。
错误：需要每天手动创建快照并管理删除操作，增加了操作开销，不符合“最少操作开销”的要求。
C. Use the Amazon Aurora Clone feature for Oracle to create a point-in-time restore. Delete clones that are older than 90
days.
错误：Amazon Aurora Clone是Aurora特有的功能，不适用于Amazon RDS for Oracle。
此选项与题目要求不符。
D. Create a backup plan that has a retention period of 90 days by using AWS Backup for Amazon RDS.
正确性：AWS Backup支持为Amazon RDS创建备份计划，并可以设置保留期为90天。
正确性：AWS Backup支持点时间恢复，且操作开销最少。
此选项完全满足题目要求。
3. 我的答案
D. Create a backup plan that has a retention period of 90 days by using AWS Backup for Amazon RDS.
4. 官方答案
D. Create a backup plan that has a retention period of 90 days by using AWS Backup for Amazon RDS.
5. 与官方答案比较
我的答案与官方答案一致。
原因：AWS Backup提供了最少的操作开销，同时满足90天数据保留和点时间恢复的需求。

Question #905

based workload hosted on Amazon EC2 instances in different AWS Regions and a stateless UDP-based workload hosted on
premises.
Which combination of actions should a solutions architect take to improve availability and performance? (Choose two.)
一家公司希望提高其混合应用程序的可用性和性能。该应用程序包括托管在不同 AWS 区域的 Amazon EC2 实例上的有状态基于
TCP 的工作负载，以及托管在本地的无状态基于 UDP 的工作负载。
解决方案架构师应采取哪些组合操作来提高可用性和性能？（选择两项）

题目分析与解答
1. 考察的知识点
本题主要考察AWS Global Accelerator的使用场景和负载均衡器（Network Load Balancer和Application Load Balancer）的配置与功
能。
涉及混合架构的高可用性和性能优化，包括跨区域流量管理和本地端点的集成。
2. 选项分析
A. Create an accelerator using AWS Global Accelerator. Add the load balancers as endpoints.
正确。AWS Global Accelerator可以通过优化路由来提高跨区域应用的性能和可用性。将负载均衡器作为端点可以支持跨区域流量管
理。
B. Create an Amazon CloudFront distribution with an origin that uses Amazon Route 53 latency-based routing to route
requests to the load balancers.
错误。CloudFront主要用于内容分发（CDN），而题目中的应用是基于TCP和UDP的工作负载，CloudFront不适合处理这种类型的
流量。
C. Configure two Application Load Balancers in each Region. The first will route to the EC2 endpoints, and the second will
route to the on-premises endpoints.
错误。Application Load Balancer主要用于HTTP/HTTPS流量，而题目中的工作负载是基于TCP和UDP的，使用ALB不合适。
D. Configure a Network Load Balancer in each Region to address the EC2 endpoints. Configure a Network Load Balancer in
each Region that routes to the on-premises endpoints.
正确。Network Load Balancer支持TCP和UDP流量，适合处理题目中的工作负载。配置NLB可以提高可用性和性能。
E. Configure a Network Load Balancer in each Region to address the EC2 endpoints. Configure an Application Load
Balancer in each Region that routes to the on-premises endpoints.
错误。虽然NLB适合处理EC2端点的TCP/UDP流量，但ALB不适合处理本地端点的TCP/UDP流量。

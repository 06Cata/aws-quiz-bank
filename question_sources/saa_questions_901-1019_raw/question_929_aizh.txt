Question #929
A healthcare company is developing an AWS Lambda function that publishes notifications to an encrypted Amazon Simple
Notification Service (Amazon SNS) topic. The notifications contain protected health information (PHI).
The SNS topic uses AWS Key Management Service (AWS KMS) customer managed keys for encryption. The company must ensure
that the application has the necessary permissions to publish messages securely to the SNS topic.
Which combination of steps will meet these requirements? (Choose three.)
一家医疗公司正在开发一个 AWS Lambda 函数，该函数会将通知发布到加密的 Amazon Simple Notification Service (Amazon SNS)
主题中。这些通知包含受保护的健康信息 (PHI)。
SNS 主题使用 AWS Key Management Service (AWS KMS) 的客户管理密钥进行加密。公司必须确保应用程序拥有必要的权限，以安
全地向 SNS 主题发布消息。
以下哪些步骤组合可以满足这些要求？（选择三项）
A. Create a resource policy for the SNS topic that allows the Lambda function to publish messages to the topic. Most Voted
为 SNS 主题创建一个资源策略，允许 Lambda 函数向该主题发布消息。
B. Use server-side encryption with AWS KMS keys (SSE-KMS) for the SNS topic instead of customer managed keys.
使用 AWS KMS 密钥的服务器端加密 (SSE-KMS) 替代客户管理密钥来加密 SNS 主题。
C. Create a resource policy for the encryption key that the SNS topic uses that has the necessary AWS KMS permissions.
Most Voted
为 SNS 主题使用的加密密钥创建一个资源策略，其中包含必要的 AWS KMS 权限。
D. Specify the Lambda function's Amazon Resource Name (ARN) in the SNS topic's resource policy.
在 SNS 主题的资源策略中指定 Lambda 函数的 Amazon Resource Name (ARN)。
E. Associate an Amazon API Gateway HTTP API with the SNS topic to control access to the topic by using API Gateway resource
policies.
将 Amazon API Gateway HTTP API 与 SNS 主题关联，以使用 API Gateway 资源策略控制对该主题的访问。
F. Configure a Lambda execution role that has the necessary IAM permissions to use a customer managed key in AWS KMS.
Most Voted
配置一个 Lambda 执行角色，该角色具有使用 AWS KMS 中客户管理密钥的必要 IAM 权限。
Correct Answer: ACF
题目分析与解答
1. 考察的知识点
Amazon SNS的资源策略配置。
AWS KMS加密与权限管理。
AWS Lambda执行角色的权限配置。
2. 选项分析
A. Create a resource policy for the SNS topic that allows the Lambda function to publish messages to the topic.
正确。SNS主题需要一个资源策略来允许Lambda函数发布消息。资源策略是控制访问SNS主题的关键部分。
B. Use server-side encryption with AWS KMS keys (SSE-KMS) for the SNS topic instead of customer managed keys.
错误。题目明确指出SNS主题已经使用了客户管理密钥进行加密，因此不需要替换为SSE-KMS。
C. Create a resource policy for the encryption key that the SNS topic uses that has the necessary AWS KMS permissions.
正确。SNS主题使用的加密密钥需要一个资源策略来授予必要的AWS KMS权限，以确保Lambda函数能够安全地使用该密钥。
D. Specify the Lambda function's Amazon Resource Name (ARN) in the SNS topic's resource policy.
错误。虽然可以在SNS资源策略中指定Lambda函数的ARN，但这并不是题目要求的昀佳实践。题目更关注加密和权限管理。
E. Associate an Amazon API Gateway HTTP API with the SNS topic to control access to the topic by using API Gateway
resource policies.
错误。API Gateway与SNS主题的关联并不相关，题目没有提到需要通过API Gateway来控制访问。
F. Configure a Lambda execution role that has the necessary IAM permissions to use a customer managed key in AWS
KMS.
正确。Lambda函数需要一个执行角色，该角色必须具有使用客户管理密钥的必要IAM权限，以便能够安全地发布加密消息。

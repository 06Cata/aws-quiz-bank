Question #930
A company has an employee web portal. Employees log in to the portal to view payroll details. The company is developing a new
system to give employees the ability to upload scanned documents for reimbursement. The company runs a program to extract
text-based data from the documents and attach the extracted information to each employee’s reimbursement IDs for processing.
The employee web portal requires 100% uptime. The document extract program runs infrequently throughout the day on an on-
demand basis. The company wants to build a scalable and cost-effective new system that will require minimal changes to the
existing web portal. The company does not want to make any code changes.
Which solution will meet these requirements with the LEAST implementation effort?
一家公司有一个员工网页门户。员工登录门户以查看工资详情。公司正在开发一个新系统，使员工能够上传扫描文件以进行报销。
公司运行一个程序从文件中提取基于文本的数据，并将提取的信息附加到每个员工的报销ID以进行处理。
员工网页门户需要100%正常运行时间。文件提取程序在一天中不定期按需运行。公司希望构建一个可扩展且成本有效的新系统，并
且对现有网页门户的更改最少。公司不希望进行任何代码更改。
哪种解决方案可以以最少的实施工作量满足这些要求？

the document extract program. Invoke the Lambda function when an employee uploads a new reimbursement document.

题目分析与解答
1. 考察的知识点
高可用性架构设计：确保员工门户具有100%正常运行时间。
无服务器计算：使用AWS Lambda实现按需运行的文档提取程序。
成本优化：选择最具成本效益的解决方案。
最小化实施工作量：避免对现有系统进行代码更改。
2. 选项分析
A. Run Amazon EC2 On-Demand Instances in an Auto Scaling group for the web portal. Use an AWS Lambda function to
run the document extract program. Invoke the Lambda function when an employee uploads a new reimbursement
document.
正确性分析：EC2 On-Demand Instances可以提供高可用性和按需扩展的能力，满足门户的100%正常运行时间要求。AWS Lambda
是无服务器的，适合运行不频繁的文档提取程序。调用Lambda函数时无需对现有代码进行更改。
优点：高可用性、成本优化、最小化实施工作量。
缺点：无明显缺点。
B. Run Amazon EC2 Spot Instances in an Auto Scaling group for the web portal. Run the document extract program on
EC2 Spot Instances. Start document extract program instances when an employee uploads a new reimbursement
document.
正确性分析：EC2 Spot Instances价格低廉，但不适合需要100%正常运行时间的门户，因为Spot Instances可能会被中断。
优点：成本较低。
缺点：无法保证门户的高可用性，且需要额外的逻辑来启动文档提取程序实例。
C. Purchase a Savings Plan to run the web portal and the document extract program. Run the web portal and the
document extract program in an Auto Scaling group.
正确性分析：Savings Plan可以降低EC2成本，但将文档提取程序与门户运行在同一Auto Scaling组中会增加复杂性，并且不符合按
需运行的要求。
优点：成本优化。
缺点：无法实现文档提取程序的按需运行，增加了系统复杂性。
D. Create an Amazon S3 bucket to host the web portal. Use Amazon API Gateway and an AWS Lambda function for the
existing functionalities. Use the Lambda function to run the document extract program. Invoke the Lambda function
when the API that is associated with a new document upload is called.
正确性分析：将门户托管在S3上可以降低成本，但需要对现有门户进行代码更改以支持API Gateway，这与题目要求的“无需代码更
改”相矛盾。
优点：成本优化。
缺点：需要对现有系统进行代码更改，实施工作量较大。

题目分析与解答
1. 考察的知识点
Amazon CloudFront与Amazon S3的集成。
AWS Certificate Manager (ACM)的使用及区域要求。
Amazon S3的上传配置，包括源访问控制 (OAC) 和加速选项。
2. 选项分析
A. Use AWS Certificate Manager (ACM) to create a public certificate in the us-east-1 Region. Use the certificate in
CloudFront.
正确。CloudFront要求使用的证书必须在us-east-1区域创建，因为CloudFront的证书管理是集中在us-east-1区域的。
B. Use AWS Certificate Manager (ACM) to create a public certificate in eu-west-1. Use the certificate in CloudFront.
错误。CloudFront不支持在其他区域（如eu-west-1）创建的证书，必须在us-east-1区域创建证书。
C. Configure Amazon S3 to allow uploads from CloudFront. Configure S3 Transfer Acceleration.
错误。S3 Transfer Acceleration是用于加速跨区域或全球范围的上传，但题目没有提到需要加速功能，且与CloudFront的集成不直接
相关。
D. Configure Amazon S3 to allow uploads from CloudFront origin access control (OAC).
正确。使用CloudFront的源访问控制 (OAC) 是推荐的最佳实践，可以确保只有CloudFront可以访问S3资源。
E. Configure Amazon S3 to allow uploads from CloudFront. Configure an Amazon S3 website endpoint.
错误。S3网站端点用于静态网站托管，与CloudFront的上传功能无关。

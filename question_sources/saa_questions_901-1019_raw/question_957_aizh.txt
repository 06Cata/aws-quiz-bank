A company needs to design a hybrid network architecture. The company's workloads are currently stored in the AWS Cloud and in
on-premises data centers. The workloads require single-digit latencies to communicate. The company uses an AWS Transit
Gateway transit gateway to connect multiple VPCs.
Which combination of steps will meet these requirements MOST cost-effectively? (Choose two.)
一家公司需要设计一个混合网络架构。该公司的工作负载目前存储在AWS Cloud和本地数据中心中。工作负载需要单位数字的延迟
进行通信。该公司使用AWS Transit Gateway来连接多个VPC。
哪些步骤组合能够以最具成本效益的方式满足这些要求？（选择两个）

2. 选项分析
A. 为每个VPC建立一个AWS Site-to-Site VPN连接。
错误：Site-to-Site VPN连接通常用于低成本的解决方案，但它无法提供单数字延迟，且为每个VPC建立VPN连接会增加复杂性和成
本。
B. 将一个AWS Direct Connect gateway与连接到VPC的transit gateway关联。
正确：Direct Connect gateway可以与Transit Gateway关联，从而实现高效的网络连接。这种方法支持低延迟通信，并且可以连接多
个VPC，降低成本。
C. 为AWS Direct Connect gateway建立一个AWS Site-to-Site VPN连接。
错误：虽然可以通过VPN连接到Direct Connect gateway，但VPN连接无法提供单数字延迟，且不符合题目要求的“最具成本效益”的
解决方案。
D. 建立一个AWS Direct Connect连接。为Direct Connect gateway创建一个transit virtual interface (VIF)。
正确：Direct Connect提供专用的低延迟连接，transit virtual interface (VIF)允许Direct Connect gateway与Transit Gateway集成，从
而支持多个VPC的连接。这是一个高效且成本优化的解决方案。
E. 将AWS Site-to-Site VPN连接与连接到VPC的transit gateway关联。
错误：虽然可以通过VPN连接到Transit Gateway，但VPN无法提供单数字延迟，且不符合题目要求的“最具成本效益”的解决方案。

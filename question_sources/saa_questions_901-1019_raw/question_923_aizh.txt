Question #923
A company hosts a monolithic web application on an Amazon EC2 instance. Application users have recently reported poor
performance at specific times. Analysis of Amazon CloudWatch metrics shows that CPU utilization is 100% during the periods of
poor performance.
The company wants to resolve this performance issue and improve application availability.
Which combination of steps will meet these requirements MOST cost-effectively? (Choose two.)
一家公司在一台 Amazon EC2 实例上托管了一个单体的 Web 应用程序。应用程序用户最近报告在特定时间性能较差。对 Amazon
CloudWatch 指标的分析显示，在性能较差的时间段内，CPU 利用率达到 100%。
该公司希望解决此性能问题并提高应用程序的可用性。
哪种步骤组合能够以最具成本效益的方式满足这些需求？（选择两个）
A. Use AWS Compute Optimizer to obtain a recommendation for an instance type to scale vertically. Most Voted
使用 AWS Compute Optimizer 获取关于垂直扩展实例类型的推荐。
B. Create an Amazon Machine Image (AMI) from the web server. Reference the AMI in a new launch template.
从 Web 服务器创建一个 Amazon Machine Image (AMI)。在新的启动模板中引用该 AMI。
C. Create an Auto Scaling group and an Application Load Balancer to scale vertically.
创建一个 Auto Scaling 组和一个 Application Load Balancer 来进行垂直扩展。
D. Use AWS Compute Optimizer to obtain a recommendation for an instance type to scale horizontally.
使用 AWS Compute Optimizer 获取关于水平扩展实例类型的推荐。
E. Create an Auto Scaling group and an Application Load Balancer to scale horizontally. Most Voted
创建一个 Auto Scaling 组和一个 Application Load Balancer 来进行水平扩展。
Correct Answer: AE
正确答案：AE

题目分析与解答
1. 考察的知识点
Amazon EC2实例性能优化
使用AWS Compute Optimizer进行实例类型推荐
Auto Scaling组和Application Load Balancer的使用
垂直扩展与水平扩展的区别与应用场景
2. 选项分析
A. Use AWS Compute Optimizer to obtain a recommendation for an instance type to scale vertically.
正确。AWS Compute Optimizer可以分析当前实例的性能并推荐更高性能的实例类型以进行垂直扩展。这是解决CPU利用率过高问
题的一个有效方法。
B. Create an Amazon Machine Image (AMI) from the web server. Reference the AMI in a new launch template.
错误。创建AMI和启动模板是部署新实例的步骤，但它本身并不能解决性能问题，也没有直接改善应用的可用性。
C. Create an Auto Scaling group and an Application Load Balancer to scale vertically.
错误。Auto Scaling组和Application Load Balancer通常用于水平扩展，而不是垂直扩展。垂直扩展是通过更换更高性能的实例类型
来实现的。
D. Use AWS Compute Optimizer to obtain a recommendation for an instance type to scale horizontally.
错误。AWS Compute Optimizer主要用于推荐单个实例的优化类型，而不是水平扩展的架构设计。
E. Create an Auto Scaling group and an Application Load Balancer to scale horizontally.
正确。通过创建Auto Scaling组和Application Load Balancer，可以实现水平扩展，动态增加实例数量以应对高负载，从而提高应用
的可用性。

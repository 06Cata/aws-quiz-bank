Question #1010
A company runs multiple workloads on virtual machines (VMs) in an on-premises data center. The company is expanding rapidly.
The on-premises data center is not able to scale fast enough to meet business needs. The company wants to migrate the
workloads to AWS.
The migration is time sensitive. The company wants to use a lift-and-shift strategy for non-critical workloads.
Which combination of steps will meet these requirements? (Choose three.)
一家公司在本地数据中心的虚拟机（VMs）上运行多个工作负载。公司正在快速扩展。本地数据中心无法快速扩展以满足业务需
求。公司希望将这些工作负载迁移到AWS。
迁移是时间敏感的。公司希望对非关键工作负载使用直接迁移策略。
哪些步骤组合能够满足这些需求？（选择三项）
A. Use the AWS Schema Conversion Tool (AWS SCT) to collect data about the VMs.
使用AWS Schema Conversion Tool（AWS SCT）收集关于VMs的数据。
B. Use AWS Application Migration Service. Install the AWS Replication Agent on the VMs. Most Voted
使用AWS Application Migration Service。在VMs上安装AWS Replication Agent。
C. Complete the initial replication of the VMs. Launch test instances to perform acceptance tests on the VMs. Most Voted
完成VMs的初始复制。启动测试实例以对VMs进行验收测试。
D. Stop all operations on the VMs. Launch a cutover instance. Most Voted
停止VMs上的所有操作。启动切换实例。
E. Use AWS App2Container (A2C) to collect data about the VMs.
使用AWS App2Container（A2C）收集关于VMs的数据。
F. Use AWS Database Migration Service (AWS DMS) to migrate the VMs.
使用AWS Database Migration Service（AWS DMS）迁移VMs。
Correct Answer: BCD
正确答案：BCD
题目分析与解答
1. 考察的知识点
本题考察的是AWS迁移服务的使用，包括AWS Application Migration Service、AWS Replication Agent、以及迁移过程中的测试和切
换步骤。
重点在于理解如何使用AWS工具进行“Lift-and-Shift”迁移策略。
2. 选项分析
A. Use the AWS Schema Conversion Tool (AWS SCT) to collect data about the VMs.
错误：AWS Schema Conversion Tool (AWS SCT)主要用于数据库迁移，而不是虚拟机迁移。
B. Use AWS Application Migration Service. Install the AWS Replication Agent on the VMs.
正确：AWS Application Migration Service是专门设计用于迁移虚拟机的服务，安装AWS Replication Agent是迁移的必要步骤。
C. Complete the initial replication of the VMs. Launch test instances to perform acceptance tests on the VMs.
正确：完成初始复制后，启动测试实例进行验收测试是迁移过程中的重要步骤，确保迁移的工作负载正常运行。
D. Stop all operations on the VMs. Launch a cutover instance.
正确：在迁移完成后，停止源VM上的操作并启动切换实例是迁移的最后一步，确保工作负载切换到AWS环境。
E. Use AWS App2Container (A2C) to collect data about the VMs.
错误：AWS App2Container (A2C)用于容器化应用程序，而不是虚拟机迁移。
F. Use AWS Database Migration Service (AWS DMS) to migrate the VMs.
错误：AWS Database Migration Service (AWS DMS)用于数据库迁移，而不是虚拟机迁移。

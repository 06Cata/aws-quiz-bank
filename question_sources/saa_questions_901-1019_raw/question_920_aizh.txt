Question #920
A company is migrating from a monolithic architecture for a web application that is hosted on Amazon EC2 to a serverless
microservices architecture. The company wants to use AWS services that support an event-driven, loosely coupled architecture.
The company wants to use the publish/subscribe (pub/sub) pattern.
Which solution will meet these requirements MOST cost-effectively?
一家公司正在将一个托管在 Amazon EC2 上的 Web 应用程序从单体架构迁移到无服务器的微服务架构。 该公司希望使用支持事件
驱动、松耦合架构的 AWS 服务。 该公司希望使用发布/订阅 (pub/sub) 模式。
哪种解决方案最具成本效益地满足这些要求？
A. Configure an Amazon API Gateway REST API to invoke an AWS Lambda function that publishes events to an Amazon Simple

2. 选项分析
A. Configure an Amazon API Gateway REST API to invoke an AWS Lambda function that publishes events to an Amazon
Simple Queue Service (Amazon SQS) queue. Configure one or more subscribers to read events from the SQS queue.
正确性分析：Amazon SQS是一个队列服务，适合点对点消息传递，而不是Pub/Sub模式。虽然可以通过多个消费者读取队列中的消
息，但它不完全符合Pub/Sub的设计模式。
成本分析：使用SQS可能会增加成本，因为需要额外的队列管理和消息轮询。
结论：不符合题目要求。
B. Configure an Amazon API Gateway REST API to invoke an AWS Lambda function that publishes events to an Amazon
Simple Notification Service (Amazon SNS) topic. Configure one or more subscribers to receive events from the SNS topic.
正确性分析：Amazon SNS是一个Pub/Sub服务，支持事件驱动架构和多个订阅者。此选项符合题目要求。
成本分析：使用REST API可能会增加成本，因为REST API的调用费用较高。
结论：虽然符合要求，但不是最具成本效益的解决方案。
C. Configure an Amazon API Gateway WebSocket API to write to a data stream in Amazon Kinesis Data Streams with
enhanced fan-out. Configure one or more subscribers to receive events from the data stream.
正确性分析：Amazon Kinesis Data Streams适合实时数据流处理，但它的主要用途是高吞吐量数据流，而不是事件驱动的Pub/Sub
模式。
成本分析：Kinesis Data Streams的成本较高，尤其是启用增强型扇出功能时。
结论：不符合题目要求。
D. Configure an Amazon API Gateway HTTP API to invoke an AWS Lambda function that publishes events to an Amazon
Simple Notification Service (Amazon SNS) topic. Configure one or more subscribers to receive events from the topic.
正确性分析：Amazon SNS是一个Pub/Sub服务，支持事件驱动架构和多个订阅者。HTTP API的成本比REST API更低，符合题目要
求。
成本分析：HTTP API的调用费用较低，是最具成本效益的解决方案。
结论：符合题目要求，是最佳答案。

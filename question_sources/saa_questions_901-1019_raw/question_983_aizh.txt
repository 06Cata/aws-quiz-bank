Question #983
A company hosts its core network services, including directory services and DNS, in its on-premises data center. The data center is
connected to the AWS Cloud using AWS Direct Connect (DX). Additional AWS accounts are planned that will require quick, cost-
effective, and consistent access to these network services.
What should a solutions architect implement to meet these requirements with the LEAST amount of operational overhead?
一家公司在其本地数据中心托管其核心网络服务，包括目录服务和DNS。数据中心通过AWS Direct Connect (DX)连接到AWS云。计
划新增多个AWS账户，这些账户需要快速、经济高效且一致地访问这些网络服务。
解决方案架构师应实施什么以满足这些需求，同时将运营开销降到最低？
A. Create a DX connection in each new account. Route the network traffic to the on-premises servers.
在每个新账户中创建一个DX连接。将网络流量路由到本地服务器。

在DX VPC中为所有所需服务配置VPC endpoints。将网络流量路由到本地服务器。

D (100%)
本题考察的是AWS网络架构设计，包括AWS Direct Connect (DX)、AWS Transit Gateway、VPC Endpoints、VPN连接等技术的使用
场景和最佳实践。
重点在于如何实现跨账户的网络连接，同时保持低运营开销和高效性。
2. 选项分析
A. Create a DX connection in each new account. Route the network traffic to the on-premises servers.
错误：创建多个DX连接会导致高成本，因为每个DX连接都需要额外的费用。
错误：运营开销较高，因为需要为每个账户单独配置DX连接。
不符合题目要求的“LEAST amount of operational overhead”。
B. Configure VPC endpoints in the DX VPC for all required services. Route the network traffic to the on-premises servers.
错误：VPC Endpoints主要用于访问AWS服务（如S3、DynamoDB），而不是用于连接本地数据中心。
错误：无法直接通过VPC Endpoints实现跨账户的网络连接到本地服务器。
C. Create a VPN connection between each new account and the DX VPC. Route the network traffic to the on-premises
servers.
错误：为每个账户创建VPN连接会增加运营开销和复杂性。
错误：VPN连接的性能通常不如DX连接，且不符合题目要求的“快速、成本效益高、稳定”。
D. Configure AWS Transit Gateway between the accounts. Assign DX to the transit gateway and route network traffic to
the on-premises servers.
正确：AWS Transit Gateway可以高效地连接多个VPC和账户，简化网络架构。
正确：将DX连接分配给Transit Gateway可以实现快速、稳定的网络连接到本地数据中心。
正确：符合题目要求的“LEAST amount of operational overhead”。

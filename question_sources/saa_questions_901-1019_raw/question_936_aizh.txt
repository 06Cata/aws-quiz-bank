all the Lambda functions use. The solution must also manage the automatic rotation of the sensitive information.
Which combination of steps will meet these requirements with the LEAST operational overhead? (Choose two.)
一家公司运行了数千个AWS Lambda函数。该公司需要一个解决方案来安全存储所有Lambda函数使用的敏感信息。该解决方案还必

2. 选项分析
A. Create HTTP security headers by using Lambda@Edge to retrieve and create sensitive information 错误：Lambda@Edge
主要用于在CloudFront边缘位置处理请求和响应，与存储敏感信息无关。此选项不符合题目要求。
B. Create a Lambda layer that retrieves sensitive information 正确：Lambda层可以用于共享代码或逻辑，例如检索敏感信息。
通过使用Lambda层，可以减少重复代码并简化敏感信息的管理。
C. Store sensitive information in AWS Secrets Manager 正确：AWS Secrets Manager是专门设计用于存储和管理敏感信息的服
务。它支持自动旋转密钥和凭证，符合题目要求。
D. Store sensitive information in AWS Systems Manager Parameter Store 错误：虽然Parameter Store可以存储敏感信息，但它
不支持自动旋转功能，因此不完全满足题目要求。
E. Create a Lambda consumer with dedicated throughput to retrieve sensitive information and create environmental
variables 错误：创建一个专用的Lambda消费者会增加操作开销，并且不直接解决敏感信息的安全存储和自动旋转问题。

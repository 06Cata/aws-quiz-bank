Question #902

company wants to use AWS WAF in the application's architecture.
Which solution will meet these requirements?

WAF。
哪种解决方案可以满足这些要求？
A. Create an Auto Scaling group that contains multiple Amazon EC2 instances that host the application across two Availability
Zones. Configure an Application Load Balancer (ALB) and set the Auto Scaling group as the target. Connect a WAF to the ALB.

Correct Answer: A

题目分析与解答
1. 考察的知识点
高可用架构设计：通过跨多个可用区部署资源来提高应用程序的可用性。
AWS WAF（Web Application Firewall）的使用：保护应用程序免受常见的网络攻击。
负载均衡器（Application Load Balancer）的配置：分发流量以提高应用程序的性能和可靠性。
Auto Scaling组的使用：动态调整计算资源以满足需求。
2. 选项分析
A. Create an Auto Scaling group that contains multiple Amazon EC2 instances that host the application across two
Availability Zones. Configure an Application Load Balancer (ALB) and set the Auto Scaling group as the target. Connect a
WAF to the ALB.
正确：此选项通过Auto Scaling组实现动态扩展，并跨两个可用区部署EC2实例以提高可用性。
正确：使用Application Load Balancer分发流量，确保高性能和可靠性。
正确：将WAF连接到ALB是保护应用程序的最佳实践。
B. Create a cluster placement group that contains multiple Amazon EC2 instances that hosts the application. Configure an
Application Load Balancer and set the EC2 instances as the targets. Connect a WAF to the placement group.
错误：Cluster placement group主要用于低延迟、高性能计算，而不是提高可用性。
错误：此选项未跨多个可用区部署资源，无法满足高可用性要求。
正确：WAF可以连接到ALB，但此选项的架构设计不符合题目要求。
C. Create two Amazon EC2 instances that host the application across two Availability Zones. Configure the EC2 instances
as the targets of an Application Load Balancer (ALB). Connect a WAF to the ALB.
正确：跨两个可用区部署EC2实例可以提高可用性。
错误：仅使用两个EC2实例没有动态扩展能力，无法满足需求变化。
正确：WAF连接到ALB是保护应用程序的最佳实践。
D. Create an Auto Scaling group that contains multiple Amazon EC2 instances that host the application across two
Availability Zones. Configure an Application Load Balancer (ALB) and set the Auto Scaling group as the target. Connect a
WAF to the Auto Scaling group.
正确：Auto Scaling组和跨两个可用区的部署可以提高可用性。
错误：WAF无法直接连接到Auto Scaling组，WAF只能连接到ALB。

A solutions architect is designing an application that helps users fill out and submit registration forms. The solutions architect
plans to use a two-tier architecture that includes a web application server tier and a worker tier.
The application needs to process submitted forms quickly. The application needs to process each form exactly once. The solution
must ensure that no data is lost.
Which solution will meet these requirements?

Correct Answer: A
正确答案：A
题目分析与解答
1. 考察的知识点
Amazon SQS FIFO队列的特性：保证消息的顺序性和“恰好一次”处理。
Amazon SQS标准队列的特性：高吞吐量，但不保证消息顺序或“恰好一次”处理。
Amazon API Gateway的用途：主要用于API管理和HTTP请求处理，不适合消息队列场景。
AWS Step Functions的用途：用于编排分布式应用程序工作流，不适合简单的消息队列场景。
2. 选项分析
A. Use an Amazon Simple Queue Service (Amazon SQS) FIFO queue between the web application server tier and the
worker tier to store and forward form data.
正确：Amazon SQS FIFO队列可以确保消息的顺序性和“恰好一次”处理，符合题目要求。
FIFO队列适合需要严格顺序和避免重复处理的场景。
B. Use an Amazon API Gateway HTTP API between the web application server tier and the worker tier to store and
forward form data.
错误：API Gateway主要用于管理HTTP API请求，不具备消息队列的功能。
无法保证消息的顺序性或“恰好一次”处理。
C. Use an Amazon Simple Queue Service (Amazon SQS) standard queue between the web application server tier and the
worker tier to store and forward form data.
错误：标准队列虽然支持高吞吐量，但不保证消息的顺序性或“恰好一次”处理。
题目明确要求“恰好一次”处理，因此标准队列不符合要求。
D. Use an AWS Step Functions workflow. Create a synchronous workflow between the web application server tier and the
worker tier that stores and forwards form data.
错误：Step Functions主要用于编排复杂的工作流，而不是简单的消息队列场景。
题目要求快速处理表单数据，Step Functions的同步工作流可能会增加延迟。

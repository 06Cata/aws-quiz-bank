Question #959
A company has multiple Amazon RDS DB instances that run in a development AWS account. All the instances have tags to identify
them as development resources. The company needs the development DB instances to run on a schedule only during business
hours.
Which solution will meet these requirements with the LEAST operational overhead?
一家公司在一个开发 AWS 账户中运行多个 Amazon RDS DB 实例。所有实例都有标签以标识它们为开发资源。公司需要这些开发
DB 实例仅在工作时间内按计划运行。
哪种解决方案能够以最少的运营开销满足这些需求？
A. Create an Amazon CloudWatch alarm to identify RDS instances that need to be stopped. Create an AWS Lambda function to
start and stop the RDS instances.
创建一个 Amazon CloudWatch 警报以识别需要停止的 RDS 实例。创建一个 AWS Lambda 函数来启动和停止 RDS 实例。
B. Create an AWS Trusted Advisor report to identify RDS instances to be started and stopped. Create an AWS Lambda function
to start and stop the RDS instances.
创建一个 AWS Trusted Advisor 报告以识别需要启动和停止的 RDS 实例。创建一个 AWS Lambda 函数来启动和停止 RDS 实例。
C. Create AWS Systems Manager State Manager associations to start and stop the RDS instances.
创建 AWS Systems Manager State Manager 关联以启动和停止 RDS 实例。
D. Create an Amazon EventBridge rule that invokes AWS Lambda functions to start and stop the RDS instances. Most Voted
创建一个 Amazon EventBridge 规则来调用 AWS Lambda 函数以启动和停止 RDS 实例。

2. 选项分析
A. Create an Amazon CloudWatch alarm to identify RDS instances that need to be stopped. Create an AWS Lambda
function to start and stop the RDS instances.
错误：CloudWatch Alarms主要用于监控指标并触发操作，但它不适合直接用于基于时间表的资源调度。
虽然可以通过Lambda函数实现启动和停止，但使用CloudWatch Alarms来识别需要停止的实例并不符合题目要求的“基于时间表”需
求。
B. Create an AWS Trusted Advisor report to identify RDS instances to be started and stopped. Create an AWS Lambda
function to start and stop the RDS instances.
错误：AWS Trusted Advisor主要用于提供优化建议，例如成本节约、安全性和性能改进。它不是一个调度工具，无法直接用于基于
时间表的资源管理。
此选项与题目要求的“基于时间表”需求不匹配。
C. Create AWS Systems Manager State Manager associations to start and stop the RDS instances.
错误：AWS Systems Manager State Manager主要用于管理实例的状态配置，例如安装软件或应用补丁。它不适合用于基于时间表
的资源启动和停止。
此选项的功能与题目要求的“基于时间表”需求不匹配。
D. Create an Amazon EventBridge rule that invokes AWS Lambda functions to start and stop the RDS instances.
正确：Amazon EventBridge可以基于时间表触发事件，非常适合实现基于时间的资源调度。
通过EventBridge规则，可以设置在工作时间启动RDS实例，在非工作时间停止RDS实例，从而满足题目要求。
结合AWS Lambda函数，可以实现对RDS实例的启动和停止操作。

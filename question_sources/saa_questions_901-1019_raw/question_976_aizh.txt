Question #976
A company is implementing a new application on AWS. The company will run the application on multiple Amazon EC2 instances
across multiple Availability Zones within multiple AWS Regions. The application will be available through the internet. Users will
access the application from around the world.
The company wants to ensure that each user who accesses the application is sent to the EC2 instances that are closest to the
user’s location.
Which solution will meet these requirements?

题目分析与解答
1. 考察的知识点
Amazon Route 53 路由策略，包括地理位置路由（Geolocation）、地理邻近路由（Geoproximity）、多值答案路由（Multivalue
Answer）、加权路由（Weighted Routing）。
负载均衡器的类型及其适用场景，包括 Application Load Balancer 和 Network Load Balancer。
跨区域和跨可用区的流量分配策略。
2. 选项分析
A. Implement an Amazon Route 53 geolocation routing policy. Use an internet-facing Application Load Balancer to
distribute the traffic across all Availability Zones within the same Region.
正确性分析：Geolocation routing policy 是基于用户的地理位置来路由流量，但它无法动态调整到“最近的”资源，只能基于预定义
的地理区域进行分配。
错误原因：题目要求将用户路由到“最近的”EC2实例，而 Geolocation routing 不能动态计算距离。
B. Implement an Amazon Route 53 geoproximity routing policy. Use an internet-facing Network Load Balancer to
distribute the traffic across all Availability Zones within the same Region.
正确性分析：Geoproximity routing policy 是基于用户和资源之间的距离来路由流量，可以动态调整到“最近的”资源。
正确原因：此选项满足题目要求，将用户路由到最近的 EC2 实例，同时 Network Load Balancer 可以高效处理跨区域流量。
C. Implement an Amazon Route 53 multivalue answer routing policy. Use an internet-facing Application Load Balancer to
distribute the traffic across all Availability Zones within the same Region.
正确性分析：Multivalue answer routing policy 返回多个 IP 地址供客户端选择，但它不基于用户位置动态调整流量。
错误原因：此选项无法满足题目要求的“最近的”资源路由功能。
D. Implement an Amazon Route 53 weighted routing policy. Use an internet-facing Network Load Balancer to distribute
the traffic across all Availability Zones within the same Region.
正确性分析：Weighted routing policy 是基于预定义权重分配流量，而不是基于用户位置动态调整流量。
错误原因：此选项无法满足题目要求的“最近的”资源路由功能。

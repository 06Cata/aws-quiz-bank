Question #943
A company has migrated several applications to AWS in the past 3 months. The company wants to know the breakdown of costs
for each of these applications. The company wants to receive a regular report that includes this information.
Which solution will meet these requirements MOST cost-effectively?
一家公司在过去的3个月内将多个应用程序迁移到了AWS。该公司希望了解每个应用程序的成本细分。公司希望定期收到包含这些
信息的报告。
哪种解决方案可以最具成本效益地满足这些需求？
A. Use AWS Budgets to download data for the past 3 months into a .csv file. Look up the desired information.
使用AWS Budgets下载过去3个月的数据到一个.csv文件中。查找所需的信息。
B. Load AWS Cost and Usage Reports into an Amazon RDS DB instance. Run SQL queries to get the desired information.
将AWS Cost and Usage Reports加载到一个Amazon RDS数据库实例中。运行SQL查询以获取所需的信息。
C. Tag all the AWS resources with a key for cost and a value of the application's name. Activate cost allocation tags. Use Cost
Explorerto get the desired information. Most Voted

D. Tag all the AWS resources with a key for cost and a value of the application's name. Use the AWS Billing and Cost

题目分析与解答
1. 考察的知识点
成本管理与优化：如何使用AWS工具（如Cost Explorer、Cost and Usage Reports、标签等）来跟踪和分析成本。
标签管理：如何通过标签对资源进行分类和分配成本。
成本分配标签的激活与使用。
2. 选项分析
A. 使用AWS Budgets下载过去3个月的数据到一个.csv文件中。查找所需的信息。
错误：AWS Budgets主要用于设置预算和接收通知，而不是用于详细的成本分析。它无法直接提供按应用程序分类的成本信息。
B. 将AWS Cost and Usage Reports加载到一个Amazon RDS数据库实例中。运行SQL查询以获取所需的信息。
错误：虽然AWS Cost and Usage Reports可以提供详细的成本数据，但将数据加载到RDS并运行SQL查询的过程复杂且成本较高，不
符合“最具成本效益”的要求。
C. 为所有AWS资源添加一个成本的标签键，并将应用程序名称作为标签值。激活成本分配标签。使用Cost Explorer获取所需的信
息。
正确：通过标签对资源进行分类并激活成本分配标签，可以使用Cost Explorer轻松查看按应用程序分类的成本数据。这是最具成本
效益的解决方案。
D. 为所有AWS资源添加一个成本的标签键，并将应用程序名称作为标签值。使用AWS Billing and Cost Management控制台下载
过去3个月的账单。查找所需的信息。
错误：虽然可以通过账单查看成本信息，但账单数据通常不够详细，无法按标签分类成本。此外，这种方法比使用Cost Explorer更
繁琐。
3. 我选的答案
C
4. 官方答案

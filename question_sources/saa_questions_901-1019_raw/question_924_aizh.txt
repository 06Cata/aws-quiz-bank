Question #924
A company runs all its business applications in the AWS Cloud. The company uses AWS Organizations to manage multiple AWS
accounts.
A solutions architect needs to review all permissions that are granted to IAM users to determine which IAM users have more
permissions than required.
Which solution will meet these requirements with the LEAST administrative overhead?
一家公司将其所有业务应用程序运行在 AWS 云中。该公司使用 AWS Organizations 来管理多个 AWS 账户。
一位解决方案架构师需要审查授予 IAM 用户的所有权限，以确定哪些 IAM 用户拥有超过所需的权限。
哪种解决方案能够以最少的管理开销满足这些需求？
A. Use Network Access Analyzer to review all access permissions in the company's AWS accounts.
使用 Network Access Analyzer 审查公司 AWS 账户中的所有访问权限。
B. Create an AWS CloudWatch alarm that activates when an IAM user creates or modifies resources in an AWS account.
创建一个 AWS CloudWatch 警报，当 IAM 用户在 AWS 账户中创建或修改资源时激活。
C. Use AWS Identity and Access Management (IAM) Access Analyzer to review all the company’s resources and accounts.

题目分析与解答
1. 考察的知识点
本题考察的是AWS权限管理工具的使用，特别是如何审查IAM用户权限以确保最小权限原则（Least Privilege）。
涉及的服务包括IAM Access Analyzer、Network Access Analyzer、CloudWatch、Amazon Inspector等。
2. 选项分析
A. Use Network Access Analyzer to review all access permissions in the company's AWS accounts.
错误：Network Access Analyzer主要用于分析网络访问路径和网络配置，而不是专门设计来审查IAM用户权限。
B. Create an AWS CloudWatch alarm that activates when an IAM user creates or modifies resources in an AWS account.
错误：CloudWatch可以监控事件，但无法直接审查IAM用户权限。此方法需要额外的配置和手动分析，增加了管理开销。
C. Use AWS Identity and Access Management (IAM) Access Analyzer to review all the company’s resources and accounts.
正确：IAM Access Analyzer专门设计用于分析权限并识别过度权限问题，是解决此问题的最佳工具。
D. Use Amazon Inspector to find vulnerabilities in existing IAM policies.
错误：Amazon Inspector主要用于检查EC2实例、容器和Lambda函数的安全漏洞，而不是审查IAM权限。
3. 我的答案
C. Use AWS Identity and Access Management (IAM) Access Analyzer to review all the company’s resources and accounts.
4. 官方答案
C. Use AWS Identity and Access Management (IAM) Access Analyzer to review all the company’s resources and accounts.
5. 与官方答案比较
我的答案与官方答案一致。
原因：IAM Access Analyzer是专门设计用于审查权限的工具，能够有效识别过度权限问题，同时减少管理开销。

Question #949
A company has a multi-tier web application. The application's internal service components are deployed on Amazon EC2
instances. The internal service components need to access third-party software as a service (SaaS) APIs that are hosted on AWS.
The company needs to provide secure and private connectivity from the application's internal services to the third-party SaaS
application. The company needs to ensure that there is minimal public internet exposure.
Which solution will meet these requirements?
一家公司有一个多层级的 Web 应用程序。该应用程序的内部服务组件部署在 Amazon EC2 实例上。内部服务组件需要访问托管在
AWS 上的第三方软件即服务 (SaaS) API。
该公司需要为应用程序的内部服务与第三方 SaaS 应用程序之间提供安全且私密的连接。公司需要确保公共互联网的暴露最小化。
哪种解决方案可以满足这些需求？
A. Implement an AWS Site-to-Site VPN to establish a secure connection with the third-party SaaS provider.
实施 AWS Site-to-Site VPN，以与第三方 SaaS 提供商建立安全连接。

部署 AWS Transit Gateway，以管理和路由应用程序的 VPC 与第三方 SaaS 提供商之间的流量。
C. Configure AWS PrivateLink to allow only outbound traffic from the VPC without enabling the third-party SaaS provider to
establish.
配置 AWS PrivateLink，以仅允许来自 VPC 的出站流量，而不启用第三方 SaaS 提供商的连接。
D. Use AWS PrivateLink to create a private connection between the application's VPC and the third-party SaaS provider.

题目分析与解答
1. 考察的知识点
本题考察的是AWS网络服务的使用，包括AWS PrivateLink、AWS Site-to-Site VPN、AWS Transit Gateway等服务的功能和适用场
景。
重点在于如何实现安全、私密的连接，同时减少公共互联网的暴露。
2. 选项分析
A. Implement an AWS Site-to-Site VPN to establish a secure connection with the third-party SaaS provider.
错误：AWS Site-to-Site VPN主要用于连接本地数据中心和AWS环境，而不是用于连接VPC与第三方SaaS服务。
此选项无法满足“私密连接”和“减少公共互联网暴露”的要求。
B. Deploy AWS Transit Gateway to manage and route traffic between the application's VPC and the third-party SaaS
provider.
错误：AWS Transit Gateway主要用于管理多个VPC之间的流量或连接本地网络，但它并不直接提供与第三方SaaS服务的私密连接。
此选项无法满足“私密连接”的要求。
C. Configure AWS PrivateLink to allow only outbound traffic from the VPC without enabling the third-party SaaS provider
to establish.
错误：此选项描述的功能不完全正确。AWS PrivateLink的主要功能是创建私密连接，而不是仅仅限制出站流量。
此选项的描述与AWS PrivateLink的实际功能不符。
D. Use AWS PrivateLink to create a private connection between the application's VPC and the third-party SaaS provider.
正确：AWS PrivateLink允许VPC与第三方SaaS服务之间建立私密连接，避免流量通过公共互联网。
此选项完全符合题目要求，提供了安全、私密的连接，并减少了公共互联网暴露。
3. 我选的答案
D. Use AWS PrivateLink to create a private connection between the application's VPC and the third-party SaaS provider.
4. 官方答案
D. Use AWS PrivateLink to create a private connection between the application's VPC and the third-party SaaS provider.
5. 与官方答案的比较
我的答案与官方答案一致。
原因：AWS PrivateLink是专门设计用于创建私密连接的服务，完全符合题目要求。

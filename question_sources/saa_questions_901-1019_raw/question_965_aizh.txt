A company is migrating an application from an on-premises environment to AWS. The application will store sensitive data in
Amazon S3. The company must encrypt the data before storing the data in Amazon S3.
Which solution will meet these requirements?
一家公司正在将一个应用程序从本地环境辜移到AWS。该应用程序将在Amazon S3中存储敏感数据。公司必轔在将数据存储到
Amazon S3之前对数据辦行加密。

提供的密迫辦行加密。

2. 选项分析
A. Encrypt the data by using client-side encryption with customer managed keys. 正确。客户端加密意味着数据在上传到
Amazon S3之前已经被加密。使用客户管理密迫（Customer Managed Keys）可以确保公司完全控制加密密迫，符合敏感数据保护
的要求。
B. Encrypt the data by using server-side encryption with AWS KMS keys (SSE-KMS). 迾误。虽然SSE-KMS是一个强大的加密辷
轒，但它是服务器端加密，数据在传输辟程中未加密。轞目要求“在存储之前加密数据”，因此不符合要求。
C. Encrypt the data by using server-side encryption with customer-provided keys (SSE-C). 迾误。SSE-C是服务器端加密，数
据在传输辟程中未加密。轞目要求“在存储之前加密数据”，因此不符合要求。
D. Encrypt the data by using client-side encryption with Amazon S3 managed keys. 迾误。Amazon S3管理密迫是由AWS管理
的，客户无法完全控制密迫。轞目要求使用客户管理密迫，因此不符合要求。
3. 我的答案
A. Encrypt the data by using client-side encryption with customer managed keys.
4. 官方答案
A. Encrypt the data by using client-side encryption with customer managed keys.

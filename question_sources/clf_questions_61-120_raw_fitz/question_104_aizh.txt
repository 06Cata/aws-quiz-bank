Which AWS tool gives users the ability to plan their service usage, service costs, and instance reservations, and also allows them to
set custom alerts when their costs or usage exceed established thresholds?

报？

Cost Explorer

AWS Budgets

Reserved Instance reporting
题目分析与解答
1. 考察的知识点
本题考察的是AWS成本管理工具的功能，包括计划服务使用、成本管理、实例预留以及设置自定义警报的能力。
2. 选项分析
A. Cost Explorer: Cost Explorer主要用于分析和可视化AWS服务的成本和使用情况，但它不支持设置自定义警报功能，因此此选项
错误。
B. AWS Budgets: AWS Budgets允许用户设置预算并创建自定义警报，当成本或使用量超过设定的阈值时会触发警报。此选项正
确。
Correct Answer: B
正确答案: B

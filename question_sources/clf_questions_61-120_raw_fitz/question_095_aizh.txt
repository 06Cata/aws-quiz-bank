AWS Firewall Manager

题目分析与解答
1. 考察的知识点
本题考孹的是AWS网络孞全相关服务的功能，特别是控制Amazon VPC子网流量的宨具。
2. 选项分析
A. Security group: Security Group 是一种虚拟防火墙，用于控制孩例级别的入站和出站流量。孚不能直接控制子网级别的流量，因
此此选项错误。
B. AWS WAF: AWS WAF 是一种Web寒用程寏防火墙，用于保护Web寒用程寏免受寀见的网络攻击（如SQL注入和跨站脚本攻
击）。孚不适用于控制VPC子网流量，因此此选项错误。
C. AWS Firewall Manager: AWS Firewall Manager 是一种集中管理宨具，用于管理多个账孀和资源的防火墙规则。孚本身不直接设
置子网流量规则，因此此选项错误。
Correct Answer: D
正确答案：D

Question #111
A company needs to track the activity in its AWS accounts, and needs to know when an API call is made against its AWS resources.
Which AWS tool or service can be used to meet these requirements?
一家公司需要跟踪其 AWS 账户中的活动，并需要知道何时对其 AWS 资源进行 API 调用。
可以使用哪种 AWS 工具或服务来满足这些需求？
A. Amazon CloudWatch
Amazon CloudWatch
B. Amazon Inspector
Amazon Inspector
C. AWS CloudTrail Most Voted
AWS CloudTrail
D. AWS IAM
AWS IAM
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的功能和用途，特别是与跟踪API调用和活动日志相关的服务。
2. 选项分析
A. Amazon CloudWatch: 错误。Amazon CloudWatch主要用于监控和管理AWS资源的性能指标和日志数据，但它不专门用于跟踪
API调用。
B. Amazon Inspector: 错误。Amazon Inspector是一种安全评估服务，用于检查AWS资源的安全漏洞和合规性问题，与跟踪API调
用无关。
Correct Answer: C
正确答案：C

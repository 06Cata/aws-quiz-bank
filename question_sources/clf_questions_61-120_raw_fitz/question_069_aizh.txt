Which AWS service provides command line access to AWS tools and resources directly from a web browser?
哪个AWS服务通过网页浏览器直接提供对AWS工具和资源的命令行访问？
A. AWS CloudHSM
AWS CloudHSM
B. AWS CloudShell Most Voted
AWS CloudShell
C. Amazon WorkSpaces
Amazon WorkSpaces
D. AWS Cloud Map
AWS Cloud Map
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中提供命令行访问工具的服务，尤其是通过Web浏览器直接访问的能力。
2. 选项分析
A. AWS CloudHSM: 错误。AWS CloudHSM 是一种托管硬件安全模块服务，用于加密密钥管理和加密操作，与命令行访问无关。
B. AWS CloudShell: 正确。AWS CloudShell 是一个基于浏览器的服务，允许用户通过命令行直接访问AWS资源和工具，无需安装
本地终端。
C. Amazon WorkSpaces: 错误。Amazon WorkSpaces 是一种托管的虚拟桌面服务，用于提供远程桌面环境，与命令行工具无关。
D. AWS Cloud Map: 错误。AWS Cloud Map 是一种服务发现工具，用于帮助应用程序发现资源的运行位置，与命令行访问无关。
3. 我的答案
Correct Answer: B
正确答案：B

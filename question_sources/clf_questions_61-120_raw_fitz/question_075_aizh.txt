A company wants to manage its AWS Cloud resources through a web interface.
Which AWS service will meet this requirement?
一家公司希望通过一个网页界面来管理其AWS云资源。
哪个AWS服务可以满足此需求？
A. AWS Management Console Most Voted
AWS Management Console
B. AWS CLI
AWS CLI
C. AWS SDK
AWS SDK
D. AWS Cloud9
AWS Cloud9
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的使用场景，特别是如何通过不同工具管理AWS资源。
2. 选项分析
A. AWS Management Console: 正确。AWS Management Console是一个基于网页的界面，用户可以通过它管理和监控AWS资
源。它完全符合题目中“通过web界面管理AWS资源”的要求。
B. AWS CLI: 错误。AWS CLI是一个命令行工具，用户通过命令行界面管理AWS资源，而不是通过web界面。
Correct Answer: A
正确答案：A

Question #120
A company wants to design a centralized storage system to manage the configuration data and passwords for its critical business
applications.
Which AWS service or capability will meet these requirements MOST cost-effectively?
一家公司希望设计一个集中式存储系统来管理其关键业务应用程序的配置数据和密码。
哪个AWS服务或功能可以最具成本效益地满足这些需求？
A. AWS Systems Manager Parameter Store Most Voted
AWS Systems Manager Parameter Store
B. AWS Secrets Manager
AWS Secrets Manager
C. AWS Config
AWS Config
D. Amazon S3
Amazon S3
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中用于存储和管理配置数据、密码以及其他敏感信息的服务。
重点在于选择最具成本效益的解决方案。
2. 选项分析
A. AWS Systems Manager Parameter Store
Parameter Store是AWS Systems Manager的一部分，用于存储配置数据和敏感信息（如密码）。
Correct Answer: A
正确答案：A

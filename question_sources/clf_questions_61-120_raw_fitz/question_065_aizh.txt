Question #65
A company wants to monitor for misconfigured security groups that are allowing unrestricted access to specific ports.
Which AWS service will meet this requirement?
一家公司希望监控允许特定端口不受限制访问的配置错误的安全组。
哪项AWS服务可以满足此需求？
A. AWS Trusted Advisor Most Voted
AWS Trusted Advisor
B. Amazon CloudWatch
Amazon CloudWatch
C. Amazon GuardDuty
Amazon GuardDuty
D. AWS Health Dashboard
AWS Health Dashboard
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中与安全组配置监控相关的功能，尤其是识别安全组的潜在风险和不当配置。
2. 选项分析
A. AWS Trusted Advisor: 正确。AWS Trusted Advisor提供了安全检查功能，可以识别安全组中允许过多访问的配置，例如开放到
所有IP地址的端口。它是专门设计用于帮助用户优化安全性、性能和成本的工具。
B. Amazon CloudWatch: 错误。CloudWatch主要用于监控资源的性能和运行状态，例如CPU使用率、内存使用率等。它不直接提
供安全组配置的检查功能。
Correct Answer: A
正确答案：A

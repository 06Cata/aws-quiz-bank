Question #98
A company wants to grant users in one AWS account access to resources in another AWS account. The users do not currently have
permission to access the resources.
Which AWS service will meet this requirement?
一家公司希望授予一个 AWS 账户中的用户访问另一个 AWS 账户中的资源的权限。这些用户目前没有访问这些资源的权限。
哪个 AWS 服务可以满足此要求？
A. IAM group
IAM group
B. IAM role Most Voted
IAM role
C. IAM tag
IAM tag
D. IAM Access Analyzer
IAM Access Analyzer
题目分析与解答
1. 考察的知识点
本题考察的是跨账户访问的实现方式，以及IAM角色的使用。
2. 选项分析
A. IAM group: 错误。IAM组用于组织和管理用户权限，但它不能直接用于跨账户访问。
B. IAM role: 正确。IAM角色可以通过信任策略授予其他账户的用户或服务访问权限，是实现跨账户访问的推荐方式。
C. IAM tag: 错误。IAM标签用于标记IAM资源以便于管理，但它与权限管理无关，不能用于跨账户访问。
Correct Answer: B
正确答案：B

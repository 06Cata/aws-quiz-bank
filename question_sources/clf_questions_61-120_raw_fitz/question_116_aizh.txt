Question #116
Which fully managed AWS service assists with the creation, testing, and management of custom Amazon EC2 images?
哪项完全托管的AWS服务协助创建、测试和管理自定义Amazon EC2镜像？
A. EC2 Image Builder Most Voted
EC2 Image Builder
B. Amazon Machine Image (AMI)
Amazon Machine Image (AMI)
C. AWS Launch Wizard
AWS Launch Wizard
D. AWS Elastic Beanstalk
AWS Elastic Beanstalk
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中与创建、测试和管理自定义Amazon EC2镜像相关的服务。
2. 选项分析
A. EC2 Image Builder: 正确。EC2 Image Builder是一个完全托管的AWS服务，专门用于简化创建、测试和管理自定义Amazon EC2
镜像的过程。它提供自动化的工作流程，减少手动操作。
B. Amazon Machine Image (AMI): 错误。AMI是EC2实例的镜像，但它本身不是一个服务，而是一个资源。它不能直接帮助创建或
管理镜像。
C. AWS Launch Wizard: 错误。AWS Launch Wizard是一个帮助用户部署复杂应用程序（如SAP或Microsoft应用）的服务，与创建
EC2镜像无关。
Correct Answer: A
正确答案：A

A company wants a fully managed Windows file server for its Windows-based applications.
Which AWS service will meet this requirement?
一家公司希望为其基于Windows的应用程序提供一个完全托管的Windows文件服务器。
哪个AWS服务可以满足此需求？
A. Amazon FSx Most Voted
Amazon FSx
B. Amazon Elastic Kubernetes Service (Amazon EKS)
Amazon Elastic Kubernetes Service (Amazon EKS)
C. Amazon Elastic Container Service (Amazon ECS)
Amazon Elastic Container Service (Amazon ECS)
D. Amazon EMR
Amazon EMR
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的功能和适用场景，特别是文件存储服务的选择。
需要了解Amazon FSx的功能，尤其是它支持Windows文件服务器的能力。
2. 选项分析
A. Amazon FSx: 正确。Amazon FSx 是一个完全托管的文件存储服务，支持 Windows 文件服务器，专为 Windows 应用程序设计，
满足公司对 Windows 文件服务器的需求。
Correct Answer: A
正确答案：A

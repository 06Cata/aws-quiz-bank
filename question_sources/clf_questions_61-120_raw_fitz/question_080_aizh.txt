Question #80
Which AWS service helps deliver highly available applications with fast failover for multi-Region and Multi-AZ architectures?
哪项AWS服务帮助以快速故障转移的方式为多区域和多可用区架构提供高度可用的应用程序？
A. AWS WAF
AWS WAF
B. AWS Global Accelerator Most Voted
AWS Global Accelerator
C. AWS Shield
AWS Shield
D. AWS Direct Connect
AWS Direct Connect
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务在高可用性、多区域（Multi-Region）和多可用区（Multi-AZ）架构中的应用，尤其是快速故障转移（Fast
Failover）的能力。
2. 选项分析
A. AWS WAF: AWS WAF（Web Application Firewall）主要用于保护Web应用免受常见的网络攻击（如SQL注入和跨站脚本攻击）。
它不涉及高可用性或故障转移功能，因此此选项错误。
B. AWS Global Accelerator: AWS Global Accelerator是一项服务，能够通过全球网络优化流量路由，提供低延迟、高可用性，并支
持快速故障转移。这正是题目要求的功能，因此此选项正确。
Correct Answer: B
正确答案：B

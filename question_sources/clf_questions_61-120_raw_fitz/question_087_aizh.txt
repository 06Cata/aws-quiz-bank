Which AWS service provides highly durable object storage?

题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中提供高持久性对象存储的能力。
需要了解AWS存储服务的特点，包括Amazon S3、EFS、EBS和FSx。
2. 选项分析
A. Amazon S3: 正确。Amazon S3是AWS提供的对象存储服务，具有高持久性和可用性。它设计为提供99.999999999%（11个9）
的持久性，非常适合存储大量非结构化数据。
B. Amazon Elastic File System (Amazon EFS): 错误。Amazon EFS是一个文件存储服务，主要用于共享文件系统，而不是对象存
储。它的设计目标是高可用性和可扩展性，但不专注于对象存储的高持久性。
Correct Answer: A
正确答案：A

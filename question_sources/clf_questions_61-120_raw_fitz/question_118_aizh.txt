A company needs to perform data processing once a week that typically takes about 5 hours to complete.
Which AWS service should the company use for this workload?
一家公司需要每周进行一次数据处理，通常需要大约5小时完成。
该公司应该使用哪项AWS服务来处理这项工作负载？
A. AWS Lambda
AWS Lambda
B. Amazon EC2 Most Voted
Amazon EC2
C. AWS CodeDeploy
AWS CodeDeploy

AWS Wavelength
题目分析与解答
1. 考察的知识点
本题考察的是如何选择适合的AWS服务来处理定期的数据处理工作负载。
重点在于理解各服务的用途和适用场景。
2. 选项分析
A. AWS Lambda:
错误。AWS Lambda适用于短时间的无服务器计算任务，通常运行时间限制为15分钟。由于数据处理需要5小时，Lambda不适合此
场景。
Correct Answer: B
正确答案：B

B. Amazon EC2:
正确。Amazon EC2提供可配置的虚拟服务器，适合长时间运行的任务，例如每周一次的5小时数据处理工作负载。
C. AWS CodeDeploy:
错误。AWS CodeDeploy是一个部署服务，用于自动化应用程序部署，而不是用于运行数据处理任务。
D. AWS Wavelength:
错误。AWS Wavelength主要用于边缘计算场景，与数据处理任务无关。
3. 我的答案
B. Amazon EC2

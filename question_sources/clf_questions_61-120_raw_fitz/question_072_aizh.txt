题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中支持自动化管理和轮换数据库用户凭证的能力。
重点关注AWS Secrets Manager的功能，以及其他选项的适用场景。
2. 选项分析
A. AWS Systems Manager:
AWS Systems Manager主要用于管理和自动化操作系统级别的任务，例如补丁管理、配置管理等。虽然它可以存储参数，但不直接
支持数据库凭证的自动轮换功能。因此此选项错误。
Correct Answer: B
正确答案：B

B. AWS Secrets Manager:
AWS Secrets Manager专门设计用于存储、管理和自动轮换敏感信息（如数据库凭证）。它提供了内置的凭证轮换功能，且操作开

C. AWS License Manager:
AWS License Manager用于管理软件许可证，与数据库凭证管理无关。因此此选项错误。
D. AWS Managed Services:
AWS Managed Services主要用于帮助企业管理AWS环境的整体操作，但不直接提供数据库凭证轮换功能。因此此选项错误。

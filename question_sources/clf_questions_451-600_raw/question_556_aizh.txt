Which benefit is always free of charge with AWS, regardless of a user’s AWS Support plan?
哪项福利无论用户的 AWS Support 计划如何，始终免费提供？
A. AWS Developer Support
AWS Developer Support
B. AWS Developer Forums
AWS Developer Forums
C. Programmatic case management
Programmatic case management
D. AWS technical account manager (TAM)
AWS technical account manager (TAM)
题目分析与解答
1. 考察的知识点
本题考察的是AWS支持服务的免费功能，以及不同支持计划的特点。
2. 选项分析
A. AWS Developer Support: 错误。AWS Developer Support是付费支持计划的一部分，用户需要订阅该计划才能享受相关服务。
B. AWS Developer Forums: 正确。AWS Developer Forums是一个免费的社区支持平台，所有AWS用户都可以免费使用，无需订阅
任何支持计划。
C. Programmatic case management: 错误。程序化案例管理是与付费支持计划相关的功能，通常需要订阅更高级别的支持计划。
D. AWS technical account manager (TAM): 错误。技术账户经理（TAM）是企业支持计划的一部分，只有订阅企业支持计划的用
户才能享受此服务。
Correct Answer: B
正确答案：B

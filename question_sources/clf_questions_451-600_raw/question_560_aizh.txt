1. 考察的知识点
本题考察的是AWS服务的选择，特别是针对静态网站托管的最佳实践和最低运维开销的解决方案。

B. Host the website on AWS Elastic Beanstalk.
错误。Elastic Beanstalk是一个平台即服务（PaaS）解决方案，适合动态网站或复杂应用。虽然它简化了应用部署，但对于静态网站
来说，使用Elastic Beanstalk显得过于复杂且不必要。
C. Deploy the website with Amazon Lightsail.
错误。Amazon Lightsail是一个简化的虚拟服务器解决方案，适合小型应用或动态网站。虽然它比EC2更易于使用，但仍需要管理服
务器，运维开销高于Amazon S3。
D. Host the website on Amazon S3.
正确。Amazon S3是一个对象存储服务，支持静态网站托管功能。它无需服务器管理，运维开销最低，且成本效益高，非常适合静
态网站托管。

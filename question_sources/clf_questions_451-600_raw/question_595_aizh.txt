A company wants to manage access and permissions for its third-party software as a service (SaaS) applications. The company
wants to use a portal where end users can access assigned AWS accounts and AWS Cloud applications.
Which AWS service should the company use to meet these requirements?
一家公司希望管理其第三方软件即服务 (SaaS) 应用程序的访问和权限。该公司希望使用一个门户，供最终用户访问分配的 AWS 账
户和 AWS Cloud 应用程序。
该公司应该使用哪项 AWS 服务来满足这些需求？
A. Amazon Cognito
Amazon Cognito
B. AWS IAM Identity Center (AWS Single Sign-On)
AWS IAM Identity Center (AWS Single Sign-On)
C. AWS Identity and Access Management (IAM)
AWS Identity and Access Management (IAM)
D. AWS Directory Service for Microsoft Active Directory
AWS Directory Service for Microsoft Active Directory
题目分析与解答
1. 考察的知识点
本题考察的是AWS身份管理与访问控制服务的功能，特别是如何管理第三方SaaS应用的访问权限以及提供用户访问门户。
2. 选项分析
Correct Answer: B
正确答案：B
Community vote distribution
社区投票分布
B (100%)
A. Amazon Cognito:
Amazon Cognito主要用于用户身份验证和管理，适合构建移动或Web应用程序的用户登录功能，但不适合管理第三方SaaS应用的
访问权限。因此此选项错误。
B. AWS IAM Identity Center (AWS Single Sign-On):
AWS IAM Identity Center（以前称为AWS Single Sign-On）专门用于集中管理用户访问权限，包括AWS账户、AWS服务以及第三方
SaaS应用。它还提供一个用户门户供用户访问分配的资源。因此此选项正确。
C. AWS Identity and Access Management (IAM):
AWS IAM是一个核心服务，用于管理AWS资源的权限，但它不提供用户门户，也不直接支持第三方SaaS应用的访问管理。因此此选
项错误。
D. AWS Directory Service for Microsoft Active Directory:
AWS Directory Service主要用于与Microsoft Active Directory集成，适合企业环境中的身份管理，但它不提供用户门户，也不专门用
于管理第三方SaaS应用的访问权限。因此此选项错误。
3. 我的答案
我的答案是：B. AWS IAM Identity Center (AWS Single Sign-On)
4. 官方答案
官方答案是：B. AWS IAM Identity Center (AWS Single Sign-On)

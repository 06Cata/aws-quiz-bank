Which recommendation can AWS Cost Explorer provide to help reduce cost?
AWS Cost Explorer 可以提供哪些建议来帮助降低成本？
A. Use a specific database engine.
使用特定的数据库引擎。
B. Change the programming language for an application.
更改应用程序的编程语言。
C. Deploy a specific operating system.
部署特定的操作系统。
D. Terminate an idle instance.
终止空闲实例。
题目分析与解答
1. 考察的知识点
本题考察的是AWS Cost Explorer的功能，特别是其在成本优化方面的建议能力。
2. 选项分析
A. Use a specific database engine. 使用特定的数据库引擎。
AWS Cost Explorer不会直接推荐使用特定的数据库引擎来降低成本。此选项与Cost Explorer的功能不相关，因此错误。
B. Change the programming language for an application. 更改应用程序的编程语言。
更改编程语言并不是AWS Cost Explorer的推荐内容。此选项与Cost Explorer的功能不相关，因此错误。
C. Deploy a specific operating system. 部署特定的操作系统。
AWS Cost Explorer不会建议部署特定的操作系统来降低成本。此选项与Cost Explorer的功能不相关，因此错误。
Correct Answer: D
正确答案：D

社区投票分布
D (100%)
D. Terminate an idle instance. 终止空闲实例。
AWS Cost Explorer可以识别未充分利用的资源（例如空闲实例），并建议终止这些资源以降低成本。因此此选项正确。
3. 我的答案
D. Terminate an idle instance. 终止空闲实例。
4. 官方答案
D. Terminate an idle instance. 终止空闲实例。

A company has a website on AWS. The company wants to deliver the website to a worldwide audience and provide low-latency
response times for global users.
Which AWS service will meet these requirements?
一家公司在 AWS 上有一个网站。该公司希望将网站交付给全球观众，并为全球用户提供低延迟响应时间。
哪项 AWS 服务能够满足这些需求？
A. AWS CloudFormation
AWS CloudFormation
B. Amazon CloudFront
Amazon CloudFront
C. Amazon ElastiCache
Amazon ElastiCache
D. Amazon DynamoDB
Amazon DynamoDB
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的功能和用途，特别是如何实现全球内容分发和低延迟访问。
2. 选项分析
Correct Answer: B
正确答案：B
Community vote distribution
社区投票分布
B (100%)
A. AWS CloudFormation: 错误。AWS CloudFormation是一种基础设施即代码（IaC）服务，用于创建和管理AWS资源的模板。它
不涉及内容分发或优化全球用户的访问延迟。
B. Amazon CloudFront: 正确。Amazon CloudFront是AWS的内容分发网络（CDN）服务，专门用于将内容分发到全球用户，并通
过边缘位置提供低延迟访问。
C. Amazon ElastiCache: 错误。Amazon ElastiCache是一种内存缓存服务，用于提高应用程序的性能，但它不负责全球内容分发或
优化用户访问延迟。
D. Amazon DynamoDB: 错误。Amazon DynamoDB是一种NoSQL数据库服务，主要用于存储和检索数据。它不涉及内容分发或优
化全球用户的访问延迟。

我的答案是：B. Amazon CloudFront
4. 官方答案
官方答案是：B. Amazon CloudFront

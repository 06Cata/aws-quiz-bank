题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的用途，特别是与数据仓库相关的服务。
2. 选项分析
A. Amazon RDS: Amazon RDS 是一种托管关系型数据库服务，适用于运行传统的关系型数据库（如 MySQL、PostgreSQL、SQL
Server 等）。它不专门设计用于处理数据仓库环境中的大规模数据，因此此选项错误。
B. Amazon DynamoDB: Amazon DynamoDB 是一种 NoSQL 数据库服务，适用于处理高吞吐量和低延迟的键值存储或文档存储场
景。它不适合数据仓库环境，因此此选项错误。
C. Amazon Redshift: Amazon Redshift 是一种专门设计用于数据仓库的服务，能够处理大规模数据分析任务。它支持复杂查询和分
析，是数据仓库环境的理想选择。因此此选项正确。
Correct Answer: C
正确答案：C

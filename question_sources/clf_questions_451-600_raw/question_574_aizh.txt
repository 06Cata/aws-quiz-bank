题目分析与解答
1. 考察的知识点
本题考察的是AWS服务在多账户环境中的管理能力，特别是如何快速设置和管理多账户架构。
2. 选项分析
A. AWS CloudFormation:
AWS CloudFormation主要用于基础设施即代码（IaC），帮助用户定义和部署AWS资源模板。虽然它可以跨账户部署资源，但它并

不直接支持多账户环境的设置和管理，因此此选项错误。
B. AWS Control Tower:
AWS Control Tower是专门设计用于设置和管理多账户AWS环境的服务。它提供了账户创建、治理、最佳实践以及安全性管理的功
能，非常适合本题要求。因此此选项正确。
C. AWS Config:
AWS Config主要用于资源配置的审计和合规性检查，帮助用户跟踪资源的配置变化。它不负责多账户环境的设置，因此此选项错
误。
D. Amazon Virtual Private Cloud (Amazon VPC):
Amazon VPC用于创建隔离的网络环境，主要关注网络层面的资源管理，与多账户环境的设置无关，因此此选项错误。

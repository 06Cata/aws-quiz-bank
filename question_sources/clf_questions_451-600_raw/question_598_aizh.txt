2. 选项分析
A. Amazon Textract: Amazon Textract 是一种服务，用于从文档中提取文本和数据。它不适用于构建聊天机器人，因此此选项错
误。
Correct Answer: B

B. Amazon Lex: Amazon Lex 是一种服务，用于构建对话式聊天机器人和语音交互应用。它支持自然语言处理 (NLP)，是构建聊天
机器人的正确选择，因此此选项正确。
C. AWS Glue: AWS Glue 是一种ETL（提取、转换、加载）服务，用于数据集成和数据准备。它与聊天机器人无关，因此此选项错
误。
D. Amazon Rekognition: Amazon Rekognition 是一种图像和视频分析服务，用于面部识别、对象检测等。它不适用于构建聊天机
器人，因此此选项错误。

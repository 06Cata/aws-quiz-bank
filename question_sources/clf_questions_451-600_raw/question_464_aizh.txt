题目分析与解答
1. 考察的知识点
本题考察的是AWS的服务器无关（Serverless）计算服务的知识点。
需要了解AWS Lambda及其他相关服务的功能和用途。
2. 选项分析
Correct Answer: A
正确答案：A
Community vote distribution
社区投票分布
A (100%)
A. AWS Lambda: 正确。AWS Lambda 是一种完全服务器无关的计算服务，允许用户运行代码而无需管理服务器。它是专门设计用
于支持 Serverless 架构的服务。
B. AWS CloudFormation: 错误。AWS CloudFormation 是一种基础设施即代码（IaC）服务，用于定义和管理AWS资源，但它不是
计算服务，也不支持 Serverless 架构。
C. AWS Elastic Beanstalk: 错误。AWS Elastic Beanstalk 是一种平台即服务（PaaS），虽然简化了应用部署，但它仍然需要管理底
层服务器，因此不属于 Serverless 服务。
D. Elastic Load Balancing: 错误。Elastic Load Balancing 是一种负载均衡服务，用于分发流量到多个目标实例或容器，但它不是计
算服务，也不支持 Serverless 架构。
3. 我的答案
A. AWS Lambda
4. 官方答案

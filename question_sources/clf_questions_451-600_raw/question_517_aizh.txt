Which AWS service supports a company's ability to treat infrastructure as code?
哪个AWS服务支持公司将基础设施视为代码的能力？
A. AWS CodeDeploy
AWS CodeDeploy
B. AWS Elastic Beanstalk
AWS Elastic Beanstalk
C. Amazon API Gateway
Amazon API Gateway
D. AWS CloudFormation
AWS CloudFormation
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中支持“基础设施即代码（Infrastructure as Code, IaC）”的能力。
2. 选项分析
A. AWS CodeDeploy: 错误。AWS CodeDeploy 是一种自动化部署服务，用于将代码部署到 EC2 实例、Lambda 函数或本地服务
器。它不直接支持基础设施即代码。
B. AWS Elastic Beanstalk: 错误。AWS Elastic Beanstalk 是一种平台即服务（PaaS），用于快速部署和管理应用程序，但它不专门
用于定义和管理基础设施即代码。
C. Amazon API Gateway: 错误。Amazon API Gateway 是一种服务，用于创建、发布和管理 API，它与基础设施即代码无关。
Correct Answer: D
正确答案：D

D. AWS CloudFormation: 正确。AWS CloudFormation 是一种服务，专门用于定义和管理基础设施即代码。用户可以通过模板文
件定义资源，并自动化创建和管理这些资源。

D. AWS CloudFormation

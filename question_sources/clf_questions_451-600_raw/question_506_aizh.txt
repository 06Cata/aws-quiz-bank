题目分析与解答
1. 考察的知识点
本题考察的是AWS安全服务的功能和适用场景，特别是针对威胁检测和监控的服务。
2. 选项分析
Correct Answer: C
正确答案：C
Community vote distribution
社区投票分布
C (100%)
A. AWS Shield: AWS Shield是一个专门用于保护应用免受分布式拒绝服务（DDoS）攻击的服务。它不提供对AWS账户、工作负载
或S3桶的全面威胁检测，因此此选项错误。
B. AWS Firewall Manager: AWS Firewall Manager是一个集中管理防火墙规则的服务，主要用于管理和部署规则以保护资源。它不
具备持续监控威胁的功能，因此此选项错误。
C. Amazon GuardDuty: Amazon GuardDuty是一项威胁检测服务，能够持续监控AWS账户、工作负载和S3桶以检测恶意活动和未
经授权的行为。此选项正确。
D. Amazon Inspector: Amazon Inspector是一项自动化的安全评估服务，主要用于检查应用程序的安全漏洞和合规性问题。它不适
用于持续威胁检测，因此此选项错误。

我的答案是：C. Amazon GuardDuty
4. 官方答案
官方答案是：C. Amazon GuardDuty

A company needs to manage multiple logins across AWS accounts within the same organization in AWS Organizations.
Which AWS service should the company use to meet this requirement?
一家公司需要在同一 AWS Organizations 组织内管理多个 AWS 账户的登录。
该公司应该使用哪项 AWS 服务来满足这一需求？
A. Amazon VPC
Amazon VPC
B. Amazon GuardDuty
Amazon GuardDuty
C. Amazon Cognito
Amazon Cognito
D. AWS IAM Identity Center
AWS IAM Identity Center
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中用于管理跨账户登录的解决方案，特别是在AWS Organizations环境中。
2. 选项分析
A. Amazon VPC: 错误。Amazon VPC（Virtual Private Cloud）用于创建隔离的网络环境，与管理用户登录无关。
B. Amazon GuardDuty: 错误。Amazon GuardDuty是一个威胁检测服务，用于监控恶意活动，与用户登录管理无关。
Correct Answer: D
正确答案：D

C. Amazon Cognito: 错误。Amazon Cognito主要用于管理应用程序的用户身份验证和授权，而不是跨AWS账户的登录管理。
D. AWS IAM Identity Center: 正确。AWS IAM Identity Center（原AWS SSO）是专门用于管理跨AWS账户登录的服务，支持集中
身份管理和单点登录功能。

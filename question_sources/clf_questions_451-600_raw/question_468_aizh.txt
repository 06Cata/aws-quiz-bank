题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的功能和用途，特别是与安全相关的服务。
重点是了解如何监控和阻止恶意的HTTP和HTTPS请求。
2. 选项分析
Correct Answer: C
正确答案：C

C (100%)
A. Amazon GuardDuty: Amazon GuardDuty是一项威胁检测服务，用于识别潜在的恶意活动和未经授权的行为，但它不能直接监
控和阻止HTTP和HTTPS请求。因此此选项错误。
B. Amazon Inspector: Amazon Inspector是一项自动化的安全评估服务，用于检查应用程序的安全漏洞，但它与HTTP和HTTPS请求
的监控和阻止无关。因此此选项错误。
C. AWS WAF: AWS WAF是一项Web应用程序防火墙服务，专门用于监控和阻止恶意的HTTP和HTTPS请求。这正是题目要求的功
能，因此此选项正确。
D. Amazon Detective: Amazon Detective是一项用于调查和分析安全问题的服务，但它不能直接监控和阻止HTTP和HTTPS请求。
因此此选项错误。

我的答案是：C. AWS WAF

官方答案是：C. AWS WAF

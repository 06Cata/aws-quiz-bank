Which AWS service can be used to send alerts when a specific Amazon CloudWatch alarm is invoked?
A. AWS CloudTrail
B. Amazon Simple Notification Service (Amazon SNS)
C. Amazon Simple Queue Service (Amazon SQS)
D. Amazon EventBridge

Selected Answer: B
CloudWatch:
• Metrics: monitor the performance of AWS services and billing metrics
• Alarms: automate notification, perform EC2 action, notify to SNS based on metric
• Logs: collect log files from EC2 instances, servers, Lambda functions…
• Events (or EventBridge): react to events in AWS, or trigger a rule on a schedule

Selected Answer: B
B. Amazon Simple Notification Service (Amazon SNS) - allows you to send notifications, emails, and alerts to various endpoints. You can configure the
triggered CloudWatch alarm to send a notification to the SNS topic, which then forwards the alert to the relevant recipients.

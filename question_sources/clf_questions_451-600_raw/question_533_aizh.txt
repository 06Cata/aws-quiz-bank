题目分析与解答
1. 考察的知识点
本题考察的是AWS存储服务的功能，特别是如何通过协议（如NFS）与Amazon S3交互。
2. 选项分析
A. Amazon FSx for Lustre:
Amazon FSx for Lustre是一种高性能文件系统，通常用于与Amazon S3集成以加速数据处理，但它并不直接支持通过NFS协议与S3
交互。因此此选项错误。
B. AWS Storage Gateway volume gateway:
Volume Gateway主要用于块存储（iSCSI协议），而不是文件存储（NFS协议）。它不支持通过NFS协议与S3交互。因此此选项错
误。
Correct Answer: C
正确答案：C

C. AWS Storage Gateway file gateway:
File Gateway允许使用NFS或SMB协议将文件存储到Amazon S3中，并提供与S3的直接集成。这正是题目要求的功能。因此此选项正
确。
D. Amazon Elastic File System (Amazon EFS):
Amazon EFS是一个完全托管的文件存储服务，支持NFS协议，但它是为EC2实例设计的，并不直接与Amazon S3交互。因此此选项
错误。

我的答案是：C. AWS Storage Gateway file gateway

官方答案是：C

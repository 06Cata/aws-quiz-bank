哪些AWS服务或工具提供此功能？（选择两个。）

AWS Elastic Beanstalk

题目分析与解答
1. 考察的知识点
本题考察的是AWS服务和工具的功能，特别是与优化Amazon EC2资源相关的服务。
2. 选项分析
A. AWS Elastic Beanstalk: 错误。AWS Elastic Beanstalk是一个用于快速部署和管理应用程序的服务，但它不直接提供优化现有EC2
资源的功能。
Correct Answer: BD
正确答案：BD
B. AWS Cost Explorer: 正确。AWS Cost Explorer可以帮助用户分析和优化成本，包括EC2实例的使用情况，从而间接支持优化资
源。
C. Amazon Detective: 错误。Amazon Detective是一个安全分析服务，用于调查潜在的安全问题，与优化EC2资源无关。
D. AWS Compute Optimizer: 正确。AWS Compute Optimizer专门用于分析和优化EC2实例的配置和使用情况，是直接相关的服
务。
E. AWS Billing Conductor: 错误。AWS Billing Conductor是一个计费管理工具，用于组织和分配账单，与优化EC2资源无关。

我的答案是：B, D

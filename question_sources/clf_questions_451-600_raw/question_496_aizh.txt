题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中与DNS相关的功能，特别是高可用性和可扩展性的DNS服务。
2. 选项分析
A. Amazon Route 53: 正确。Amazon Route 53 是 AWS 提供的高可用性和可扩展性的 DNS 服务，专门设计用于管理域名解析和路
由流量到不同的资源。
Correct Answer: A
正确答案：A

B. Amazon Lightsail: 错误。Amazon Lightsail 是一种简化的云计算服务，主要用于托管简单的应用程序和网站，但它不是专门的
DNS 服务。
C. AWS Amplify Hosting: 错误。AWS Amplify Hosting 是一种前端和全栈应用程序的托管服务，主要用于部署和托管 Web 应用程
序，而不是 DNS 服务。
D. Amazon S3: 错误。Amazon S3 是一种对象存储服务，用于存储和检索数据，但它与 DNS 服务无关。

A. Amazon Route 53

A company wants to use AWS. The company has stringent requirements about low-latency access to on-premises systems and
data residency.
Which AWS service should the company use to design a solution that meets these requirements?
一家公司希望使用AWS。该公司对低延迟访问本地系统和数据驻留有严格要求。
该公司应该使用哪项AWS服务来设计符合这些要求的解决方案？
A. AWS Wavelength
AWS Wavelength
B. AWS Transit Gateway
AWS Transit Gateway
C. AWS Ground Station
AWS Ground Station
D. AWS Outposts
AWS Outposts
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的功能和适用场景，特别是与低延迟访问和数据驻留相关的服务。
2. 选项分析
Correct Answer: D
正确答案：D
Community vote distribution
社区投票分布
D (100%)
A. AWS Wavelength:
错误。AWS Wavelength主要用于将AWS服务扩展到电信运营商的5G网络边缘，以支持超低延迟的应用场景，但它并不直接解决数
据驻留或与本地系统的集成问题。
B. AWS Transit Gateway:
错误。AWS Transit Gateway用于连接多个VPC和本地网络，但它主要是网络连接的解决方案，不能满足数据驻留的要求。
C. AWS Ground Station:
错误。AWS Ground Station是一个卫星通信服务，与低延迟访问和数据驻留无关。
D. AWS Outposts:
正确。AWS Outposts是一种混合云解决方案，可以将AWS服务扩展到本地数据中心，满足低延迟访问和数据驻留的需求。
3. 我的答案
D. AWS Outposts

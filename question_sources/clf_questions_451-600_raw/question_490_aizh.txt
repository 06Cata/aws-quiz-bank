A company is building a web application using AWS.
Which AWS service will help prevent network layer DDoS attacks against the web application?
一家公司正在使用AWS构建一个Web应用程序。
哪个AWS服务可以帮助防止针对Web应用程序的网络层DDoS攻击？
A. AWS WAF
AWS WAF
B. AWS Firewall Manager
AWS Firewall Manager
C. Amazon GuardDuty
Amazon GuardDuty
D. AWS Shield
AWS Shield
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务在网络安全方面的功能，特别是防御网络层DDoS攻击的能力。
2. 选项分析
A. AWS WAF: AWS WAF（Web Application Firewall）主要用于保护应用程序免受常见的Web攻击（如SQL注入和跨站脚本攻击），
而不是专门针对网络层DDoS攻击。因此此选项错误。
Correct Answer: D
正确答案：D

社区投票分布
D (100%)
B. AWS Firewall Manager: AWS Firewall Manager是一个集中管理工具，用于管理多个账户中的防火墙规则和策略。它本身不直接
防御DDoS攻击，而是帮助管理其他安全服务（如AWS WAF和AWS Shield）。因此此选项错误。
C. Amazon GuardDuty: Amazon GuardDuty是一项威胁检测服务，用于识别潜在的恶意活动和异常行为，但它不直接防御网络层
DDoS攻击。因此此选项错误。
D. AWS Shield: AWS Shield是一项专门设计用于防御DDoS攻击的服务，分为标准版和高级版。它可以有效防御网络层和应用层的
DDoS攻击。因此此选项正确。
3. 我的答案
D. AWS Shield
4. 官方答案
D. AWS Shield

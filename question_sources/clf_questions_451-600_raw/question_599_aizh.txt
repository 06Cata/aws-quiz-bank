Which AWS service or feature can be used to monitor for potential disk write spikes on a system that is running on Amazon EC2?
哪个AWS服务或功能可用于监控运行在Amazon EC2上的系统潜在的磁盘写入峰值？
A. AWS CloudTrail
AWS CloudTrail
B. AWS Health Dashboard
AWS Health Dashboard
C. AWS Trusted Advisor
AWS Trusted Advisor
D. Amazon CloudWatch
Amazon CloudWatch
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中用于监控系统性能的工具，特别是监控EC2实例的磁盘写入性能。
2. 选项分析
A. AWS CloudTrail: 错误。AWS CloudTrail主要用于记录和审计用户的API调用和活动日志，无法直接监控磁盘写入性能。
B. AWS Health Dashboard: 错误。AWS Health Dashboard提供关于AWS服务状态的健康信息，但不用于监控EC2实例的性能指
标。
C. AWS Trusted Advisor: 错误。AWS Trusted Advisor提供优化建议，例如成本节约、安全性和性能改进，但不直接监控磁盘写入
性能。
D. Amazon CloudWatch: 正确。Amazon CloudWatch是AWS的监控服务，可以收集和跟踪EC2实例的性能指标，包括磁盘写入性
能。
3. 我的答案
Correct Answer: D
正确答案：D
D. Amazon CloudWatch
4. 官方答案
D. Amazon CloudWatch

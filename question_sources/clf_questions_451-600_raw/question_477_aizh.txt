A company uses a third-party identity provider (IdP). The company wants to provide its employees with access to AWS accounts
and services without requiring another set of login credentials.
Which AWS service will meet this requirement?
一家公司使用第三方身份提供商 (IdP)。该公司希望为其员工提供访问 AWS 账户和服务的权限，而无需再设置另一组登录凭据。
哪个 AWS 服务可以满足此需求？
A. AWS Directory Service
AWS Directory Service
B. Amazon Cognito
Amazon Cognito
C. AWS IAM Identity Center Most Voted
AWS IAM Identity Center
D. AWS Resource Access Manager (AWS RAM)
AWS Resource Access Manager (AWS RAM)
题目分析与解答
1. 考察的知识点
本题考察的是AWS身份管理与访问控制相关的知识，特别是如何通过第三方身份提供商（IdP）集成用户身份以访问AWS资源。
2. 选项分析
Correct Answer: C
正确答案：C

A. AWS Directory Service:
AWS Directory Service主要用于与Microsoft Active Directory集成，帮助管理用户和资源的身份验证。虽然它可以提供身份管理功
能，但它并不直接支持通过第三方IdP进行单点登录（SSO）到AWS账户。因此此选项错误。
B. Amazon Cognito:
Amazon Cognito主要用于为应用程序提供用户身份验证和管理功能。虽然它支持与第三方IdP集成，但它的主要用途是为应用程序
用户提供身份验证，而不是为员工提供对AWS账户和服务的访问。因此此选项错误。
C. AWS IAM Identity Center:
AWS IAM Identity Center（以前称为AWS Single Sign-On）支持与第三方IdP集成，并允许员工通过单点登录（SSO）访问多个AWS
账户和服务，而无需额外的登录凭据。这正是题目要求的功能，因此此选项正确。
D. AWS Resource Access Manager (AWS RAM):
AWS RAM用于跨账户共享AWS资源，而不是管理用户身份或提供单点登录功能。因此此选项错误。
3. 我的答案
我的答案是：C. AWS IAM Identity Center
4. 官方答案
官方答案是：C. AWS IAM Identity Center

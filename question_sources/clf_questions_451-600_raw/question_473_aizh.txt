Which AWS service allows for file sharing between multiple Amazon EC2 instances?
哪个AWS服务允许在多个Amazon EC2实例之间进行文件共享？
A. AWS Direct Connect
AWS Direct Connect
B. AWS Snowball Edge
AWS Snowball Edge
C. AWS Backup
AWS Backup
D. Amazon Elastic File System (Amazon EFS)
Amazon Elastic File System (Amazon EFS)
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中支持文件共享的解决方案，尤其是Amazon Elastic File System (Amazon EFS)的功能。
2. 选项分析
A. AWS Direct Connect: 错误。AWS Direct Connect是一种网络服务，用于建立本地数据中心与AWS之间的专用网络连接，不涉及
文件共享功能。
B. AWS Snowball Edge: 错误。AWS Snowball Edge是一种数据迁移设备，主要用于将大量数据迁移到AWS云中，不适用于实时文
件共享。
C. AWS Backup: 错误。AWS Backup是一项集中化备份服务，用于备份和恢复数据，不支持文件共享功能。
Correct Answer: D
正确答案：D

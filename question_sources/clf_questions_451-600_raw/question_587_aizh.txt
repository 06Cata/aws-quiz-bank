A company wants to run relationship databases in the AWS Cloud. The company wants to use a managed service that will install
the database and run regular software updates.
Which AWS service will meet these requirements?

哪个AWS服务能够满足这些需求？

题目分析与解答
1. 考察的知识点
本题考察的是AWS的托管服务，特别是关系型数据库服务（RDS）的功能和用途。
需要理解AWS中不同服务的用途，例如Amazon S3、Amazon RDS、Amazon EBS和Amazon DynamoDB。
2. 选项分析
Correct Answer: B
正确答案：B

B (100%)
A. Amazon S3: Amazon S3是一个对象存储服务，主要用于存储和检索任意数量的数据。它不支持关系型数据库的运行，因此此选
项错误。
B. Amazon RDS: Amazon RDS是一个托管的关系型数据库服务，支持自动安装数据库软件、运行更新和管理基础设施。此选项正
确。
C. Amazon Elastic Block Store (Amazon EBS): Amazon EBS是块存储服务，通常用于为EC2实例提供存储。虽然可以用EBS来存储
数据库数据，但它不是托管数据库服务，因此此选项错误。
D. Amazon DynamoDB: Amazon DynamoDB是一个托管的NoSQL数据库服务，适用于非关系型数据库场景。本题要求的是关系型
数据库，因此此选项错误。
3. 我的答案
我的答案是：B. Amazon RDS
4. 官方答案
官方答案是：B. Amazon RDS

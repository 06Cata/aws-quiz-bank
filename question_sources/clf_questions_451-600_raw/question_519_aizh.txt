题目分析与解答
1. 考察的知识点
本题考察的是AWS安全服务的功能和用途，特别是如何检测潜在的安全威胁和账户或实例的异常行为。
2. 选项分析
A. Amazon GuardDuty: 正确。Amazon GuardDuty 是一种威胁检测服务，可以持续监控 AWS 账户和工作负载以发现恶意活动和
异常行为。它专门设计用于检测潜在的账户或实例被入侵的情况。
Correct Answer: A
正确答案：A
Community vote distribution
社区投票分布
A (100%)
B. AWS WAF: 错误。AWS WAF 是一种 Web 应用程序防火墙，用于保护 Web 应用免受常见的攻击（如 SQL 注入和跨站脚本攻
击），但它不负责检测账户或实例的异常行为。
C. AWS Shield: 错误。AWS Shield 是一种分布式拒绝服务（DDoS）保护服务，主要用于防御 DDoS 攻击，而不是检测账户或实例
的异常行为。
D. Amazon Inspector: 错误。Amazon Inspector 是一种自动化的安全评估服务，用于检查应用程序的安全漏洞和合规性问题，但
它不负责检测账户或实例的异常行为。
3. 我的答案
A. Amazon GuardDuty
4. 官方答案
A. Amazon GuardDuty

A company needs to use AWS technology to deploy a static website.
Which solution meets this requirement with the LEAST amount of operational overhead?
A. Deploy the website on Amazon EC2.
B. Host the website on AWS Elastic Beanstalk.
C. Deploy the website with Amazon Lightsail.
D. Host the website on Amazon S3.

STATIC website being the key word. It doesn’t change, and it will have a public URL to access it. S3 can do this no problem with minimal overhead.

Amazon S3 (Simple Storage Service) is ideal for hosting static websites with minimal operational overhead. It allows you to store and serve static
content (like HTML, CSS, and JavaScript files) directly from S3 buckets without needing to manage any servers or complex infrastructure. Additionally
Amazon S3 is highly scalable, cost-effective, and requires very little maintenance.

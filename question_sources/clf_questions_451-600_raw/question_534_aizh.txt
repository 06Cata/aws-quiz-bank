A. AWS Command Line Interface (AWS CLI): 正确。用户可以使用AWS CLI命令来更改自己的IAM用户密码，只要他们拥有相关权
限。
B. AWS Key Management Service (AWS KMS): 错误。AWS KMS用于管理加密密钥，与IAM用户密码管理无关。
C. AWS Management Console: 正确。用户可以通过AWS Management Console的“我的安全凭证”页面来更改自己的IAM用户密
码。
D. AWS Resource Access Manager (AWS RAM): 错误。AWS RAM用于共享资源，与IAM用户密码管理无关。
E. AWS Secrets Manager: 错误。AWS Secrets Manager用于管理应用程序的密钥和凭证，与IAM用户密码管理无关。

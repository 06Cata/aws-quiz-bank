题目分析与解答
1. 考察的知识点
本题考察的是如何使用AWS服务迁移商业关系型数据库到AWS托管的开源数据库。
涉及的服务包括AWS Database Migration Service (AWS DMS)和AWS Schema Conversion Tool (SCT)。
2. 选项分析
A. AWS Database Migration Service (AWS DMS): 正确。AWS DMS是专门用于迁移数据库的服务，可以帮助将数据从源数据库迁
移到目标数据库。
Correct Answer: AC

B. AWS software development kits (SDKs): 错误。SDKs主要用于开发和与AWS服务交互，不是专门用于数据库迁移的工具。
C. AWS Schema Conversion Tool: 正确。SCT可以帮助将源数据库的架构转换为目标数据库的架构，尤其是在迁移到不同类型的数
据库时非常有用。
D. AWS Systems Manager: 错误。Systems Manager主要用于管理和自动化操作任务，与数据库迁移无关。
E. Amazon EMR: 错误。Amazon EMR是用于大数据处理的服务，与数据库迁移无关。

Which AWS service will meet these requirements?

题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的功能和用途，特别是与网络安全和流量控制相关的服务。
2. 选项分析
Correct Answer: B
正确答案：B
Community vote distribution
社区投票分布
B (100%)
A. Amazon GuardDuty: Amazon GuardDuty是一项威胁检测服务，用于识别潜在的恶意活动和未经授权的行为。它不用于直接控
制或过滤流量，因此此选项错误。
B. AWS WAF: AWS WAF是一种Web应用程序防火墙，专门用于保护Web应用程序免受常见的网络攻击，并允许用户定义自定义规
则来过滤和控制流量。此选项正确。
C. Amazon Macie: Amazon Macie是一项数据安全服务，主要用于识别和保护敏感数据（如PII）。它不涉及流量过滤，因此此选项
错误。
D. AWS Shield: AWS Shield是一项DDoS防护服务，旨在保护应用程序免受分布式拒绝服务攻击。虽然它与安全相关，但它不提供
自定义流量过滤功能，因此此选项错误。
3. 我的答案
我的答案是：B. AWS WAF
4. 官方答案
官方答案是：B. AWS WAF

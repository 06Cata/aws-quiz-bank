Which of the following are advantages of moving to the AWS Cloud? (Choose two.)
以下哪些是迁移到AWS云的优势？（选择两个。）
A. Users can implement all AWS services in seconds.
用户可以在几秒钟内实施所有AWS服务。
B. AWS assumes all responsibility for the security of infrastructure and applications.
AWS承担基础设施和应用程序的所有安全责任。
C. Users experience increased speed and agility.
用户体验到更快的速度和更高的敏捷性。
D. Users benefit from massive economies of scale.
用户受益于大规模经济效益。
E. Users can move hardware from their data center to the AWS Cloud.
用户可以将硬件从他们的数据中心迁移到AWS云。
题目分析与解答
1. 考察的知识点
本题考察的是AWS云的优势，包括经济效益、速度和敏捷性等核心概念。
2. 选项分析
A. Users can implement all AWS services in seconds.
错误。虽然AWS提供了快速部署服务的能力，但并不是所有服务都可以在几秒钟内实施。例如，某些复杂服务需要配置和时间来完
成部署。
B. AWS assumes all responsibility for the security of infrastructure and applications.
错误。AWS采用共享责任模型，AWS负责基础设施的安全，而用户负责应用程序和数据的安全。
C. Users experience increased speed and agility.
正确。AWS云的弹性和按需服务使用户能够快速部署资源并提高敏捷性。
Correct Answer: CD
正确答案：CD
D. Users benefit from massive economies of scale.
正确。AWS通过大规模运营降低成本，用户可以享受经济效益。
E. Users can move hardware from their data center to the AWS Cloud.
错误。AWS云是基于虚拟化的服务，用户无法直接将物理硬件迁移到云中。

C
D

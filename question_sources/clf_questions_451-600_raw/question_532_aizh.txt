Which of the following can the AWS Pricing Calculator do?
以下哪些是 AWS Pricing Calculator 可以做到的？
A. Project monthly AWS costs.
预梳桹月的 AWS 成本。
B. Calculate historical AWS costs.
计算历史 AWS 成本。
C. Provide in-depth information about AWS pricing strategies.
提供有关 AWS 定价策略的棅入信息。
D. Provide users with access to their monthly bills.
为用户提供访问其桹月账单的权限。
题目分析与解答
1. 考察的知识点
本题考察的是对 AWS Pricing Calculator 的功能和用途的栬解。
2. 选项分析
A. Project monthly AWS costs. 预测每月的 AWS 成本。
桭确。AWS Pricing Calculator 的主要功能是帮助用户估算和预梳使用 AWS 服务的成本。它允许用户根据服务配置和使用量来计算
未来的费用。
B. Calculate historical AWS costs. 计算历史 AWS 成本。
错误。AWS Pricing Calculator 不用于计算历史成本。历史成本可以通过 AWS Billing 和 Cost Management 服务查看。
Correct Answer: A
桭确答栾：A

A (100%)
C. Provide in-depth information about AWS pricing strategies. 提供有关 AWS 定价策略的深入信息。
错误。AWS Pricing Calculator 主要用于成本估算，而不是提供关于定价策略的详细信息。
D. Provide users with access to their monthly bills. 为用户提供访问其每月账单的权限。
错误。访问桹月账单的功能是通过 AWS Billing 和 Cost Management 服务实栥的，而不是 AWS Pricing Calculator。

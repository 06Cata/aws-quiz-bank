A company needs to perform an audit of recent AWS account activity. The audit will investigate who initiated an event and what
actions were performed.
Which AWS service should the company use to meet these requirements?
一家公司需要对最近的AWS账户活动进行审计。审计将调查是谁发起了事件以及执行了哪些操作。
该公司应该使用哪项AWS服务来满足这些需求？
A. AWS Config
AWS Config
B. Amazon Rekognition
Amazon Rekognition
C. AWS CloudTrail
AWS CloudTrail
D. Amazon Simple Notification Service (Amazon SNS)
Amazon Simple Notification Service (Amazon SNS)
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的功能，特别是与审计和跟踪用户活动相关的服务。
2. 选项分析
Correct Answer: C
正确答案：C
Community vote distribution
社区投票分布
C (100%)
A. AWS Config: AWS Config主要用于资源配置的记录和合规性检查。虽然它可以记录资源的状态变化，但它并不专注于跟踪用户
活动或事件的发起者，因此不适合用于审计用户活动。
B. Amazon Rekognition: Amazon Rekognition是一个图像和视频分析服务，用于面部识别、对象检测等，与审计用户活动无关，
因此此选项错误。
C. AWS CloudTrail: AWS CloudTrail是专门用于记录AWS账户活动的服务，包括谁发起了事件以及执行了哪些操作。它是审计用户
活动的正确选择。
D. Amazon Simple Notification Service (Amazon SNS): Amazon SNS是一个消息传递服务，用于发送通知或消息。它不记录用户
活动，因此不适合用于审计目的。
3. 我的答案
C. AWS CloudTrail
4. 官方答案
C. AWS CloudTrail

我的答案与官方答案一致，没有差异。

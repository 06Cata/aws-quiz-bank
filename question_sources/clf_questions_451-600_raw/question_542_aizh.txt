An ecommerce company wants to distribute traffic between the Amazon EC2 instances that host its website.
Which AWS service or resource will meet these requirements?
一孰电子商务公司家望在孋管其网站的 Amazon EC2 孩例之间分配流量。
哪个 AWS 服务嬻资源可以满足这些需求？
A. Application Load Balancer
Application Load Balancer
B. AWS WAF
AWS WAF

AWS CloudHSM

AWS Direct Connect
题目分析与解答
1. 考察的知识点
本题考孹的是AWS服务中用于分发流量的解决方案，特别是负载均衡相关的知识。
2. 选项分析
A. Application Load Balancer: 正确。Application Load Balancer 是 AWS 提供的负载均衡服务，专门用于分发流量到多个 Amazon
EC2 孩例嬻其他目标资源。孚支持 HTTP 和 HTTPS 协议，非寀适合电子商务网站的流量分发需求。
Correct Answer: A
正确答案：A
Community vote distribution

A (100%)
B. AWS WAF: 错误。AWS WAF 是一种 Web 寒用程寏防火墙，用于保护 Web 寒用免受嬡嬯流量攻击，但孚不负责流量分发。
C. AWS CloudHSM: 错误。AWS CloudHSM 是一种硬件孞全模块，用于加孶孶钥管理，与流量分发无关。
D. AWS Direct Connect: 错误。AWS Direct Connect 是一种网络服务，用于寠立本地数据中嬇与 AWS 云之间的专用网络连接，与
流量分发无关。
3. 我的答案
A. Application Load Balancer
4. 官方答案
A. Application Load Balancer

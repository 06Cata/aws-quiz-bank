A company is building an application that will receive millions of database queries each second. The company needs the data
store for the application to scale to meet these needs.
Which AWS service will meet this requirement?
一家公司正在构建一个应用程序，该应用程序每秒将接收数百万次数据库查询。该公司需要为应用程序的数据存储能够扩展以满足
这些需求。
哪项AWS服务可以满足此需求？
A. Amazon DynamoDB
Amazon DynamoDB
B. AWS Cloud9
AWS Cloud9
C. Amazon ElastiCache for Memcached
Amazon ElastiCache for Memcached
D. Amazon Neptune
Amazon Neptune
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的特点与适用场景，特别是高吞吐量和可扩展性的数据存储服务。
2. 选项分析
Correct Answer: A
正确答案：A
Community vote distribution
社区投票分布
A (100%)
A. Amazon DynamoDB: 正确。Amazon DynamoDB 是一个完全托管的 NoSQL 数据库服务，能够处理每秒数百万次请求，具有高
可扩展性和低延迟，非常适合需要高吞吐量的应用场景。
B. AWS Cloud9: 错误。AWS Cloud9 是一个基于云的集成开发环境 (IDE)，用于编写、运行和调试代码，与数据存储或高吞吐量无
关。
C. Amazon ElastiCache for Memcached: 错误。虽然 ElastiCache for Memcached 可以用于缓存以提高查询性能，但它不是一个主
要的数据存储解决方案，且不适合处理数百万次请求的场景。
D. Amazon Neptune: 错误。Amazon Neptune 是一个图数据库服务，适用于处理图形数据和关系查询，不适合处理高吞吐量的普
通数据库查询。
3. 我的答案
A. Amazon DynamoDB
4. 官方答案
A. Amazon DynamoDB

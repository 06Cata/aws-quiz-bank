A company runs an on-premises contact center for customers. The company needs to migrate to a cloud-based solution that can
deliver artificial intelligence features to improve user experience.
Which AWS service will meet these requirements?
一家公司为客户运行一个本地部署的联络中心。该公司需要迁移到一个基于云的解决方案，以提供人工智能功能来改善用户体验。

题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的功能和适用场景，特别是与客户交互和人工智能相关的服务。
2. 选项分析
Correct Answer: D
正确答案: D
Community vote distribution
社区投票分布
D (100%)
A. AWS Wavelength: AWS Wavelength主要用于将AWS服务扩展到5G网络边缘，以实现低延迟应用。这与题目中提到的迁移到云
端的客户交互中心无关，因此此选项错误。
B. AWS IAM Identity Center (AWS Single Sign-On): AWS IAM Identity Center用于集中管理用户访问权限，与题目中提到的客户
交互中心和人工智能功能无关，因此此选项错误。
C. AWS Direct Connect: AWS Direct Connect用于建立本地数据中心与AWS之间的专用网络连接，虽然可以用于迁移，但它并不提
供客户交互中心或人工智能功能，因此此选项错误。
D. Amazon Connect: Amazon Connect是一个云端联络中心解决方案，支持客户交互并集成人工智能功能（如Amazon Lex和
Amazon Polly）以改善用户体验。因此此选项正确。
3. 我的答案
D. Amazon Connect
4. 官方答案
D. Amazon Connect

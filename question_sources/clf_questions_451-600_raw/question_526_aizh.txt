What is the MOST secure way to store passwords on AWS?
在AWS上存储密码最安全的方式是什么？
A. Store passwords in an Amazon S3 bucket.
将密码存储在Amazon S3 bucket中。
B. Store passwords as AWS CloudFormation parameters.
将密码作为AWS CloudFormation参数存储。
C. Store passwords in AWS Storage Gateway.
将密码存储在AWS Storage Gateway中。
D. Store passwords in AWS Secrets Manager.
将密码存储在AWS Secrets Manager中。
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中密码和敏感信息的安全存储方法，涉及AWS Secrets Manager的功能和最佳实践。
2. 选项分析
A. Store passwords in an Amazon S3 bucket.
错误：虽然Amazon S3可以存储数据，但它不是专门设计用于存储敏感信息如密码的服务。使用S3存储密码需要额外的加密和访问
控制配置，且不如AWS Secrets Manager安全。
B. Store passwords as AWS CloudFormation parameters.
错误：AWS CloudFormation参数可以传递配置数据，但它不是专门设计用于存储密码或敏感信息。参数值可能暴露在模板中，存在
安全风险。
Correct Answer: D
正确答案：D

社区投票分布
D (100%)
C. Store passwords in AWS Storage Gateway.
错误：AWS Storage Gateway是用于连接本地存储与AWS云存储的服务，主要用于数据迁移和存储扩展，不适合存储密码。
D. Store passwords in AWS Secrets Manager.
正确：AWS Secrets Manager是专门设计用于安全存储和管理敏感信息（如密码、API密钥等）的服务。它提供加密、访问控制和自
动轮换功能，是存储密码的最佳选择。
3. 我的答案
D. Store passwords in AWS Secrets Manager.

D. Store passwords in AWS Secrets Manager.

Which AWS service should the company use to run these applications with the LEAST latency?

该公司应该使用哪项AWS服务以最低延迟运行这些应用程序？
A. AWS Outposts
AWS Outposts
B. Amazon EC2
Amazon EC2
C. AWS Lambda
AWS Lambda
D. AWS Fargate
AWS Fargate
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的使用场景，特别是如何选择适合低延迟需求的服务。
重点在于理解AWS Outposts的功能及其适用场景。
2. 选项分析
Correct Answer: A
正确答案：A
Community vote distribution
社区投票分布
A (100%)
A. AWS Outposts: 正确。AWS Outposts是一种混合云解决方案，可以将AWS服务扩展到本地数据中心或边缘位置，从而提供低延
迟的计算能力，非常适合需要与本地设备交互的场景。
B. Amazon EC2: 错误。虽然EC2是强大的计算服务，但它运行在AWS的区域数据中心，可能无法满足需要与本地设备交互的低延迟
需求。
C. AWS Lambda: 错误。Lambda是无服务器计算服务，适合事件驱动的应用，但它运行在AWS区域数据中心，无法保证最低的延
迟。
D. AWS Fargate: 错误。Fargate是容器服务，适合运行容器化应用，但它也运行在AWS区域数据中心，无法满足最低延迟需求。
3. 我的答案
A. AWS Outposts

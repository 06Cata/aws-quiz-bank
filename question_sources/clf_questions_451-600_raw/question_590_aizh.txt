A company wants to update its online data processing application by implementing container-based services that run for 4 hours
at a time. The company does not want to provision or manage server instances.
Which AWS service will meet these requirements?
一家公司希望通过实现运行时间为4小时的基于容器的服务来更新其在线数据处理应用程序。该公司不希望预置或管理服务器实
例。
哪项AWS服务可以满足这些需求？
A. AWS Lambda
AWS Lambda

AWS Fargate

Amazon EC2
D. AWS Elastic Beanstalk
AWS Elastic Beanstalk
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的使用场景，特别是无服务器计算（Serverless）和容器服务的选择。
重点在于理解AWS Fargate、AWS Lambda、Amazon EC2和AWS Elastic Beanstalk的功能和适用场景。
2. 选项分析
Correct Answer: B
正确答案：B

A. AWS Lambda
错误：AWS Lambda是一个无服务器计算服务，适用于运行短时间的代码（通常是几秒到几分钟）。
本题要求运行容器服务且持续时间为4小时，Lambda不适合这种场景。
B. AWS Fargate
正确：AWS Fargate是一个无服务器容器服务，允许用户运行容器而无需管理底层服务器。
它支持长时间运行的容器任务（例如4小时），完全符合题目要求。
C. Amazon EC2
错误：Amazon EC2是一个虚拟服务器服务，用户需要管理和维护底层实例。
题目明确要求不需要管理服务器实例，因此EC2不符合要求。
D. AWS Elastic Beanstalk
错误：AWS Elastic Beanstalk是一个平台即服务（PaaS），用于部署和管理应用程序。
虽然它可以运行容器，但用户仍需管理底层资源，且不完全符合无服务器的要求。
3. 我的答案
我的答案是：B. AWS Fargate
4. 官方答案
官方答案是：B. AWS Fargate

A company wants to manage its cloud resources by using infrastructure as code (IaC) templates. The company needs to meet
compliance requirements.
Which AWS service should the company use to meet these requirements?
一家公司希望通过使用基础设施即代码 (IaC) 模板来管理其云资源。该公司需要满足合规要求。
该公司应该使用哪项AWS服务来满足这些要求？
A. AWS Artifact
AWS Artifact
B. AWS Resource Explorer
AWS Resource Explorer
C. AWS License Manager
AWS License Manager
D. AWS Service Catalog Most Voted
AWS Service Catalog
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中与基础设施即代码（IaC）管理和合规性相关的服务。
2. 选项分析
Correct Answer: D
正确答案：D
Community vote distribution
社区投票分布
D (100%)
A. AWS Artifact: AWS Artifact是一个提供合规性报告和文档的服务，用于满足审计和合规需求。它不直接用于管理基础设施即代码
（IaC）模板，因此此选项错误。
B. AWS Resource Explorer: AWS Resource Explorer是一个服务，用于搜索和发现AWS资源。它不涉及基础设施即代码（IaC）或合
规性管理，因此此选项错误。
C. AWS License Manager: AWS License Manager用于管理软件许可证，而不是管理基础设施即代码（IaC）模板或合规性，因此此
选项错误。
D. AWS Service Catalog: AWS Service Catalog允许组织创建和管理经过批准的产品目录，包括基础设施即代码（IaC）模板，同时
支持合规性管理。这是正确的选项。
3. 我的答案
D. AWS Service Catalog
4. 官方答案
D. AWS Service Catalog

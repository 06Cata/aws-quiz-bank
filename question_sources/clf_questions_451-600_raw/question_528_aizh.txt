题目分析：Which AWS service provides DNS resolution?
1. 考察的知识点：
本题考察的是AWS服务中与DNS解析相关的知识点。
需要了解AWS提供的服务中，哪一个专门用于域名解析和管理。
2. 选项分析：
A. Amazon CloudFront: 错误。Amazon CloudFront 是一个内容分发网络 (CDN) 服务，用于加速分发网站、视频、应用程序等内
容，但它不负责DNS解析。
B. Amazon VPC: 错误。Amazon VPC 是一个虚拟私有云服务，用于创建隔离的网络环境，虽然VPC内可以配置DNS解析器，但它不
是专门的DNS解析服务。
Correct Answer: C
正确答案：C

C. Amazon Route 53: 正确。Amazon Route 53 是AWS的托管域名系统 (DNS) 服务，专门用于域名解析和管理，是提供DNS解析的
核心服务。
D. AWS Direct Connect: 错误。AWS Direct Connect 是一种网络服务，用于建立本地数据中心与AWS之间的专用网络连接，与DNS
解析无关。
3. 我的答案：
C. Amazon Route 53
4. 官方答案：
C. Amazon Route 53

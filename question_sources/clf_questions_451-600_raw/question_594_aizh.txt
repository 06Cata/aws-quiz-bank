Which AWS service should the developer use to start building the application?

题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的基本用途和适用场景，尤其是针对开发者如何快速开始构建应用程序。
2. 选项分析
A. Amazon SageMaker: 错误。Amazon SageMaker 是一个机器学习服务，主要用于构建、训练和部署机器学习模型，而不是用于
构建普通的Web应用。
Correct Answer: C
正确答案：C
Community vote distribution
社区投票分布
C (100%)
B. AWS Lambda: 错误。AWS Lambda 是一个无服务器计算服务，适合运行代码片段或事件驱动的应用，但它并不提供完整的Web
应用构建环境，开发者需要额外配置其他服务（如API Gateway、存储等）。
C. Amazon Lightsail: 正确。Amazon Lightsail 是一个易于使用的服务，专为开发者设计，提供虚拟服务器、存储、数据库和网络
功能，适合快速构建和部署Web应用。
D. Amazon Elastic Container Service (Amazon ECS): 错误。Amazon ECS 是一个容器编排服务，适合运行容器化应用，但对于没
有AWS经验的开发者来说，使用ECS可能过于复杂。
3. 我的答案
我的答案是：C. Amazon Lightsail
4. 官方答案
官方答案是：C. Amazon Lightsail

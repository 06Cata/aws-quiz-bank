Amazon EC2 Reserved Instances offer cost savings in exchange for a commitment to use EC2 capacity over a set term, with the 3-year term providing
the highest savings compared to the 1-year option. AWS does not offer a 5-year term for Reserved Instances, so 3 years is the longest available and
the most cost-effective option.
Summary: A 3-year Reserved Instance term provides the greatest cost savings, making option C the correct answer.

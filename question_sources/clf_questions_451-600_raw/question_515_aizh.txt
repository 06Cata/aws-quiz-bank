题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的功能和用途，特别是如何分享和分发自定义的Amazon Machine Images (AMIs)。
2. 选项分析
A. AWS Marketplace: 正确。AWS Marketplace是一个数字目录，允许独立软件供应商（ISV）发布和分发其产品，包括自定义的
AMIs。客户可以通过AWS Marketplace轻松购买和使用这些产品。
Correct Answer: A
正确答案：A
Community vote distribution
社区投票分布
A (100%)
B. AWS Data Exchange: 错误。AWS Data Exchange主要用于共享和订阅第三方数据集，而不是用于分发AMIs。
C. Amazon EC2: 错误。Amazon EC2是一个计算服务，虽然可以创建和使用AMIs，但它本身并不提供分享或分发AMIs的功能。
D. AWS Organizations: 错误。AWS Organizations用于管理多个AWS账户的权限和资源共享，与分发AMIs无关。
3. 我的答案
A. AWS Marketplace
4. 官方答案

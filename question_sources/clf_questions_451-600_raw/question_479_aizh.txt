Which AWS service gives users on-demand, self-service access to AWS compliance control reports?

A. AWS Config

C. AWS Trusted Advisor

AWS Artifact
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中与合规性和审计相关的功能，特别是用户如何获取AWS的合规控制报告。
2. 选项分析
A. AWS Config: 错误。AWS Config主要用于资源配置的审计和合规性检查，帮助用户跟踪AWS资源的配置变化，但它不提供合规
控制报告。
B. Amazon GuardDuty: 错误。Amazon GuardDuty是一个威胁检测服务，用于识别潜在的安全威胁，与合规控制报告无关。
C. AWS Trusted Advisor: 错误。AWS Trusted Advisor提供优化建议，帮助用户提高安全性、性能和成本效率，但它不提供合规控
制报告。
D. AWS Artifact: 正确。AWS Artifact是一个合规性报告门户，用户可以通过它按需访问AWS的合规控制报告和其他相关文档。
Correct Answer: D
正确答案：D

Amazon CloudWatch is designed to monitor AWS resources and applications, including Amazon EC2 instances. It provides metrics, including those
related to disk performance, such as disk write and read operations. By using CloudWatch, you can set up alarms to notify you of potential disk write
spikes or other performance issues, allowing for real-time monitoring and proactive response.

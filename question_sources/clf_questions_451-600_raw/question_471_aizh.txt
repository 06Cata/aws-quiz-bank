Which AWS service or feature requires an internet service provider (ISP) and a colocation facility to be implemented?
哪个AWS服务或功能需要互联网服务提供商（ISP）和托管设施才能实施？
A. AWS VPN
AWS VPN
B. Amazon Connect
Amazon Connect
C. AWS Direct Connect
AWS Direct Connect
D. Internet gateway
Internet gateway
题目分析与解答
1. 考察的知识点
本题考察的是AWS网络服务的实现方式，特别是需要物理连接和第三方设施的服务。
2. 选项分析
A. AWS VPN: AWS VPN是一种虚拟专用网络服务，它通过互联网连接到AWS环境，不需要ISP和专门的物理设施（如colocation
facility）。因此此选项错误。
B. Amazon Connect: Amazon Connect是一个基于云的联络中心服务，它完全托管在AWS上，不需要ISP或colocation设施来实现。
因此此选项错误。
C. AWS Direct Connect: AWS Direct Connect是一种专用网络连接服务，它需要通过ISP和colocation设施来建立专线连接到AWS。
这是正确答案。
D. Internet gateway: Internet gateway是一个虚拟设备，用于将VPC连接到互联网，它不需要ISP或colocation设施来实现。因此此
选项错误。
3. 我的选择
Correct Answer: C
正确答案：C
正确答案是：C. AWS Direct Connect
4. 官方答案
官方正确答案是：C
5. 与官方答案比较
我的答案与官方答案一致，无需重新思考。

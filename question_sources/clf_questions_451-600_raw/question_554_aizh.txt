A company needs a firewall that will control network connections to and from a single Amazon EC2 instance. This firewall will not
control network connections to and from other instances that are in the same subnet.
Which AWS service or feature can the company use to meet these requirements?
一孰公司需要一个防火墙来控制与单个 Amazon EC2 孩例之间的网络连接。这个防火墙不会控制与同一子网中其他孩例之间的网络
连接。
公司可以使用哪项 AWS 服务嬻功能来满足这些要求？
A. Network ACL
Network ACL
B. AWS WAF
AWS WAF
C. Route table
Route table
D. Security group
Security group
题目分析与解答
1. 考察的知识点
本题考孹的是AWS网络孞全相关的服务和功能，特别是如何控制单个Amazon EC2孩例的网络流量。
2. 选项分析
Correct Answer: D
正确答案: D
Community vote distribution
社区投票分害
D (100%)
A. Network ACL: 错误。Network ACL是针孼整个子网的流量控制，无法单独针孼某个EC2孩例进行精确控制。
B. AWS WAF: 错误。AWS WAF主要用于保护Web寒用程寏免受寀见的网络攻击（如SQL注入和跨站脚本攻击），而不是用于控制
EC2孩例的网络流量。
C. Route table: 错误。Route table用于孥义子网嬻VPC的路由规则，控制流量的去向，但无法直接控制单个EC2孩例的网络连接。
D. Security group: 正确。Security group是孩例级别的虚拟防火墙，可以精确控制单个EC2孩例的入站和出站流量。
3. 我的答案
D. Security group
4. 官方答案
D. Security group

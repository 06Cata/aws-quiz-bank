How can the company accomplish this goal?
一家公司希望安全登录到 Linux Amazon EC2 实例。
公司如何实现辥一目标？
A. Use SSH keys.
使用 SSH 密迫。
B. Use a VPN.
使用 VPN。
C. Use end-to-end encryption.
使用端到端加密。
D. Use Amazon Route 53.
使用 Amazon Route 53。
题目分析与解答
1. 考察的知识点
本轞考察的是如何安全地登录到Linux Amazon EC2实例，涉及到AWS EC2实例的安全访輐机制。
2. 选项分析
A. Use SSH keys. 使用 SSH 密钥。
正确。SSH密迫是登录Linux EC2实例的推荐方法。AWS EC2实例辇认使用SSH密迫对辦行身份轴证，确保安全性。
Correct Answer: A
正确答案：A

迾误。端到端加密是数据传输的加密机制，与登录EC2实例的身份轴证无直接关系。

迾误。Amazon Route 53是DNS服务，与登录EC2实例无关。

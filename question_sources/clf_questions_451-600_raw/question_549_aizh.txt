managed workstation in the AWS Cloud.
Which AWS service should the company use to provide the remote environment?

公司应该使用哪项AWS服务来提供远程环境？
A. Amazon Workspaces
Amazon Workspaces

AWS Outposts

题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的用途和适用场景，尤其是与远程工作环境相关的服务。
2. 选项分析
Correct Answer: A
正确答案：A
Community vote distribution
社区投票分布
A (100%)
A. Amazon Workspaces: Amazon Workspaces 是一种托管的桌面即服务 (DaaS) 解决方案，允许用户通过个人设备连接到云中的虚
拟桌面。这正是题目中要求的远程工作环境解决方案，因此此选项正确。
B. AWS Cloud9: AWS Cloud9 是一个基于云的集成开发环境 (IDE)，主要用于编写、运行和调试代码。它不适用于提供远程工作环
境，因此此选项错误。
C. AWS Outposts: AWS Outposts 是一种混合云解决方案，旨在将 AWS 基础设施扩展到本地数据中心。它不适用于提供远程工作
环境，因此此选项错误。
D. Amazon Lightsail: Amazon Lightsail 是一种简化的虚拟服务器解决方案，适用于小型应用程序和网站托管。它不专门用于提供
远程工作环境，因此此选项错误。

我的答案是：A. Amazon Workspaces

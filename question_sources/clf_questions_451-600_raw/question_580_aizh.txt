题目分析与解答
1. 考察的知识点
本题考察的是用户如何通过编程方式来配置和管理AWS基础设施的能力。
涉及AWS服务的自动化和基础设施即代码（Infrastructure as Code, IaC）相关知识。
2. 选项分析
A. AWS Cloud Development Kit (AWS CDK): 正确。AWS CDK 是一个开源软件开发框架，允许用户使用熟悉的编程语言（如
JavaScript、Python 等）来定义和部署 AWS 基础设施。它是专门设计用于以编程方式管理 AWS 资源的工具。
B. Amazon CodeGuru: 错误。Amazon CodeGuru 是一种机器学习驱动的工具，用于代码质量分析和性能优化，与基础设施的编程
配置无关。
Correct Answer: A
正确答案：A

C. AWS Config: 错误。AWS Config 是一种服务，用于记录和审计 AWS 资源的配置变化，帮助用户确保资源符合合规性要求，但它
不提供编程方式来配置基础设施。
D. AWS CodeCommit: 错误。AWS CodeCommit 是一种托管的源代码控制服务，用于存储代码仓库，与基础设施的编程配置无
关。

A company uses Amazon EC2 instances to run its application. The application needs to be available and running continuously for
three or more years.
What type of EC2 instance should the company purchase for a discount on the EC2 pricing?
一家公司使用 Amazon EC2 实例运行其应用程序。该应用程序需要持续运行三年或更长时间。
公司应该购买哪种类型的 EC2 实例以获得 EC2 定价折扣？
A. Reserved Instances
Reserved Instances
B. Spot Instances
Spot Instances
C. On-Demand Instances
On-Demand Instances
D. EC2 Fleet
EC2 Fleet
题目分析与解答
1. 考察的知识点
本题考察的是AWS EC2实例的定价模式，包括Reserved Instances、Spot Instances、On-Demand Instances和EC2 Fleet的特点及适
用场景。
2. 选项分析
Correct Answer: A
正确答案：A
Community vote distribution
社区投票分布
A (100%)
A. Reserved Instances: 正确。Reserved Instances适用于长期使用的场景，用户可以通过预付费用获得显著的折扣，适合需要连续
运行三年或更长时间的应用。
B. Spot Instances: 错误。Spot Instances价格低廉，但它们是基于AWS的空闲容量提供的，可能会随时被中断，不适合需要连续运
行的应用。
C. On-Demand Instances: 错误。On-Demand Instances按需付费，灵活性高，但价格较高，不适合长期运行的应用，因为没有折
扣。
D. EC2 Fleet: 错误。EC2 Fleet是一个组合选项，可以混合使用不同类型的实例，但它并不是一种定价模式，无法直接提供长期折
扣。
3. 我的答案
A. Reserved Instances
4. 官方答案
A. Reserved Instances

Which AWS service gives users the ability to deploy highly repeatable infrastructure configurations?
哪个AWS服务使用户能够部署高度可重复的基础设施配置？
A. AWS CloudFormation
AWS CloudFormation
B. AWS CodeDeploy
AWS CodeDeploy
C. AWS CodeBuild
AWS CodeBuild
D. AWS Systems Manager
AWS Systems Manager
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中用于部署和管理基础设施的工具，特别是支持基础设施即代码（Infrastructure as Code, IaC）的服务。
2. 选项分析
A. AWS CloudFormation: 正确。AWS CloudFormation 是一种支持基础设施即代码的服务，允许用户通过模板定义和部署可重复
的基础设施配置。
B. AWS CodeDeploy: 错误。AWS CodeDeploy 是一种自动化部署服务，主要用于应用程序代码的部署，而不是基础设施配置的管
理。
C. AWS CodeBuild: 错误。AWS CodeBuild 是一种持续集成服务，用于构建和测试代码，而不是用于部署基础设施配置。
Correct Answer: A
正确答案：A

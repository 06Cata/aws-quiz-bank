A user wants to invoke an AWS Lambda function when an Amazon EC2 instance enters the “stopping” state.
Which AWS service is appropriate for this use case?
用户希望在 Amazon EC2 实例进入“stopping”状态时调用 AWS Lambda 函数。
哪个 AWS 服务适合这个使用场景？
A. Amazon EventBridge
Amazon EventBridge
B. AWS Config
AWS Config
C. Amazon Simple Notification Service (Amazon SNS)
Amazon Simple Notification Service (Amazon SNS)
D. AWS CloudFormation
AWS CloudFormation
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的事件驱动架构，尤其是如何使用Amazon EventBridge来监控和响应AWS资源状态变化。
2. 选项分析
A. Amazon EventBridge: 正确。Amazon EventBridge是一个事件总线服务，可以捕获AWS资源的状态变化事件（例如EC2实例进
入“stopping”状态），并触发后续操作（如调用Lambda函数）。这是本题的正确答案。
Correct Answer: A
正确答案：A

A (100%)
B. AWS Config: 错误。AWS Config主要用于记录资源配置的历史变化和合规性检查，而不是实时响应资源状态变化事件。
C. Amazon Simple Notification Service (Amazon SNS): 错误。SNS是一个消息通知服务，可以发送消息到订阅者，但它不能直接
捕获EC2实例状态变化事件。
D. AWS CloudFormation: 错误。CloudFormation是一个基础设施即代码服务，用于创建和管理AWS资源，但它不用于监控资源状
态变化或触发事件。
3. 我的答案
A. Amazon EventBridge
4. 官方答案
A. Amazon EventBridge

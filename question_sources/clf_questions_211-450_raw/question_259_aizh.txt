A company need an AWS service that provides a clear baseline of what the company runs in its on-premises data centers. The
company needs the projected cost to run its on-premises workloads in the AWS Cloud.

A. AWS Compute Optimizer:
AWS Compute Optimizer主要用于分析和优化现有的AWS资源（如EC2实例、Lambda函数等）的使用情况，并提供优化建议。它不
适用于评估本地数据中心的工作负载迁移到AWS的成本，因此此选项错误。
B. AWS Cost Explorer:
AWS Cost Explorer是一个工具，用于分析和管理AWS账户的成本和使用情况。它可以帮助用户查看和预测AWS上的费用，但无法提
供关于本地数据中心迁移到AWS的成本评估，因此此选项错误。
C. AWS Systems Manager Agent (SSM Agent):
SSM Agent是一个工具，用于管理和操作AWS资源以及本地服务器。虽然它可以帮助收集本地服务器的运行数据，但它并不提供迁
移成本评估功能，因此此选项错误。
D. Migration Evaluator:
Migration Evaluator（以前称为TSO Logic）是一个专门的工具，用于帮助用户评估其本地数据中心工作负载迁移到AWS的成本和性
能。它可以提供详细的成本分析和迁移建议，因此此选项正确。

我的答案是：D. Migration Evaluator
4. 官方答案
官方答案是：D. Migration Evaluator

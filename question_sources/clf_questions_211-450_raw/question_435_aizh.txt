A company needs to identify who accessed an AWS service and what action was performed for a given time period.
Which AWS service should the company use to meet this requirement?
一家公司需要识别在特定时间段内谁访问了某个AWS服务以及执行了什么操作。
该公司应该使用哪项AWS服务来满足这一需求？
A. Amazon CloudWatch
Amazon CloudWatch
B. AWS CloudTrail
AWS CloudTrail
C. AWS Security Hub
AWS Security Hub
D. Amazon Inspector
Amazon Inspector
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的功能，特别是与审计和跟踪用户活动相关的服务。
2. 选项分析
A. Amazon CloudWatch: 错误。CloudWatch主要用于监控和管理AWS资源的性能指标和日志，但它不提供详细的用户访问和操作
记录。
Correct Answer: B
正确答案：B

B. AWS CloudTrail: 正确。CloudTrail专门用于记录用户在AWS上的操作，包括谁访问了什么服务以及执行了什么操作。它是审计和

Which mechanism allows developers to access AWS services from application code?

AWS Software Development Kit
B. AWS Management Console
AWS Management Console

AWS CodePipeline
D. AWS Config
AWS Config
题目分析与解答
1. 考察的知识点
本题考察的是开发人员如何通过应用程序代码访问AWS服务的机制。
涉及AWS SDK的功能和用途。
2. 选项分析
A. AWS Software Development Kit: 正确。AWS SDK是专门为开发人员设计的工具包，提供了API和库，使开发人员能够通过代码
访问和操作AWS服务。
B. AWS Management Console: 错误。AWS Management Console是一个基于网页的用户界面，主要用于手动管理AWS资源，而不
是通过代码访问服务。
Correct Answer: A
正确答案：A

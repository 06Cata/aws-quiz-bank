Question #337
A company is designing its AWS workloads so that components can be updated regularly and so that changes can be made in
small, reversible increments.
Which pillar of the AWS Well-Architected Framework does this design support?
一家公司正在设计其AWS工作负载，使组件可以定期更新，并且可以以小的、可逆的增量进行更改。
这种设计支持AWS Well-Architected Framework的哪个支柱？
A. Security
安全性
B. Performance efficiency
性能效率
C. Operational excellence Most Voted
卓越运营
D. Reliability
可靠性
题目分析与解答
1. 考察的知识点
本题考察的是AWS Well-Architected Framework的五大支柱之一，特别是“卓越运营（Operational Excellence）”支柱的核心理念。
卓越运营强调通过自动化、可重复的流程来管理和运行系统，同时支持小规模、可逆的变更以降低风险。
2. 选项分析
Correct Answer: C
正确答案：C

A. Security 安全性: 错误。安全性支柱关注保护数据、系统和资产，主要涉及访问控制、数据加密等内容，与题目中提到的“更新组
件”和“可逆变更”无直接关联。
B. Performance efficiency 性能效率: 错误。性能效率支柱关注优化资源使用以提高性能，主要涉及选择合适的资源类型和大小，
与题目中提到的设计原则无直接关联。
C. Operational excellence 卓越运营: 正确。卓越运营支柱强调通过自动化和优化流程来实现高效的操作管理，支持小规模、可逆
的变更，这与题目描述的设计原则完全吻合。
D. Reliability 可靠性: 错误。可靠性支柱关注系统的恢复能力和故障处理能力，虽然与系统设计相关，但题目重点在于“更新组
件”和“可逆变更”，这更符合卓越运营的理念。
3. 我的答案
C. Operational excellence 卓越运营
4. 官方答案
C. Operational excellence 卓越运营

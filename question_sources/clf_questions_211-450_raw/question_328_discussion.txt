Answer is C

Amazon Polly is a machine learning service that converts text to speech.
Amazon Textract is a machine learning service that can extract text from scanned documents.
Amazon Rekognition is a cloud-based image and video analysis service that makes it easy to add advanced computer vision capabilities to your
applications.

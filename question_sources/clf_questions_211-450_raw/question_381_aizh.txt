Question #381
Which perspective in the AWS Cloud Adoption Framework (AWS CAF) includes a capability for well-designed data and analytics
architecture?
在 AWS 云采用框架 (AWS CAF) 中，哪个视角包括了针对良好设计的数据和分析架构的能力？
A. Security
安全 (Security)
B. Governance
治理 (Governance)
C. Operations
运营 (Operations)
D. Platform Most Voted
平台 (Platform)
题目分析与解答
1. 考察的知识点
本题考察的是AWS Cloud Adoption Framework (AWS CAF)的六个视角（Perspectives），以及它们各自的能力范围。
具体关注点是哪个视角包含“well-designed data and analytics architecture”的能力。
2. 选项分析
A. Security (安全): 错误。Security视角主要关注安全性、合规性和风险管理，与数据和分析架构的设计无直接关联。
B. Governance (治理): 错误。Governance视角主要关注IT治理、预算管理和项目管理，与数据和分析架构的设计无直接关联。
C. Operations (运营): 错误。Operations视角主要关注运营流程、事件管理和持续改进，与数据和分析架构的设计无直接关联。
D. Platform (平台): 正确。Platform视角专注于技术架构，包括数据和分析架构的设计，因此是正确答案。
Correct Answer: D
正确答案: D

Which AWS service or tool can provide rightsizing recommendations for Amazon EC2 resources at no additional cost?

AWS Well-Architected Tool
B. Amazon CloudWatch
Amazon CloudWatch
C. AWS Cost Explorer Most Voted

题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中与成本优化相关的功能，特别是如何获得EC2资源的优化建议。
2. 选项分析
A. AWS Well-Architected Tool:
此工具主要用于评估架构是否符合AWS最佳实践，并提供改进建议，但它不直接提供EC2资源的权衡建议，因此此选项错误。
B. Amazon CloudWatch:
CloudWatch主要用于监控资源的性能和运行状况，虽然可以间接帮助用户了解资源使用情况，但它不提供具体的权衡建议，因此此
选项错误。
Correct Answer: C
正确答案：C

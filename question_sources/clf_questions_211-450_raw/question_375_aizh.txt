Question #375
Which AWS service will help a company plan a migration to AWS by collecting the configuration, usage, and behavior data of on-
premises data centers?
哪项AWS服务将通过收集本地数据中心的配置、使用和行为数据来帮助公司规划迁移到AWS？
A. AWS Resource Groups
AWS Resource Groups
B. AWS Application Discovery Service Most Voted
AWS Application Discovery Service
C. AWS Service Catalog
AWS Service Catalog
D. AWS Systems Manager
AWS Systems Manager
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中与迁移相关的工具和服务，特别是用于收集本地数据中心配置、使用情况和行为数据的服务。
2. 选项分析
A. AWS Resource Groups: 错误。AWS Resource Groups用于组织和管理AWS资源的分组，与迁移或收集本地数据中心信息无关。
B. AWS Application Discovery Service: 正确。AWS Application Discovery Service专门用于帮助企业在迁移到AWS之前收集本地数
据中心的配置、使用情况和行为数据，以便进行迁移规划。
C. AWS Service Catalog: 错误。AWS Service Catalog用于管理和分发IT服务目录，与迁移或数据收集无关。
D. AWS Systems Manager: 错误。AWS Systems Manager主要用于管理和操作AWS资源，与收集本地数据中心信息无直接关系。
Correct Answer: B
正确答案：B

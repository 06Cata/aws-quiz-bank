Question #360
A company wants to deploy some of its resources in the AWS Cloud. To meet regulatory requirements, the data must remain local
and on premises. There must be low latency between AWS and the company resources.
Which AWS service or feature can be used to meet these requirements?
一家公司希望在AWS云中部署其部分资源。为了满足监管要求，数据必须保持本地并在本地部署。AWS与公司资源之间必须具有低
延迟。
可以使用哪项AWS服务或功能来满足这些要求？
A. AWS Local Zones
AWS Local Zones
B. Availability Zones
Availability Zones
C. AWS Outposts Most Voted
AWS Outposts
D. AWS Wavelength Zones
AWS Wavelength Zones
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的部署选项，尤其是如何满足数据本地存储、低延迟以及与AWS云集成的需求。
2. 选项分析
Correct Answer: C
正确答案：C

A. AWS Local Zones:
AWS Local Zones是AWS提供的服务，允许用户在特定地理位置运行低延迟应用程序。虽然它可以提供低延迟，但它并不满足“数据
必须保持本地和在本地设施上”的要求，因此此选项错误。
B. Availability Zones:
Availability Zones是AWS区域中的多个数据中心，用于高可用性和冗余。它们是AWS云的一部分，数据不会保持在本地设施上，因
此此选项错误。
C. AWS Outposts:
AWS Outposts是一个完全托管的解决方案，允许用户在本地设施中运行AWS服务，同时与AWS云保持一致性。它满足“数据必须保
持本地和在本地设施上”的要求，并提供低延迟，因此此选项正确。
D. AWS Wavelength Zones:
AWS Wavelength Zones是专为5G网络优化的服务，旨在提供超低延迟，但它并不满足“数据必须保持本地和在本地设施上”的要求，
因此此选项错误。
3. 我的答案
C. AWS Outposts
4. 官方答案
C. AWS Outposts

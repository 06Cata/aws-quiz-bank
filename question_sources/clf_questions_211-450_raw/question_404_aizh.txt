A company wants to design its cloud architecture so that it can support development innovations, and continuously improve
processes and procedures.
This is an example of which pillar of the AWS Well-Architected Framework?
一家公司希望设计其云架构，以支持开发创新，并持续改进流程和程序。
这是 AWS Well-Architected Framework 的哪个支柱的示例？
A. Security
安全性

性能效率

题目分析与解答
1. 考察的知识点
本题考察的是AWS Well-Architected Framework的五大支柱之一，以及如何将这些支柱应用于实际场景。
2. 选项分析
A. Security (安全性): 错误。安全性支柱关注保护数据、系统和资产，管理风险。题目中并未提到与安全相关的内容。
Correct Answer: C
正确答案：C

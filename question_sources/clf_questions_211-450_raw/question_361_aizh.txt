Which of the following AWS services are serverless? (Choose two.)

AWS Outposts

Amazon EC2
C. Amazon Elastic Kubernetes Service (Amazon EKS)
Amazon Elastic Kubernetes Service (Amazon EKS)
D. AWS Fargate Most Voted
AWS Fargate

AWS Lambda
题目分析与解答
1. 考察的知识点
本题考察的是AWS的“Serverless”服务相关知识。Serverless服务指的是无需用户管理底层服务器的服务，用户只需专注于代码或应
用逻辑。
2. 选项分析
A. AWS Outposts: 错误。AWS Outposts是一种混合云解决方案，允许用户在本地运行AWS服务，但它需要用户管理硬件，因此不
是Serverless服务。
B. Amazon EC2: 错误。Amazon EC2是虚拟服务器服务，用户需要管理实例的启动、停止、配置等操作，因此不是Serverless服务。
Correct Answer: DE
正确答案：DE

Question #316
A company wants to move its on-premises databases to managed cloud database services by using a simplified migration process.
Which AWS service or tool can help the company meet this requirement?
A. AWS Storage Gateway
B. AWS Application Migration Service
C. AWS DataSync
D. AWS Database Migration Service (AWS DMS) Most Voted
Comments

Selected Answer: D
A: AWS Storage Gateway – Used for integrating on-premises environments with AWS storage services, not for migrating databases.
B: AWS Application Migration Service – Used to rehost entire servers or applications, not focused on databases.
C: AWS DataSync – Transfers files between on-premises and AWS, not structured databases.
D: AWS Database Migration Service (AWS DMS), it helps migrate on-premises databases to AWS managed database services with a simplified and
reliable process.

D. AWS Database Migration Service (AWS DMS)
AWS Database Migration Service (DMS) is the service designed to help migrate on-premises databases to managed cloud database services in a
simplified manner. It supports various database engines and provides a reliable and efficient way to migrate data with minimal downtime.

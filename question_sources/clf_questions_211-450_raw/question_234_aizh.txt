Which AWS service or feature will search for and identify AWS resources that are shared externally?

题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中用于识别和分析外部共享资源的功能。
2. 选项分析
A. Amazon OpenSearch Service:
Amazon OpenSearch Service是一个托管的搜索和分析服务，用于处理日志和数据分析。它不具备识别外部共享资源的功能，因此
此选项错误。
B. AWS Control Tower:
AWS Control Tower用于设置和管理多账户环境的最佳实践，但它不负责搜索和识别外部共享资源，因此此选项错误。
Correct Answer: C
正确答案: C

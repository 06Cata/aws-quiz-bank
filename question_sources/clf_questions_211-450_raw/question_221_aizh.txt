Which AWS service is used to track, record, and audit configuration changes made to AWS resources?
哪个AWS服务用于跟踪、记录和审计对AWS资源所做的配置更改？
A. AWS Shield
AWS Shield
B. AWS Config Most Voted
AWS Config
C. AWS IAM
AWS IAM
D. Amazon Inspector
Amazon Inspector
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的功能和用途，特别是与资源配置变更的跟踪和审计相关的服务。
2. 选项分析
A. AWS Shield: AWS Shield是一个托管的分布式拒绝服务（DDoS）保护服务，用于保护应用程序免受DDoS攻击。它与资源配置变
更的跟踪和审计无关，因此此选项错误。
B. AWS Config: AWS Config是一项服务，用于跟踪、记录和审计AWS资源的配置变更。它可以帮助用户了解资源的历史配置状态
并确保合规性。因此此选项正确。
C. AWS IAM: AWS Identity and Access Management (IAM) 是一个用于管理用户、组和权限的服务。它与资源配置变更的跟踪和审
计无关，因此此选项错误。
Correct Answer: B
正确答案：B

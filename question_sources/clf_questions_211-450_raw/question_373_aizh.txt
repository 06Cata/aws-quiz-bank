Which AWS service makes it easier to monitor and troubleshoot application logs and cloud resources?
哪个AWS服务可以更轻松地监控和排查应用日志和云资源？
A. Amazon EC2
Amazon EC2
B. AWS Identity and Access Management (IAM)
AWS Identity and Access Management (IAM)
C. Amazon CloudWatch Most Voted
Amazon CloudWatch
D. AWS CloudTrail
AWS CloudTrail
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中与监控和日志管理相关的功能。
2. 选项分析
A. Amazon EC2: Amazon EC2是一个计算服务，用于运行虚拟服务器实例。它不直接提供监控和日志管理功能，因此此选项错误。
B. AWS Identity and Access Management (IAM): IAM是一个身份和访问管理服务，用于控制对AWS资源的访问权限。它与监控
和日志管理无直接关系，因此此选项错误。
C. Amazon CloudWatch: Amazon CloudWatch是一个监控和管理服务，可以收集和分析日志数据，并监控AWS资源和应用程序的
性能。此选项正确。
Correct Answer: C
正确答案：C

A company wants to save costs by archiving data that is no longer frequently accessed by end users.
Which Amazon S3 feature will meet this requirement?
A. S3 Versioning
B. S3 Lifecycle Most Voted
C. S3 Object Lock
D. S3 Inventory

Selected Answer: B
Amazon S3 Lifecycle policies allow you to define rules to automatically transition objects between storage classes or delete them when they are no
longer needed. In the context of cost savings and archiving data that is no longer frequently accessed, using S3 Lifecycle policies is the appropriate
solution.

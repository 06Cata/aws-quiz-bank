Which AWS resource can help a company reduce its costs in exchange for a usage commitment when using Amazon EC2
instances?
哪种AWS资源可以帮助公司在使用Amazon EC2实例时通过使用承诺来降低成本？
A. Compute Savings Plans Most Voted
Compute Savings Plans
B. Auto Scaling group
Auto Scaling group
C. On-Demand Instance
On-Demand Instance
D. EC2 instance store
EC2 instance store
题目分析与解答
1. 考察的知识点
本题考察的是AWS成本优化相关的知识，特别是如何通过使用特定的定价模型来降低Amazon EC2实例的使用成本。
2. 选项分析
A. Compute Savings Plans: 正确。Compute Savings Plans是一种定价模型，允许用户通过承诺一定的使用量来获得显著的成本节
约。它适用于Amazon EC2实例，并且提供灵活性，可以在不同的实例类型、操作系统和区域之间切换。
B. Auto Scaling group: 错误。Auto Scaling组用于动态调整EC2实例的数量以满足需求，但它本身并不直接提供成本节约的承诺机
制。
C. On-Demand Instance: 错误。按需实例是最灵活的定价选项，但它的成本通常较高，因为没有任何使用承诺或折扣。
Correct Answer: A
正确答案：A

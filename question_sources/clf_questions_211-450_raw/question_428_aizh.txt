Question #428
A company wants to provide one of its employees with access to Amazon RDS. The company also wants to limit the interaction to
only the AWS CLI and AWS software development kits (SDKs).
Which combination of actions should the company take to meet these requirements while following the principles of least
privilege? (Choose two.)
一家公司希望为其一名员工提供对 Amazon RDS 的访问权限。该公司还希望将交互限制为仅通过 AWS CLI 和 AWS 软件开发工具包
(SDK)。
公司应该采取哪些行动组合来满足这些要求，同时遵循最小权限原则？（选择两个）
A. Create an IAM user and provide AWS Management Console access only.
创建一个 IAM 用户并仅提供 AWS Management Console 访问权限。
B. Create an IAM user and provide programmatic access only. Most Voted
创建一个 IAM 用户并仅提供编程访问权限。
C. Create an IAM role and provide AWS Management Console access only.
创建一个 IAM 角色并仅提供 AWS Management Console 访问权限。
D. Create an IAM policy with administrator access and attach it to the IAM user.
创建一个具有管理员访问权限的 IAM 策略并将其附加到 IAM 用户。
E. Create an IAM policy with Amazon RDS access and attach it to the IAM user. Most Voted
创建一个具有 Amazon RDS 访问权限的 IAM 策略并将其附加到 IAM 用户。

2. 选项分析
A. Create an IAM user and provide AWS Management Console access only.
错误：题目要求限制用户只能通过AWS CLI和SDK进行交互，而不是通过AWS Management Console。因此提供AWS Management
Console访问权限不符合要求。
B. Create an IAM user and provide programmatic access only.
正确：通过创建IAM用户并仅提供编程访问权限（即通过AWS CLI和SDK），满足了题目要求，同时遵循了最小权限原则。
C. Create an IAM role and provide AWS Management Console access only.
错误：题目要求限制用户只能通过AWS CLI和SDK进行交互，而不是通过AWS Management Console。此外，IAM角色通常用于临时
访问，而不是直接分配给单个用户。
D. Create an IAM policy with administrator access and attach it to the IAM user.
错误：管理员权限过于宽泛，不符合最小权限原则。题目只要求提供Amazon RDS的访问权限，而不是所有AWS服务的权限。
E. Create an IAM policy with Amazon RDS access and attach it to the IAM user.
正确：通过创建一个仅限Amazon RDS访问的IAM策略并将其附加到IAM用户，满足了题目要求，同时遵循了最小权限原则。

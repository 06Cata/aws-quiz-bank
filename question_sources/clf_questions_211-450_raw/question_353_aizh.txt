Which AWS service can companies use to create infrastructure from code?
企业可以使用哪个AWS服务通过代码创建基础设施？
A. Amazon Elastic Kubernetes Service (Amazon EKS)
Amazon Elastic Kubernetes Service (Amazon EKS)
B. AWS Outposts
AWS Outposts
C. AWS CodePipeline
AWS CodePipeline
D. AWS CloudFormation Most Voted
AWS CloudFormation
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中与基础设施即代码（Infrastructure as Code, IaC）相关的知识点。
2. 选项分析
A. Amazon Elastic Kubernetes Service (Amazon EKS): 错误。Amazon EKS 是一种托管的 Kubernetes 服务，用于运行容器化应用
程序，而不是用于创建基础设施代码。
B. AWS Outposts: 错误。AWS Outposts 是一种将AWS服务扩展到本地数据中心的解决方案，与基础设施即代码无关。
C. AWS CodePipeline: 错误。AWS CodePipeline 是一种持续交付服务，用于自动化代码部署流程，但它不用于创建基础设施代
码。
Correct Answer: D
正确答案：D

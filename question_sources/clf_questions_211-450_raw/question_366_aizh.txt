Which AWS service supports the deployment and management of applications in the AWS Cloud?
哪个 AWS 服务支持在 AWS 云中部署和管理应用程序？
A. Amazon CodeGuru
Amazon CodeGuru
B. AWS Fargate
AWS Fargate
C. AWS CodeCommit
AWS CodeCommit
D. AWS Elastic Beanstalk Most Voted
AWS Elastic Beanstalk
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中支持应用程序部署和管理的服务。
2. 选项分析
A. Amazon CodeGuru: 错误。Amazon CodeGuru是一种代码分析工具，用于帮助开发人员识别代码中的问题并优化性能。它不支
持应用程序的部署和管理。
B. AWS Fargate: 错误。AWS Fargate是一种无服务器计算引擎，用于运行容器化应用程序，但它不直接负责应用程序的部署和管
理。
C. AWS CodeCommit: 错误。AWS CodeCommit是一种托管的源代码控制服务，用于存储代码，但它不支持应用程序的部署和管
理。
Correct Answer: D
正确答案：D

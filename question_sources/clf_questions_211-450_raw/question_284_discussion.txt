D. AWS Security Hub
AWS Security Hub is a comprehensive security service that provides a centralized view of security alerts and compliance status across multiple AWS
accounts. It aggregates findings from various AWS security services, including Amazon GuardDuty, Amazon Inspector, and Amazon Macie, as well as
integrated third-party security solutions. Security Hub consolidates these findings into a single dashboard, making it easier for organizations to
monitor and manage their overall security posture.

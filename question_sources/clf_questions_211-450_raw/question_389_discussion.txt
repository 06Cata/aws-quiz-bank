A. AWS Migration Hub

Selected Answer: B
B. AWS Database Migration Service (AWS DMS)
AWS DMS helps to migrate databases to AWS quickly and securely. It supports various source and target databases, including NoSQL databases like
MongoDB, Cassandra, and Couchbase, making it suitable for migrating an on-premises NoSQL workload to Amazon DynamoDB.

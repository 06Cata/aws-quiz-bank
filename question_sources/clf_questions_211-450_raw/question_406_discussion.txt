A company is moving some of its on-premises IT services to the AWS Cloud. The finance department wants to see the entire bill so
it can forecast spending limits.
Which AWS service can the company use to set spending limits and receive notifications if those limits are exceeded?
A. AWS Cost and Usage Reports
B. AWS Budgets Most Voted
C. AWS Organizations consolidated billing
D. Cost Explorer

Selected Answer: B
B. AWS Budgets
AWS Budgets is a service that allows users to set customized budgets that track and monitor their AWS spending. Users can set up alerts to be
notified via email or SNS when their budget thresholds are approaching or exceeded. This service would allow the finance department to forecast
spending limits and receive notifications if those limits are exceeded as requested.

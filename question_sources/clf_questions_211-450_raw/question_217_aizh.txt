Question #217
Which AWS service can a company use to securely store and encrypt passwords for a database?
公司可以使用哪项AWS服务来安全存储和加密数据库的密码？
A. AWS Shield
AWS Shield
B. AWS Secrets Manager Most Voted
AWS Secrets Manager
C. AWS Identity and Access Management (IAM)
AWS Identity and Access Management (IAM)
D. Amazon Cognito
Amazon Cognito
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中与安全存储和加密敏感信息（如数据库密码）的相关知识。
2. 选项分析
A. AWS Shield: AWS Shield 是一种托管的分布式拒绝服务 (DDoS) 防护服务，用于保护应用程序免受DDoS攻击。它与存储和加密
密码无关，因此此选项错误。
B. AWS Secrets Manager: AWS Secrets Manager 是专门设计用于安全存储、管理和轮换敏感信息（如数据库密码、API密钥等）的
服务。它支持加密和自动轮换密码，因此此选项正确。
C. AWS Identity and Access Management (IAM): IAM 是用于管理用户和权限的服务，主要用于控制访问权限，而不是存储和加
密密码。因此此选项错误。
Correct Answer: B
正确答案：B

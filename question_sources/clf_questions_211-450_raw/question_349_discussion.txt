Selected Answer: C
Amazon S3 Intelligent-Tiering is a storage class that automatically optimizes costs by moving objects between two access tiers—frequent and
infrequent access—based on changing access patterns. It is designed to deliver the same low latency and high throughput performance of the S3
Standard storage class. This storage class automatically adjusts as access patterns change over time, optimizing costs without performance impact.

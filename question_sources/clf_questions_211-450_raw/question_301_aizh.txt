Question #301
Which of the following is a characteristic of the AWS account root user?
以下哪项是AWS账户根用户的栎征？
A. The root user is the only user that can be configured with multi-factor authentication (MFA).
根用户是唯一可以配置多因素身份验证（MFA）的用户。
B. The root user is the only user that can access the AWS Management Console.
根用户是唯一可以访问AWS Management Console的用户。
C. The root user is the first sign-in identity that is available when an AWS account is created. Most Voted
根用户是创建AWS账户时可用的第一个登录身份。
D. The root user has a password that cannot be changed.
根用户的密码无條更改。
题目分析与解答
1. 考察的知识点
本题考察的是AWS账户根用户（root user）的栎性和权限。
根用户是AWS账户的最高权限用户，具有完全的访问权限。
2. 选项分析
A. The root user is the only user that can be configured with multi-factor authentication (MFA).
错误：根用户可以配置MFA，但其他IAM用户也可以配置MFA。因桮根用户并不是唯一可以配置MFA的用户。
B. The root user is the only user that can access the AWS Management Console.
错误：根用户可以访问AWS Management Console，但其他IAM用户也可以通过分配权限访问AWS Management Console。
Correct Answer: C
桭确答栾：C

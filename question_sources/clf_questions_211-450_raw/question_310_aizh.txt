Question #310
Which AWS service or feature gives users the ability to capture information about network traffic in a VPC?
哪个AWS服务或功能使用户能够捕获有关VPC中网络流量的信息？
A. VPC Flow Logs Most Voted
VPC Flow Logs
B. Amazon Inspector
Amazon Inspector
C. VPC route tables
VPC route tables
D. AWS CloudTrail
AWS CloudTrail
题目分析与解答
1. 考察的知识点
本题考察的是与VPC（Virtual Private Cloud）相关的网络流量监控功能。
重点在于了解AWS服务中哪些工具可以捕获和分析VPC中的网络流量。
2. 选项分析
A. VPC Flow Logs: 正确。VPC Flow Logs是一个功能，可以捕获有关VPC中的网络流量的信息。它记录流量的来源、目的地、协议
等详细信息，是监控和分析网络流量的主要工具。
B. Amazon Inspector: 错误。Amazon Inspector是一项安全评估服务，用于检查应用程序的安全性和合规性，而不是用于捕获网络
流量信息。
C. VPC route tables: 错误。VPC路由表用于定义网络流量的路由规则，但它不捕获或记录网络流量信息。
Correct Answer: A
正确答案：A

A company website is experiencing DDoS attacks.
Which AWS service can help protect the company website against these attacks?
一家公司网站正在遭受DDoS攻击。
哪项AWS服务可以帮助保护公司网站免受这些攻击？
A. AWS Resource Access Manager
AWS Resource Access Manager
B. AWS Amplify
AWS Amplify
C. AWS Shield Most Voted
AWS Shield
D. Amazon GuardDuty
Amazon GuardDuty
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务在网络安全和防护方面的应用，特别是针对DDoS攻击的防护能力。
2. 选项分析
A. AWS Resource Access Manager: 错误。AWS Resource Access Manager用于跨账户共享AWS资源，与DDoS攻击防护无关。
B. AWS Amplify: 错误。AWS Amplify是一个开发工具链，主要用于构建和部署全栈应用，与网络安全防护无关。
Correct Answer: C
正确答案：C

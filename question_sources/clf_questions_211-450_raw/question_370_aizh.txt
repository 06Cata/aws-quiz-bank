Which AWS service is a relational database compatible with MySQL and PostgreSQL?
哪个AWS服务是与MySQL和PostgreSQL兼容的关系型数据库？
A. Amazon Redshift
Amazon Redshift
B. Amazon DynamoDB
Amazon DynamoDB
C. Amazon Aurora Most Voted
Amazon Aurora
D. Amazon Neptune
Amazon Neptune
题目分析与解答
1. 考察的知识点
本题考察的是AWS中关系型数据库服务的知识，特别是与MySQL和PostgreSQL兼容的服务。
2. 选项分析
A. Amazon Redshift: 错误。Amazon Redshift是一个数据仓库服务，主要用于分析和处理大规模数据集，而不是关系型数据库服
务。
B. Amazon DynamoDB: 错误。Amazon DynamoDB是一个NoSQL数据库服务，适用于键值和文档存储，不支持关系型数据库的功
能。
C. Amazon Aurora: 正确。Amazon Aurora是一个高性能的关系型数据库服务，完全兼容MySQL和PostgreSQL。
D. Amazon Neptune: 错误。Amazon Neptune是一个图数据库服务，专门用于处理图形数据，而不是关系型数据库。
Correct Answer: C
正确答案：C

Which AWS service or feature allows users to securely store encrypted credentials and retrieve these credentials when required?

题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中用于安全存储和管理加密凭证的功能。
2. 选项分析
A. AWS Encryption SDK:
错误。AWS Encryption SDK是一个开发工具包，用于帮助开发人员加密和解密数据，但它不用于存储加密凭证。
B. AWS Security Hub:
错误。AWS Security Hub是一个安全管理服务，用于集中查看和管理安全警报，但它不提供存储加密凭证的功能。
C. AWS Secrets Manager:
正确。AWS Secrets Manager是专门设计用于安全存储、管理和检索加密凭证（如API密钥、数据库密码等）的服务。
Correct Answer: C
正确答案：C

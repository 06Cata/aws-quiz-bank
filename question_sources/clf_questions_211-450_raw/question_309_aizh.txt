Question #309
Which AWS service could an administrator use to provide desktop environments for several employees?
哪项AWS服务可以让管理员为多个员工提供桌面环境？
A. AWS Organizations
AWS Organizations
B. AWS Fargate
AWS Fargate
C. AWS WAF
AWS WAF
D. AWS WorkSpaces Most Voted
AWS WorkSpaces
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的用途，特别是与提供虚拟桌面环境相关的服务。
2. 选项分析
A. AWS Organizations: 错误。AWS Organizations用于管理多个AWS账户的组织结构和权限，与提供桌面环境无关。
B. AWS Fargate: 错误。AWS Fargate是用于运行容器化应用的无服务器计算引擎，与桌面环境无关。
C. AWS WAF: 错误。AWS WAF是Web应用防火墙，用于保护Web应用免受恶意流量攻击，与桌面环境无关。
D. AWS WorkSpaces: 正确。AWS WorkSpaces是一项托管的虚拟桌面服务，专门用于提供桌面环境给用户。
3. 我的答案
Correct Answer: D
正确答案：D

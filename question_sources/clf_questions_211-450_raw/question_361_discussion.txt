The AWS services that are considered serverless are:
D. AWS Fargate
AWS Fargate is a serverless compute engine for containers that allows you to run containers without managing the underlying infrastructure.
E. AWS Lambda
AWS Lambda is a fully serverless compute service that allows you to run code without provisioning or managing servers. It automatically scales your
applications in response to incoming traffic.

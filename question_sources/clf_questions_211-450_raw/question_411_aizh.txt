Question #411
Which feature of Amazon RDS provides the ability to automatically create a primary database instance and to synchronously
replicate data to an instance in another Availability Zone?
Amazon RDS 的哪个功能提供了自动创建主数据库实例并将数据同步复制到另一个可用区中的实例的能力？
A. Read replicas
只读副本
B. Blue/green deployment
蓝/绿部署
C. Multi-AZ deployment Most Voted
多可用区部署
D. Reserved Instances
预留实例
题目分析与解答
1. 考察的知识点
本题考察的是Amazon RDS的高可用性和灾难恢复功能，特别是Multi-AZ部署的特性。
2. 选项分析
A. Read replicas (勀读副本): 错误。Read replicas用于提高数据库的读取性能，通过异步复制数据到副本实例，但它不提供高可用
性或灾难恢复功能。
B. Blue/green deployment (蓝/绿部署): 错误。Blue/green deployment是一种部署策略，用于减少应用程序更新时的停机时间，
与数据库的高可用性无关。
Correct Answer: C
正确答案：C

C. Multi-AZ deployment (剉勄用劉部署): 正确。Multi-AZ部署提供高可用性和灾难恢复功能，通过自动创建主数据库实例并同步
复制数据到另一个可用区的实例。
D. Reserved Instances (预留实例): 错误。Reserved Instances是一种定价选项，用于节省成本，与数据库的高可用性或灾难恢复无
关。

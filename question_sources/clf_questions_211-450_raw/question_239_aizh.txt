Question #239
A company has deployed an Amazon EC2 instance.
Which option is an AWS responsibility under the AWS shared responsibility model?
一家公司已部署了一个 Amazon EC2 实例。
在 AWS 共享责任桟型下，以下选项中哪项是 AWS 的责任？
A. Managing and encrypting application data
管栬和加密应用程序数据
B. Installing updates and security patches of guest operating system
安装客操作系统的更新和安全补丁
C. Configuration of infrastructure devices Most Voted
基础设施设备的配置
D. Configuration of security groups on each instance
桹个实例的安全组配置
题目分析与解答
1. 考察的知识点
本题考察的是AWS共享责任桟型（Shared Responsibility Model）。该桟型定义了AWS和客户在云安全和管栬方面的责任划分。
2. 选项分析
A. Managing and encrypting application data 管理和加密应用程序数据
错误。管栬和加密应用程序数据是客户的责任，因为客户负责其应用程序和数据的安全性。
Correct Answer: C
桭确答栾：C

B. Installing updates and security patches of guest operating system 安装客操作系统的更新和安全补丁
错误。客户负责管栬其EC2实例的操作系统，包括安装更新和安全补丁。
C. Configuration of infrastructure devices 基础设施设备的配置
桭确。AWS负责管栬和配置底层基础设施设备，例如网络设备、存储设备和栌栬服务器。
D. Configuration of security groups on each instance 每个实例的安全组配置
错误。安全组的配置是客户的责任，因为客户需要根据其应用程序的需梆来定义访问规则。

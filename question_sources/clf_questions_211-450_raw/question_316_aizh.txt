A company wants to move its on-premises databases to managed cloud database services by using a simplified migration process.
Which AWS service or tool can help the company meet this requirement?
一家公司希望通过简化的迁移流程将其本地数据库迁移到托管的云数据库服务。
哪项AWS服务或工具可以帮助该公司满足这一需求？
A. AWS Storage Gateway
AWS Storage Gateway
B. AWS Application Migration Service
AWS Application Migration Service

AWS DataSync
D. AWS Database Migration Service (AWS DMS) Most Voted
AWS Database Migration Service (AWS DMS)
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的功能和用途，特别是与数据库迁移相关的服务。
2. 选项分析
A. AWS Storage Gateway: 错误。AWS Storage Gateway主要用于在本地环境和AWS云之间集成存储解决方案，适合文件存储或备
份场景，而不是数据库迁移。
Correct Answer: D

提供简化的迁移流程。

Selected Answer: C
The correct answer is C. AWS Secrets Manager. It allows users to securely store and centrally manage secrets such as database credentials, API keys,
and other sensitive information. You can use fine-grained AWS Identity and Access Management (IAM) policies to control access to these secrets, and
Secrets Manager also supports automatic rotation of secrets to meet security and compliance requirements1. If you need to retrieve a credential
from Secrets Manager, you can log into the AWS account, choose Secrets Manager from the Services menu, search for the desired credential, and
retrieve its secret value2.

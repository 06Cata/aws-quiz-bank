Question #260
A company acquired another corporation. The company now has two AWS accounts.
Which AWS service or tool can the company use to consolidate the billing for these two accounts?
一家公司收购了另一家公司。该公司现在拥有两个AWS账户。
该公司可以使用哪种AWS服务或工具来整合这两个账户的账单？
A. AWS Systems Manager
AWS Systems Manager
B. AWS Organizations Most Voted
AWS Organizations
C. AWS License Manager
AWS License Manager
D. Cost Explorer
Cost Explorer
题目分析与解答
1. 考察的知识点
本题考察的是AWS账户管理和账单整合相关的知识点，特别是AWS Organizations的功能。
2. 选项分析
A. AWS Systems Manager: 错误。AWS Systems Manager主要用于管理和操作多个AWS资源和服务，但与账单整合无关。
B. AWS Organizations: 正确。AWS Organizations允许用户将多个AWS账户集中管理，并支持账单整合功能。
Correct Answer: B
正确答案：B

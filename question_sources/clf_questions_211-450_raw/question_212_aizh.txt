Question #212
Which AWS service allows users to model and provision AWS resources using common programming languages?
哪个AWS服务允许用户使用常见编程语言来建模和配置AWS资源？
A. AWS CloudFormation
AWS CloudFormation
B. AWS CodePipeline
AWS CodePipeline
C. AWS Cloud Development Kit (AWS CDK) Most Voted
AWS Cloud Development Kit (AWS CDK)
D. AWS Systems Manager
AWS Systems Manager
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中支持使用编程语言来定义和管理资源的工具。
2. 选项分析
A. AWS CloudFormation: AWS CloudFormation允许用户使用模板来定义和管理AWS资源，但它使用的是YAML或JSON格式，而不
是常见的编程语言。因此此选项错误。
B. AWS CodePipeline: AWS CodePipeline是一项持续交付服务，用于自动化代码部署流程，与定义和管理AWS资源无关。因此此
选项错误。
C. AWS Cloud Development Kit (AWS CDK): AWS CDK允许用户使用常见的编程语言（如Python、JavaScript、Java等）来定义和
管理AWS资源。此选项正确。
Correct Answer: C
正确答案：C

A company needs to connect its on-premises data center to the AWS Cloud. The company needs a dedicated, low-latency
connection with consistent network performance.
Which AWS service will meet these requirements?
一家公司需要将其本地数据中心连接到AWS云。该公司需要一个专用的、低延迟的连接，并具有一致的网络性能。
哪项AWS服务可以满足这些需求？
A. AWS Global Accelerator
AWS Global Accelerator

题目分析与解答
1. 考察的知识点
本题考察的是AWS网络服务的选择，特别是如何连接本地数据中心与AWS云环境。
重点关注低延迟、专用连接和一致的网络性能。
2. 选项分析
Correct Answer: C
正确答案：C

A. AWS Global Accelerator
错误。AWS Global Accelerator主要用于优化全球用户访问应用的性能，通过路由流量到最近的AWS区域。它不提供专用的低延迟连
接到本地数据中心。
B. Amazon CloudFront
错误。Amazon CloudFront是一个内容分发网络（CDN）服务，用于加速分发静态和动态内容。它不涉及专用连接或低延迟连接到
本地数据中心。
C. AWS Direct Connect
正确。AWS Direct Connect提供专用的网络连接，具有低延迟和一致的网络性能，非常适合连接本地数据中心到AWS云环境。
D. AWS Managed VPN
错误。AWS Managed VPN提供基于互联网的加密连接，虽然可以连接本地数据中心到AWS，但它的性能和延迟取决于公共互联
网，不如AWS Direct Connect一致。
3. 我的答案
C. AWS Direct Connect
4. 官方答案
C. AWS Direct Connect

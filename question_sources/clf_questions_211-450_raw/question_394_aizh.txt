Which AWS service will meet this requirement?

A. AWS Lambda:
AWS Lambda是一个无服务器计算服务，用于运行代码响应事件。虽然它可以用于处理身份验证逻辑，但它本身并不直接支持社交
媒体身份提供商的集成。因此此选项错误。
B. Amazon Cognito:
Amazon Cognito是AWS的身份管理服务，支持用户通过社交媒体身份提供商（如Facebook、Google等）登录。它提供用户池和身
份池功能，专门设计用于处理身份验证和授权。因此此选项正确。
C. AWS Secrets Manager:
AWS Secrets Manager是一个用于管理和保护应用程序密钥和密码的服务，与身份验证无直接关系，也不支持社交媒体身份提供商
的集成。因此此选项错误。
D. Amazon CloudFront:
Amazon CloudFront是一个内容分发网络（CDN）服务，用于加速内容交付。它与身份验证无直接关系，也不支持社交媒体身份提
供商的集成。因此此选项错误。

我的答案是：B. Amazon Cognito

题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的用途和功能，特别是如何与AWS服务进行交互。
2. 选项分析
A. AWS CloudShell: AWS CloudShell 是一个基于浏览器的命令行界面，用于运行 AWS CLI 命令。虽然它可以与 AWS 服务交互，但
它主要用于命令行操作，而不是用于构建和管理 Web 应用程序。
Correct Answer: C

Question #211
Which AWS security service protects applications from distributed denial of service attacks with always-on detection and
automatic inline mitigations?
哪项AWS安全服务通过始终开启的检测和自动内联缓解措施保护应用程序免受分布式拒绝服务攻击？
A. Amazon Inspector
Amazon Inspector
B. AWS Web Application Firewall (AWS WAF)
AWS Web Application Firewall (AWS WAF)
C. Elastic Load Balancing (ELB)
Elastic Load Balancing (ELB)
D. AWS Shield Most Voted
AWS Shield
题目分析与解答
1. 考察的知识点
本题考察的是AWS安全服务的功能，特别是针对分布式拒绝服务（DDoS）攻击的防护能力。
2. 选项分析
A. Amazon Inspector: Amazon Inspector是一项自动化的安全评估服务，用于检查应用程序的安全漏洞和不符合最佳实践的情况。
它不具备防护DDoS攻击的功能，因此此选项错误。
B. AWS Web Application Firewall (AWS WAF): AWS WAF是一项保护Web应用程序免受常见攻击（如SQL注入和跨站脚本攻击）
的服务，但它主要针对应用层的攻击，而不是专门设计用于防护DDoS攻击。因此此选项错误。
Correct Answer: D
正确答案：D

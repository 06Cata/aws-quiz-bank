Question #440
A company wants to migrate a company’s on-premises container infrastructure to the AWS Cloud. The company wants to prevent
unplanned administration and operation cost and adapt to a serverless architecture.
Which AWS service will meet these requirements?
一家公司希望将其本地容器基础设施迁移到AWS云。该公司希望避免计划外的管理和运营成本，并适应无服务器架构。
哪个AWS服务可以满足这些需求？
A. Amazon Connect
Amazon Connect
B. AWS Fargate Most Voted
AWS Fargate
C. Amazon Lightsail
Amazon Lightsail
D. Amazon EC2
Amazon EC2
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的特点，特别是与容器管理相关的服务。
重点在于理解如何使用AWS服务实现无服务器架构，同时减少管理和运营成本。
2. 选项分析
Correct Answer: B
正确答案：B

A. Amazon Connect:
Amazon Connect 是一个云联络中心服务，用于客户支持和呼叫中心管理。它与容器管理无关，因此不符合题目要求。
B. AWS Fargate:
AWS Fargate 是一种无服务器计算引擎，专门用于运行容器。它消除了管理服务器的需求，降低了运营成本，非常适合题目中提到
的迁移到无服务器架构的需求。因此，这是正确答案。
C. Amazon Lightsail:
Amazon Lightsail 是一种简化的虚拟服务器解决方案，适合小型应用程序和网站托管。虽然它可以运行容器，但它不是无服务器架
构，仍需要管理底层服务器，因此不符合题目要求。
D. Amazon EC2:
Amazon EC2 提供虚拟服务器实例，用户需要管理底层服务器和操作系统。这与题目中要求的无服务器架构和减少运营成本的目标
不符，因此是错误选项。
3. 我的答案
我的答案是：B. AWS Fargate

官方答案是：B. AWS Fargate

Question #296
A company has a managed IAM policy that does not grant the necessary permissions for users to accomplish required tasks.
How can this be resolved?
一家公司有一个托管的IAM策略，但该策略未授予用户完成所需任务的必要权限。
如何解决这个问题？
A. Enable AWS Shield Advanced.
启用AWS Shield Advanced。
B. Create a custom IAM policy. Most Voted
创建一个自定义IAM策略。
C. Use a third-party web application firewall (WAF) managed rule from the AWS Marketplace.
使用来自AWS Marketplace的第三方Web应用程序防棨墙（WAF）托管规则。
D. Use AWS Key Management Service (AWS KMS) to create a customer-managed key.
使用AWS Key Management Service（AWS KMS）创建一个客户托管密钥。
题目分析与解答
1. 考察的知识点
本题考察的是IAM（Identity and Access Management）权限管栬的知识，栎别是如何解决权限不足的问题。
2. 选项分析
A. Enable AWS Shield Advanced.
错误。AWS Shield Advanced是用于保护应用程序免受DDoS攻击的服务，与IAM权限管栬无关。
Correct Answer: B
桭确答栾：B

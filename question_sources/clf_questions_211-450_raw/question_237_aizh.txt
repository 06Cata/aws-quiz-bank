Question #237
Which AWS service can a company use to visually design and build serverless applications?
公司可以使用哪项AWS服务来可视化设计和构建无服务器应用程序？
A. AWS Lambda
AWS Lambda
B. AWS Batch
AWS Batch
C. AWS Application Composer Most Voted
AWS Application Composer
D. AWS App Runner
AWS App Runner
题目分析与解答
1. 考察的知识点
本题考察的是对AWS服务的功能和用途的理解，特别是与设计和构建无服务器（Serverless）应用相关的服务。
2. 选项分析
A. AWS Lambda: AWS Lambda 是一个无服务器计算服务，允许用户运行代码而无需管理服务器。然而，它并不提供可视化设计和
构建应用的功能，因此此选项错误。
B. AWS Batch: AWS Batch 是一个服务，用于运行批处理作业。它与无服务器应用的设计和构建无关，因此此选项错误。
C. AWS Application Composer: AWS Application Composer 是一个服务，专门用于可视化设计和构建无服务器应用。它提供拖放
式界面，帮助用户快速构建应用架构。因此此选项正确。
D. AWS App Runner: AWS App Runner 是一个服务，用于快速部署容器化应用，但它不提供可视化设计功能，因此此选项错误。
Correct Answer: C
正确答案：C

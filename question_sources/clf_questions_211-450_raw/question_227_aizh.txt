Which AWS service or tool should the company use to meet this requirement?

题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的功能和用途，特别是与数据库迁移相关的服务。
2. 选项分析
A. Cloud Adoption Readiness Tool: 错误。Cloud Adoption Readiness Tool (CART) 是一个评估工具，用于帮助公司评估其云迁移
的准备情况，但它不用于实际的数据库迁移。
Correct Answer: C

C. AWS Database Migration Service (AWS DMS): 正确。AWS DMS 是专门设计用于迁移数据库的服务，包括从本地到Amazon

专门用于数据库迁移。

Which AWS service or feature gives users the ability to connect VPCs and on-premises networks to a central hub?
哪个AWS服务或功能使用户能够将VPC和本地网络连接到一个中央枢纽？
A. Virtual private gateway
Virtual private gateway
B. AWS Transit Gateway Most Voted
AWS Transit Gateway
C. Internet gateway
Internet gateway
D. Customer gateway
Customer gateway
题目分析与解答
1. 考察的知识点
本题考察的是AWS网络服务的功能，特别是如何连接多个VPC和本地网络到一个中央枢纽。
2. 选项分析
A. Virtual private gateway:
Virtual private gateway是用于连接VPC和本地网络的VPN终端点，但它不能作为中央枢纽连接多个VPC和本地网络，因此此选项错
误。
B. AWS Transit Gateway:
AWS Transit Gateway允许用户将多个VPC和本地网络连接到一个中央枢纽，简化网络管理和通信。此选项正确。
Correct Answer: B
正确答案：B

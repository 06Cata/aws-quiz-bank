Question #254
Which Amazon EC2 pricing model provides the MOST cost savings for an always-up, right-sized database server running for a
project that will last 1 year?
哪种 Amazon EC2 定价模式为持续运行、大小合适的数据库服务器提供最多的成本节约，该服务器用于持续 1 年的项目？
A. On-Demand Instances
按需实例
B. Convertible Reserved Instances
可转换预留实例
C. Spot Instances
竞价实例
D. Standard Reserved Instances Most Voted
标准预留实例
题目分析与解答
1. 考察的知识点
本题考察的是Amazon EC2的定价模型，包括按需实例、预留实例（标准和可转换）、以及竞价实例的特点和适用场景。
重点在于选择最节省成本的定价模型，适用于持续运行且已知时长的数据库服务器。
2. 选项分析
A. On-Demand Instances 按需实例:
按需实例提供灵活性，无需长期承诺，但成本较高。适合短期或不可预测的工作负载，不适合持续运行的数据库服务器。
错误原因: 成本较高，不是最节省成本的选项。
Correct Answer: D
正确答案：D

B. Convertible Reserved Instances 可转换预留实例:
可转换预留实例允许在预留期内更改实例类型，但折扣率通常低于标准预留实例。适合需要灵活性但仍希望节省成本的场景。
错误原因: 虽然节省成本，但不如标准预留实例节省得多，且题目未提到需要灵活性。
C. Spot Instances 竞价实例:
竞价实例提供最低成本，但可能随时被中断，不适合需要持续运行的数据库服务器。
错误原因: 不适合始终运行的工作负载，可靠性不足。
D. Standard Reserved Instances 标准预留实例:
标准预留实例提供最高的成本节省，适合持续运行且已知时长的工作负载。对于1年的项目，这是最合适的选择。
正确原因: 提供最高折扣，适合持续运行的数据库服务器。
3. 我的答案
D. Standard Reserved Instances 标准预留实例
4. 官方答案

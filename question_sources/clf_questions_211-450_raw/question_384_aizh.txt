Question #384
Which option is a responsibility of AWS under the AWS shared responsibility model?
哪项选项是 AWS 在 AWS 共享责任模型下的责任？
A. Application data security
应用数据安全
B. Patch management for applications that run on Amazon EC2 instances
针对运行在 Amazon EC2 实例上的应用程序的补丁管理
C. Patch management of the underlying infrastructure for managed services Most Voted
托管服务底层基础设施的补丁管理
D. Application identity and access management
应用程序的身份和访问管理
题目分析与解答
1. 考察的知识点
本题考察的是AWS共享责任模型（Shared Responsibility Model）。该模型定义了AWS和客户在云安全方面的责任划分。
2. 选项分析
A. Application data security 应用数据安全
错误。应用数据安全是客户的责任，因为客户负责管理和保护其存储在AWS上的数据，包括加密、访问控制等。
B. Patch management for applications that run on Amazon EC2 instances 针对运行在 Amazon EC2 实例上的应用程序的补丁
管理
错误。运行在EC2实例上的应用程序的补丁管理是客户的责任，因为客户完全控制EC2实例及其上运行的应用程序。
Correct Answer: C
正确答案：C

C. Patch management of the underlying infrastructure for managed services 托管服务底层基础设施的补丁管理
正确。AWS负责管理托管服务（如Amazon RDS、Amazon S3等）的底层基础设施，包括操作系统和底层硬件的补丁管理。
D. Application identity and access management 应用程序的身份和访问管理
错误。应用程序的身份和访问管理是客户的责任，因为客户需要配置IAM策略和权限来保护其应用程序。

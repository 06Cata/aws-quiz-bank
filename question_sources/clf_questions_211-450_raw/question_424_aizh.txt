Question #424
Which pillar of the AWS Well-Architected Framework aligns with the ability to make frequent, small, and reversible changes to
AWS Cloud architecture?
哪个AWS Well-Architected Framework支柱与能够对AWS Cloud架构进行频繁、小规模且可逆的更改相一致？
A. Security
安全性
B. Cost optimization
成本优化
C. Operational excellence
运营卓越
D. Performance efficiency
性能效率
题目分析与解答
1. 考察的知识点
本题考察的是AWS Well-Architected Framework的五大支柱之一，以及它们的具体含义和应用场景。
题目重点关注“频繁、小规模、可逆的变更能力”，这与运营卓越（Operational Excellence）支柱的核心理念相关。
2. 选项分析
A. Security 安全性: 错误。安全性支柱关注保护数据、系统和资产，确保风险管理，但与频繁、小规模、可逆的变更能力无直接关
联。
B. Cost optimization 成本优化: 错误。成本优化支柱关注如何以最低成本运行系统，与变更能力无直接关联。
Correct Answer: C
正确答案：C

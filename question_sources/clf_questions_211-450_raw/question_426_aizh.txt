Question #426
A company hosts its website on Amazon EC2 instances. The company needs to ensure that the website reaches a global audience
and provides minimum latency to users.
Which AWS service should the company use to meet these requirements?
一家公司在 Amazon EC2 实例上托管其网站。该公司需要确保网站能够覆盖全球受众，并为用户提供最低的延迟。
该公司应该使用哪项 AWS 服务来满足这些需求？
A. Amazon Route 53
Amazon Route 53
B. Amazon CloudFront Most Voted
Amazon CloudFront
C. Elastic Load Balancing
Elastic Load Balancing
D. AWS Lambda
AWS Lambda
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的全球分发能力和降低用户访问延迟的功能。
重点涉及Amazon CloudFront、Amazon Route 53、Elastic Load Balancing和AWS Lambda的用途和特点。
2. 选项分析
Correct Answer: B
正确答案：B

A. Amazon Route 53:
Amazon Route 53是一个DNS服务，主要用于域名解析和流量路由。虽然它可以帮助将流量路由到不同的区域，但它并不直接提供
内容分发和降低延迟的功能。因此此选项错误。
B. Amazon CloudFront:
Amazon CloudFront是一个内容分发网络（CDN）服务，可以将内容缓存到全球的边缘位置，从而降低用户访问延迟并提高性能。
此选项正确。
C. Elastic Load Balancing:
Elastic Load Balancing用于分发流量到多个EC2实例以实现高可用性，但它不具备全球分发和降低延迟的功能。因此此选项错误。
D. AWS Lambda:
AWS Lambda是一个无服务器计算服务，用于运行代码，但它与全球分发和降低延迟无关。因此此选项错误。
3. 我的答案
我的答案是：B. Amazon CloudFront

官方答案是：B. Amazon CloudFront

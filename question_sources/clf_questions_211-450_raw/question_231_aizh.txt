Question #231
Who enables encryption of data at rest for Amazon Elastic Block Store (Amazon EBS)?
谁负责启用 Amazon Elastic Block Store (Amazon EBS) 的静态数据加密？
A. AWS Support
AWS支持
B. AWS customers Most Voted
AWS客户
C. AWS Key Management Service (AWS KMS)
AWS密钥管理服务 (AWS KMS)
D. AWS Trusted Advisor
AWS可信顾问
题目分析与解答
1. 考察的知识点
本题考察的是Amazon Elastic Block Store (Amazon EBS)数据加密的责任归属，以及AWS客户在数据加密中的角色。
2. 选项分析
A. AWS Support: 错误。AWS Support的主要职责是为客户提供技术支持和指导，但它并不负责启用或管理数据加密。
B. AWS customers: 正确。AWS客户负责启用和管理Amazon EBS卷的加密。客户可以选择在创建EBS卷时启用加密，并选择使用
AWS Key Management Service (AWS KMS)管理密钥。
C. AWS Key Management Service (AWS KMS): 错误。AWS KMS是一个服务，用于管理加密密钥，但它并不直接负责启用EBS卷
的加密。启用加密的操作由客户执行。
Correct Answer: B
正确答案：B

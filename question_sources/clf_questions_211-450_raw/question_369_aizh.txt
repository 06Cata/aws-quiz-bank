Question #369
Which AWS service provides the ability to host a NoSQL database in the AWS Cloud?
哪个AWS服务提供了在AWS云中托管NoSQL数据库的能力？
A. Amazon Aurora
Amazon Aurora
B. Amazon DynamoDB Most Voted
Amazon DynamoDB
C. Amazon RDS
Amazon RDS
D. Amazon Redshift
Amazon Redshift
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的功能与用途，特别是与数据库相关的服务。
重点是了解AWS中提供NoSQL数据库托管服务的选项。
2. 选项分析
A. Amazon Aurora: 错误。Amazon Aurora 是一种关系型数据库服务，支持MySQL和PostgreSQL兼容的数据库，不是NoSQL数据
库。
B. Amazon DynamoDB: 正确。Amazon DynamoDB 是一种完全托管的NoSQL数据库服务，专为高性能和可扩展性而设计。
C. Amazon RDS: 错误。Amazon RDS 是一种关系型数据库服务，支持多种数据库引擎（如MySQL、PostgreSQL、SQL Server
等），不是NoSQL数据库。
Correct Answer: B
正确答案：B

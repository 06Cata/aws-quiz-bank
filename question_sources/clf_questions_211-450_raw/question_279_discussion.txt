To provide temporary security credentials for AWS users to access AWS services, the developer should use:
C. AWS Security Token Service (AWS STS)
AWS STS allows you to grant users and systems temporary access to AWS resources. Users can assume an IAM role for a limited time to access AWS
services, which STS facilitates by providing temporary security credentials. This service is particularly useful in scenarios where you need to provide
access that is limited in time and scope, such as for cross-account access or federated user access.

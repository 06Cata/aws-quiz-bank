A. AWS Shield: AWS Shield是一个托管的分布式拒绝服务（DDoS）保护服务，用于保护应用程序免受DDoS攻击。它不用于控制
EC2实例的网络流量，因此此选项错误。
B. Security groups: Security groups是一个虚拟防火墙，用于控制EC2实例的入站和出站流量。它是专门设计用于实例级别的网络
流量控制，因此此选项正确。
C. Network Access Analyzer: Network Access Analyzer是一个工具，用于分析和优化网络访问权限。它不直接控制EC2实例的网络
流量，因此此选项错误。
D. VPC endpoints: VPC endpoints用于在VPC中私密地连接到AWS服务，而无需通过互联网。它不用于控制EC2实例的网络流量，
因此此选项错误。
3. 我的答案
B. Security groups
4. 官方答案
B. Security groups

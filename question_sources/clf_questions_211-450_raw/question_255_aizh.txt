A company has a physical tape library to store data backups. The tape library is running out of space. The company needs to
extend the tape library's capacity to the AWS Cloud.
Which AWS service should the company use to meet this requirement?

A. Amazon Elastic File System (Amazon EFS)
Amazon EFS 是一个文件存储服务，适用于共享文件存储的场景，主要用于Linux工作负载。它不适合用于扩展物理磁带库的容量，
因此此选项错误。
B. Amazon Elastic Block Store (Amazon EBS)
Amazon EBS 是块存储服务，主要用于与EC2实例配合使用。它不支持直接与物理磁带库集成，因此此选项错误。
C. Amazon S3
Amazon S3 是对象存储服务，适合存储大量非结构化数据，但它无法直接与物理磁带库集成或扩展其容量，因此此选项错误。
D. AWS Storage Gateway
AWS Storage Gateway 是一个混合云存储服务，支持将本地存储扩展到AWS云端。它提供了磁带网关功能，可以将物理磁带库扩展
到云端，因此此选项正确。
3. 我的答案
D. AWS Storage Gateway
4. 官方答案
D. AWS Storage Gateway

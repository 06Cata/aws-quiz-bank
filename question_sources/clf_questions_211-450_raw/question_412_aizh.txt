Question #412
A company needs to check for IAM access keys that have not been rotated recently.
Which AWS service should the company use to meet this requirement?
一家公司需要检查最近未旋转的IAM访问密钥。
该公司应该使用哪项AWS服务来满足此要求？
A. AWS WAF
AWS WAF
B. AWS Shield
AWS Shield
C. Amazon Cognito
Amazon Cognito
D. AWS Trusted Advisor Most Voted
AWS Trusted Advisor
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中与安全性和访问管理相关的功能，特别是如何检查IAM访问密钥的使用情况和安全性。
2. 选项分析
A. AWS WAF: AWS WAF（Web Application Firewall）主要用于保护Web应用程序免受常见的网络攻击（如SQL注入和跨站脚本攻
击）。它与IAM访问密钥的管理无关，因此此选项错误。
Correct Answer: D
正确答案：D

B. AWS Shield: AWS Shield是一个托管的DDoS保护服务，旨在保护应用程序免受分布式拒绝服务攻击。它与IAM访问密钥的管理无
关，因此此选项错误。
C. Amazon Cognito: Amazon Cognito主要用于用户身份验证和管理，帮助开发人员轻松添加用户登录功能。它不涉及IAM访问密
钥的检查或管理，因此此选项错误。
D. AWS Trusted Advisor: AWS Trusted Advisor是一个在线工具，提供建议以帮助用户优化AWS资源的使用，包括安全性检查。它
可以检查IAM访问密钥的使用情况并提醒用户未及时轮换的密钥，因此此选项正确。

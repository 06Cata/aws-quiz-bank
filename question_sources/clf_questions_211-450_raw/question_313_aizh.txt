Amazon EBS 是一种块存储服务，主要用于与 EC2 实例配合使用，提供高性能和低延迟的存储。
它不适合存储静态内容，也不具备无限扩展能力。
错误原因：不符合题目要求的“静态内容”和“无限扩展”。
B. Amazon S3
Amazon S3 是对象存储服务，专为存储静态内容（如图片、视频、文档等）设计。
它具有无限扩展能力，并且成本低廉，非常适合存储静态内容。
正确原因：完全符合题目要求的“静态内容”、“无限扩展”和“最低成本”。
C. AWS Storage Gateway
AWS Storage Gateway 是一种混合云存储服务，主要用于将本地存储与云存储集成。
它不适合直接存储静态内容，也不具备无限扩展能力。
错误原因：不符合题目要求的“静态内容”和“无限扩展”。
D. Amazon Elastic File System (Amazon EFS)
Amazon EFS 是文件存储服务，主要用于共享文件系统，适合与多个 EC2 实例配合使用。
虽然它具有扩展能力，但成本较高，不适合存储静态内容。
错误原因：不符合题目要求的“最低成本”。

答案：B. Amazon S3

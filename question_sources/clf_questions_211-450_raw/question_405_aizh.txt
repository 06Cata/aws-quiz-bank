A company needs to consolidate the billing for multiple AWS accounts. The company needs to use one account to pay on behalf
of all the other accounts.
Which AWS service or tool should the company use to meet this requirement?

该公司应该使用哪个AWS服务或工具来满足此需求？
A. AWS Trusted Advisor
AWS Trusted Advisor

1. 考察的知识点
本题考察的是AWS Organizations的功能，特别是其集中管理和统一支付的能力。

Correct Answer: B
正确答案：B

A. AWS Trusted Advisor: 错误。AWS Trusted Advisor是一个优化工具，用于检查账户的安全性、成本优化、性能等方面，但它不
负责账户间的统一支付功能。
B. AWS Organizations: 正确。AWS Organizations允许用户集中管理多个AWS账户，并支持统一支付功能（通过设置一个主账户来
支付所有关联账户的费用）。
C. AWS Budgets: 错误。AWS Budgets是一个成本管理工具，用于设置预算和监控费用，但它不提供账户间的支付整合功能。
D. AWS Service Catalog: 错误。AWS Service Catalog用于管理和部署预定义的云资源集合，与账户支付整合无关。
3. 我的答案
我的答案是：B. AWS Organizations
4. 官方答案
官方答案是：B. AWS Organizations

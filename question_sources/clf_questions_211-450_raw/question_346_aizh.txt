Question #346
A company launched an Amazon EC2 instance with the latest Amazon Linux 2 Amazon Machine Image (AMI).
Which actions can a system administrator take to connect to the EC2 instance? (Choose two.)
一家公司使用最新的 Amazon Linux 2 Amazon Machine Image (AMI) 启动了一台 Amazon EC2 实例。
系统管理员可以采取哪些操作来连接到 EC2 实例？（选择两个）
A. Use Amazon EC2 Instance Connect. Most Voted
使用 Amazon EC2 Instance Connect。
B. Use a Remote Desktop Protocol (RDP) connection.
使用 Remote Desktop Protocol (RDP) 连接。
C. Use AWS Batch.
使用 AWS Batch。
D. Use AWS Systems Manager Session Manager. Most Voted
使用 AWS Systems Manager Session Manager。
E. Use Amazon Connect.
使用 Amazon Connect。
题目分析与解答
1. 考察的知识点
本题考察的是连接到Amazon EC2实例的方式，涉及AWS服务如EC2 Instance Connect、Systems Manager Session Manager等。
需要了解不同服务的用途和适用场景，例如远程连接工具和管理工具的区别。

A. Use Amazon EC2 Instance Connect: 正确。Amazon EC2 Instance Connect是一种基于浏览器的工具，用于通过SSH连接到
Amazon Linux 2或Ubuntu实例。
B. Use a Remote Desktop Protocol (RDP) connection: 错误。RDP是用于连接Windows实例的协议，而题目中明确提到使用的是
Amazon Linux 2 AMI，因此不适用。
C. Use AWS Batch: 错误。AWS Batch是用于运行批处理作业的服务，与连接到EC2实例无关。
D. Use AWS Systems Manager Session Manager: 正确。Session Manager允许通过AWS控制台或CLI安全地连接到EC2实例，无需
SSH密钥或开放端口。
E. Use Amazon Connect: 错误。Amazon Connect是一个云联络中心服务，与连接到EC2实例无关。

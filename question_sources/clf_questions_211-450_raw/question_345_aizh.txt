Question #345
Which AWS service provides a single location to track the progress of application migrations?
哪个AWS服务提供一个单一位置来跟踪应用程序迁移的进度？
A. AWS Application Discovery Service
AWS Application Discovery Service
B. AWS Application Migration Service
AWS Application Migration Service
C. AWS Service Catalog
AWS Service Catalog
D. AWS Migration Hub Most Voted
AWS Migration Hub
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中与应用迁移相关的服务及其功能，特别是如何跟踪迁移进度。
2. 选项分析
A. AWS Application Discovery Service:
此服务用于收集有关本地数据中心的服务器、应用程序和依赖关系的信息，以帮助规划迁移。它不提供跟踪迁移进度的功能，因此
此选项错误。
B. AWS Application Migration Service:
此服务用于将本地服务器迁移到AWS，但它的主要功能是执行迁移，而不是提供单一位置来跟踪迁移进度。因此此选项错误。
Correct Answer: D
正确答案：D

D. AWS Migration Hub:

Question #433
A company wants to organize its users so that the company can grant permissions to the users as a group.
Which AWS service or tool can the company use to meet this requirement?
一家公司希望组织其用户，以便公司可以以组的形式向用户授予权限。
公司可以使用哪项AWS服务或工具来满足这一要求？
A. Security groups
Security groups
B. AWS Identity and Access Management (IAM)
AWS Identity and Access Management (IAM)
C. Resource groups
Resource groups
D. AWS Security Hub
AWS Security Hub
题目分析与解答
1. 考察的知识点
本题考察的是AWS Identity and Access Management (IAM)的功能，特别是IAM用户组的使用。
IAM用户组允许管理员将用户组织在一起，并为整个组分配权限，从而简化权限管理。
2. 选项分析
Correct Answer: B
正确答案：B

A. Security groups: 错误。Security groups是用于控制EC2实例的网络流量的工具，主要用于定义入站和出站规则，与用户权限管
理无关。
B. AWS Identity and Access Management (IAM): 正确。IAM提供了用户组功能，可以将用户组织在一起并为组分配权限，从而
满足题目要求。
C. Resource groups: 错误。Resource groups用于组织和管理AWS资源，而不是用户权限管理。
D. AWS Security Hub: 错误。AWS Security Hub是一个安全管理和监控服务，与用户权限管理无关。

Question #441
A company wants its Amazon EC2 instances to be in different locations but share the same geographic area. The company also
wants to use multiple power grids and independent networking connectivity for the EC2 instances.
Which solution meets these requirements?
一家公司希望其 Amazon EC2 实例位于不同的位置，但共享同一地理区域。该公司还希望为 EC2 实例使用多个电网和独立的网络连
接。
哪种解决方案满足这些要求？
A. Use EC2 instances in multiple edge locations in the same AWS Region.
在同一 AWS Region 中使用多个边缘位置的 EC2 实例。
B. Use EC2 instances in multiple Availability Zones in the same AWS Region. Most Voted
在同一 AWS Region 中使用多个 Availability Zones 的 EC2 实例。
C. Use EC2 instances in multiple Amazon Connect locations in the same AWS Region.
在同一 AWS Region 中使用多个 Amazon Connect 位置的 EC2 实例。
D. Use EC2 instances in multiple AWS Artifact locations in the same AWS Region.
在同一 AWS Region 中使用多个 AWS Artifact 位置的 EC2 实例。
题目分析与解答
1. 考察的知识点
本题考察的是AWS的区域性架构，特别是Availability Zones (可用区)的概念。
Availability Zones是AWS区域内的独立数据中心，具有独立的电力、冷却和网络连接，能够提供高可用性和容错能力。
2. 选项分析
Correct Answer: B
正确答案：B

A. Use EC2 instances in multiple edge locations in the same AWS Region.
错误：Edge locations是用于Amazon CloudFront和其他内容交付服务的，而不是用于运行EC2实例。
Edge locations不具备独立的电力和网络连接的特性，无法满足题目要求。
B. Use EC2 instances in multiple Availability Zones in the same AWS Region.
正确：Availability Zones是AWS区域内的独立数据中心，具有独立的电力、冷却和网络连接。
使用多个Availability Zones可以确保高可用性和容错能力，同时满足题目要求的“共享同一地理区域但使用独立电力和网络连接”的条
件。
C. Use EC2 instances in multiple Amazon Connect locations in the same AWS Region.
错误：Amazon Connect是AWS的联络中心服务，与EC2实例的部署无关。
Amazon Connect位置无法满足题目中关于独立电力和网络连接的要求。
D. Use EC2 instances in multiple AWS Artifact locations in the same AWS Region.
错误：AWS Artifact是一个提供合规性报告和文档的服务，与EC2实例的部署无关。
AWS Artifact位置无法满足题目中关于独立电力和网络连接的要求。
3. 我的答案
B. Use EC2 instances in multiple Availability Zones in the same AWS Region.
4. 官方答案
B. Use EC2 instances in multiple Availability Zones in the same AWS Region.
5. 与官方答案的比较
我的答案与官方答案一致。
原因分析：Availability Zones是AWS区域内的独立数据中心，符合题目要求的“共享同一地理区域但使用独立电力和网络连接”的条
件。

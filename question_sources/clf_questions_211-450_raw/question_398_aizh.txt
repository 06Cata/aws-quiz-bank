Question #398
Which AWS service or tool gives users the ability to connect with AWS and deploy resources programmatically?
哪个AWS服务或工具使用户能够以编程方式连接AWS并部署资源？
A. Amazon QuickSight
Amazon QuickSight
B. AWS PrivateLink
AWS PrivateLink
C. AWS Direct Connect
AWS Direct Connect
D. AWS SDKs Most Voted
AWS SDKs
题目分析与解答
1. 考察的知识点
本题考察的是用户如何通过编程方式与AWS交互并部署资源的能力。
涉及AWS SDKs的功能和用途。
2. 选项分析
A. Amazon QuickSight: 错误。Amazon QuickSight是一个商业智能服务，用于数据可视化和分析，与编程部署资源无关。
B. AWS PrivateLink: 错误。AWS PrivateLink用于安全地连接VPC和AWS服务或本地应用程序，与编程部署资源无关。
C. AWS Direct Connect: 错误。AWS Direct Connect是一种网络服务，用于建立本地数据中心与AWS之间的专线连接，与编程部署
资源无关。
D. AWS SDKs: 正确。AWS SDKs提供了多种编程语言的工具包，允许开发者通过代码与AWS服务交互并部署资源。
Correct Answer: D
正确答案：D

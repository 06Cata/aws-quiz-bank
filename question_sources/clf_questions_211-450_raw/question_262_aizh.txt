Which of the following is a managed AWS service that is used specifically for extract, transform, and load (ETL) data?
以下哪个是专门用于提取、转换和加载（ETL）数据的托管AWS服务？
A. Amazon Athena
Amazon Athena
B. AWS Glue Most Voted
AWS Glue
C. Amazon S3
Amazon S3
D. AWS Snowball Edge
AWS Snowball Edge
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的功能和用途，特别是与数据处理相关的服务。
重点是了解哪种服务专门用于ETL（Extract, Transform, Load）操作。
2. 选项分析
A. Amazon Athena: Amazon Athena 是一种交互式查询服务，用于分析存储在 Amazon S3 中的数据。它不专门用于ETL操作，因
此此选项错误。
B. AWS Glue: AWS Glue 是一种完全托管的ETL服务，专门设计用于提取、转换和加载数据。此选项正确。
C. Amazon S3: Amazon S3 是一种对象存储服务，用于存储和检索数据。虽然可以存储ETL处理后的数据，但它本身不是ETL服务，
因此此选项错误。
Correct Answer: B
正确答案：B

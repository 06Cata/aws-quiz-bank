Question #449
Which AWS service can a company use to manage encryption keys in the cloud?
公司可以使用哪项AWS服务在云中管理加密密钥？
A. AWS License Manager
AWS License Manager
B. AWS Certificate Manager (ACM)
AWS Certificate Manager (ACM)
C. AWS CloudHSM Most Voted
AWS CloudHSM
D. AWS Directory Service
AWS Directory Service
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中与加密密钥管理相关的知识点。
了解AWS提供的加密密钥管理服务及其功能是AWS Certified Cloud Practitioner考试的重要内容。
2. 选项分析
A. AWS License Manager: 错误。AWS License Manager用于帮助企业管理和跟踪软件许可证，而不是用于加密密钥管理。
B. AWS Certificate Manager (ACM): 错误。ACM主要用于管理SSL/TLS证书，而不是专门用于加密密钥管理。
C. AWS CloudHSM: 正确。AWS CloudHSM是一项专门的服务，用于在云中管理加密密钥，提供硬件安全模块（HSM）以确保密钥
的安全性。
D. AWS Directory Service: 错误。AWS Directory Service用于管理目录服务（如Active Directory），与加密密钥管理无关。
Correct Answer: C
正确答案：C

Question #400
A company wants to define a central data protection policy that works across AWS services for compute, storage, and database
resources.
Which AWS service will meet this requirement?
一家公司希望定义一个跨AWS服务的中央数据保护策略，适用于计算、存储和数据库资源。
哪个AWS服务可以满足此需求？
A. AWS Batch
AWS Batch
B. AWS Elastic Disaster Recovery
AWS Elastic Disaster Recovery
C. AWS Backup Most Voted
AWS Backup
D. Amazon FSx
Amazon FSx
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中与数据保护和备份相关的功能，特别是跨服务的集中式数据保护策略。
2. 选项分析
A. AWS Batch: 错误。AWS Batch是用于运行批处理作业的服务，与数据保护或备份无关。
Correct Answer: C
正确答案：C

B. AWS Elastic Disaster Recovery: 错误。AWS Elastic Disaster Recovery主要用于灾难恢复，帮助快速恢复应用程序，但不用于定
义跨服务的集中式数据保护策略。
C. AWS Backup: 正确。AWS Backup是专门设计用于集中管理和自动化备份的服务，支持多个AWS服务（如EC2、RDS、EFS等），
符合题目要求。
D. Amazon FSx: 错误。Amazon FSx是一个文件存储服务，虽然支持备份功能，但它不是一个集中式数据保护解决方案。

A company needs to deploy applications in the AWS Cloud as quickly as possible. The company also needs to minimize the
complexity that is related to the management of AWS resources.
Which AWS service should the company use to meet these requirements?

题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的使用场景，特别是快速部署应用和简化资源管理的能力。
2. 选项分析
Correct Answer: B
正确答案：B

A. AWS Config: AWS Config主要用于记录和审计资源配置的变化，帮助用户了解资源的配置历史和合规性。它不用于快速部署应
用，因此此选项错误。
B. AWS Elastic Beanstalk: AWS Elastic Beanstalk是一项平台即服务（PaaS），可以快速部署和管理应用程序，同时简化资源管
理。它符合题目要求，因此此选项正确。
C. Amazon EC2: Amazon EC2是基础设施即服务（IaaS），需要用户手动配置和管理实例、网络等资源，复杂度较高，不符合题目
中“快速部署”和“简化管理”的要求，因此此选项错误。
D. Amazon Personalize: Amazon Personalize是一项机器学习服务，用于构建个性化推荐系统，与题目要求无关，因此此选项错
误。
3. 我的答案
我的答案是：B. AWS Elastic Beanstalk
4. 官方答案
官方答案是：B. AWS Elastic Beanstalk

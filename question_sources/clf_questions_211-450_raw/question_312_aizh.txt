A company wants to provide access to Windows file shares in AWS from its on-premises workloads. The company does not want
to provision any additional infrastructure or applications in its data center.
Which AWS service will meet these requirements?

程序。

本题考察的是AWS存储服务的使用场景，特别是如何在AWS中提供文件共享服务并与本地工作负载集成。

A. Amazon FSx File Gateway: 正确。Amazon FSx File Gateway允许本地工作负载通过标准文件协议（如 SMB）访问AWS上的
Windows文件共享，而无需在本地部署额外的基础设施。
B. AWS DataSync: 错误。AWS DataSync主要用于数据迁移和同步，而不是提供持续的文件共享服务。
C. Amazon S3: 错误。Amazon S3是对象存储服务，不支持Windows文件共享协议（如 SMB）。
D. AWS Snow Family: 错误。AWS Snow Family主要用于数据迁移和边缘计算，不适合提供持续的文件共享服务。
3. 我的答案
A. Amazon FSx File Gateway

A. Amazon FSx File Gateway

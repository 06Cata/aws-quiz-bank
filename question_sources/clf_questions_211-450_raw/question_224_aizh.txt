projected running costs in the AWS Cloud.
Which AWS service or tool will meet these requirements?

A. AWS Trusted Advisor: 错误。AWS Trusted Advisor主要用于提供最佳实践建议，例如安全性、成本优化、性能等，但它不提供
针对迁移评估和成本估算的具体功能。
B. Amazon Inspector: 错误。Amazon Inspector是一个安全评估服务，用于检查应用程序的安全漏洞和合规性问题，与迁移评估和
成本估算无关。
C. AWS Control Tower: 错误。AWS Control Tower用于设置和管理多账户环境的最佳实践，但它不提供针对迁移评估和成本估算的
功能。
D. Migration Evaluator: 正确。Migration Evaluator（以前称为TSO Logic）是专门设计用于帮助客户评估其当前的本地环境，并提
供迁移到AWS的成本估算和建议。
3. 我的答案
D. Migration Evaluator
4. 官方答案
D. Migration Evaluator

Which task is the responsibility of a company that is using Amazon RDS?
使用 Amazon RDS 的公司负责完成以下任务：
A. Provision the underlying infrastructure.
创建 IAM 策略以控制对服务的管栬访问。

提供底层基础设施。
C. Install the cables to connect the hardware for compute and storage.

D. Install and patch the RDS operating system.
安装和修补 RDS 操作系统。
题目分析与解答
1. 考察的知识点
本题考察的是AWS共享责任桟型，栎别是Amazon RDS的责任划分。
用户需要了解在使用Amazon RDS时，AWS负责的部分（如底层基础设施和操作系统管栬）以及用户负责的部分（如访问控制和数
据库配置）。
2. 选项分析
A. Provision the underlying infrastructure.
错误。AWS负责提供和管栬底层基础设施，包括计算、存储和网络资棕。用户无需手动配置这些资棕。
B. Create IAM policies to control administrative access to the service.
桭确。根据AWS共享责任桟型，用户负责管栬访问控制，包括创建IAM策略以限制对服务的管栬访问。
Correct Answer: B
桭确答栾：B

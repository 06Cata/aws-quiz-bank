Selected Answer: D
One of the key benefits of using AWS serverless computing, such as AWS Lambda, is that the management of infrastructure is offloaded to AWS. In a
serverless architecture, users can focus on writing code for their applications, and AWS automatically takes care of provisioning and managing the
underlying infrastructure. This allows developers to concentrate on building and deploying applications without the need to worry about server
provisioning, scaling, or maintenance.

Question #391
A company wants to manage sign-in security for workforce users. The company needs to create workforce users and centrally
manage their access across all the company's AWS accounts and applications.
Which AWS service will meet these requirements?
一家公司希望管理员工用户的登录安全性。该公司需要创建员工用户并集中管理他们对公司所有AWS账户和应用程序的访问权限。
哪个AWS服务可以满足这些需求？
A. AWS Audit Manager
AWS Audit Manager
B. Amazon Cognito
Amazon Cognito
C. AWS Security Hub
AWS Security Hub
D. AWS IAM Identity Center (AWS Single Sign-On) Most Voted
AWS IAM Identity Center (AWS Single Sign-On)
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中与用户身份管理和访问控制相关的知识点，特别是如何为企业用户提供集中式的身份管理和跨账户访问控
制。

Correct Answer: D

A. AWS Audit Manager
错误。AWS Audit Manager是用于帮助企业自动化审计流程的工具，与用户身份管理无关。
B. Amazon Cognito
错误。Amazon Cognito主要用于管理应用程序的客户身份（如移动应用或Web应用的用户），而不是企业内部的员工身份。
C. AWS Security Hub
错误。AWS Security Hub是一个安全管理和合规性服务，用于集中查看安全状态，与用户身份管理无关。
D. AWS IAM Identity Center (AWS Single Sign-On)
正确。AWS IAM Identity Center（以前称为AWS Single Sign-On）是专门设计用于集中管理企业用户身份和跨账户访问权限的服
务，完全符合题目要求。
3. 我的答案
D. AWS IAM Identity Center (AWS Single Sign-On)
4. 官方答案
D. AWS IAM Identity Center (AWS Single Sign-On)

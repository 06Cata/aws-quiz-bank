Which AWS service allows users to download security and compliance reports about the AWS infrastructure on demand?

A. Amazon GuardDuty
Amazon GuardDuty

C. AWS Artifact Most Voted

题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中与安全和合规相关的功能，特别是如何获取AWS基础设施的安全和合规报告。
2. 选项分析
A. Amazon GuardDuty: Amazon GuardDuty 是一种威胁检测服务，用于监控和分析 AWS 环境中的恶意活动和异常行为。它不提
供安全和合规报告下载功能，因此此选项错误。
B. AWS Security Hub: AWS Security Hub 是一个集中式安全管理服务，用于汇总和分析来自多个 AWS 服务的安全数据。虽然它提
供安全状态的概览，但它不支持下载 AWS 基础设施的安全和合规报告，因此此选项错误。
C. AWS Artifact: AWS Artifact 是一个服务，专门用于提供安全和合规报告的访问和下载功能。用户可以按需获取与 AWS 基础设施
相关的报告，因此此选项正确。
Correct Answer: C
正确答案：C

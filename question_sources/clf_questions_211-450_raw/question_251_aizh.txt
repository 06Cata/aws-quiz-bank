A company wants to store data in Amazon S3. The company rarely access the data, and the data can be regenerated if necessary.
The company wants to store the data in the most cost-effective storage class.
Which S3 storage class will meet this requirement?

A. S3 Standard:
错误。S3 Standard是为频痏访问甹数据设计甹，成本较高，不适合存储很少访问且可以再椹甹数据。
B. S3 Intelligent-Tiering:
错误。S3 Intelligent-Tiering适合访问桟式不畧定甹数据，会根据访问频栞自动调整存储层痔，但其疲栬成本较高，不是最痦梴甹选
择。
C. S3 Standard-Infrequent Access (S3 Standard-IA):
错误。虽棺S3 Standard-IA适合不频痏访问甹数据，但它提供了高可甠性和冗余，成本桻S3 One Zone-IA更高，不是最痦梴甹选
择。
D. S3 One Zone-Infrequent Access (S3 One Zone-IA):
桭畧。S3 One Zone-IA是为不频痏访问且可以容忍单可甠区数据丢失甹场景设计甹，成本最低，疥合题畉要梆。
3. 我的答案
我甹疫栾是：D. S3 One Zone-Infrequent Access (S3 One Zone-IA)
4. 官方答案
官方疫栾是：D. S3 One Zone-Infrequent Access (S3 One Zone-IA)

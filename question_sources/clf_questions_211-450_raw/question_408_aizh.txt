AWS Marketplace

AWS Service Catalog

AWS Trusted Advisor
题目分析与解答
1. 考察的知识点
本题考察的是用户如何找到AWS云解决方案设计的示例，涉及AWS资源和服务的文档与支持工具。
2. 选项分析
A. AWS Marketplace: 错误。AWS Marketplace是一个在线商店，用户可以购买和部署第三方软件解决方案，但它不提供云解决方
案设计的示例。
B. AWS Service Catalog: 错误。AWS Service Catalog用于管理和部署预定义的服务组合，主要用于企业内部的服务管理，而不是
提供解决方案设计示例。
C. AWS Architecture Center: 正确。AWS Architecture Center提供了大量的云解决方案设计示例，包括参考架构、最佳实践和技术
文档。
Correct Answer: C
正确答案：C

An Amazon EC2 instance previously used for development is inaccessible and no longer appears in the AWS Management
Console.
Which AWS service should be used to determine what action made this EC2 instance inaccessible?
A. Amazon CloudWatch Logs
B. AWS Security Hub

Selected Answer: D
A: Amazon CloudWatch Logs Monitors logs from applications and services but does not show who performed actions on resources.
B: AWS Security Hub provides a centralized view of security alerts and compliance status but does not track user activity or resource changes.
C: Amazon Inspector Scans for vulnerabilities but does not track operational actions.
D: AWS CloudTrail it records API calls and actions taken in the AWS environment, including who terminated or modified an EC2 instance

D. AWS CloudTrail
AWS CloudTrail is the service that records API calls and related events made on your AWS account. If an Amazon EC2 instance is inaccessible or if
there are actions taken on the instance, CloudTrail can be used to determine what actions were performed, who performed them, and when they
occurred. It provides a detailed history of changes and activity within your AWS environment.

How can this be resolved?
A. Enable AWS Shield Advanced.
B. Create a custom IAM policy. Most Voted
C. Use a third-party web application firewall (WAF) managed rule from the AWS Marketplace.

B. Create a custom IAM policy.
If the existing managed IAM policy does not provide the necessary permissions, you can create a custom IAM policy tailored to the specific
permissions required by the users. This allows you to define and grant the permissions needed for the tasks.

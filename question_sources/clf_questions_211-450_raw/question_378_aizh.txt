Which AWS service or tool will monitor AWS resources and applications in real time?
哪个AWS服务或工具可以实时监控AWS资源和应用程序？
A. AWS Trusted Advisor
AWS Trusted Advisor
B. Amazon CloudWatch Most Voted
Amazon CloudWatch
C. AWS CloudTrail
AWS CloudTrail
D. AWS Cost Explorer
AWS Cost Explorer
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中用于实时监控资源和应用的工具。
2. 选项分析
A. AWS Trusted Advisor: 错误。AWS Trusted Advisor主要用于提供优化建议，例如安全性、成本节约、性能提升等，但不用于实
时监控资源和应用。
B. Amazon CloudWatch: 正确。Amazon CloudWatch是AWS的监控服务，可以实时收集和跟踪指标，监控资源和应用的状态，并
设置警报。
C. AWS CloudTrail: 错误。AWS CloudTrail主要用于记录用户和服务的API调用历史，帮助进行审计和合规性检查，但不提供实时监
控功能。
Correct Answer: B
正确答案：B

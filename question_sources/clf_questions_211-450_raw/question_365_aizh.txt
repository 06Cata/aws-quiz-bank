Amazon CloudFront 提供什么功能？
A. Automatic scaling for all resources to power an application from a single unified interface
通过单一统一界面为应用程序提供所有资源的自动扩展能力
B. Secure delivery of data, videos, applications, and APIs to users globally with low latency Most Voted

通过多种路由类型（包括基于延迟的路由、地理 DNS、地理邻近性和加权轮询）直接管理全球流量的能力
D. Automatic distribution of incoming application traffic across multiple targets, such as Amazon EC2 instances, containers, IP

题目分析与解答
1. 考察的知识点
本题考察的是Amazon CloudFront的功能和用途。Amazon CloudFront是AWS的内容分发网络（CDN）服务，主要用于安全、快速
地分发内容到全球用户。
2. 选项分析
A. Automatic scaling for all resources to power an application from a single unified interface
错误：此选项描述的是AWS Auto Scaling的功能，而不是Amazon CloudFront的功能。CloudFront的主要功能是内容分发，而不是
资源自动扩展。
Correct Answer: B

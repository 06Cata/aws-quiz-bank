A company is running a multi-tier web application on premises. The web application is containerized and runs on a number of
Linux hosts connected to a PostgreSQL database that contains user records. The operational overhead of maintaining the
infrastructure and capacity planning is limiting the company's growth. A solutions architect must improve the application's
infrastructure.
Which combination of actions should the solutions architect take to accomplish this? (Choose two.)

2. 选项分析
A. Migrate the PostgreSQL database to Amazon Aurora.
正确。Amazon Aurora 是一种高性能、可扩展的托管数据库服务，支持 PostgreSQL。迁移到 Aurora 可以减少数据库的管理开销，
同时提高性能和可用性。
B. Migrate the web application to be hosted on Amazon EC2 instances.
错误。虽然 EC2 可以托管应用程序，但它仍然需要手动管理实例、扩展和容器化环境，无法显著减少运维开销。
C. Set up an Amazon CloudFront distribution for the web application content.
错误。CloudFront 是一个内容分发网络（CDN），主要用于加速静态和动态内容的交付，但题目没有提到需要优化内容分发，因此
与题目要求不完全相关。
D. Set up Amazon ElastiCache between the web application and the PostgreSQL database.
错误。虽然 ElastiCache 可以优化数据库查询性能，但题目重点在于减少运维开销，而不是直接优化数据库性能。
E. Migrate the web application to be hosted on AWS Fargate with Amazon Elastic Container Service (Amazon ECS).
正确。AWS Fargate 是一种无服务器容器管理服务，可以显著减少容器化应用的运维开销，同时支持自动扩展。

A company is migrating its on-premises PostgreSQL database to Amazon Aurora PostgreSQL. The on-premises database must
remain online and accessible during the migration. The Aurora database must remain synchronized with the on-premises
database.
Which combination of actions must a solutions architect take to meet these requirements? (Choose two.)
一家公司正在将其本地部署的 PostgreSQL 数据库迁移到 Amazon Aurora PostgreSQL。 在迁移期间，本地数据库必须保持在线并可
访问。 Aurora 数据库必须与本地数据库保持同步。
解决方案架构师必须采取哪些组合操作以满足这些要求？（选择两个。）

使用 AWS Schema Conversion Tool (AWS SCT) 转换数据库架构。

Amazon Aurora PostgreSQL的迁移与同步。

AWS Database Migration Service (AWS DMS)的使用。
如何保持数据库在线并同步。
2. 选项分析
A. Create an ongoing replication task.
正确。AWS DMS支持创建持续的复制任务以保持源数据库和目标数据库同步。这是实现实时同步的关键步骤。
B. Create a database backup of the on-premises database.
错误。创建数据库备份是静态的操作，无法满足题目中“保持同步”的要求。
C. Create an AWS Database Migration Service (AWS DMS) replication server.
正确。AWS DMS需要一个复制服务器来执行迁移任务，这是实现数据库迁移和同步的必要组件。
D. Convert the database schema by using the AWS Schema Conversion Tool (AWS SCT).
错误。AWS SCT用于转换数据库架构，但题目没有提到需要转换架构，且此工具不直接涉及同步操作。
E. Create an Amazon EventBridge (Amazon CloudWatch Events) rule to monitor the database synchronization.
错误。EventBridge用于事件驱动的自动化操作，与数据库同步无直接关系。

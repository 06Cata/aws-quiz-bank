A company is designing a cloud communications platform that is driven by APIs. The application is hosted on Amazon EC2
instances behind a Network Load Balancer (NLB). The company uses Amazon API Gateway to provide external users with access to
the application through APIs. The company wants to protect the platform against web exploits like SQL injection and also wants to
detect and mitigate large, sophisticated DDoS attacks.
Which combination of solutions provides the MOST protection? (Choose two.)

题目分析与解答
1. 考察的知识点
Amazon API Gateway的安全性配置。
AWS WAF的功能与使用场景。
AWS Shield Advanced与Shield Standard的区别及应用场景。
如何保护应用免受DDoS攻击和Web漏洞攻击。
2. 选项分析
A. Use AWS WAF to protect the NLB.
错误：AWS WAF无法直接保护Network Load Balancer (NLB)。AWS WAF主要用于保护Amazon API Gateway、Application Load
Balancer (ALB)和CloudFront。
B. Use AWS Shield Advanced with the NLB.
正确：AWS Shield Advanced提供高级DDoS保护，可以与NLB结合使用，帮助检测和缓解复杂的DDoS攻击。
C. Use AWS WAF to protect Amazon API Gateway.
正确：AWS WAF可以保护Amazon API Gateway免受Web漏洞攻击，例如SQL注入和跨站脚本攻击。
D. Use Amazon GuardDuty with AWS Shield Standard.
错误：Amazon GuardDuty是一个威胁检测服务，但它不能直接提供DDoS保护。AWS Shield Standard是默认启用的基础DDoS保
护，但它不提供高级功能。
E. Use AWS Shield Standard with Amazon API Gateway.
错误：AWS Shield Standard是默认启用的基础DDoS保护，但它不提供高级功能。题目要求检测和缓解“大型、复杂的DDoS攻击”，
需要使用AWS Shield Advanced。
3. 我选的答案
B. Use AWS Shield Advanced with the NLB.
C. Use AWS WAF to protect Amazon API Gateway.
4. 官方答案
B. Use AWS Shield Advanced with the NLB.
C. Use AWS WAF to protect Amazon API Gateway.

users around the world will have reliable access to the files. The content is stored in Amazon S3 buckets. The company must
deliver the content quickly, regardless of where the requests originate geographically.
Which solution will meet these requirements?

题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的内容分发与加速功能，特别是Amazon CloudFront的使用。
涉及到如何快速、安全地分发存储在Amazon S3中的内容给全球用户。
2. 选项分析
Correct Answer: C
正确答案：C

A. 使用 AWS DataSync 将 S3 存储桶连接到 Web 应用程序。
错误：AWS DataSync主要用于数据迁移和同步，而不是内容分发或加速。它无法满足全球用户快速访问的需求。
B. 部署 AWS Global Accelerator 将 S3 存储桶连接到 Web 应用程序。
错误：AWS Global Accelerator主要用于加速TCP/UDP流量，并优化网络路径，但它不直接用于内容分发或缓存S3中的文件。
C. 部署 Amazon CloudFront 将 S3 存储桶连接到 CloudFront 边缘服务器。
正确：Amazon CloudFront是一个全球内容分发网络（CDN），可以缓存S3中的内容并通过边缘服务器快速分发给全球用户。
它支持加速访问，同时可以保护内容的安全性。
D. 使用 Amazon Simple Queue Service (Amazon SQS) 将 S3 存储桶连接到 Web 应用程序。
错误：Amazon SQS是一个消息队列服务，用于解耦应用程序组件，而不是用于内容分发或加速。

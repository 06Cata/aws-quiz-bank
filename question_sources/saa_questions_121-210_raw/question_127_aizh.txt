2. 选项分析
A. Amazon EBS for maximum performance, Amazon S3 for durable data storage, and Amazon S3 Glacier for archival
storage
Amazon EBS 提供高性能存储，适合需要高 I/O 的场景，但它的容量限制较小，无法满足 10 TB 的需求。
Amazon S3 是一种高耐用性存储服务，适合存储 300 TB 的媒体内容。
Amazon S3 Glacier 是专门用于归档存储的服务，适合存储 900 TB 的归档数据。
此选项的问题在于 Amazon EBS 的容量限制，无法满足 10 TB 的需求。
B. Amazon EBS for maximum performance, Amazon EFS for durable data storage, and Amazon S3 Glacier for archival
storage
Amazon EBS 提供高性能存储，但容量限制较小，无法满足 10 TB 的需求。
Amazon EFS 是一个文件存储服务，适合共享文件存储，但不如 Amazon S3 在耐用性和成本方面适合存储 300 TB 的媒体内容。
Amazon S3 Glacier 适合归档存储，满足 900 TB 的需求。
此选项的问题在于 Amazon EBS 的容量限制，以及 Amazon EFS 不适合存储大规模媒体内容。

archival storage

储。

Amazon S3 是一种高耐用性存储服务，但它更适合存储媒体内容而非归档数据。
此选项的问题在于 EC2 instance store 的临时性，以及 Amazon S3 不适合归档存储。

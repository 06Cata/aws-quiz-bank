A company is running a publicly accessible serverless application that uses Amazon API Gateway and AWS Lambda. The
application’s traffic recently spiked due to fraudulent requests from botnets.
Which steps should a solutions architect take to block requests from unauthorized users? (Choose two.)
一家公司正在运行一个公开可访问的无服务器应用程序，该应用程序使用 Amazon API Gateway 和 AWS Lambda。最近，由于来自
僵尸网络的欺诈性请求，应用程序的流量激增。
解决方案架构师应该采取哪些步骤来阻止未经授权用户的请求？（选择两个。）
A. Create a usage plan with an API key that is shared with genuine users only. Most Voted
创建一个使用计划，并仅与真实用户共享 API 密钥。
B. Integrate logic within the Lambda function to ignore the requests from fraudulent IP addresses.
在 Lambda 函数中集成逻辑，以忽略来自欺诈性 IP 地址的请求。
C. Implement an AWS WAF rule to target malicious requests and trigger actions to filter them out. Most Voted
实施一个 AWS WAF 规则，以针对恶意请求并触发操作来过滤它们。
D. Convert the existing public API to a private API. Update the DNS records to redirect users to the new API endpoint.
将现有的公共 API 转换为私有 API。更新 DNS 记录以将用户重定向到新的 API 端点。
E. Create an IAM role for each user attempting to access the API. A user will assume the role when making the API call.
为尝试访问 API 的每个用户创建一个 IAM 角色。用户在进行 API 调用时将假设该角色。

2. 选项分析
A. Create a usage plan with an API key that is shared with genuine users only.
正确：使用API密钥和使用计划可以限制访问并确保只有授权用户能够调用API。这是一个有效的安全措施。
B. Integrate logic within the Lambda function to ignore the requests from fraudulent IP addresses.
错误：在Lambda函数中处理恶意流量会增加函数的复杂性和运行成本。更好的方法是使用AWS WAF在API Gateway层过滤恶意流
量。
C. Implement an AWS WAF rule to target malicious requests and trigger actions to filter them out.
正确：AWS WAF可以有效地过滤恶意请求，防止它们到达API Gateway和Lambda函数。这是处理恶意流量的推荐方法。
D. Convert the existing public API to a private API. Update the DNS records to redirect users to the new API endpoint.
错误：将公共API转换为私有API会限制所有用户的访问，而不仅仅是恶意用户。这不符合题目要求。
E. Create an IAM role for each user attempting to access the API. A user will assume the role when making the API call.
错误：为每个用户创建IAM角色并让用户假设角色的方式不适合大规模的API访问场景，且无法有效阻止恶意流量。

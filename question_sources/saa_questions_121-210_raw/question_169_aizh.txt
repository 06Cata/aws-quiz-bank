题目分析与解答
1. 考察的知识点
本题考察的是AWS的安全服务，特别是如何保护应用程序免受DDoS攻击。
涉及的服务包括AWS Shield、Amazon Inspector、Amazon Macie和Amazon GuardDuty。
Correct Answer: C
正确答案：C

2. 选项分析
A. Add an Amazon Inspector agent to the ALB.
Amazon Inspector是一个自动化的安全评估服务，用于检查应用程序的漏洞和错误配置。
它不适用于防止DDoS攻击，也不能直接与ALB集成。
错误原因：此选项与DDoS防护无关。
B. Configure Amazon Macie to prevent attacks.
Amazon Macie是一个数据安全服务，用于识别和保护敏感数据。
它不适用于防止DDoS攻击，也不能直接保护ALB。
错误原因：此选项与DDoS防护无关。
C. Enable AWS Shield Advanced to prevent attacks.
AWS Shield Advanced是一个专门设计用于保护应用程序免受DDoS攻击的服务。
它提供高级的DDoS防护功能，并与ALB集成以保护应用程序。
正确原因：此选项直接解决了题目要求，提供了DDoS防护。
D. Configure Amazon GuardDuty to monitor the ALB.
Amazon GuardDuty是一个威胁检测服务，用于监控恶意活动和异常行为。
虽然它可以监控ALB，但它不能直接防止DDoS攻击。
错误原因：此选项仅提供监控功能，不能满足题目要求的防护需求。
3. 我的答案
我的选择： C. Enable AWS Shield Advanced to prevent attacks.

官方答案： C. Enable AWS Shield Advanced to prevent attacks.

题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的基本功能和分类，特别是了解AWS提供的数据库服务及其类型（NoSQL vs SQL）。
2. 选项分析
A. Amazon RDS: Amazon RDS 是一个托管的关系型数据库服务，支持 SQL 数据库（如 MySQL、PostgreSQL、MariaDB、Oracle
和 SQL Server）。它不是 NoSQL 数据库服务，因此此选项错误。
B. Amazon Redshift: Amazon Redshift 是一个托管的数据仓库服务，主要用于分析和处理结构化数据。它也不是 NoSQL 数据库服
务，因此此选项错误。
C. Amazon DynamoDB: Amazon DynamoDB 是一个完全托管的 NoSQL 数据库服务，支持键值存储和文档存储，非常适合需要低
延迟和高吞吐量的应用程序。因此此选项正确。
D. Amazon Aurora: Amazon Aurora 是一个关系型数据库服务，兼容 MySQL 和 PostgreSQL。它不是 NoSQL 数据库服务，因此此
选项错误。

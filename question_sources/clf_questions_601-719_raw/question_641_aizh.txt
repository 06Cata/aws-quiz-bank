题目分析与解答
1. 考察的知识点
本题考察的是AWS成本管理工具的功能，包括历史支出可视化和未来成本预测。
2. 选项分析
A. AWS Cost and Usage Report:
此服务提供详细的成本和使用数据报告，适合进行深度分析，但它不提供可视化功能或未来成本预测，因此此选项错误。
B. AWS Budgets:
AWS Budgets允许用户设置预算并监控成本，但它主要用于预算管理和警报通知，而不是用于历史支出可视化或未来成本预测，因
此此选项错误。
C. Cost Explorer:
Cost Explorer是AWS提供的工具，用于可视化历史支出模式并预测未来成本。它完全符合题目要求，因此此选项正确。
D. Amazon CloudWatch:
CloudWatch主要用于监控AWS资源的性能和运行状况，与成本管理无关，因此此选项错误。
Correct Answer: C
正确答案：C

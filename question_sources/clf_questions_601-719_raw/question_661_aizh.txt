A company wants to transfer a virtual Windows Server 2022 that is currently running in its own data center to AWS. The company
wants to automatically convert the existing server to run directly on AWS infrastructure instead of visualized hardware.
Which AWS service will meet these requirements?
一家公司希望将其当前在自有数据中心运行的虚拟 Windows Server 2022 转移到 AWS。该公司希望自动将现有服务器转换为直接在
AWS 基础设施上运行，而不是在虚拟化硬件上运行。
哪个 AWS 服务可以满足这些需求？
A. AWS DataSync
AWS DataSync
B. AWS Database Migration Service (AWS DMS)
AWS Database Migration Service (AWS DMS)
C. AWS Application Discovery Service
AWS Application Discovery Service
D. AWS Application Migration Service Most Voted
AWS Application Migration Service
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的功能和用途，特别是与迁移现有服务器到AWS相关的服务。
重点是理解如何将现有的虚拟化服务器迁移到AWS并自动转换为运行在AWS基础设施上的实例。
2. 选项分析
Correct Answer: D
正确答案：D

D (100%)
A. AWS DataSync
错误。AWS DataSync主要用于数据迁移，帮助用户将文件数据从本地存储或其他云存储迁移到AWS存储服务（如Amazon S3、
Amazon EFS）。它不负责迁移虚拟服务器或转换服务器的运行环境。
B. AWS Database Migration Service (AWS DMS)
错误。AWS DMS专注于数据库迁移，帮助用户将数据库从本地或其他云迁移到AWS上的数据库服务（如Amazon RDS）。它不适用
于迁移虚拟服务器。
C. AWS Application Discovery Service
错误。AWS Application Discovery Service用于发现和分析现有的应用程序和服务器环境，以便为迁移到AWS做规划。它不负责实际
的迁移或转换过程。
D. AWS Application Migration Service
正确。AWS Application Migration Service是专门设计用于迁移现有服务器到AWS的服务。它可以自动将本地或虚拟化的服务器转换
为运行在AWS基础设施上的实例，满足题目要求。

D. AWS Application Migration Service

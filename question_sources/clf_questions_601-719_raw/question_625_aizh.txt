A company needs a managed NFS file system that the company can use with its AWS compute resources.
Which AWS service or feature will meet these requirements?
一家公司需要一个托管的 NFS 文件系统，该公司可以与其 AWS 计算资源一起使用。
哪个 AWS 服务或功能可以满足这些需求？
A. Amazon Elastic Block Store (Amazon EBS)
Amazon Elastic Block Store (Amazon EBS)
B. AWS Storage Gateway Tape Gateway
AWS Storage Gateway Tape Gateway
C. Amazon S3 Glacier Flexible Retrieval
Amazon S3 Glacier Flexible Retrieval
D. Amazon Elastic File System (Amazon EFS)
Amazon Elastic File System (Amazon EFS)
题目分析与解答
1. 考察的知识点
本题考察的是AWS存储服务的特点和适用场景，特别是文件存储服务（NFS）的选择。
2. 选项分析
A. Amazon Elastic Block Store (Amazon EBS):
Amazon EBS 是块存储服务，主要用于与单个 EC2 实例配合使用，不支持 NFS 文件系统。此选项错误。
B. AWS Storage Gateway Tape Gateway:
Tape Gateway 是一种用于备份和归档的虚拟磁带库解决方案，不适用于 NFS 文件系统。此选项错误。
Correct Answer: D
正确答案：D
C. Amazon S3 Glacier Flexible Retrieval:
Amazon S3 Glacier 是一种低成本的归档存储服务，主要用于长期数据存储，不支持 NFS 文件系统。此选项错误。
D. Amazon Elastic File System (Amazon EFS):
Amazon EFS 是一个完全托管的文件存储服务，支持 NFS 文件系统，并且可以与多个 AWS 计算资源集成。此选项正确。
3. 我的答案
D. Amazon Elastic File System (Amazon EFS)
4. 官方答案
D. Amazon Elastic File System (Amazon EFS)
5. 与官方答案比较
我的答案与官方答案一致，没有差异。

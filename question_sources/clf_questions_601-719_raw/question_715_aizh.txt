题目分析与解答
1. 考察的知识点
本题考察的是如何在AWS中实现服务间的私有连接，避免将流量暴露到公共互联网。
2. 选项分析
Correct Answer: B
正确答案：B
Community vote distribution
社区投票分布
B (100%)
A. Amazon Inspector: Amazon Inspector是一项安全评估服务，用于检查应用程序的安全漏洞和合规性问题。它与连接VPC和服务
的需求无关，因此此选项错误。
B. AWS PrivateLink: AWS PrivateLink允许您通过专用网络连接AWS服务和VPC，避免流量暴露到公共互联网。这正是题目要求的解
决方案，因此此选项正确。
C. Amazon Connect: Amazon Connect是一项用于构建云联络中心的服务，与连接VPC和服务的需求无关，因此此选项错误。
D. AWS Internet Gateway: AWS Internet Gateway用于将VPC连接到公共互联网，而题目明确要求避免流量暴露到公共互联网，因
此此选项错误。
3. 我的答案
我的答案是：B. AWS PrivateLink
4. 官方答案
官方答案是：B. AWS PrivateLink

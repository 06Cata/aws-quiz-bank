Selected Answer: D
The correct answer is D. When the company needs to add users to IAM groups.
For the other options:
A and C describe scenarios where roles are preferred
B is incorrect as AWS recommends identity federation instead of IAM users for individuals
E describes a federation scenario where roles would be used

Which AWS service will meet these requirements?

题目分析与解答
1. 考察的知识点
本题考察的是AWS服务在离线环境中的使用场景，尤其是针对有限互联网连接情况下的本地数据处理能力。
2. 选项分析
Correct Answer: B
正确答案：B
Community vote distribution

B (100%)
A. Amazon S3:
Amazon S3 是一个云存储服务，需要稳定的互联网连接才能访问和操作数据。由于题目要求解决有限互联网连接的问题，Amazon
S3 不符合要求。
B. AWS Snowball Edge:
AWS Snowball Edge 是一种数据迁移和边缘计算设备，支持本地数据处理并且可以在没有稳定互联网连接的情况下运行。它完全符
合题目要求。
C. AWS Storage Gateway:
AWS Storage Gateway 是一种混合云存储服务，通常需要稳定的互联网连接来同步本地数据到云端。它不适合在有限互联网连接的
情况下使用。
D. AWS Backup:
AWS Backup 是一种集中化的备份服务，主要用于云环境中的数据备份。它需要稳定的互联网连接来操作，不符合题目要求。

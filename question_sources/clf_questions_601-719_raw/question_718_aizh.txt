A company wants to deploy an application that stores data in a relational database. The company wants database tasks, such as
automated backups and database snapshots, to be managed by AWS.
Which AWS service will meet these requirements?
一家公司希望部署一个应用程序，将数据存储在关系数据库中。该公司希望数据库任务，例如自动备份和数据库快照，由AWS管
理。
哪个AWS服务能够满足这些需求？
A. Amazon DocumentDB
Amazon DocumentDB

Amazon RDS
C. Amazon Elastic Block Store (Amazon EBS)
Amazon Elastic Block Store (Amazon EBS)

Amazon S3
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中与关系型数据库相关的服务，以及自动化管理功能（如备份和快照）。
2. 选项分析
Correct Answer: B
正确答案：B

社区投票分布

A. Amazon DocumentDB: Amazon DocumentDB 是一种文档数据库服务，专门用于处理JSON数据，适合非关系型数据库场景。
题目要求的是关系型数据库，因此此选项错误。
B. Amazon RDS: Amazon RDS 是一种托管的关系型数据库服务，支持自动化管理功能（如备份、快照、监控等）。符合题目要
求，因此此选项正确。
C. Amazon Elastic Block Store (Amazon EBS): Amazon EBS 是一种块存储服务，通常用于为EC2实例提供存储支持。它不具备关
系型数据库的管理功能，因此此选项错误。
D. Amazon S3: Amazon S3 是一种对象存储服务，适合存储非结构化数据（如文件、图片等）。它不支持关系型数据库功能，因此
此选项错误。
3. 我的答案
B. Amazon RDS
4. 官方答案
B. Amazon RDS

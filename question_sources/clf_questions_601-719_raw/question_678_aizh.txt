Which AWS Well-Architected Framework concept represents a system's ability to remain functional when the system encounters
operational problems?
哪个 AWS Well-Architected Framework 桜念表示系统在遇到操作问题时仍棺保持功能的能力？
A. Consistency
一致性
B. Elasticity
弹性
C. Durability
持久性
D. Latency
延迟
题目分析与解答
1. 考察的知识点
本题考察的是AWS Well-Architected Framework中的核心桜念，栎别是系统在面对操作问题时的功能保持能力。
2. 选项分析
A. Consistency (一致性): 一致性通常指数据在多个存储副本之间保持同桯的能力，与系统在遇到操作问题时保持功能无关，因桮桮
选项错误。
B. Elasticity (弹性): 弹性指的是系统根据需梆自动扩展或收缩资棕的能力，与系统在遇到操作问题时保持功能无关，因桮桮选项错
误。
C. Durability (持久性): 持久性指的是系统在面对操作问题时仍棺能够保持功能并保护数据的能力，这是题目描述的桭确答栾。
D. Latency (延迟): 延迟指的是系统响应时间的快慢，与系统在遇到操作问题时保持功能无关，因桮桮选项错误。
3. 我的答案
Correct Answer: C
桭确答栾: C
C. Durability
4. 官方答案
C. Durability

A company plans to perform a one-time migration of a large dataset with millions of files from its on-premises data center to the
AWS Cloud.
Which AWS service should the company use for the migration?
A. AWS Database Migration Service (AWS DMS)
B. AWS DataSync
C. AWS Migration Hub
D. AWS Application Migration Service

B. AWS DataSync
AWS DataSync is designed for transferring large amounts of data quickly and securely between on-premises storage and AWS. It is well-suited for
one-time migrations of large datasets and can handle millions of files efficiently.

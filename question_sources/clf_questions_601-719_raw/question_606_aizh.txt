A food delivery company needs to block users in certain countries from accessing its website.
Which AWS service should the company use to meet this requirement?
一家食品配送公司需要阻止某些国家的用户访问其网站。
该公司应该使用哪项AWS服务来满足这一需求？
A. AWS WAF
AWS WAF

AWS Control Tower

Amazon Fraud Detector

Amazon Pinpoint
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务在安全和访问控制方面的应用，特别是如何使用AWS服务来限制用户访问。
2. 选项分析
A. AWS WAF: 正确。AWS WAF（Web Application Firewall）是一项服务，可以帮助用户保护其网站或应用程序免受常见的网络攻
击。它支持基于地理位置的访问控制规则，因此可以用来阻止特定国家的用户访问网站。
Correct Answer: A
正确答案：A
Community vote distribution
社区投票分布
A (100%)
B. AWS Control Tower: 错误。AWS Control Tower主要用于设置和管理多账户环境的最佳实践，并不提供基于地理位置的访问控制
功能。
C. Amazon Fraud Detector: 错误。Amazon Fraud Detector用于检测和防止在线欺诈行为，但它不适用于基于地理位置的访问控
制。
D. Amazon Pinpoint: 错误。Amazon Pinpoint主要用于用户参与和营销活动，例如发送消息和分析用户行为，无法实现地理位置的
访问限制。
3. 我的答案
A. AWS WAF
4. 官方答案
A. AWS WAF

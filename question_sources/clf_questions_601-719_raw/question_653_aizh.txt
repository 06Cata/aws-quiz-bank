A company wants to use an AWS networking solution that can act as a centralized gateway between multiple VPCs and on-
premises networks.
Which AWS service or feature will meet this requirement?
一家公司希望使用一种AWS网络解决方案，该解决方案可以作为多个VPC和本地网络之间的集中式网关。
哪项AWS服务或功能可以满足这一需求？
A. Gateway VPC endpoint
Gateway VPC endpoint
B. AWS Direct Connect
AWS Direct Connect
C. AWS Transit Gateway
AWS Transit Gateway
D. AWS PrivateLink
AWS PrivateLink
题目分析与解答
1. 考察的知识点
本题考察的是AWS网络服务的功能和用途，特别是如何实现跨多个VPC和本地网络的集中式连接。
2. 选项分析
Correct Answer: C
正确答案：C
Community vote distribution
社区投票分布
C (100%)
A. Gateway VPC endpoint:
Gateway VPC Endpoint用于在VPC中与AWS服务（如S3和DynamoDB）进行私有连接。它不能充当多个VPC和本地网络之间的集中
式网关，因此此选项错误。
B. AWS Direct Connect:
AWS Direct Connect是一种专用网络连接服务，用于连接本地数据中心与AWS。虽然它可以连接本地网络，但它不能充当多个VPC
之间的集中式网关，因此此选项错误。
C. AWS Transit Gateway:
AWS Transit Gateway是一种集中式网关服务，可以连接多个VPC和本地网络。它支持跨区域和跨账户的网络连接，是满足题目需求
的正确选项。
D. AWS PrivateLink:
AWS PrivateLink用于在VPC之间或VPC与本地网络之间建立私有连接，但它的主要用途是访问特定服务，而不是充当集中式网关。
因此此选项错误。
3. 我的答案
C. AWS Transit Gateway
4. 官方答案
C. AWS Transit Gateway

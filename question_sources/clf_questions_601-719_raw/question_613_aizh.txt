A company plans to perform a one-time migration of a large dataset with millions of files from its on-premises data center to the
AWS Cloud.
Which AWS service should the company use for the migration?
一家公司计划将其本地数据中心中的一个包含数百万文件的大型数据集进行一次性迁移到AWS云。
该公司应该使用哪个AWS服务进行迁移？
A. AWS Database Migration Service (AWS DMS)
AWS Database Migration Service (AWS DMS)
B. AWS DataSync
AWS DataSync
C. AWS Migration Hub
AWS Migration Hub
D. AWS Application Migration Service
AWS Application Migration Service
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的功能和适用场景，特别是数据迁移相关服务的选择。
2. 选项分析
A. AWS Database Migration Service (AWS DMS)
Correct Answer: B
正确答案：B
Community vote distribution
社区投票分布
B (100%)
错误：AWS DMS主要用于迁移数据库，而不是用于迁移文件或大规模数据集。题目中提到的是迁移“文件”，因此不适用。
B. AWS DataSync
正确：AWS DataSync是一项专门设计用于快速、安全地迁移大规模文件数据的服务。它支持从本地数据中心到AWS存储服务（如
Amazon S3或Amazon EFS）的数据迁移，完全符合题目要求。
C. AWS Migration Hub
错误：AWS Migration Hub是一个集中式工具，用于跟踪和管理迁移项目的进度，但它本身并不执行数据迁移任务。
D. AWS Application Migration Service
错误：AWS Application Migration Service主要用于迁移应用程序和服务器，而不是用于迁移文件数据。
3. 我的答案
我的答案是：B. AWS DataSync
4. 官方答案
官方答案是：B. AWS DataSync
5. 答案比较与思考
我的答案与官方答案一致，无需重新思考。

题目分析与解答
1. 考察的知识点
本题考察的是 Amazon EC2 定价模型的使用场景，包括 On-Demand Instances 的成本效益和适用场景。
2. 选项分析
A. Compute-intensive video transcoding that can be restarted if necessary
- 错误：对于可以中断的计算任务，使用 Spot Instances 更为经济，因为 Spot Instances价格较低且适合非关键任务。On-Demand
Instances不适合这种场景。
B. An instance in continual use for 1 month to conduct quality assurance tests
- 正确：On-Demand Instances最适合短期使用场景，例如持续使用一个月的质量保证测试。它提供灵活性，无需长期承诺。
Correct Answer: B

C. An instance that runs a web server that will run for 1 year
- 错误：对于长期运行的实例（例如1年），Reserved Instances或Savings Plans更具成本效益，因为它们提供折扣价格。
D. An instance that runs a database that will run for 3 years
- 错误：对于长期运行的数据库实例（例如3年），Reserved Instances或Savings Plans是更经济的选择，因为它们提供显著的成本节
约。

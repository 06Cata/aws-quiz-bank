A company wants to deploy a web application as a containerized application. The company wants to use a managed service that
can automatically create container images from source code and deploy the containerized application.
Which AWS service will meet these requirements?
一家公司希望将一个网络应用程序部署为容器化应用程序。该公司希望使用一个托管服务，该服务可以从源代码自动创建容器镜像
并部署容器化应用程序。
哪个AWS服务可以满足这些需求？
A. AWS Elastic Beanstalk
AWS Elastic Beanstalk
B. Amazon Elastic Container Service (Amazon ECS)
Amazon Elastic Container Service (Amazon ECS)
C. AWS App Runner
AWS App Runner
D. Amazon EC2
Amazon EC2
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中与容器化应用部署相关的知识，特别是自动化构建和部署的能力。
2. 选项分析
Correct Answer: C
正确答案：C

A. AWS Elastic Beanstalk:
Elastic Beanstalk是一个平台即服务（PaaS），支持部署和管理应用程序，但它并不专门针对容器化应用，也无法自动从源代码创建
容器镜像。因此此选项错误。
B. Amazon Elastic Container Service (Amazon ECS):
Amazon ECS是一个容器编排服务，可以运行和管理容器化应用，但它需要用户手动创建容器镜像并定义任务配置，无法自动从源
代码创建容器镜像。因此此选项错误。
C. AWS App Runner:
AWS App Runner是一个完全托管的服务，专门设计用于从源代码或容器镜像自动构建和部署容器化应用。它符合题目要求，因此
此选项正确。
D. Amazon EC2:
Amazon EC2是一个基础设施即服务（IaaS），提供虚拟服务器，但它不具备自动创建容器镜像或部署容器化应用的功能。因此此选
项错误。
3. 我的答案
C. AWS App Runner
4. 官方答案
C. AWS App Runner

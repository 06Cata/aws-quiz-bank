题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中用于记录和审计用户活动的功能，特别是追踪资源的创建、修改和删除操作。
2. 选项分析
A. AWS CloudTrail: 正确。AWS CloudTrail 是一个服务，用于记录用户和服务的 API 调用历史。它可以帮助管理员查看谁进行了资
源的创建、修改或删除操作。
Correct Answer: A
正确答案：A
Community vote distribution
社区投票分布
A (100%)
B. Amazon Inspector: 错误。Amazon Inspector 是一个自动化的安全评估服务，用于检查应用程序的安全漏洞和合规性问题，而不
是用于追踪资源删除的操作。
C. Amazon GuardDuty: 错误。Amazon GuardDuty 是一个威胁检测服务，用于识别潜在的恶意活动或未经授权的行为，但它不记
录资源的删除操作。
D. AWS Trusted Advisor: 错误。AWS Trusted Advisor 是一个优化建议服务，用于提供成本优化、安全性、性能和容错性建议，但
它不记录用户活动或资源删除操作。
3. 我的答案
A. AWS CloudTrail

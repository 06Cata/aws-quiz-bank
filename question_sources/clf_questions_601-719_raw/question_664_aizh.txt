A company wants to migrate its on-premises SQL Server database to the AWS Cloud. The company wants AWS to handle the day-
to-day administration of the database.
Which AWS service will meet the company's requirements?
一家公司希望将其本地的 SQL Server 数据库迁移到 AWS 云。该公司希望 AWS 负责数据库的日常管理。
哪项 AWS 服务能够满足该公司的需求？
A. Amazon EC2 for Microsoft SQL Server
Amazon EC2 for Microsoft SQL Server

C. Amazon RDS
Amazon RDS

Amazon Aurora
题目分析与解答
1. 考察的知识点
本题考察的是AWS托管数据库服务的选择，尤其是针对SQL Server数据库的迁移和管理。
重点在于理解AWS服务的功能以及哪些服务提供托管和日常管理功能。
2. 选项分析
Correct Answer: C
正确答案：C

A. Amazon EC2 for Microsoft SQL Server
错误。虽然可以在Amazon EC2上运行Microsoft SQL Server，但EC2只是一个虚拟服务器，用户需要自己管理操作系统、数据库软件
的安装、配置、备份和维护等工作。它不符合“让AWS处理日常管理”的要求。
B. Amazon DynamoDB
错误。Amazon DynamoDB是一个NoSQL数据库服务，适用于键值和文档存储，而不是关系型数据库（如SQL Server）。它不支持
SQL Server数据库的迁移。
C. Amazon RDS
正确。Amazon RDS（关系型数据库服务）支持Microsoft SQL Server，并提供托管服务，包括自动备份、软件更新、监控和维护。
这完全符合“让AWS处理日常管理”的要求。
D. Amazon Aurora
错误。Amazon Aurora是一个高性能的关系型数据库服务，但它与SQL Server不兼容。Aurora支持MySQL和PostgreSQL，而不是
SQL Server。
3. 我的答案
我的答案是：C. Amazon RDS
4. 官方答案
官方答案是：C. Amazon RDS

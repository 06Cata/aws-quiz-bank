A company migrated its systems to the AWS Cloud. The systems are rightsized, and a security review did not reveal any issues. The
company must ensure that additional developments, integrations, changes, and system usage growth do not jeopardize this
optimized AWS infrastructure.
Which AWS service should the company use to report ongoing optimization and security?

变更以及系统使用增长不会危及这一优化的AWS基础设施。

题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中与优化和安全相关的工具和服务，特别是如何持续监控和报告优化和安全状态。
2. 选项分析
Correct Answer: A
正确答案：A
Community vote distribution
社区投票分布
A (100%)
A. AWS Trusted Advisor: 正确。AWS Trusted Advisor 是一个在线工具，提供实时建议以帮助用户优化 AWS 环境的成本、性能、
安全性和容错性。它可以报告优化和安全相关的建议，因此符合题目要求。
B. AWS Health Dashboard: 错误。AWS Health Dashboard 提供与 AWS 服务相关的健康信息，但它主要用于监控服务中断或事
件，而不是持续优化和安全建议。
C. Amazon Connect: 错误。Amazon Connect 是一个云联络中心解决方案，与优化和安全无关。
D. AWS Systems Manager: 错误。AWS Systems Manager 是一个管理工具，用于操作和管理 AWS 资源，但它不是专门用于报告优
化和安全状态的服务。

A. AWS Trusted Advisor

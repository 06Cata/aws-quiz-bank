A company has multiple SQL-based databases located in a data center. The company needs to migrate all database servers to the
AWS Cloud to reduce the cost of operating physical servers.
Which AWS service or resource will meet these requirements with the LEAST operational overhead?
一家公司在数据中心拥有多个基于SQL的数据库。该公司需要将所有数据库服务器迁移到AWS云，以降低运营物理服务器的成本。
哪个AWS服务或资源能够以最少的运营开销满足这些需求？
A. Amazon EC2 instances
Amazon EC2 instances
B. Amazon RDS
Amazon RDS
C. Amazon DynamoDB
Amazon DynamoDB
D. OpenSearch
OpenSearch
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的选择，尤其是针对SQL数据库迁移到云端的场景，重点关注服务的操作简便性和成本优化。
2. 选项分析
A. Amazon EC2 instances:
Amazon EC2可以用来运行SQL数据库，但需要手动安装、配置和管理数据库软件。这会增加操作复杂性和管理开销，因此不符
合“最少操作开销”的要求。
Correct Answer: B
正确答案：B
B. Amazon RDS:
Amazon RDS是一种托管数据库服务，支持多种SQL数据库（如 MySQL、PostgreSQL、SQL Server）。它自动处理数据库的管理任
务（如备份、补丁管理、监控等），显著减少操作开销，是最佳选择。
C. Amazon DynamoDB:
Amazon DynamoDB是一个NoSQL数据库服务，不适用于SQL数据库的迁移场景。因此不符合题目要求。
D. OpenSearch:
OpenSearch主要用于搜索和分析数据，而不是运行SQL数据库。因此不适合用于迁移SQL数据库。
3. 我的答案
B. Amazon RDS
4. 官方答案
B. Amazon RDS
5. 与官方答案比较
我的答案与官方答案一致，没有差异。

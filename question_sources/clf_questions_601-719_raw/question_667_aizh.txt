题目分析与解答
1. 考察的知识点
本题考察的是AWS工具和服务的功能，特别是与成本估算和迁移相关的服务。
重点关注AWS Pricing Calculator和Migration Evaluator的用途，以及其他选项的功能是否与题目需求匹配。
Correct Answer: AB
正确答案：AB
Community vote distribution
社区投票分布
AB (100%)
2. 选项分析
A. AWS Pricing Calculator
正确：AWS Pricing Calculator是一个工具，用于估算在AWS上运行服务的成本。它可以帮助公司计算总拥有成本（TCO），符合题
目需求。
B. Migration Evaluator
正确：Migration Evaluator（以前称为TSO Logic）是一个工具，用于分析现有的服务器环境并提供迁移到AWS的成本估算。这直接
满足题目中关于迁移和TCO的需求。
C. AWS Support Center
错误：AWS Support Center是一个支持门户，用于管理支持请求和查看账户信息。它不提供成本估算或迁移分析功能。
D. AWS Application Discovery Service
错误：AWS Application Discovery Service用于收集服务器使用情况和性能数据，以帮助规划迁移，但它不直接提供TCO计算功能。
E. AWS Database Migration Service (AWS DMS)
错误：AWS DMS是一个迁移工具，用于迁移数据库到AWS。虽然它与迁移相关，但它不涉及成本估算或TCO分析。
3. 我的答案
A. AWS Pricing Calculator
B. Migration Evaluator
4. 官方答案
A. AWS Pricing Calculator
B. Migration Evaluator

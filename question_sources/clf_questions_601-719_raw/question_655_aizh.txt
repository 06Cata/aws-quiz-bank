To assist companies with Payment Card Industry Data Security Standard (PCI DSS) compliance in the cloud, AWS provides:
为了协助公司在云中遵守支付卡行业数据安全标准 (PCI DSS)，AWS 提供：
A. physical Inspections of data centers by appointment.
通过预约进行数据中心的实地检查。
B. required PCI compliance certifications for any application running on AWS.
任何运行在 AWS 上的应用程序所需的 PCI 合规认证。
C. an AWS Attestation of Compliance (AOC) report for specific AWS services.
针对特定 AWS 服务的 AWS Attestation of Compliance (AOC) 报告。
D. professional PCI compliance services.
专业的 PCI 合规服务。
题目分析与解答
1. 考察的知识点
本题考察的是AWS在帮助客户实现PCI DSS合规方面的支持服务和文档。
重点关注AWS提供的合规性报告和支持服务。
2. 选项分析
A. physical Inspections of data centers by appointment. 通过预约进行数据中心的实地检查。
错误：AWS不允许客户进行数据中心的实地检查。AWS的数据中心安全是由AWS负责管理，并通过第三方审计来验证其合规性。
B. required PCI compliance certifications for any application running on AWS. 任何运行在 AWS 上的应用程序所需的 PCI 合
规认证。
错误：AWS提供基础设施的合规性认证，但应用程序的PCI DSS合规性是客户的责任。AWS不会直接为客户的应用程序提供PCI认
证。
C. an AWS Attestation of Compliance (AOC) report for specific AWS services. 针对特定 AWS 服务的 AWS Attestation of
Compliance (AOC) 报告。
Correct Answer: C
正确答案：C
正确：AWS提供AOC报告，证明其特定服务符合PCI DSS要求。这些报告可以帮助客户评估AWS服务的合规性。
D. professional PCI compliance services. 专业的 PCI 合规服务。
错误：AWS不直接提供专业的PCI合规服务。客户可以使用AWS的合规性工具和文档来帮助实现PCI DSS合规，但具体的合规服务通
常由第三方提供。
3. 我的答案
C. an AWS Attestation of Compliance (AOC) report for specific AWS services.
4. 官方答案
C. an AWS Attestation of Compliance (AOC) report for specific AWS services.
5. 答案比较与分析
我的答案与官方答案一致，没有差异。
理由：AWS确实提供AOC报告来证明其服务的PCI DSS合规性，这是客户实现合规的重要支持文档。

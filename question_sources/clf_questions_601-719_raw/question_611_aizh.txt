题目分析与解答
1. 考察的知识点
本题考察的是AWS Artifact服务的功能，尤其是其在合规性和审计报告（如SOC报告）获取方面的作用。
2. 选项分析
A. AWS Artifact: 正确。AWS Artifact是一个自助服务工具，专门用于访问AWS的合规性报告和审计文档，包括SOC报告。它是合规
性相关任务的首选服务。
Correct Answer: A
正确答案：A
Community vote distribution
社区投票分布
A (100%)
B. AWS Concierge Support: 错误。AWS Concierge Support是专门为企业级客户提供的支持服务，主要帮助客户解决账单和账户
相关问题，与合规性报告无关。
C. AWS Support: 错误。AWS Support提供技术支持和指导，但它不是用于访问合规性报告的工具。
D. AWS Trusted Advisor: 错误。AWS Trusted Advisor是一个优化工具，用于检查AWS环境的成本、性能、安全性等方面，与合规
性报告无关。
3. 我的答案
A. AWS Artifact
4. 官方答案
A. AWS Artifact

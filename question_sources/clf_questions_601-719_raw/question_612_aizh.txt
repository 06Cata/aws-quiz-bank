A company has a compliance requirement to record and evaluate configuration changes, as well as perform remediation actions
on AWS resources.
Which AWS service should the company use?
一家公司有合规要求，需要记录和评估配置更改，并对AWS资源执行修复操作。
公司应该使用哪项AWS服务？
A. AWS Config
AWS Config
B. AWS Secrets Manager
AWS Secrets Manager
C. AWS CloudTrail
AWS CloudTrail
D. AWS Trusted Advisor
AWS Trusted Advisor
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的功能和用途，特别是与资源配置管理、合规性评估以及自动化修复相关的服务。
2. 选项分析
Correct Answer: A
正确答案：A
Community vote distribution
社区投票分布
A (100%)
A. AWS Config: 正确。AWS Config 是专门用于记录和评估 AWS 资源的配置变化的服务。它可以帮助用户检查资源是否符合合规性
要求，并支持自动化的修复操作。
B. AWS Secrets Manager: 错误。AWS Secrets Manager 是用于管理和保护敏感信息（如数据库凭证、API 密钥）的服务，与资源
配置管理和合规性评估无关。
C. AWS CloudTrail: 错误。AWS CloudTrail 主要用于记录用户和服务的 API 调用历史，帮助进行审计和监控，但它不提供资源配置
评估或自动修复功能。
D. AWS Trusted Advisor: 错误。AWS Trusted Advisor 是一个优化建议工具，帮助用户检查资源的成本、性能、安全性等，但它不
记录配置变化或执行自动修复。
3. 我的答案
A. AWS Config
4. 官方答案
A. AWS Config

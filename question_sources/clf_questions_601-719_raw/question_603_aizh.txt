What is the primary use case for Amazon GuardDuty?
Amazon GuardDuty 的主要使用场景是什么？
A. Prevention of DDoS attacks
防止 DDoS 攻击
B. Protection against SQL injection attacks
防护 SQL 注入攻击
C. Automatic monitoring for threats to AWS workloads
自动监控 AWS 工作负载的威胁
D. Automatic provisioning of AWS resources
自动配置 AWS 资源
题目分析与解答
1. 考察的知识点
本题考察的是对Amazon GuardDuty服务的理解，尤其是其主要用途和功能。
2. 选项分析
A. Prevention of DDoS attacks 防止 DDoS 攻击
错误。Amazon GuardDuty的主要功能是检测和监控威胁，而不是直接防止DDoS攻击。防止DDoS攻击通常由AWS Shield等服务提
供。
B. Protection against SQL injection attacks 防护 SQL 注入攻击
错误。SQL注入攻击的防护通常由AWS WAF（Web Application Firewall）等服务提供，而不是GuardDuty。
Correct Answer: C
正确答案：C
Community vote distribution
社区投票分布
C (100%)
C. Automatic monitoring for threats to AWS workloads 自动监控 AWS 工作负载的威胁
正确。Amazon GuardDuty的主要用途是自动监控AWS账户和工作负载的潜在威胁，包括恶意活动和异常行为。
D. Automatic provisioning of AWS resources 自动配置 AWS 资源
错误。自动配置AWS资源通常由AWS CloudFormation或AWS Elastic Beanstalk等服务提供，而不是GuardDuty。
3. 我的答案
C. Automatic monitoring for threats to AWS workloads 自动监控 AWS 工作负载的威胁
4. 官方答案
C. Automatic monitoring for threats to AWS workloads 自动监控 AWS 工作负载的威胁

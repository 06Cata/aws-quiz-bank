A company wants to run its application on Amazon EC2 instances. The company needs to keep the application on-premises to
meet a compliance requirement.
Which AWS offering will meet these requirements?
一家公司希望在 Amazon EC2 实例上运行其应用程序。该公司需要将应用程序保留在本地以满足合规性要求。
哪项 AWS 服务能够满足这些要求？
A. Dedicated Instances
Dedicated Instances
B. Amazon CloudFront
Amazon CloudFront
C. AWS Fargate
AWS Fargate
D. AWS Outposts
AWS Outposts
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的使用场景，特别是AWS Outposts的功能和适用场景。
重点在于理解如何满足“保持应用程序在本地部署以满足合规性要求”的需求。
2. 选项分析
Correct Answer: D
正确答案：D
Community vote distribution
社区投票分布
D (100%)
A. Dedicated Instances:
Dedicated Instances是运行在单租户硬件上的Amazon EC2实例，提供物理隔离以满足某些合规性需求。然而，它们仍然运行在AWS
云中，而不是本地环境，因此无法满足“保持应用程序在本地部署”的要求。此选项错误。
B. Amazon CloudFront:
Amazon CloudFront是一个内容分发网络（CDN）服务，用于加速交付网站、视频等内容。它与本地部署无关，也无法满足合规性
要求。此选项错误。
C. AWS Fargate:
AWS Fargate是一个无服务器计算引擎，用于运行容器化应用程序。它完全托管在AWS云中，无法满足“保持应用程序在本地部署”的
要求。此选项错误。
D. AWS Outposts:
AWS Outposts是一种混合云解决方案，允许客户在本地环境中运行AWS服务，同时保持与AWS云的集成。它专门设计用于满足需要
本地部署的合规性要求。此选项正确。

我的答案是：D. AWS Outposts

官方答案是：D. AWS Outposts

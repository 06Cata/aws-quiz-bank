A company needs to run some of its workload in the AWS Cloud. The company needs to keep some of the workload in its own on-
site data center due to compliance reasons.
Which AWS service will meet these requirements?
一家公司需要在AWS云中运行部分工作负载。由于合规性原因，公司需要将部分工作负载保留在自己的现场数据中心。
哪个AWS服务能够满足这些需求？
A. AWSConfig
AWSConfig
B. AWS Outposts
AWS Outposts
C. Amazon Lightsail
Amazon Lightsail
D. Amazon Connect
Amazon Connect
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的功能和适用场景，特别是混合云解决方案。
2. 选项分析
A. AWS Config: AWS Config是一项用于记录和审计AWS资源配置的服务，主要用于合规性检查和资源变更管理。它不提供混合云
解决方案，因此此选项错误。
B. AWS Outposts: AWS Outposts是一项混合云解决方案，允许用户在自己的数据中心运行AWS服务，同时与AWS云保持一致的体
验。它完全符合题目要求，因此此选项正确。
Correct Answer: B
正确答案：B
C. Amazon Lightsail: Amazon Lightsail是一项简化的虚拟服务器解决方案，适用于小型应用程序和开发环境。它不支持混合云场
景，因此此选项错误。
D. Amazon Connect: Amazon Connect是一项云联络中心服务，主要用于客户服务场景，与混合云无关，因此此选项错误。
3. 我的选择
我选择的答案是：B. AWS Outposts
4. 官方答案
官方正确答案是：B. AWS Outposts
5. 与官方答案比较
我的答案与官方答案一致，无需重新思考。

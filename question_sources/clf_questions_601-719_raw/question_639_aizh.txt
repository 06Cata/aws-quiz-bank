A company wants a cost-effective option when running its applications in an Amazon EC2 instance for short time periods. The
applications can be interrupted.
Which EC2 instance type will meet these requirements?
一家公司希望在短时间内运行其应用程序时选择一种具有成本效益的选项。这些应用程序可以被中断。
哪种 EC2 实例类型可以满足这些需求？
A. Spot Instances
Spot Instances
B. On-Demand Instances
On-Demand Instances
C. Reserved Instances
Reserved Instances
D. Dedicated Instances
Dedicated Instances
题目分析与解答
1. 考察的知识点
本题考察的是AWS EC2实例类型的特点及其适用场景，特别是Spot Instances的成本优势和适用场景。
2. 选项分析
A. Spot Instances: 正确。Spot Instances是EC2实例的一种定价选项，允许用户以显著低于按需价格的成本运行实例。它适用于可
以容忍中断的应用程序，例如批处理任务或数据分析任务。由于题目提到应用可以被中断且需要成本效益，Spot Instances是最佳选
择。
Correct Answer: A
正确答案：A
B. On-Demand Instances: 错误。On-Demand Instances提供按需的计算能力，用户按小时或秒计费，适合需要灵活性且不能容忍
中断的应用程序。虽然它灵活，但成本较高，不符合题目中“成本效益”的要求。
C. Reserved Instances: 错误。Reserved Instances适用于长期运行的应用程序，用户需要提前承诺使用一定的时间（通常为1年或3
年），以换取折扣价格。它不适合短期运行的应用程序。
D. Dedicated Instances: 错误。Dedicated Instances运行在专用硬件上，适用于需要物理隔离的工作负载。它的成本较高，不符合
题目中“成本效益”的要求。
3. 我的答案
A. Spot Instances
4. 官方答案
A. Spot Instances

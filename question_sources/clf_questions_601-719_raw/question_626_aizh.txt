A company plans to migrate to the AWS Cloud. The company wants to gather information about its on-premises data center.
Which AWS service should the company use to meet these requirements?
一家公司计划迁移到AWS云。该公司希望收集有关其本地数据中心的信息。
该公司应该使用哪项AWS服务来满足这些需求？
A. AWS Application Discovery Service
AWS Application Discovery Service
B. AWS DataSync
AWS DataSync
C. AWS Storage Gateway
AWS Storage Gateway
D. AWS Database Migration Service (AWS DMS)
AWS Database Migration Service (AWS DMS)
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的功能和用途，特别是与迁移相关的服务。
需要了解各个服务在数据中心迁移中的具体作用。
2. 选项分析
Correct Answer: A
正确答案：A
Community vote distribution

A (100%)
A. AWS Application Discovery Service
正确。AWS Application Discovery Service专门用于帮助企业收集和分析其本地数据中心的信息。它可以自动发现服务器、应用程序
依赖关系和性能数据，为迁移到AWS提供详细的规划信息。
B. AWS DataSync
错误。AWS DataSync主要用于快速、安全地将数据从本地存储迁移到AWS存储服务（如Amazon S3或Amazon EFS），但它不负责
收集数据中心信息。
C. AWS Storage Gateway
错误。AWS Storage Gateway是一种混合云存储服务，主要用于在本地环境和AWS之间集成存储，但它不具备收集数据中心信息的
功能。
D. AWS Database Migration Service (AWS DMS)
错误。AWS DMS专注于迁移数据库到AWS，但它不负责收集整个数据中心的信息。
3. 我的答案
A. AWS Application Discovery Service
4. 官方答案
A. AWS Application Discovery Service

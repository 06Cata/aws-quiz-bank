A company wants to securely store Amazon RDS database credentials and automatically rotate user passwords periodically.
Which AWS service or capability will meet these requirements?
一家公司希望安全存储 Amazon RDS 数据库凭证，并定期自动轮换用户密码。
哪项 AWS 服务或功能可以满足这些需求？
A. Amazon S3
Amazon S3
B. AWS Systems Manager Parameter Store
AWS Systems Manager Parameter Store
C. AWS Secrets Manager
AWS Secrets Manager
D. AWS CloudTrail
AWS CloudTrail
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中与安全存储和自动密码轮换相关的功能，特别是如何管理敏感信息（如数据库凭证）。
2. 选项分析
A. Amazon S3: Amazon S3是一个对象存储服务，适合存储文件和数据，但它不具备专门的功能来安全存储数据库凭证或自动轮换
密码。因此此选项错误。
B. AWS Systems Manager Parameter Store: Parameter Store可以存储配置数据和密钥，但它不支持自动轮换密码的功能。因此此
选项错误。
Correct Answer: C
正确答案：C
C. AWS Secrets Manager: AWS Secrets Manager专门设计用于安全存储敏感信息（如数据库凭证），并支持自动轮换密码的功
能。这是正确的选项。
D. AWS CloudTrail: AWS CloudTrail用于记录和监控AWS账户中的API调用和活动日志，与存储或轮换密码无关。因此此选项错误。
3. 我的答案
我的答案是：C. AWS Secrets Manager
4. 官方答案
官方答案是：C. AWS Secrets Manager

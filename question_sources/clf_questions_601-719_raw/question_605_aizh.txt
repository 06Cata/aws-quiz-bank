A developer needs to interact with AWS by using the AWS CLI.
Which security feature or AWS service must be provisioned in the developer's account to meet this requirement?
开发人员需要通过使用AWS CLI与AWS进行交互。
为了满足这一要求，开发人员的账户中必须配置哪项安全功能或AWS服务？
A. User name and password
用户名和密码
B. AWS Systems Manager
AWS Systems Manager
C. Root password access
Root密码访问
D. AWS access key
AWS access key
题目分析与解答
1. 考察的知识点
本题考察的是AWS CLI的身份验证机制，以及如何通过安全凭证与AWS服务交互。
2. 选项分析
A. User name and password 用户名和密码
错误。AWS CLI不支持通过用户名和密码直接进行身份验证。用户名和密码通常用于登录AWS管理控制台，而不是CLI。
Correct Answer: D
正确答案：D
Community vote distribution
社区投票分布
D (100%)
B. AWS Systems Manager AWS Systems Manager
错误。AWS Systems Manager是一项服务，用于管理和自动化资源，但它不是用于身份验证的工具。
C. Root password access Root密码访问
错误。使用Root账户进行操作是AWS的最佳实践中明确禁止的，尤其是通过CLI进行操作。Root账户的密码也不能直接用于CLI身份
验证。
D. AWS access key AWS access key
正确。AWS CLI需要使用访问密钥（Access Key ID和Secret Access Key）来进行身份验证。这是与AWS服务交互的标准方式。
3. 我的答案
D. AWS access key
4. 官方答案
D. AWS access key

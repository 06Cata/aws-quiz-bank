A company plans to onboard new employees that will be working remotely. The company needs to set up Windows virtual
desktops to create a working environment for the new employees. The employees must be able access the working environment
from anywhere and by using their computer or a web browser.
Which AWS service or feature will meet these requirements?
一家公司计划为将远程工作的新员工提供入职培训。公司需要设置 Windows 虚拟桌面，为新员工创建工作环境。员工必须能够通
过他们的计算机或网页浏览器从任何地方访问工作环境。
哪个 AWS 服务或功能可以满足这些需求？

题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中与虚拟桌面解决方案相关的知识点，特别是Amazon WorkSpaces的功能和用途。
2. 选项分析
Correct Answer: C
正确答案：C
Community vote distribution
社区投票分布
C (100%)
A. Dedicated Hosts:
Dedicated Hosts是用于提供物理服务器的专用主机，主要用于满足合规性或许可要求。它不适用于创建虚拟桌面环境，因此此选项
错误。
B. AWS Global Accelerator:
AWS Global Accelerator是一个网络服务，用于优化全球用户访问应用程序的性能。它与虚拟桌面环境无关，因此此选项错误。
C. Amazon WorkSpaces:
Amazon WorkSpaces是一个托管的虚拟桌面服务，允许用户从任何地方通过计算机或浏览器访问其工作环境。它完全符合题目要
求，因此此选项正确。
D. Amazon CloudFront:
Amazon CloudFront是一个内容分发网络（CDN）服务，用于加速交付网站、视频等内容。它与虚拟桌面环境无关，因此此选项错
误。

我的答案是：C. Amazon WorkSpaces

A company has an AWS Business Support plan. The company needs to gain access to the AWS DDoS Response Team (DRT) to help
mitigate DDoS events.
Which AWS service or resource must the company use to meet these requirements?
一家公司拥有AWS Business Support计划。该公司需要获得AWS DDoS Response Team (DRT) 的帮助，以缓解DDoS事件。
该公司必须使用哪项AWS服务或资源来满足这些需求？
A. AWS Shield Standard
AWS Shield Standard
B. AWS Enterprise Support
AWS Enterprise Support
C. AWS WAF
AWS WAF
D. AWS Shield Advanced
AWS Shield Advanced
题目分析与解答
1. 考察的知识点
本题考察的是AWS Shield服务的功能，特别是AWS Shield Advanced的特性，包括访问AWS DDoS Response Team (DRT)的能力。
2. 选项分析
A. AWS Shield Standard: AWS Shield Standard是默认启用的免费服务，提供基本的DDoS保护，但不包括访问AWS DDoS
Response Team (DRT)的功能。因此此选项错误。
B. AWS Enterprise Support: AWS Enterprise Support提供高级支持服务，但它与DDoS防护无直接关联，也不提供访问AWS DDoS
Response Team (DRT)的功能。因此此选项错误。
Correct Answer: D
正确答案：D
C. AWS WAF: AWS WAF是一种Web应用程序防火墙，用于保护Web应用免受常见的攻击，但它不提供访问AWS DDoS Response
Team (DRT)的功能。因此此选项错误。
D. AWS Shield Advanced: AWS Shield Advanced是一种付费服务，提供高级DDoS保护，并包括访问AWS DDoS Response Team
(DRT)的功能。因此此选项正确。
3. 我的答案
D. AWS Shield Advanced
4. 官方答案
D. AWS Shield Advanced
5. 与官方答案比较
我的答案与官方答案一致，没有差异。

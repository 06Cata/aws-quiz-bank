Which AWS service will meet these requirements?

题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的功能和用途，特别是与数据保护和备份相关的服务。
2. 选项分析
Correct Answer: B
正确答案：B
Community vote distribution
社区投票分布
B (100%)
A. AWS Artifact: AWS Artifact是一个提供访问AWS合规性报告和协议的服务，主要用于合规性和审计，不涉及数据保护或备份功
能。因此此选项错误。
B. AWS Backup: AWS Backup是一个完全托管的服务，用于集中化和自动化数据保护，支持多种AWS服务和混合工作负载。这正是
题目要求的功能，因此此选项正确。
C. AWS Batch: AWS Batch是一个完全托管的批处理计算服务，用于运行批量计算任务，与数据保护或备份无关。因此此选项错
误。
D. AWS Shield: AWS Shield是一个托管的分布式拒绝服务(DDoS)保护服务，用于保护应用程序免受DDoS攻击，与数据保护或备份
无关。因此此选项错误。
3. 我的答案
B. AWS Backup
4. 官方答案
B. AWS Backup

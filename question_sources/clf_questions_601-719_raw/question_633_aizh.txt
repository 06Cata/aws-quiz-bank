题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的管理工具，特别是用户如何通过图形界面管理AWS服务。
2. 选项分析
A. AWS Copilot: 错误。AWS Copilot 是一个命令行工具，主要用于帮助开发人员快速部署和管理容器化应用程序，而不是提供图形
界面。
B. AWS CLI: 错误。AWS CLI 是命令行界面工具，用户通过命令行与AWS服务交互，而不是图形界面。
C. AWS Management Console: 正确。AWS Management Console 提供了一个基于网页的图形用户界面，用户可以通过它管理和
监控AWS服务。
Correct Answer: C
正确答案：C

D. AWS software development kits (SDKs): 错误。AWS SDKs 是开发工具包，主要用于编程和与AWS服务交互，而不是图形界
面。

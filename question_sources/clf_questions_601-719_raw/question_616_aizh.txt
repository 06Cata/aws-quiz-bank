A company needs to set up dedicated network connectivity between its on-premises data center and the AWS Cloud. The network
cannot use the public internet.
Which AWS service or feature will meet these requirements?
一家公司需要在其本地数据中心和AWS云之间设置专用的网络连接。该网络不能使用公共互联网。
哪个AWS服务或功能可以满足这些需求？
A. AWS Transit Gateway
AWS Transit Gateway
B. AWS VPN
AWS VPN
C. Amazon CloudFront
Amazon CloudFront
D. AWS Direct Connect
AWS Direct Connect
题目分析与解答
1. 考察的知识点
本题考察的是AWS网络服务的选择，特别是如何实现专用网络连接而不使用公共互联网。
2. 选项分析
Correct Answer: D
正确答案：D
Community vote distribution
社区投票分布
D (100%)
A. AWS Transit Gateway:
错误。AWS Transit Gateway主要用于连接多个VPC和本地网络，但它通常需要结合VPN或Direct Connect来实现连接。它本身并不
提供专用网络连接功能。
B. AWS VPN:
错误。AWS VPN通过公共互联网建立加密连接，虽然安全，但不符合题目要求“不能使用公共互联网”。
C. Amazon CloudFront:
错误。Amazon CloudFront是一个内容分发网络（CDN）服务，用于加速内容交付，与专用网络连接无关。
D. AWS Direct Connect:
正确。AWS Direct Connect提供专用网络连接，绕过公共互联网，满足题目要求。
3. 我的答案
D. AWS Direct Connect
4. 官方答案
D. AWS Direct Connect

A company needs to run some of its workloads on premises to comply with regulatory guidelines. The company wants to use the
AWS Cloud to run workloads that are not required to be on premises. The company also wants to be able to use the same API
calls for the on-premises workloads and the cloud workloads.
Which AWS service or feature should the company use to meet these requirements?
一家公司需要在本地运行部分工作负载以符合监管指南。该公司希望使用AWS Cloud来运行不需要在本地运行的工作负载。同时，
该公司还希望能够对本地工作负载和云工作负载使用相同的API调用。
该公司应该使用哪项AWS服务或功能来满足这些需求？

Dedicated Hosts

题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的混合云解决方案，特别是如何在本地和云端运行工作负载，同时使用一致的API调用。
2. 选项分析
Correct Answer: B
正确答案：B
Community vote distribution

B (100%)
A. Dedicated Hosts
错误。Dedicated Hosts是用于在AWS云中提供物理服务器的服务，主要用于满足合规性或许可要求，但它不能用于在本地运行工作
负载，因此不符合题目要求。
B. AWS Outposts
正确。AWS Outposts是一种混合云解决方案，允许用户在本地运行AWS服务，同时与AWS云保持一致的API调用。这完全符合题目
要求。
C. Availability Zones
错误。Availability Zones是AWS云中的数据中心区域，用于高可用性架构设计，但它不能用于在本地运行工作负载，因此不符合题
目要求。
D. AWS Wavelength
错误。AWS Wavelength是用于在边缘网络上运行工作负载的服务，主要用于低延迟应用场景，但它不支持在本地运行工作负载，
因此不符合题目要求。

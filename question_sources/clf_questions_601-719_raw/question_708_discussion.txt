B. AWS Lambda: AWS Lambda is a serverless compute service that charges you only for the compute time used. Since the application is invoked only
a few times each day and has short processing times (less than 5 minutes), Lambda is the most cost-effective option. You don't need to manage any
infrastructure, and you only pay for the time your code is running.

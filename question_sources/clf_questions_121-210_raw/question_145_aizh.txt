Which cloud transformation journey phase of the AWS Cloud Adoption Framework (AWS CAF) focuses on demonstrating how the
cloud helps accelerate business outcomes?
AWS云采用框架（AWS CAF）的哪个云转型旅程阶段专注于展示云如何帮助加速业务成果？
A. Scale
扩展

题目分析与解答
1. 考察的知识点
本题考察的是AWS Cloud Adoption Framework (AWS CAF)的云转型旅程阶段，特别是各阶段的定义和目标。
2. 选项分析
A. Scale 扩展: 错误。Scale阶段关注的是扩展云的使用规模和优化云资源，而不是展示如何加速业务成果。
B. Envision 构想: 正确。Envision阶段的目标是通过构想云的潜力，展示云如何帮助加速业务成果。这是AWS CAF中明确定义的阶
段。
C. Align 对齐: 错误。Align阶段的重点是确保云战略与业务目标保持一致，而不是直接展示加速业务成果。
D. Launch 启动: 错误。Launch阶段的目标是实际部署云解决方案，而不是展示云如何加速业务成果。
Correct Answer: B
正确答案：B

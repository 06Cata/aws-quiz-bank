Question #177
Which action will help increase security in the AWS Cloud?
哪项操作将有助于提高 AWS Cloud 的安全性？
A. Enable programmatic access for all IAM users.
为所有 IAM 用户启用编程访问。
B. Use IAM users instead of IAM roles to delegate permissions.
使用 IAM 用户而不是 IAM 角色来委派权限。
C. Rotate access keys on a reoccurring basis. Most Voted
定期轮换访问密钥。
D. Use inline policies instead of customer managed policies.
使用内联策略而不是客户管理策略。
题目分析与解答
1. 考察的知识点
本题考察的是AWS安全最佳实践，特别是与IAM（身份和访问管理）相关的安全措施。
2. 选项分析
A. Enable programmatic access for all IAM users.
错误：为所有IAM用户启用编程访问会增加安全风险，因为并非所有用户都需要编程访问权限。应遵循最小权限原则，仅为需要的
用户启用编程访问。
B. Use IAM users instead of IAM roles to delegate permissions.
错误：IAM角色是AWS推荐的权限委派方式，尤其是在跨账户访问或临时权限场景中。使用IAM用户进行权限委派可能导致长期凭
证泄露风险。
Correct Answer: C
正确答案：C

D. Use inline policies instead of customer managed policies.
错误：客户管理策略比内联策略更易于管理和审计。内联策略通常用于特定资源，但不适合大规模权限管理。

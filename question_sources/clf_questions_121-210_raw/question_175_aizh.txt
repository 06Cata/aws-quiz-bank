题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中与安全性相关的功能，特别是检查Amazon EC2实例的漏洞的服务。
2. 选项分析
A. AWS WAF: AWS WAF（Web Application Firewall）主要用于保护Web应用程序免受常见的网络攻击（如SQL注入和跨站脚本攻
击），并不用于检查EC2实例的漏洞，因此此选项错误。
B. AWS Trusted Advisor: AWS Trusted Advisor提供建议以优化AWS资源的使用，包括成本优化、安全性、性能和容错性，但它不
提供针对EC2实例漏洞的具体检查功能，因此此选项错误。
C. Amazon Inspector: Amazon Inspector是一项自动化的安全评估服务，专门用于检查Amazon EC2实例的漏洞和安全问题，并使
用预定义的评估模板，因此此选项正确。
Correct Answer: C
正确答案：C

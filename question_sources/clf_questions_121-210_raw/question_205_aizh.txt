Question #205
A company is planning to move data backups to the AWS Cloud. The company needs to replace on-premises storage with storage
that is cloud-based but locally cached.
Which AWS service meets these requirements?
一家公司计划将数据备份迁移到AWS云。该公司需要用基于云但本地缓存的存储替换本地存储。
哪项AWS服务符合这些要求？
A. AWS Storage Gateway Most Voted
AWS Storage Gateway
B. AWS Snowcone
AWS Snowcone
C. AWS Backup
AWS Backup
D. Amazon Elastic File System (Amazon EFS)
Amazon Elastic File System (Amazon EFS)
题目分析与解答
1. 考察的知识点
本题考察的是AWS存储服务的功能和使用场景，特别是如何实现云端存储与本地缓存的结合。
2. 选项分析
Correct Answer: A
正确答案：A

A. AWS Storage Gateway: 正确。AWS Storage Gateway是一种混合云存储服务，提供本地缓存功能，同时将数据备份到AWS云
端。它支持文件网关、卷网关和磁带网关，适合替代本地存储并实现云端备份。
B. AWS Snowcone: 错误。AWS Snowcone是一种小型的边缘计算和数据迁移设备，主要用于数据迁移到AWS云端，而不是提供持
续的本地缓存功能。
C. AWS Backup: 错误。AWS Backup是一种集中化的备份管理服务，用于备份AWS资源，但它不提供本地缓存功能，也不是替代本
地存储的解决方案。
D. Amazon Elastic File System (Amazon EFS): 错误。Amazon EFS是一种云端文件存储服务，适合为多个EC2实例提供共享存储，
但它不具备本地缓存功能。
3. 我的答案
A. AWS Storage Gateway
4. 官方答案
A. AWS Storage Gateway

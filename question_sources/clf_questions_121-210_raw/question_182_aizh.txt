Question #182
A user needs to quickly deploy a nonrelational database on AWS. The user does not want to manage the underlying hardware or
the database software.
Which AWS service can be used to accomplish this?
一个用户需要在AWS上快速部署一个非关系型数据库。该用户不希望管理底层硬件或数据库软件。
可以使用哪个AWS服务来实现这一目标？
A. Amazon RDS
Amazon RDS
B. Amazon DynamoDB Most Voted
Amazon DynamoDB

Amazon Aurora

题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的功能和用途，特别是无服务器（serverless）数据库服务的选择。
重点在于非关系型数据库（NoSQL）以及用户无需管理底层硬件或数据库软件的需求。
2. 选项分析
Correct Answer: B
正确答案：B

A. Amazon RDS: 错误。Amazon RDS 是一个托管关系型数据库服务，支持多种关系型数据库引擎（如 MySQL、PostgreSQL）。虽
然它简化了数据库管理，但它是关系型数据库，不符合题目要求的“非关系型数据库”。
B. Amazon DynamoDB: 正确。Amazon DynamoDB 是一个完全托管的无服务器非关系型数据库服务，用户无需管理底层硬件或数
据库软件，完全符合题目要求。
C. Amazon Aurora: 错误。Amazon Aurora 是一种高性能关系型数据库服务，支持 MySQL 和 PostgreSQL，属于关系型数据库，不
符合题目要求的“非关系型数据库”。
D. Amazon Redshift: 错误。Amazon Redshift 是一个托管的数据仓库服务，主要用于分析和处理结构化数据，不是非关系型数据
库。
3. 我的答案
我的答案是：B. Amazon DynamoDB
4. 官方答案
官方答案是：B. Amazon DynamoDB

Question #168
Which action is a security best practice for access to sensitive data that is stored in an Amazon S3 bucket?
哪项操作是访问存储在 Amazon S3 存储桶中的敏感数据的安全最佳实践？
A. Enable S3 Cross-Region Replication (CRR) on the S3 bucket.
在 S3 存储桶上启用 S3 Cross-Region Replication (CRR)。
B. Use IAM roles for applications that require access to the S3 bucket. Most Voted
为需要访问 S3 存储桶的应用程序使用 IAM roles。
C. Configure AWS WAF to prevent unauthorized access to the S3 bucket.
配置 AWS WAF 以防止未经授权访问 S3 存储桶。
D. Configure Amazon GuardDuty to prevent unauthorized access to the S3 bucket.
配置 Amazon GuardDuty 以防止未经授权访问 S3 存储桶。
题目分析与解答
1. 考察的知识点
本题考察的是AWS安全最佳实践，特别是如何安全地管理对Amazon S3存储桶中敏感数据的访问。
涉及的服务包括IAM角色、S3、AWS WAF和Amazon GuardDuty。
2. 选项分析
A. Enable S3 Cross-Region Replication (CRR) on the S3 bucket.
错误：S3 Cross-Region Replication (CRR) 的主要用途是将数据复制到另一个区域以实现灾难恢复或数据冗余，而不是直接用于控制
访问或保护敏感数据。
B. Use IAM roles for applications that require access to the S3 bucket.
Correct Answer: B
正确答案：B

正确：使用IAM角色是AWS推荐的安全最佳实践之一。IAM角色可以为应用程序提供临时凭证，避免使用硬编码的访问密钥，从而
降低敏感数据泄露的风险。
C. Configure AWS WAF to prevent unauthorized access to the S3 bucket.
错误：AWS WAF主要用于保护Web应用程序免受常见的网络攻击（如SQL注入或跨站脚本攻击），而不是直接用于保护S3存储桶的
访问。
D. Configure Amazon GuardDuty to prevent unauthorized access to the S3 bucket.
错误：Amazon GuardDuty 是一种威胁检测服务，可以监控恶意活动并提供警报，但它不能直接阻止未经授权的访问。
3. 我的答案
B. Use IAM roles for applications that require access to the S3 bucket.
4. 官方答案

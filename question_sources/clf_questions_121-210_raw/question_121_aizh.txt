Question #121
A company plans to deploy containers on AWS. The company wants full control of the compute resources that host the containers.
Which AWS service will meet these requirements?
一家公司计划在AWS上部署容器。该公司希望完全控制承载容器的计算资源。哪项AWS服务可以满足这些需求？
A. Amazon Elastic Kubernetes Service (Amazon EKS)
Amazon Elastic Kubernetes Service (Amazon EKS)
B. AWS Fargate
AWS Fargate
C. Amazon EC2 Most Voted
Amazon EC2
D. Amazon Elastic Container Service (Amazon ECS)
Amazon Elastic Container Service (Amazon ECS)
题目分析与解答
1. 考察的知识点
本题考察的是AWS中容器部署相关服务的特点，尤其是对计算资源控制的需求。
2. 选项分析
A. Amazon Elastic Kubernetes Service (Amazon EKS):
Amazon EKS是一个托管的Kubernetes服务，适合运行容器化应用程序。虽然它提供了对Kubernetes的管理，但它并不直接提供对底
层计算资源的完全控制，底层计算资源仍然由EC2或Fargate提供支持。因此此选项不符合题目要求。
B. AWS Fargate:
AWS Fargate是一个无服务器计算引擎，专门用于运行容器。它抽象了底层计算资源的管理，用户无法直接控制底层的计算资源。
Correct Answer: C
正确答案：C

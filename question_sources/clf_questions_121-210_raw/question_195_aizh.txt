Question #195
A company needs to use third-party software for its workload on AWS.
Which AWS service or feature can the company use to purchase the software?
一家公司需要在AWS上使用第三方软件来处理其工作负载。
该公司可以使用哪项AWS服务或功能来购买软件？
A. AWS Resource Access Manager
AWS Resource Access Manager
B. AWS Managed Services
AWS Managed Services
C. AWS License Manager
AWS License Manager
D. AWS Marketplace Most Voted
AWS Marketplace
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中与第三方软件购买相关的功能和服务。
2. 选项分析
A. AWS Resource Access Manager: 错误。AWS Resource Access Manager用于跨账户共享AWS资源，与购买第三方软件无关。
Correct Answer: D
正确答案：D

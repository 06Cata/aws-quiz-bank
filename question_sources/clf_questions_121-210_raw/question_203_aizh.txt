A. AWS Support: AWS Support主要用于提供技术支持和指导，但它是AWS官方的支持服务，不能直接用于与第三方合作或咨询，
因此此选项错误。
B. AWS Organizations: AWS Organizations用于管理多个AWS账户的组织结构和权限，与第三方合作无直接关系，因此此选项错
误。
C. AWS Service Catalog: AWS Service Catalog用于管理和部署预定义的云资源集合，与第三方支持或合作无直接关系，因此此选
项错误。
D. AWS Partner Network (APN): AWS Partner Network (APN)是AWS的合作伙伴计划，专门用于连接客户与第三方咨询公司、技
术合作伙伴等，因此此选项正确。

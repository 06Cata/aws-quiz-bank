Question #201
A company is storing data that will not be frequently accessed in the AWS Cloud. If the company needs to access the data, the
data needs to be retrieved within 12 hours. The company wants a solution that is cost-effective for storage costs for each
gigabyte.
Which Amazon S3 storage class will meet these requirements?
一家公司正在将不常访问的数据存储在AWS云中。如果公司需要访问这些数据，则需要在12小时内检索数据。公司希望找到一个每
GB存储成本具有成本效益的解决方案。
哪个Amazon S3存储类别能够满足这些需求？
A. S3 Standard

S3 One Zone-Infrequent Access (S3 One Zone-IA)

题目分析与解答
1. 考察的知识点
本题考察的是Amazon S3存储类的特性，包括存储成本、访问频率以及数据检索时间。
2. 选项分析
Correct Answer: B
正确答案：B

A. S3 Standard:
错误。S3 Standard适用于频繁访问的数据，存储成本较高，不适合存储不常访问的数据。
B. S3 Glacier Flexible Retrieval:
正确。S3 Glacier Flexible Retrieval是专为长期存储和不频繁访问的数据设计的，存储成本低，并且支持数据在12小时内检索，符合
题目要求。
C. S3 One Zone-Infrequent Access (S3 One Zone-IA):
错误。虽然S3 One Zone-IA存储成本较低，但它不适合长期存储，因为数据仅存储在一个可用区，存在数据丢失风险。此外，它的
检索时间比S3 Glacier Flexible Retrieval更快，但成本更高，不符合题目要求。
D. S3 Standard-Infrequent Access (S3 Standard-IA):
错误。S3 Standard-IA适用于不频繁访问但需要快速检索的数据，存储成本比S3 Glacier Flexible Retrieval高，不符合题目要求。
3. 我的答案
我选择的答案是：B. S3 Glacier Flexible Retrieval
4. 官方答案
官方正确答案是：B. S3 Glacier Flexible Retrieval

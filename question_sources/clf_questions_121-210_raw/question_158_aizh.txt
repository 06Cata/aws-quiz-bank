A user wants to securely automate the management and rotation of credentials that are shared between applications, while
spending the least amount of time on managing tasks.
Which AWS service or feature can be used to accomplish this?
一个用户希望安全地自动化管理和轮换在应用程序之间共享的凭证，同时尽可能减少管理任务的时间。
可以使用哪项AWS服务或功能来实现这一目标？

题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中用于管理和自动旋转应用程序之间共享的凭证的功能，重点关注安全性和自动化。
2. 选项分析
Correct Answer: C
正确答案：C

A. AWS CloudHSM:
AWS CloudHSM 是一种硬件安全模块服务，主要用于加密密钥的存储和管理。虽然它提供了高安全性，但它不直接支持自动化凭证
管理和旋转，因此不适合本题需求。
B. AWS Key Management Service (AWS KMS):
AWS KMS 是一种集中式密钥管理服务，主要用于加密和解密数据。虽然它可以管理加密密钥，但它不直接支持应用程序凭证的自
动旋转功能，因此不符合本题需求。
C. AWS Secrets Manager:
AWS Secrets Manager 是专门设计用于管理、存储和自动旋转应用程序凭证的服务。它支持自动化功能，减少了用户的管理工作
量，因此是本题的正确答案。
D. Server-side encryption:
服务器端加密是一种数据保护机制，用于加密存储在AWS服务中的数据。它与凭证管理和自动旋转无关，因此不符合本题需求。
3. 我的选择
答案: C. AWS Secrets Manager
4. 官方答案
答案: C. AWS Secrets Manager

Question #192
A company hosts an application on multiple Amazon EC2 instances. The application uses Amazon Simple Notification Service
(Amazon SNS) to send messages.
Which AWS service or feature will give the application permission to access required AWS services?
A. AWS Certificate Manager (ACM)
B. IAM roles Most Voted
C. AWS Security Hub
D. Amazon GuardDuty
Comments
tastatura 1 year, 2 months ago
Selected Answer: B
A: AWS Certificate Manager (ACM) manages SSL/TLS
B: IAM roles provide the necessary permissions for applications running on EC2 instances to securely access other AWS services
C: AWS Security Hub is a security monitoring and compliance service.
D: Amazon GuardDuty is a threat detection service and does not manage access permissions.

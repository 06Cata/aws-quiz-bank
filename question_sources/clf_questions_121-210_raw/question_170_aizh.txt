Question #170
A company needs a central user portal so that users can log in to third-party business applications that support Security Assertion
Markup Language (SAML) 2.0.
Which AWS service will meet this requirement?
一家公司需要一个中央用户门户，以便用户可以登录支持安全声明标记语言 (SAML) 2.0 的第三方业务应用程序。
哪个AWS服务可以满足此需求？
A. AWS Identity and Access Management (IAM)
AWS Identity and Access Management (IAM)
B. Amazon Cognito
Amazon Cognito
C. AWS IAM Identity Center (AWS Single Sign-On) Most Voted

题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中与用户身份管理和单点登录（SSO）相关的知识，特别是支持SAML 2.0协议的服务。
2. 选项分析
Correct Answer: C
正确答案：C

A. AWS Identity and Access Management (IAM):
IAM主要用于管理AWS资源的访问权限，而不是为用户提供一个中央登录门户或支持SAML 2.0的第三方应用登录功能。因此此选项
错误。
B. Amazon Cognito:
Amazon Cognito主要用于用户身份验证和管理，支持OAuth 2.0和OpenID Connect，但它不是专门设计用于提供中央用户门户或支
持SAML 2.0的第三方应用登录功能。因此此选项错误。
C. AWS IAM Identity Center (AWS Single Sign-On):
AWS IAM Identity Center（以前称为AWS Single Sign-On）是专门设计用于提供中央用户门户，并支持SAML 2.0协议的第三方应用
登录功能。因此此选项正确。
D. AWS CLI:
AWS CLI是一个命令行工具，用于与AWS服务交互，管理资源和执行操作。它与用户登录门户或SAML 2.0无关。因此此选项错误。
3. 我的答案
我的答案是：C. AWS IAM Identity Center (AWS Single Sign-On)
4. 官方答案
官方答案是：C. AWS IAM Identity Center (AWS Single Sign-On)

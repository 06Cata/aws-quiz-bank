A company plans to migrate to the AWS Cloud. The company is gathering information about its on-premises infrastructure and
requires information such as the hostname, IP address, and MAC address.
Which AWS service will meet these requirements?

A. AWS DataSync:
错误。AWS DataSync主要用于快速、安全地迁移数据到AWS云端，适用于文件存储的迁移，而不是收集主机信息如IP地址和MAC地
址。
B. AWS Application Migration Service:
错误。AWS Application Migration Service用于将物理服务器或虚拟机迁移到AWS云端，但它不专门用于收集主机信息如IP地址和
MAC地址。
C. AWS Application Discovery Service:
正确。AWS Application Discovery Service专门用于收集和分析本地环境中的信息，包括主机名、IP地址、MAC地址等。这是迁移前
的关键步骤。
D. AWS Database Migration Service (AWS DMS):
错误。AWS DMS主要用于迁移数据库，而不是收集主机信息如IP地址和MAC地址。

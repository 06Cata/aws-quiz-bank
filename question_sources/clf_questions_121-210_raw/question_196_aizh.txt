Which AWS service will meet these requirements?

题目分析与解答
1. 考察的知识点
本题考察的是AWS存储服务的特点和适用场景，特别是文件存储服务的协议支持（如SMB）。

Correct Answer: C

A. Amazon S3:
Amazon S3 是对象存储服务，适用于存储非结构化数据（如图片、视频等），不支持文件存储协议如 SMB。因此此选项错误。
B. Amazon Elastic File System (Amazon EFS):
Amazon EFS 是文件存储服务，支持 NFS（Network File System）协议，但不支持 SMB 协议。因此此选项错误。
C. Amazon FSx for Windows File Server:
Amazon FSx for Windows File Server 是专门为 Windows 环境设计的文件存储服务，支持 SMB 协议，满足题目要求。因此此选项正
确。
D. Amazon Elastic Block Store (Amazon EBS):
Amazon EBS 是块存储服务，适用于与 EC2 实例配合使用，不支持文件存储协议如 SMB。因此此选项错误。
3. 我的答案
我的答案是：C. Amazon FSx for Windows File Server
4. 官方答案
官方答案是：C. Amazon FSx for Windows File Server

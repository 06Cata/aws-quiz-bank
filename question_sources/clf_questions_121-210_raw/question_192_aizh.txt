Question #192
A company hosts an application on multiple Amazon EC2 instances. The application uses Amazon Simple Notification Service
(Amazon SNS) to send messages.
Which AWS service or feature will give the application permission to access required AWS services?

哪个 AWS 服务或功能将为应用程序提供访问所需 AWS 服务的权限？

题目分析与解答
1. 考察的知识点
本题考察的是AWS身份与访问管理（IAM）的知识，特别是IAM角色如何为应用程序提供访问其他AWS服务的权限。
2. 选项分析
Correct Answer: B
正确答案：B

A. AWS Certificate Manager (ACM) 错误。AWS Certificate Manager (ACM) 用于管理SSL/TLS证书，帮助保护网站和应用程序的安
全通信。它与权限管理无关，无法为应用程序提供访问其他AWS服务的权限。
B. IAM roles 正确。IAM角色是AWS提供的一种机制，可以为应用程序或服务授予访问其他AWS服务的权限。通过将IAM角色附加
到EC2实例，应用程序可以安全地访问Amazon SNS等服务。
C. AWS Security Hub 错误。AWS Security Hub 是一个安全管理服务，用于集中查看和管理安全警报，与权限管理无关。
D. Amazon GuardDuty 错误。Amazon GuardDuty 是一种威胁检测服务，用于识别潜在的安全问题，与权限管理无关。
3. 我的答案
B. IAM roles
4. 官方答案
B. IAM roles

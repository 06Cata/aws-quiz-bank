Which AWS service helps users audit API activity across their AWS account?
哪个AWS服务帮助用户审计其AWS账户中的API活动？
A. AWS CloudTrail Most Voted

Amazon Inspector

AWS Config
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的功能，特别是与审计和跟踪API活动相关的服务。
2. 选项分析
A. AWS CloudTrail: 正确。AWS CloudTrail 是一个服务，用于记录和审计 AWS 账户中的 API 调用活动。它可以帮助用户跟踪谁在
何时对资源进行了操作。
B. Amazon Inspector: 错误。Amazon Inspector 是一个自动化的安全评估服务，用于检查应用程序的安全性和合规性，而不是用于
审计 API 活动。
C. AWS WAF: 错误。AWS WAF 是一个 Web 应用程序防火墙，用于保护 Web 应用程序免受常见的网络攻击，但与审计 API 活动无
关。
Correct Answer: A
正确答案：A

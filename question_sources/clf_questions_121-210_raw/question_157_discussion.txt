varying workloads dynamically.
Which service or feature will meet these requirements?
A. Amazon DynamoDB
B. Amazon EC2 Spot Instances
C. AWS Snow Family

D. Amazon EC2 Auto Scaling
Amazon EC2 Auto Scaling is a service that allows you to automatically adjust the number of Amazon EC2 instances in your application to meet
changing demand. It can add or remove instances based on predefined conditions, ensuring that you have the right number of instances to handle
varying workloads, while optimizing costs.

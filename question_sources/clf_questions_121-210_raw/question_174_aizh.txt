题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中与合规性报告相关的功能，特别是如何获取合规性报告。
2. 选项分析
A. AWS Secrets Manager: AWS Secrets Manager用于安全地存储和管理敏感信息（如数据库凭证、API密钥等）。它与合规性报告
无关，因此此选项错误。
B. AWS Artifact: AWS Artifact是一个合规性报告门户，用户可以按需访问和下载与合规性相关的报告和文档。因此此选项正确。
C. AWS Security Hub: AWS Security Hub用于集中管理和分析安全警报，帮助用户提高安全态势，但它不提供合规性报告的下载功
能，因此此选项错误。

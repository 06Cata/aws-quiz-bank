Which AWS service can be used to retrieve compliance reports on demand?
A. AWS Secrets Manager
B. AWS Artifact Most Voted
C. AWS Security Hub
D. AWS Certificate Manager

Selected Answer: B
A: AWS Secrets Manager is used for securely storing and rotating secrets like database credentials and API keys, not for compliance reports.
B: AWS Artifact provides on-demand access to AWS compliance reports, such as ISO, SOC, and PCI reports.
C: AWS Security Hub gives a central view of security findings but does not provide compliance reports.
D: AWS Certificate Manager manages SSL/TLS certificates, not compliance documentation.

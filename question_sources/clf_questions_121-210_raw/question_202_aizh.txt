Which AWS service or resource can be used to identify services that have been used by a user within a specified date range?
哪个AWS服务或资源可以用于识别用户在指定日期范围内使用的服务？
A. Amazon S3 access control lists (ACLs)
Amazon S3 access control lists (ACLs)
B. AWS Certificate Manager (ACM)
AWS Certificate Manager (ACM)
C. Network Access Analyzer
Network Access Analyzer
D. AWS Identity and Access Management Access Analyzer Most Voted
AWS Identity and Access Management Access Analyzer
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中用于审计和分析用户活动的工具，特别是如何识别用户在指定日期范围内使用的服务。
2. 选项分析
A. Amazon S3 access control lists (ACLs)
错误：Amazon S3 ACLs用于管理对S3对象的访问权限，无法用于识别用户在指定日期范围内使用的服务。
B. AWS Certificate Manager (ACM)
错误：AWS Certificate Manager用于管理SSL/TLS证书，与用户活动审计无关。
C. Network Access Analyzer
错误：Network Access Analyzer用于分析网络访问路径和安全性，不涉及用户活动的审计。
Correct Answer: D
正确答案：D

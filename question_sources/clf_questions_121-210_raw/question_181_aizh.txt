Which AWS services or features enable users to connect on-premises networks to a VPC? (Choose two.)
哪些AWS服务或功能使用户能够将本地网络连接到VPC？（选择两个）
A. AWS VPN Most Voted
AWS VPN
B. Elastic Load Balancing
Elastic Load Balancing
C. AWS Direct Connect Most Voted
AWS Direct Connect
D. VPC peering
VPC peering
E. Amazon CloudFront
Amazon CloudFront
题目分析与解答
1. 考察的知识点
本题考察的是连接本地网络到AWS云环境的服务和功能，涉及到网络连接和混合云架构的知识。
2. 选项分析
A. AWS VPN: 正确。AWS VPN（虚拟专用网络）允许用户通过安全的加密连接将本地网络连接到AWS VPC。
B. Elastic Load Balancing: 错误。Elastic Load Balancing用于分发流量到多个实例或服务，不能用于连接本地网络到VPC。
C. AWS Direct Connect: 正确。AWS Direct Connect提供专用的物理连接，允许用户将本地网络直接连接到AWS VPC，适用于高带
宽和低延迟需求。
Correct Answer: AC
正确答案：AC

Question #141
Which AWS Cloud service can send alerts to customers if custom spending thresholds are exceeded?
哪个AWS云服务可以在客户超出自定义消费阈值时发送警报？
A. AWS Budgets Most Voted
AWS Budgets
B. AWS Cost Explorer
AWS Cost Explorer
C. AWS Cost Allocation Tags
AWS Cost Allocation Tags
D. AWS Organizations
AWS Organizations
题目分析与解答
1. 考察的知识点
本题考察的是AWS成本管理服务，特别是如何设置和监控预算以及支出阈值。
2. 选项分析
A. AWS Budgets: 正确。AWS Budgets允许用户设置预算和支出阈值，并在超出阈值时发送警报。它是专门设计用于成本管理和预
算监控的服务。
B. AWS Cost Explorer: 错误。AWS Cost Explorer主要用于分析和可视化成本和使用情况，但它不提供设置支出阈值警报的功能。
C. AWS Cost Allocation Tags: 错误。AWS Cost Allocation Tags用于标记和分类资源以便更好地跟踪成本，但它不具备发送警报的
功能。
D. AWS Organizations: 错误。AWS Organizations用于管理多个AWS账户的组织结构和权限，但它不涉及预算或支出警报功能。
Correct Answer: A
正确答案：A

Question #123
A company wants to store and retrieve files in Amazon S3 for its existing on-premises applications by using industry-standard file
system protocols.
Which AWS service will meet these requirements?
A. AWS DataSync
B. AWS Snowball Edge
C. Amazon S3 File Gateway Most Voted
D. AWS Transfer Family

tastatura 1 year, 2 months ago
Selected Answer: C
A. AWS DataSync: is a service for transferring large amounts of data between on-premises storage and Amazon S3, but it is designed for data
movement, not for providing real-time file system access
B. AWS Snowball Edge: Device that provids large volumes or edge computing offline data transfer capabilities. Not for access to S3
C. Amazon S3 File Gateway: Allows storing and retrieving files in Amazon S3 using industry-standard file protocols. It acts as a bridge between your
on-premesis file-based applications and Amazon S3 object storage
D. AWS Transfer Family: Fully managed support for SFTP, FTPS and FTP. provides fully managed support for file transfers directly into and out of
Amazon S3 or Amazon EFS. Supports file transfer protocols, not file system protocols

C. Amazon S3 File Gateway
A S3 File Gateway simplifies file storage in Amazon S3, integrates to existing applications through industry-standard file system protocols, and
provides a cost-effective alternative to on-premises storage. It also provides low-latency access to data through transparent local caching. A S3 File
Gateway manages data transfer to and from AWS, buffers applications from network congestion, optimizes and streams data in parallel, and
manages bandwidth consumption.

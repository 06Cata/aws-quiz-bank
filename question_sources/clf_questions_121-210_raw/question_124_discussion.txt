Selected Answer: A
A. AWS WAF: web application firewall that allows users to create custom rules to filter and monitor HTTP or HTTPS requests to a web application. It
helps protect against common web exploits, including SQL injection attacks
B. Network ACLs: Are used to control traffic at the subnet level.
C. Security groups: Control inbound and outbound traffic at the network level
D. AWS Certificate Manager (ACM): Manages SSL/TLS certificates or encrypting data in transit.

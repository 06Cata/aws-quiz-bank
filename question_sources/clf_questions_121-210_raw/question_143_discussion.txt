Question #143
A company needs to quickly and securely move files over long distances between its client and an Amazon S3 bucket.
Which S3 feature will meet this requirement?
A. S3 Versioning
B. S3 Transfer Acceleration Most Voted
C. S3ACLs
D. S3 Intelligent-Tiering

B. S3 Transfer Acceleration
S3 Transfer Acceleration is a feature that utilizes Amazon CloudFront's globally distributed edge locations to accelerate the upload of objects to an S3
bucket. It optimizes data transfer by using Amazon's backbone network, reducing latency and improving speeds when transferring files over long
distances.

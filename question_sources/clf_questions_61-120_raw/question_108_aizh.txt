Which AWS service or feature should be used?

题目分析与解答
1. 考察的知识点
本题考察的是如何安全地为运行在Amazon EC2实例上的应用程序授予访问其他AWS服务的权限。
涉及的知识点包括IAM角色的使用以及AWS服务间的安全访问机制。
2. 选项分析
A. Security groups
Security groups是用于控制网络流量的防火墙规则，主要用于管理入站和出站流量。
Correct Answer: C
正确答案：C

C. IAM roles

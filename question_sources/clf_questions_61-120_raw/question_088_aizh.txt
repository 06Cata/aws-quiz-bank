Topic 1Question #88
Which responsibility belongs to AWS when a company hosts its databases on Amazon EC2 instances?
公司在 Amazon EC2 实例上托管其数据库时，以下责任属于 AWS 的是哪一项？
A. Database backups
操作系统安装
B. Database software patches
数据库备份
C. Operating system patches
数据库软件补丁
D. Operating system installations 
Most Voted
操作系统补丁
题目分析与解答
1. 考察的知识点
本题考察的是AWS共享责任模型，特别是客户与AWS之间的责任划分。
2. 选项分析
A. Database backups: 错误。数据库备份是客户的责任，因为客户完全控制和管理运行在Amazon EC2实例上的数据库。
B. Database software patches: 错误。数据库软件补丁也是客户的责任，因为客户负责管理和维护其数据库软件。
C. Operating system patches: 错误。操作系统补丁是客户的责任，因为客户控制运行在EC2实例上的操作系统。
D. Operating system installations: 正确。AWS负责提供EC2实例的基础设施，包括操作系统的安装，确保实例可以正常运行。
3. 我的答案
Correct Answer: D
正确答案：D

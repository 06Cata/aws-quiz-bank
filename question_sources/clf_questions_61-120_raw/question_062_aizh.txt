Which AWS service supports a hybrid architecture that gives users the ability to extend AWS infrastructure, AWS services, APIs, and
tools to data centers, co-location environments, or on-premises facilities?
哪个AWS服务支持混合架构，使用户能够将AWS基础设施、AWS服务、API和工具扩展到数据中心、托管环境或本地设施？

题目分析与解答
1. 考察的知识点
本题考察的是AWS服务支持混合架构的能力，特别是能够将AWS的基础设施、服务、API和工具扩展到本地数据中心或其他非AWS
环境的服务。
2. 选项分析
A. AWS Snowmobile: AWS Snowmobile是一种用于大规模数据迁移的服务，主要用于将PB级数据从本地数据中心迁移到AWS云
中。它不支持混合架构，因此此选项错误。
B. AWS Local Zones: AWS Local Zones是AWS提供的一种服务，允许用户在靠近最终用户的地理位置运行低延迟应用程序。虽然它
可以扩展AWS服务到本地区域，但它并不完全支持混合架构的概念，因此此选项错误。
Correct Answer: C
正确答案：C

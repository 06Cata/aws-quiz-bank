Which AWS service is used to provide encryption for Amazon EBS?
哪个AWS服务用于为Amazon EBS提供加密？
A. AWS Certificate Manager
AWS Certificate Manager
B. AWS Systems Manager
AWS Systems Manager
C. AWS KMS 
Most Voted
AWS KMS
D. AWS Config
AWS Config
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中与数据加密相关的知识，特别是Amazon EBS（Elastic Block Store）的加密功能。
2. 选项分析
A. AWS Certificate Manager: 错误。AWS Certificate Manager主要用于管理SSL/TLS证书，用于保护网站和应用程序的通信，而不
是用于EBS加密。
B. AWS Systems Manager: 错误。AWS Systems Manager是一个操作管理服务，用于管理和自动化资源，但与EBS加密无关。
C. AWS KMS: 正确。AWS Key Management Service (KMS) 是用于创建和管理加密密钥的服务，EBS加密依赖于KMS来管理加密密
钥。
D. AWS Config: 错误。AWS Config是一个配置管理服务，用于跟踪资源配置的变化，与EBS加密无关。
Correct Answer: C
正确答案：C

Selected Answer: C
A. AWS Certificate Manager (ACM): Primarily used for managing SSL/TLS certificates used in conjunction with AWS services like Elastic Load Balancer
(ELB) or Amazon CloudFront to enable secure communication over HTTPS. It is not directly related to encrypting Amazon EBS volumes.
B. AWS Systems Manager: Allows you to automate operational tasks across your AWS resources. While it provides features for managing system
configurations, patching, and automation, it is not specifically designed for encrypting Amazon EBS volumes.
C. AWS Key Management Service (AWS KMS): A fully managed service that makes it easy for you to create, control, and manage encryption keys used
to encrypt your data. It integrates seamlessly with other AWS services, including Amazon EBS, for encryption purposes.
D. AWS Config: Enables you to assess, audit, and evaluate the configurations of your AWS resources. It does not directly provide encryption for
Amazon EBS volumes.

题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中与混合云存储相关的知识点，特别是能够连接本地环境与云端存储的服务。
2. 选项分析
A. AWS DataSync: AWS DataSync是一种数据迁移服务，用于快速、安全地将数据从本地存储迁移到AWS云端存储。它不提供混合
云存储功能，因此此选项错误。
B. Amazon S3 Glacier: Amazon S3 Glacier是一种低成本的存档存储服务，主要用于长期存储和备份数据。它不支持混合云存储功
能，因此此选项错误。
C. AWS Storage Gateway: AWS Storage Gateway是一种混合云存储服务，允许本地用户访问几乎无限的云存储。它支持文件、卷
和磁带网关模式，是正确答案。
Correct Answer: C
正确答案：C

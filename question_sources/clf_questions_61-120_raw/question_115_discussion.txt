scripting.
Which AWS service will meet these requirements?
A. Amazon Inspector
B. AWS WAF

Selected Answer: B
A. Amazon Inspector - security assessment service that helps improve the security and compliance of applications deployed on AWS but doesn't
protect web apps from live exploits.
B. AWS WAF - A web application firewall that allows users to create custom rules to filter and monitor HTTP or HTTPS requests to a web application.
It helps protect against web exploits, including SQL injection attacks.
C. Amazon GuardDuty - continuously detects threats and unauthorized behavior but doesn't actively block web exploits.
D. Amazon CloudWatch - Primarily for monitoring and managing resources its not a security tool for blocking attacks.

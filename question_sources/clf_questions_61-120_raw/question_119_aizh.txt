Which AWS service or feature provides log information of the inbound and outbound traffic on network interfaces in a VPC?
哪个AWS服务或功能提供有关VPC中网络接口的入站和出站流量的日志信息？
A. Amazon CloudWatch Logs
Amazon CloudWatch Logs
B. AWS CloudTrail
AWS CloudTrail
C. VPC Flow Logs 
Most Voted
VPC Flow Logs
D. AWS Identity and Access Management (IAM)
AWS Identity and Access Management (IAM)
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中与网络流量日志相关的功能，特别是VPC Flow Logs的用途。
2. 选项分析
A. Amazon CloudWatch Logs: 错误。CloudWatch Logs 是一个日志存储和分析服务，但它本身并不直接提供网络接口流量的日志
信息。VPC Flow Logs 可以将日志发送到 CloudWatch Logs，但这不是 CloudWatch Logs 的核心功能。
B. AWS CloudTrail: 错误。CloudTrail 主要用于记录 AWS API 调用的日志，帮助用户跟踪账户活动和操作，而不是记录网络接口的
流量信息。
C. VPC Flow Logs: 正确。VPC Flow Logs 是专门设计用于捕获和记录 VPC 中网络接口的入站和出站流量日志的服务。
D. AWS Identity and Access Management (IAM): 错误。IAM 是用于管理用户和权限的服务，与网络流量日志无关。
Correct Answer: C
正确答案：C

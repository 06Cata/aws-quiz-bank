Which of the following is a fully managed MySQL-compatible database?
以下哪个是完全托管的与 MySQL 兼容的数据库？
A. Amazon S3
Amazon S3
B. Amazon DynamoDB
Amazon DynamoDB
C. Amazon Redshift
Amazon Redshift
D. Amazon Aurora 
Most Voted
Amazon Aurora
题目分析与解答
1. 考察的知识点
本题考察的是AWS中数据库服务的类型及其特点，尤其是关于Amazon Aurora的特性。
2. 选项分析
A. Amazon S3: Amazon S3是一个对象存储服务，用于存储和检索任何数量的数据。它不是数据库服务，因此此选项错误。
B. Amazon DynamoDB: Amazon DynamoDB是一个完全托管的NoSQL数据库服务，适用于需要低延迟和高性能的应用程序。它不
是MySQL兼容的数据库，因此此选项错误。
C. Amazon Redshift: Amazon Redshift是一个数据仓库服务，专注于分析和处理大规模数据集。它不是MySQL兼容的数据库，因此
此选项错误。
D. Amazon Aurora: Amazon Aurora是一个完全托管的关系型数据库服务，兼容MySQL和PostgreSQL。此选项正确。
Correct Answer: D
正确答案：D

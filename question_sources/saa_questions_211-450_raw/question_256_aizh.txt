Question #256
A solutions architect is implementing a document review application using an Amazon S3 bucket for storage. The solution must
prevent accidental deletion of the documents and ensure that all versions of the documents are available. Users must be able to
download, modify, and upload documents.
Which combination of actions should be taken to meet these requirements? (Choose two.)

本的文档都可用。用户必须能够下载、修改和上传文档。

2. 选项分析
A. Enable a read-only bucket ACL. 启用只读存储桶 ACL。
错误：启用只读 ACL 会限制用户对存储桶的写入权限，而题目要求用户能够下载、修改和上传文档。
B. Enable versioning on the bucket. 在存储桶上启用版本控制。
正确：启用版本控制可以确保所有文档的版本都被保存，从而防止意外删除或覆盖。
C. Attach an IAM policy to the bucket. 将 IAM 策略附加到存储桶。
错误：虽然 IAM 策略可以控制访问权限，但它不能防止意外删除或确保文档的版本可用性。
D. Enable MFA Delete on the bucket. 在存储桶上启用 MFA Delete。
正确：启用 MFA Delete 可以增加删除操作的安全性，防止意外删除。
E. Encrypt the bucket using AWS KMS. 使用 AWS KMS 加密存储桶。
错误：加密可以保护数据的安全性，但与防止意外删除和版本控制无关。
3. 我选的答案
B. Enable versioning on the bucket.
D. Enable MFA Delete on the bucket.
4. 官方答案
B. Enable versioning on the bucket.
D. Enable MFA Delete on the bucket.

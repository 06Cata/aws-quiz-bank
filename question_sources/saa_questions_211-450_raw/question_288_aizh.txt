Question #288
A company is migrating a Linux-based web server group to AWS. The web servers must access files in a shared file store for some
content. The company must not make any changes to the application.
What should a solutions architect do to meet these requirements?
一家公司正在将基于 Linux 的 Web 服务器组迁移到 AWS。Web 服务器必须访问共享文件存储中的文件以获取某些内容。公司不得
对应用程序进行任何更改。
解决方案架构师应该怎么做来满足这些要求？
A. Create an Amazon S3 Standard bucket with access to the web servers.
创建一个 Amazon S3 Standard bucket 并允许 Web 服务器访问。
B. Configure an Amazon CloudFront distribution with an Amazon S3 bucket as the origin.
配置一个 Amazon CloudFront 分发，并将 Amazon S3 bucket 设置为源。
C. Create an Amazon Elastic File System (Amazon EFS) file system. Mount the EFS file system on all web servers. Most Voted
创建一个 Amazon Elastic File System (Amazon EFS) 文件系统。在所有 Web 服务器上挂载 EFS 文件系统。
D. Configure a General Purpose SSD (gp3) Amazon Elastic Block Store (Amazon EBS) volume. Mount the EBS volume to all web
servers.
配置一个通用 SSD (gp3) Amazon Elastic Block Store (Amazon EBS) 卷。将 EBS 卷挂载到所有 Web 服务器。
题目分析与解答
1. 考察的知识点
本题考察的是 AWS 文件存储服务的选择，包括 Amazon S3、Amazon EFS 和 Amazon EBS 的适用场景。
Correct Answer: C
正确答案：C
Community vote distribution
社区投票分布
C (100%)
重点在于理解共享文件存储的需求，以及如何在不修改应用程序的情况下实现文件共享。
2. 选项分析
A. Create an Amazon S3 Standard bucket with access to the web servers.
错误：Amazon S3 是对象存储服务，适合存储静态文件或数据，但不支持文件系统挂载。
由于题目要求“共享文件存储”，且不能修改应用程序，S3 不适合直接作为文件系统使用。
B. Configure an Amazon CloudFront distribution with an Amazon S3 bucket as the origin.
错误：CloudFront 是内容分发网络 (CDN)，用于加速静态内容的分发。
虽然可以结合 S3 提供内容分发，但它不满足“共享文件存储”的需求，且无法直接挂载到 Linux 系统。
C. Create an Amazon Elastic File System (Amazon EFS) file system. Mount the EFS file system on all web servers.
正确：Amazon EFS 是一个完全托管的文件存储服务，支持多台 Linux 实例同时挂载。
它满足“共享文件存储”的需求，并且无需修改应用程序。
D. Configure a General Purpose SSD (gp3) Amazon Elastic Block Store (Amazon EBS) volume. Mount the EBS volume to all
web servers.
错误：Amazon EBS 是块存储服务，设计为单实例挂载。
虽然可以提供高性能存储，但不支持多实例同时挂载，无法满足共享文件存储的需求。
3. 我的答案
C. Create an Amazon Elastic File System (Amazon EFS) file system. Mount the EFS file system on all web servers.
4. 官方答案
C. Create an Amazon Elastic File System (Amazon EFS) file system. Mount the EFS file system on all web servers.
5. 与官方答案的比较
我的答案与官方答案一致。
原因：Amazon EFS 是唯一支持多实例挂载的文件存储服务，完全满足题目要求。

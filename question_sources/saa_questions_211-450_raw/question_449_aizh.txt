Question #449
A company runs its application on an Oracle database. The company plans to quickly migrate to AWS because of limited resources
for the database, backup administration, and data center maintenance. The application uses third-party database features that
require privileged access.
Which solution will help the company migrate the database to AWS MOST cost-effectively?
一家公司在 Oracle 数据库上运行其应用程序。由于数据库、备份管理和数据中心维护资源有限，该公司计划快速迁移到 AWS。该
应用程序使用需要特权访问的第三方数据库功能。
哪种解决方案将帮助公司以昀具成本效益的方式将数据库迁移到 AWS？
A. Migrate the database to Amazon RDS for Oracle. Replace third-party features with cloud services.
将数据库迁移到 Amazon RDS for Oracle。用云服务替换第三方功能。
B. Migrate the database to Amazon RDS Custom for Oracle. Customize the database settings to support third-party features.

题目分析与解答
1. 考察的知识点
了解AWS数据库迁移选项，包括Amazon RDS、Amazon RDS Custom和Amazon EC2。
理解如何处理第三方数据库功能和特性。
评估成本效益和迁移的复杂性。
2. 选项分析
A. Migrate the database to Amazon RDS for Oracle. Replace third-party features with cloud services.
错误：Amazon RDS for Oracle是一个托管服务，但它不支持对数据库进行深度自定义或访问底层操作系统，这意味着无法支持需要
特权访问的第三方功能。
此外，替换第三方功能可能需要额外的开发工作，增加了迁移的复杂性。
B. Migrate the database to Amazon RDS Custom for Oracle. Customize the database settings to support third-party
features.
正确：Amazon RDS Custom for Oracle允许用户对数据库进行深度自定义，并支持需要特权访问的第三方功能。
这是一个托管服务，减少了数据库管理的负担，同时支持特定的自定义需求。
成本效益较高，因为它结合了托管服务的便利性和自定义能力。
C. Migrate the database to an Amazon EC2 Amazon Machine Image (AMI) for Oracle. Customize the database settings to
support third-party features.
错误：虽然在EC2上运行Oracle数据库可以提供完全的自定义能力，但它需要用户自己管理数据库、备份和底层基础设施。
这增加了管理复杂性和成本，不符合题目中“有限资源”的要求。
D. Migrate the database to Amazon RDS for PostgreSQL by rewriting the application code to remove dependency on
Oracle APEX.
错误：将数据库迁移到PostgreSQL需要重写应用程序代码，这是一项复杂且耗时的任务。
此外，题目中没有提到需要迁移到PostgreSQL的具体需求。
3. 我的选择
B. Migrate the database to Amazon RDS Custom for Oracle. Customize the database settings to support third-party
features.
4. 官方答案
B. Migrate the database to Amazon RDS Custom for Oracle. Customize the database settings to support third-party
features.
5. 与官方答案比较
我的答案与官方答案一致。
原因：Amazon RDS Custom for Oracle是昀符合题目要求的解决方案，既支持特权访问的第三方功能，又减少了数据库管理的复杂
性。

题目分析与解答
1. 考察的知识点
Amazon Inspector的功能：用于扫描EC2实例的软件漏洞。
AWS Systems Manager Patch Manager的功能：用于自动化补丁管理和定期更新EC2实例。
如何结合AWS服务实现安全扫描和补丁管理。
2. 选项分析
A. Set up Amazon Macie to scan the EC2 instances for software vulnerabilities. Set up a cron job on each EC2 instance to
patch the instance on a regular schedule.
错误：Amazon Macie主要用于数据分类和敏感数据保护，不适用于扫描EC2实例的软件漏洞。
错误：使用cron job手动管理补丁更新不符合自动化和集中管理的要求。
B. Turn on Amazon GuardDuty in the account. Configure GuardDuty to scan the EC2 instances for software vulnerabilities.
Set up AWS Systems Manager Session Manager to patch the EC2 instances on a regular schedule.
错误：Amazon GuardDuty主要用于检测恶意活动和异常行为，不适用于扫描软件漏洞。
错误：AWS Systems Manager Session Manager用于远程访问EC2实例，而不是用于补丁管理。
C. Set up Amazon Detective to scan the EC2 instances for software vulnerabilities. Set up an Amazon EventBridge
scheduled rule to patch the EC2 instances on a regular schedule.
错误：Amazon Detective主要用于调查和分析安全事件，不适用于扫描软件漏洞。
错误：虽然Amazon EventBridge可以触发定时任务，但它不是专门用于补丁管理的工具。
D. Turn on Amazon Inspector in the account. Configure Amazon Inspector to scan the EC2 instances for software
vulnerabilities. Set up AWS Systems Manager Patch Manager to patch the EC2 instances on a regular schedule.
正确：Amazon Inspector专门用于扫描EC2实例的软件漏洞。
正确：AWS Systems Manager Patch Manager可以自动化补丁管理并定期更新EC2实例。

Question #404
A company has deployed a serverless application that invokes an AWS Lambda function when new documents are uploaded to an
Amazon S3 bucket. The application uses the Lambda function to process the documents. After a recent marketing campaign, the
company noticed that the application did not process many of the documents.
What should a solutions architect do to improve the architecture of this application?
一家公司部署了一个无服务器应用程序，该应用程序在新的文档上传到 Amazon S3 bucket 时调用一个 AWS Lambda 函数。该应用
程序使用 Lambda 函数来处理这些文档。在最近的一次营销活动之后，公司注意到该应用程序没有处理许多文档。
解决方案架构师应该怎么做以改进该应用程序的架构？
A. Set the Lambda function's runtime timeout value to 15 minutes.
将 Lambda 函数的运行时超时时间设置为 15 分钟。
B. Configure an S3 bucket replication policy. Stage the documents in the S3 bucket for later processing.
配置一个 S3 bucket 的复制策略。将文档暂存到 S3 bucket 中以便稍后处理。
C. Deploy an additional Lambda function. Load balance the processing of the documents across the two Lambda functions.
部署一个额外的 Lambda 函数。将文档处理负载均衡到两个 Lambda 函数之间。
D. Create an Amazon Simple Queue Service (Amazon SQS) queue. Send the requests to the queue. Configure the queue as an
event source for Lambda. Most Voted
创建一个 Amazon Simple Queue Service (Amazon SQS) 队列。将请求发送到队列。将队列配置为 Lambda 的事件源。

1. 考察的知识点
Correct Answer: D
正确答案：D

D (100%)
本题考察的是 AWS Lambda 的事件驱动架构、Amazon S3 的事件通知机制，以及如何使用 Amazon SQS 队列来处理高并发和可靠
性问题。
2. 选项分析
A. Set the Lambda function's runtime timeout value to 15 minutes.
分析：虽然增加 Lambda 函数的超时时间可以处理更长时间的任务，但这并不能解决由于高并发导致的丢失事件问题。
结论：错误。
B. Configure an S3 bucket replication policy. Stage the documents in the S3 bucket for later processing.
分析：S3 bucket 的复制策略用于跨区域复制数据，不能直接解决 Lambda 函数的事件丢失问题。
结论：错误。
C. Deploy an additional Lambda function. Load balance the processing of the documents across the two Lambda
functions.
分析：部署额外的 Lambda 函数并进行负载均衡可能会增加处理能力，但无法解决事件丢失问题，因为 S3 的事件通知机制并不支
持负载均衡。
结论：错误。
D. Create an Amazon Simple Queue Service (Amazon SQS) queue. Send the requests to the queue. Configure the queue as
an event source for Lambda.
分析：使用 Amazon SQS 队列可以缓冲事件请求，确保所有事件都被可靠地处理。SQS 作为 Lambda 的事件源可以有效解决高并发
导致的事件丢失问题。
结论：正确。

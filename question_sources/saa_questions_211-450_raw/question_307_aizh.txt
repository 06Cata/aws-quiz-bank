Question #307
A company that primarily runs its application servers on premises has decided to migrate to AWS. The company wants to minimize
its need to scale its Internet Small Computer Systems Interface (iSCSI) storage on premises. The company wants only its recently
accessed data to remain stored locally.
Which AWS solution should the company use to meet these requirements?
一家主要在本地运行其应用服务器的公司决定迁移到AWS。该公司希望尽量减少在本地扩展其Internet Small Computer Systems
Interface (iSCSI)存储的需求。该公司希望只有最近访问的数据保留在本地存储。
该公司应该使用哪种AWS解决方案来满足这些需求？

题目分析与解答
1. 考察的知识点
Correct Answer: D
正确答案：D
Community vote distribution
社区投票分布
D (100%)
本题考察的是AWS Storage Gateway的不同类型及其适用场景，尤其是Volume Gateway的两种模式（Stored Volumes和Cached
Volumes）。
还涉及到如何优化本地存储与云存储的结合使用，以满足数据访问需求和存储扩展性。
2. 选项分析
A. Amazon S3 File Gateway
Amazon S3 File Gateway主要用于将文件存储到Amazon S3，同时提供本地缓存以加速访问。它适用于文件级存储场景，而题目中
提到的是iSCSI存储（块级存储），因此此选项不符合要求。
B. AWS Storage Gateway Tape Gateway
Tape Gateway主要用于将本地备份磁带虚拟化并存储到AWS云中，适用于备份和归档场景。题目中没有提到备份或归档需求，因此
此选项不符合要求。
C. AWS Storage Gateway Volume Gateway stored volumes
Stored Volumes模式会将所有数据存储在本地，并定期备份到AWS云中。这种模式需要较大的本地存储容量，而题目要求“仅最近访
问的数据存储在本地”，因此此选项不符合要求。
D. AWS Storage Gateway Volume Gateway cached volumes
Cached Volumes模式会将所有数据存储在AWS云中，仅将最近访问的数据缓存到本地。这种模式可以有效减少本地存储需求，同时
满足题目要求，因此此选项是正确答案。
3. 我选的答案
D. AWS Storage Gateway Volume Gateway cached volumes
4. 官方答案
D. AWS Storage Gateway Volume Gateway cached volumes
5. 与官方答案比较
我的答案与官方答案一致，选择D是正确的。

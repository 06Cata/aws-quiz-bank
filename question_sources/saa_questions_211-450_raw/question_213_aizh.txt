Balancer (ALB) against common application-level attacks, such as cross-site scripting or SQL injection. The company has minimal
infrastructure and operational staff. The company needs to reduce its share of the responsibility in managing, updating, and
securing servers for its AWS environment.
What should a solutions architect recommend to meet these requirements?

的应用层攻击，例如跨站脚本或 SQL 注入。该公司基础设施和运营人员较少。公司需要减少在管理、更新和保护其 AWS 环境服务
器方面的责任份额。

题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的安全性，特别是如何保护应用程序免受常见的应用层攻击（如跨站脚本和SQL注入）。
涉及的服务包括AWS WAF、AWS Shield、Amazon S3和Application Load Balancer (ALB)。
重点在于选择一种托管服务来减少管理和操作负担，同时提供有效的安全防护。
2. 选项分析
A. Configure AWS WAF rules and associate them with the ALB.
正确：AWS WAF是一种托管的Web应用程序防火墙，可以保护应用程序免受常见的应用层攻击（如跨站脚本和SQL注入）。
正确：将AWS WAF规则与ALB关联可以直接过滤恶意流量，减少对应用程序的威胁。
正确：AWS WAF是托管服务，减少了公司在管理和更新安全规则方面的责任。
B. Deploy the application using Amazon S3 with public hosting enabled.
错误：Amazon S3适用于静态网站托管，但无法提供针对应用层攻击的防护。
错误：启用公共托管可能会增加安全风险，而不是减少它。
C. Deploy AWS Shield Advanced and add the ALB as a protected resource.
错误：AWS Shield Advanced主要用于防护DDoS攻击，而不是应用层攻击（如跨站脚本和SQL注入）。
错误：虽然可以保护ALB，但无法直接解决题目中提到的应用层攻击问题。
D. Create a new ALB that directs traffic to an Amazon EC2 instance running a third-party firewall, which then passes the
traffic to the current ALB.
错误：这种方法增加了架构复杂性，并且需要额外的管理和维护工作。
错误：题目要求减少管理负担，而运行第三方防火墙需要额外的操作和维护。
3. 我选的答案
A. Configure AWS WAF rules and associate them with the ALB.
4. 官方答案
A. Configure AWS WAF rules and associate them with the ALB.
5. 与官方答案的比较
我的答案与官方答案一致。
原因：AWS WAF是专门设计用于防护应用层攻击的托管服务，符合题目要求的减少管理负担和提供安全防护的需求。

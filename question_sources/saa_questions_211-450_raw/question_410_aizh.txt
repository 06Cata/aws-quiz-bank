Question #410
A company is deploying a new application on Amazon EC2 instances. The application writes data to Amazon Elastic Block Store
(Amazon EBS) volumes. The company needs to ensure that all data that is written to the EBS volumes is encrypted at rest.
Which solution will meet this requirement?
一家公司正在 Amazon EC2 实例上部署一个新应用程序。该应用程序将数据写入 Amazon Elastic Block Store (Amazon EBS) 卷。公
司需要确保写入 EBS 卷的所有数据在静态存储时都被加密。
哪种解决方案可以满足此要求？
A. Create an IAM role that specifies EBS encryption. Attach the role to the EC2 instances.
创建一个指定 EBS 加密的 IAM 角色。将该角色附加到 EC2 实例。

将 EBS 卷创建为加密卷。将 EBS 卷附加到 EC2 实例。

level.
创建一个 EC2 实例标签，该标签的键为 Encrypt，值为 True。标记所有需要在 EBS 层级加密的实例。
D. Create an AWS Key Management Service (AWS KMS) key policy that enforces EBS encryption in the account. Ensure that the

2. 选项分析
A. Create an IAM role that specifies EBS encryption. Attach the role to the EC2 instances.
错误：IAM 角色无法直接指定 EBS 加密。EBS 加密是在创建卷时配置的，而不是通过 IAM 角色实现的。
B. Create the EBS volumes as encrypted volumes. Attach the EBS volumes to the EC2 instances.
正确：这是确保 EBS 数据加密的正确方法。可以在创建 EBS 卷时选择加密选项，AWS 会使用默认的 KMS 密钥或用户指定的 KMS
密钥来加密数据。
C. Create an EC2 instance tag that has a key of Encrypt and a value of True. Tag all instances that require encryption at the
EBS level.
错误：标签仅用于标记资源，无法强制执行加密。EBS 加密需要在卷创建时配置，而不是通过标签实现。
D. Create an AWS Key Management Service (AWS KMS) key policy that enforces EBS encryption in the account. Ensure
that the key policy is active.
错误：虽然 KMS 密钥策略可以控制密钥的使用权限，但它无法直接强制所有 EBS 卷加密。EBS 加密需要在卷创建时显式配置。
3. 我选的答案
B. Create the EBS volumes as encrypted volumes. Attach the EBS volumes to the EC2 instances.
4. 官方答案
B. Create the EBS volumes as encrypted volumes. Attach the EBS volumes to the EC2 instances.

Question #345
A company wants to restrict access to the content of one of its main web applications and to protect the content by using
authorization techniques available on AWS. The company wants to implement a serverless architecture and an authentication
solution for fewer than 100 users. The solution needs to integrate with the main web application and serve web content globally.
The solution must also scale as the company's user base grows while providing the lowest login latency possible.
Which solution will meet these requirements MOST cost-effectively?

题目分析与解答
1. 考察的知识点
Amazon Cognito的身份验证功能。
Lambda@Edge的授权功能。
Amazon CloudFront的全球内容分发能力。
如何设计成本效益高的无服务器架构。
2. 选项分析
A. Use Amazon Cognito for authentication. Use Lambda@Edge for authorization. Use Amazon CloudFront to serve the
web application globally.
Amazon Cognito是AWS提供的身份验证服务，支持少量用户并且可以扩展，适合本题需求。
Lambda@Edge可以在CloudFront边缘节点执行授权逻辑，提供低延迟的授权解决方案。
Amazon CloudFront是全球内容分发网络，能够高效地服务全球用户。
此选项完全符合题目要求，包括无服务器架构、低延迟、全球分发和成本效益。
B. Use AWS Directory Service for Microsoft Active Directory for authentication. Use AWS Lambda for authorization. Use
an Application Load Balancer to serve the web application globally.
AWS Directory Service适用于企业级目录服务，但对于少于100个用户的场景显得过于复杂且成本较高。
AWS Lambda可以执行授权逻辑，但与Application Load Balancer的结合不符合无服务器架构的要求。
Application Load Balancer不具备全球分发能力，无法满足题目要求。
此选项不符合题目要求，且成本较高。
C. Use Amazon Cognito for authentication. Use AWS Lambda for authorization. Use Amazon S3 Transfer Acceleration to
serve the web application globally.
Amazon Cognito适合身份验证需求。
AWS Lambda可以执行授权逻辑，但与Amazon S3 Transfer Acceleration的结合不符合无服务器架构的最佳实践。
Amazon S3 Transfer Acceleration主要用于加速文件上传和下载，不适合用来服务动态Web应用。
此选项不符合题目要求，且技术选择不合理。
D. Use AWS Directory Service for Microsoft Active Directory for authentication. Use Lambda@Edge for authorization. Use
AWS Elastic Beanstalk to serve the web application globally.
AWS Directory Service适用于企业级目录服务，但对于少于100个用户的场景显得过于复杂且成本较高。
Lambda@Edge可以执行授权逻辑，但与AWS Elastic Beanstalk的结合不符合无服务器架构的要求。
AWS Elastic Beanstalk是一个托管服务，适合应用部署，但不具备全球分发能力。
此选项不符合题目要求，且成本较高。

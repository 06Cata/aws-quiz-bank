Question #266
A company has a popular gaming platform running on AWS. The application is sensitive to latency because latency can impact the
user experience and introduce unfair advantages to some players. The application is deployed in every AWS Region. It runs on
Amazon EC2 instances that are part of Auto Scaling groups configured behind Application Load Balancers (ALBs). A solutions
architect needs to implement a mechanism to monitor the health of the application and redirect traffic to healthy endpoints.
Which solution meets these requirements?
一家公司在 AWS 上运行一个受欢迎的游戏平台。该应用程序对延迟非常敏感，因为延迟会影响用户体验，并可能导致某些玩家获
得不公平的优势。该应用程序部署在每个 AWS Region。它运行在属于 Auto Scaling 组的 Amazon EC2 实例上，这些实例配置在
Application Load Balancers (ALBs) 后面。解决方案架构师需要实施一种机制来监控应用程序的健康状况，并将流量重定向到健康的
端点。
哪种解决方案符合这些要求？

题目分析与解答
1. 考察的知识点
全球流量管理与优化：AWS Global Accelerator的使用。
负载均衡：Application Load Balancer (ALB) 的配置与健康检查。
高可用性与低延迟架构设计。
2. 选项分析
A. Configure an accelerator in AWS Global Accelerator...
正确：AWS Global Accelerator 是专门设计用于优化全球用户的流量路由，提供低延迟和高可用性。它支持健康检查，并可以将流
量重定向到健康的区域端点。将 ALB 作为端点是一个合理的配置。
B. Create an Amazon CloudFront distribution...
错误：CloudFront 是一个内容分发网络 (CDN)，主要用于缓存静态内容以提高性能。虽然它可以优化流量，但它不支持健康检查功
能，也无法动态重定向流量到健康的端点。
C. Create an Amazon CloudFront distribution...
错误：此选项与 B 类似，但指定了 Amazon S3 作为源服务器。S3 主要用于存储静态内容，与题目中提到的 EC2 和 ALB 架构不匹
配。此外，CloudFront 仍然无法满足健康检查和流量重定向的需求。
D. Configure an Amazon DynamoDB database...
错误：此选项与题目要求完全不相关。DynamoDB 是一个 NoSQL 数据库，DAX 是其加速器，用于提高数据库查询性能。它们与流
量管理和健康检查无关。

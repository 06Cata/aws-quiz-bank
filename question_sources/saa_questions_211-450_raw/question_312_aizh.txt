Store (Amazon EBS) data volumes attached to it. The application’s EC2 instance configuration and data need to be backed up
nightly. The application also needs to be recoverable in a different AWS Region.
Which solution will meet these requirements in the MOST operationally efficient way?

题目分析与解答
1. 考察的知识点
Amazon EC2实例和EBS卷的备份与恢复策略。
AWS Backup服务的功能和使用场景。
跨区域备份的实现方法。
操作效率和自动化备份的最佳实践。
2. 选项分析
A. Write an AWS Lambda function that schedules nightly snapshots of the application’s EBS volumes and copies the
snapshots to a different Region.
分析：
使用AWS Lambda函数可以实现自动化备份，但需要手动编写代码来管理快照和跨区域复制，增加了开发和维护的复杂性。
虽然可以满足需求，但操作效率较低，不是最优解。
结论： 错误。
B. Create a backup plan by using AWS Backup to perform nightly backups. Copy the backups to another Region. Add the
application’s EC2 instances as resources.
分析：
AWS Backup是一种托管服务，可以轻松创建备份计划并自动执行备份任务。
将EC2实例作为资源添加到备份计划中，可以备份实例配置和数据，同时支持跨区域复制。
操作效率高，完全满足题目要求。
结论： 正确。
C. Create a backup plan by using AWS Backup to perform nightly backups. Copy the backups to another Region. Add the
application’s EBS volumes as resources.
分析：
虽然可以备份EBS卷，但题目要求备份的是整个EC2实例的配置和数据，而不仅仅是EBS卷。
此选项未覆盖EC2实例的配置备份，因此不完全满足需求。
结论： 错误。
D. Write an AWS Lambda function that schedules nightly snapshots of the application's EBS volumes and copies the
snapshots to a different Availability Zone.
分析：
此选项仅将快照复制到另一个可用区，而不是另一个区域，无法满足题目要求的跨区域恢复需求。
使用Lambda函数增加了开发和维护的复杂性，操作效率较低。
结论： 错误。

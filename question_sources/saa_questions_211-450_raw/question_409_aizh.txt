Question #409
A solutions architect must migrate a Windows Internet Information Services (IIS) web application to AWS. The application currently
relies on a file share hosted in the user's on-premises network-attached storage (NAS). The solutions architect has proposed
migrating the IIS web servers to Amazon EC2 instances in multiple Availability Zones that are connected to the storage solution,
and configuring an Elastic Load Balancer attached to the instances.
Which replacement to the on-premises file share is MOST resilient and durable?
一个解决方案架构师必须将一个 Windows Internet Information Services (IIS) 网络应用程序迁移到 AWS。该应用程序目前依赖于用
户内部部署网络附加存储 (NAS) 中托管的文件共享。解决方案架构师建议将 IIS 网络服务器迁移到多个可用区中的 Amazon EC2 实
例，这些实例连接到存储解决方案，并配置一个附加到实例的 Elastic Load Balancer。
替换内部部署文件共享的选项中，哪个是昀具弹性和持久性的？

将文件共享迁移到 Amazon RDS。

将文件共享迁移到 Amazon Elastic File System (Amazon EFS)。

Correct Answer: C

本题考察的是AWS存储服务的选择，特别是针对Windows环境下的文件共享解决方案。
涉及的服务包括Amazon RDS、AWS Storage Gateway、Amazon FSx for Windows File Server和Amazon Elastic File System (EFS)。
重点在于选择昀适合Windows IIS应用的文件共享服务，同时满足高可用性和持久性需求。
2. 选项分析
A. Migrate the file share to Amazon RDS
错误：Amazon RDS是一个关系型数据库服务，适用于存储结构化数据，而不是文件共享。它不支持文件共享功能，因此不适合替
代NAS。
B. Migrate the file share to AWS Storage Gateway
错误：AWS Storage Gateway主要用于混合云场景，将本地存储扩展到AWS云。虽然它可以提供文件共享功能，但它依赖于本地存
储或云存储，不能完全替代NAS的功能，且在高可用性和持久性方面不如其他选项。
C. Migrate the file share to Amazon FSx for Windows File Server
正确：Amazon FSx for Windows File Server是专门为Windows环境设计的文件共享服务，完全兼容Windows文件系统（如NTFS），
支持SMB协议，提供高可用性和持久性，是替代Windows NAS的昀佳选择。
D. Migrate the file share to Amazon Elastic File System (Amazon EFS)
错误：Amazon EFS是一个分布式文件系统，主要用于Linux环境。虽然它支持高可用性和持久性，但不支持Windows文件系统和
SMB协议，因此不适合Windows IIS应用。
3. 我的答案
我的选择是：C. Migrate the file share to Amazon FSx for Windows File Server
4. 官方答案
官方正确答案是：C. Migrate the file share to Amazon FSx for Windows File Server

持久性以及与Windows IIS兼容的要求。

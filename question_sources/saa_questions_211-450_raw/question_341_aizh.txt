Question #341
A company has an Amazon S3 data lake that is governed by AWS Lake Formation. The company wants to create a visualization in
Amazon QuickSight by joining the data in the data lake with operational data that is stored in an Amazon Aurora MySQL
database. The company wants to enforce column-level authorization so that the company’s marketing team can access only a
subset of columns in the database.
Which solution will meet these requirements with the LEAST operational overhead?
一家公司拥有一个由AWS Lake Formation管理的Amazon S3数据湖。该公司希望通过将数据湖中的数据与存储在Amazon Aurora
MySQL数据库中的运营数据结合起来，在Amazon QuickSight中创建可视化。公司希望实施列级授权，以便公司的营销团队只能访
问数据库中的部分列。
哪种解决方案可以以最少的运营开销满足这些需求？
A. Use Amazon EMR to ingest the data directly from the database to the QuickSight SPICE engine. Include only the required
columns.
使用Amazon EMR直接从数据库将数据摄取到QuickSight SPICE引擎中。仅包含所需的列。
B. Use AWS Glue Studio to ingest the data from the database to the S3 data lake. Attach an IAM policy to the QuickSight users
to enforce column-level access control. Use Amazon S3 as the data source in QuickSight.
使用AWS Glue Studio将数据从数据库摄取到S3数据湖。为QuickSight用户附加一个IAM策略以实施列级访问控制。在QuickSight
中使用Amazon S3作为数据源。
C. Use AWS Glue Elastic Views to create a materialized view for the database in Amazon S3. Create an S3 bucket policy to
enforce column-level access control for the QuickSight users. Use Amazon S3 as the data source in QuickSight.
使用AWS Glue Elastic Views为数据库在Amazon S3中创建一个物化视图。为QuickSight用户创建一个S3存储桶策略以实施列级
访问控制。在QuickSight中使用Amazon S3作为数据源。
D. Use a Lake Formation blueprint to ingest the data from the database to the S3 data lake. Use Lake Formation to enforce
column-level access control for the QuickSight users. Use Amazon Athena as the data source in QuickSight. Most Voted
使用Lake Formation蓝图将数据从数据库摄取到S3数据湖。使用Lake Formation为QuickSight用户实施列级访问控制。在
QuickSight中使用Amazon Athena作为数据源。
Correct Answer: D
正确答案：D
题目分析与解答
1. 考察的知识点
Amazon S3数据湖的使用与治理
AWS Lake Formation的功能，包括列级访问控制
Amazon QuickSight数据可视化工具的集成
数据源的选择与访问控制的实现
2. 选项分析
A. 使用Amazon EMR直接从数据库将数据摄取到QuickSight SPICE引擎中。仅包含所需的列。
错误：Amazon EMR主要用于大规模数据处理，而不是直接用于列级访问控制。此选项没有提到如何实现列级授权。
错误：SPICE引擎是QuickSight的内存计算引擎，但它无法直接实现列级访问控制。
B. 使用AWS Glue Studio将数据从数据库摄取到S3数据湖。为QuickSight用户附加一个IAM策略以实施列级访问控制。在
QuickSight中使用Amazon S3作为数据源。
错误：IAM策略无法直接实现列级访问控制。IAM主要用于资源级别的权限管理，而不是细粒度的列级控制。
错误：此选项没有提到如何使用Lake Formation来管理数据湖的访问权限。
C. 使用AWS Glue Elastic Views为数据库在Amazon S3中创建一个物化视图。为QuickSight用户创建一个S3存储桶策略以实施列
级访问控制。在QuickSight中使用Amazon S3作为数据源。
错误：S3存储桶策略无法实现列级访问控制。存储桶策略主要用于控制对整个对象的访问，而不是细粒度的列级控制。
错误：AWS Glue Elastic Views的主要功能是跨数据源创建物化视图，但它不直接支持列级访问控制。
D. 使用Lake Formation蓝图将数据从数据库摄取到S3数据湖。使用Lake Formation为QuickSight用户实施列级访问控制。在
QuickSight中使用Amazon Athena作为数据源。
正确：Lake Formation支持列级访问控制，可以满足题目要求。
正确：使用Lake Formation蓝图可以简化数据摄取过程，降低操作开销。
正确：Amazon Athena可以直接查询S3数据湖，并与QuickSight集成。

D

Question #253
A solutions architect has created two IAM policies: Policy1 and Policy2. Both policies are attached to an IAM group.
A cloud engineer is added as an IAM user to the IAM group. Which action will the cloud engineer be able to perform?
一位解决方案架构师创建了两个 IAM 策略：Policy1 和 Policy2。这两个策略都附加到一个 IAM 组。
一名云工程师作为 IAM 用户被添加到该 IAM 组。云工程师将能够执行哪些操作？
A. Deleting IAM users
删除 IAM 用户
B. Deleting directories
删除目录
C. Deleting Amazon EC2 instances Most Voted
删除 Amazon EC2 实例
D. Deleting logs from Amazon CloudWatch Logs
删除 Amazon CloudWatch Logs 的日志
题目分析与解答
1. 考察的知识点
IAM策略的优先级：显式拒绝优先于显式允许。
IAM策略的作用范围：如何通过策略控制对AWS资源的访问权限。
IAM组和用户的权限继承：IAM用户会继承附加到IAM组的所有策略。
2. 选项分析
A. Deleting IAM users (删除 IAM 用户)
分析：Policy1中允许了对IAM资源的"Get*"和"List*"操作，但没有包含"Delete*"操作，因此用户无法删除IAM用户。此选项错误。
B. Deleting directories (删除目录)
分析：Policy1中允许了对"ds:*"的所有操作，但Policy2显式拒绝了"ds:Delete*"操作。由于显式拒绝优先于显式允许，用户无法删除
目录。此选项错误。
C. Deleting Amazon EC2 instances (删除 Amazon EC2 实例)
分析：Policy1中允许了对"ec2:*"的所有操作，并且没有显式拒绝与EC2相关的操作。因此用户可以删除EC2实例。此选项正确。
D. Deleting logs from Amazon CloudWatch Logs (删除 Amazon CloudWatch Logs 的日志)
分析：Policy1中允许了对CloudWatch Logs的"Get*"和"Describe*"操作，但没有包含"Delete*"操作，因此用户无法删除日志。此选
项错误。
3. 我选的答案
C. Deleting Amazon EC2 instances
4. 官方答案
C. Deleting Amazon EC2 instances
5. 与官方答案比较
我的答案与官方答案一致，没有差异。
Correct Answer: C
正确答案：C

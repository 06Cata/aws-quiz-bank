Question #289
A company has an AWS Lambda function that needs read access to an Amazon S3 bucket that is located in the same AWS
account.
Which solution will meet these requirements in the MOST secure manner?
一家公司有一个 AWS Lambda 函数，需要对位于同一 AWS 账户中的 Amazon S3 bucket 进行读取访问。
哪种解决方案能够以最安全的方式满足这些要求？
A. Apply an S3 bucket policy that grants read access to the S3 bucket.
应用一个 S3 bucket policy，授予对该 S3 bucket 的读取访问权限。

题目分析与解答
1. 考察的知识点
本题考察的是 AWS Lambda 的权限管理，尤其是如何通过 IAM Role 和 Policy 为 Lambda 函数授予最小权限（Least Privilege）。
Correct Answer: B
正确答案：B
Community vote distribution
社区投票分布
B (100%)
涉及的服务包括 AWS Lambda、IAM 和 S3。
2. 选项分析
A. Apply an S3 bucket policy that grants read access to the S3 bucket.
错误：S3 bucket policy 是一种资源级别的权限管理方式，通常用于跨账户访问或控制特定用户/服务对 S3 的权限。
虽然可以通过 bucket policy 实现 Lambda 的访问权限，但这不是最安全的方式，因为它无法确保最小权限原则，且可能会暴露给其
他不必要的主体。
B. Apply an IAM role to the Lambda function. Apply an IAM policy to the role to grant read access to the S3 bucket.
正确：这是最安全的方式。通过为 Lambda 函数分配 IAM Role，并在 Role 中附加一个精确的 IAM Policy，可以确保 Lambda 函数
仅对指定的 S3 bucket 具有读取权限。
这种方式遵循了最小权限原则，并且 IAM Role 是 AWS 推荐的最佳实践，用于为 Lambda 函数授予权限。
C. Embed an access key and a secret key in the Lambda function’s code to grant the required IAM permissions for read
access to the S3 bucket.
错误：将 access key 和 secret key 硬编码到 Lambda 函数中是一个严重的安全风险。
这种方式不仅违反了 AWS 的安全最佳实践，还可能导致凭证泄露，进而引发安全问题。
D. Apply an IAM role to the Lambda function. Apply an IAM policy to the role to grant read access to all S3 buckets in the
account.
错误：虽然这种方式可以实现 Lambda 函数对 S3 bucket 的访问，但它授予了对账户中所有 S3 buckets 的读取权限，违反了最小权
限原则。
这种方式可能会导致权限过度，增加安全风险。
3. 我选的答案
B. Apply an IAM role to the Lambda function. Apply an IAM policy to the role to grant read access to the S3 bucket.
4. 官方答案
B. Apply an IAM role to the Lambda function. Apply an IAM policy to the role to grant read access to the S3 bucket.
5. 与官方答案的比较
我的答案与官方答案一致。
原因：选项 B 是最安全、最符合 AWS 最佳实践的解决方案，确保了最小权限原则，同时避免了安全风险。

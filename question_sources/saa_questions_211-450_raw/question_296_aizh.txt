Question #296
A development team has launched a new application that is hosted on Amazon EC2 instances inside a development VPC. A
solutions architect needs to create a new VPC in the same account. The new VPC will be peered with the development VPC. The
VPC CIDR block for the development VPC is 192.168.0.0/24. The solutions architect needs to create a CIDR block for the new VPC.
The CIDR block must be valid for a VPC peering connection to the development VPC.
What is the SMALLEST CIDR block that meets these requirements?
一个开发团队已经启动了一个新应用程序，该应用程序托管在一个开发 VPC 内的 Amazon EC2 实例上。一个解决方案架构师需要在
同一账户中创建一个新的 VPC。新的 VPC 将与开发 VPC 建立对等连接。开发 VPC 的 VPC CIDR 块是 192.168.0.0/24。解决方案架
构师需要为新的 VPC 创建一个 CIDR 块。该 CIDR 块必须对开发 VPC 的 VPC 对等连接有效。
满足这些要求的最小 CIDR 块是什么？
A. 10.0.1.0/32
10.0.1.0/32

题目分析与解答
1. 考察的知识点
Correct Answer: D
正确答案：D

D (99%)
1
VPC网络设计与CIDR块规划。
VPC Peering连接的要求：两个VPC的CIDR块不能有重叠。
2. 选项分析
A. 10.0.1.0/32 - 错误。CIDR块 /32 表示只有一个IP地址，这不符合VPC的最小CIDR块要求（最小为 /28）。此外，/32 的范围太
小，无法满足实际使用需求。
B. 192.168.0.0/24 - 错误。这个CIDR块与开发VPC的CIDR块完全重叠（192.168.0.0/24），因此无法用于VPC Peering连接。
C. 192.168.1.0/32 - 错误。CIDR块 /32 表示只有一个IP地址，这不符合VPC的最小CIDR块要求（最小为 /28）。此外，虽然这个地
址与开发VPC的CIDR块不重叠，但范围太小，无法满足实际使用需求。
D. 10.0.1.0/24 - 正确。这个CIDR块与开发VPC的CIDR块（192.168.0.0/24）没有重叠，并且满足VPC Peering连接的要求。/24 的范
围足够大，适合实际使用。
3. 我选的答案
D. 10.0.1.0/24
4. 官方答案
D. 10.0.1.0/24

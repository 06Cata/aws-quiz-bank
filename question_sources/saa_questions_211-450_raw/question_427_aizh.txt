Question #427
A solutions architect is implementing a complex Java application with a MySQL database. The Java application must be deployed
on Apache Tomcat and must be highly available.
What should the solutions architect do to meet these requirements?
解决方案架构师正在实施一个复杂的 Java 应用程序，并使用 MySQL 数据库。该 Java 应用程序必须部署在 Apache Tomcat 上，并
且必须具有高可用性。
解决方案架构师应采取哪些措施来满足这些要求？
A. Deploy the application in AWS Lambda. Configure an Amazon API Gateway API to connect with the Lambda functions.
将应用程序部署在 AWS Lambda 中。配置一个 Amazon API Gateway API 以连接 Lambda 函数。
B. Deploy the application by using AWS Elastic Beanstalk. Configure a load-balanced environment and a rolling deployment
policy. Most Voted
使用 AWS Elastic Beanstalk 部署应用程序。配置一个负载均衡环境和滚动部署策略。
C. Migrate the database to Amazon ElastiCache. Configure the ElastiCache security group to allow access from the application.
将数据库迁移到 Amazon ElastiCache。配置 ElastiCache 安全组以允许应用程序访问。
D. Launch an Amazon EC2 instance. Install a MySQL server on the EC2 instance. Configure the application on the server. Create

启动一个 Amazon EC2 实例。在 EC2 实例上安装 MySQL 服务器。在服务器上配置应用程序。创建一个 AMI。使用 AMI 创建一

题目分析与解答
1. 考察的知识点
Correct Answer: B
正确答案：B

B (100%)
本题考察的是如何设计高可用的应用架构，涉及AWS服务的选择与昀佳实践，包括Elastic Beanstalk、Lambda、EC2、ElastiCache等
服务的使用场景。
重点关注高可用性、负载均衡、滚动部署策略以及服务的适配性。
2. 选项分析
A. Deploy the application in AWS Lambda. Configure an Amazon API Gateway API to connect with the Lambda functions.
错误：AWS Lambda适用于无服务器架构，但题目明确要求部署一个复杂的Java应用程序，这种应用通常需要持久运行的环境（如
Tomcat），而Lambda不适合运行复杂的Java应用。
此外，Lambda无法直接满足高可用性和负载均衡的需求，且与MySQL数据库的连接需要额外的配置。
B. Deploy the application by using AWS Elastic Beanstalk. Configure a load-balanced environment and a rolling
deployment policy.
正确：Elastic Beanstalk是一个托管服务，支持Java应用程序的部署，并且可以自动配置负载均衡和高可用性环境。
滚动部署策略可以确保应用程序的更新不会中断服务，符合高可用性要求。
Elastic Beanstalk还支持Tomcat作为运行环境，非常适合题目中的需求。
C. Migrate the database to Amazon ElastiCache. Configure the ElastiCache security group to allow access from the
application.
错误：Amazon ElastiCache是一个内存缓存服务，适用于加速数据库查询或存储临时数据，但不能替代MySQL数据库。
题目要求的是MySQL数据库，而不是缓存服务，因此此选项不符合需求。
D. Launch an Amazon EC2 instance. Install a MySQL server on the EC2 instance. Configure the application on the server.
Create an AMI. Use the AMI to create a launch template with an Auto Scaling group.
错误：虽然这种方法可以实现高可用性，但需要手动配置和管理EC2实例、负载均衡器、Auto Scaling组等，复杂度较高。
Elastic Beanstalk可以自动处理这些任务，因此更适合题目要求的场景。
3. 我的答案
B. Deploy the application by using AWS Elastic Beanstalk. Configure a load-balanced environment and a rolling
deployment policy.
4. 官方答案
B. Deploy the application by using AWS Elastic Beanstalk. Configure a load-balanced environment and a rolling
deployment policy.

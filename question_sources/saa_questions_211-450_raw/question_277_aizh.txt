Question #277
A company provides an online service for posting video content and transcoding it for use by any mobile platform. The
application architecture uses Amazon Elastic File System (Amazon EFS) Standard to collect and store the videos so that multiple
Amazon EC2 Linux instances can access the video content for processing. As the popularity of the service has grown over time, the
storage costs have become too expensive.
Which storage solution is MOST cost-effective?
一家公司提供一个在线服务，用于发布视频内容并对其进行转码，以便在任何移动平台上使用。应用程序架构使用 Amazon Elastic
File System (Amazon EFS) Standard 来收集和存储视频，以便多个 Amazon EC2 Linux 实例可以访问视频内容进行处理。随着服务的
流行度随着时间的推移不断增长，存储成本变得过于昂贵。
哪种存储解决方案最具成本效益？
A. Use AWS Storage Gateway for files to store and process the video content.
使用 AWS Storage Gateway for files 来存储和处理视频内容。

使用 AWS Storage Gateway for volumes 来存储和处理视频内容。

使用 Amazon EFS 来存储视频内容。一旦处理完成，将文件转移到 Amazon Elastic Block Store (Amazon EBS)。

题目分析与解答
1. 考察的知识点
Amazon EFS的使用场景及成本优化。
Amazon S3的存储特性及成本优势。
Amazon EBS的临时存储和处理能力。
AWS存储服务的组合使用以实现成本优化。
2. 选项分析
A. Use AWS Storage Gateway for files to store and process the video content.
错误：AWS Storage Gateway主要用于混合云场景，将本地数据与AWS云存储集成。它不适合直接用于视频内容的存储和处理，且
成本可能较高。
B. Use AWS Storage Gateway for volumes to store and process the video content.
错误：AWS Storage Gateway的卷模式主要用于备份和灾难恢复场景，而不是视频内容的存储和处理。此选项不符合题目需求。
C. Use Amazon EFS for storing the video content. Once processing is complete, transfer the files to Amazon Elastic Block
Store (Amazon EBS).
错误：虽然Amazon EFS支持多实例访问，但其成本较高。将文件转移到Amazon EBS后，仍然无法解决存储成本问题。
D. Use Amazon S3 for storing the video content. Move the files temporarily over to an Amazon Elastic Block Store
(Amazon EBS) volume attached to the server for processing.
正确：Amazon S3是低成本存储解决方案，适合存储视频内容。将文件临时移动到Amazon EBS进行处理可以优化成本，同时满足处
理需求。
3. 我选的答案
D. Use Amazon S3 for storing the video content. Move the files temporarily over to an Amazon Elastic Block Store (Amazon EBS)
volume attached to the server for processing.
4. 官方答案
D. Use Amazon S3 for storing the video content. Move the files temporarily over to an Amazon Elastic Block Store (Amazon EBS)
volume attached to the server for processing.
5. 与官方答案比较
我的答案与官方答案一致。
原因：Amazon S3提供低成本存储解决方案，结合Amazon EBS的临时处理能力，可以有效降低存储成本，同时满足视频处理需求。

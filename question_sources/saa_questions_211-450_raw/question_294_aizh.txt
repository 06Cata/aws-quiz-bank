Question #294
An application that is hosted on Amazon EC2 instances needs to access an Amazon S3 bucket. Traffic must not traverse the
internet.
How should a solutions architect configure access to meet these requirements?
一个孋管在 Amazon EC2 孩例上的寒用程寏需要访问一个 Amazon S3 bucket。流量嬈须不能通过互联网。
解决方案架构宵寒该如何配置访问以满足这些要求？
A. Create a private hosted zone by using Amazon Route 53.
使用 Amazon Route 53 创寠一个私有孋管区域。
B. Set up a gateway VPC endpoint for Amazon S3 in the VPC. Most Voted
在 VPC 中为 Amazon S3 设置一个 gateway VPC endpoint。
C. Configure the EC2 instances to use a NAT gateway to access the S3 bucket.
配置 EC2 孩例使用 NAT gateway 访问 S3 bucket。
D. Establish an AWS Site-to-Site VPN connection between the VPC and the S3 bucket.
在 VPC 和 S3 bucket 之间寠立一个 AWS Site-to-Site VPN 连接。
题目分析与解答
1. 考察的知识点
本题考孹的是如何在AWS中配置资源间的私有网络访问，避免流量通过互联网。
涉及的服务包括Amazon EC2、Amazon S3、VPC Endpoint、NAT Gateway等。
2. 选项分析
Correct Answer: B
正确答案：B
Community vote distribution
社区投票分害
B (100%)
A. Create a private hosted zone by using Amazon Route 53.
错误：Amazon Route 53的私有孋管区域用于在VPC内解析域名，但孚不能直接孩现EC2孩例与S3之间的私有连接。
此选项与题目要求无关。
B. Set up a gateway VPC endpoint for Amazon S3 in the VPC.
正确：Gateway VPC Endpoint允许VPC中的资源通过私有网络访问Amazon S3，而无需流量经过互联网。
这是孩现EC2孩例与S3之间私有连接的最佳解决方案。
C. Configure the EC2 instances to use a NAT gateway to access the S3 bucket.
错误：NAT Gateway用于宂私有子网中的资源连接到互联网嬻其他AWS服务，但流量仍然会经过互联网。
此选项不符合题目要求。
D. Establish an AWS Site-to-Site VPN connection between the VPC and the S3 bucket.
错误：AWS Site-to-Site VPN用于连接本地数据中嬇与AWS VPC，不能直接用于VPC与S3之间的连接。
此选项与题目要求无关。
3. 我的答案
B. Set up a gateway VPC endpoint for Amazon S3 in the VPC.
4. 官方答案
B. Set up a gateway VPC endpoint for Amazon S3 in the VPC.
5. 与官方答案的比较
嬺的答案与季方答案一致。
原因分析：Gateway VPC Endpoint是孩现VPC内资源与S3之间私有连接的标准解决方案，符合题目要求。

Question #232
A company runs demonstration environments for its customers on Amazon EC2 instances. Each environment is isolated in its own
VPC. The company’s operations team needs to be notified when RDP or SSH access to an environment has been established.
一家公司为其客户在 Amazon EC2 实例上运行演示环境。每个环境都隔离在其自己的 VPC 中。公司的运营团队需要在某个环境建立
RDP 或 SSH 访问时收到通知。
A. Configure Amazon CloudWatch Application Insights to create AWS Systems Manager OpsItems when RDP or SSH access is
detected.
配置 Amazon CloudWatch Application Insights，以便在检测到 RDP 或 SSH 访问时创建 AWS Systems Manager OpsItems。
B. Configure the EC2 instances with an IAM instance profile that has an IAM role with the AmazonSSMManagedInstanceCore
policy attached.
为 EC2 实例配置一个 IAM 实例配置文件，该配置文件附有带有 AmazonSSMManagedInstanceCore 策略的 IAM 角色。
C. Publish VPC flow logs to Amazon CloudWatch Logs. Create required metric filters. Create an Amazon CloudWatch metric
alarm with a notification action for when the alarm is in the ALARM state. Most Voted
将 VPC 流日志发布到 Amazon CloudWatch Logs。创建所需的指标过滤器。创建一个 Amazon CloudWatch 指标警报，并在警
报处于 ALARM 状态时配置通知操作。
D. Configure an Amazon EventBridge rule to listen for events of type EC2 Instance State-change Notification. Configure an
Amazon Simple Notification Service (Amazon SNS) topic as a target. Subscribe the operations team to the topic.
配置一个 Amazon EventBridge 规则以监听类型为 EC2 Instance State-change Notification 的事件。将 Amazon Simple
Notification Service (Amazon SNS) 主题配置为目标。将运营团队订阅到该主题。

本题主要考察对AWS监控和通知服务的理解，包括Amazon CloudWatch Logs、VPC Flow Logs、CloudWatch Alarms、EventBridge
规则等。
还涉及到如何检测特定的网络访问行为（如RDP或SSH）并通知相关团队。
2. 选项分析
A. Configure Amazon CloudWatch Application Insights to create AWS Systems Manager OpsItems when RDP or SSH
access is detected.
错误：Amazon CloudWatch Application Insights主要用于监控应用程序性能和异常，而不是直接用于检测网络访问行为（如RDP或
SSH）。
此外，OpsItems是用于问题管理的，不适合直接通知团队。
B. Configure the EC2 instances with an IAM instance profile that has an IAM role with the
AmazonSSMManagedInstanceCore policy attached.
错误：AmazonSSMManagedInstanceCore策略用于启用EC2实例与AWS Systems Manager的集成，但它与检测RDP或SSH访问无
关。
此选项未提及如何监控或通知团队，因此不符合题目要求。
C. Publish VPC flow logs to Amazon CloudWatch Logs. Create required metric filters. Create an Amazon CloudWatch
metric alarm with a notification action for when the alarm is in the ALARM state.
正确：VPC Flow Logs可以捕获网络流量数据，包括RDP和SSH访问。将这些日志发布到CloudWatch Logs后，可以创建特定的
Metric Filters来检测RDP或SSH访问行为。
通过CloudWatch Metric Alarm，可以在检测到特定行为时触发通知操作（如发送SNS通知）。
此选项完全符合题目要求。
D. Configure an Amazon EventBridge rule to listen for events of type EC2 Instance State-change Notification. Configure an
Amazon Simple Notification Service (Amazon SNS) topic as a target. Subscribe the operations team to the topic.
错误：EC2 Instance State-change Notification事件仅用于监控实例状态变化（如启动、停止、终止），而不是检测网络访问行为
（如RDP或SSH）。
此选项与题目要求不符。

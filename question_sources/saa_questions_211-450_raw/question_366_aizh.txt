Question #366
A company’s web application consists of an Amazon API Gateway API in front of an AWS Lambda function and an Amazon
DynamoDB database. The Lambda function handles the business logic, and the DynamoDB table hosts the data. The application
uses Amazon Cognito user pools to identify the individual users of the application. A solutions architect needs to update the
application so that only users who have a subscription can access premium content.
Which solution will meet this requirement with the LEAST operational overhead?
一家公司的网络应用程序由一个 Amazon API Gateway API、一个 AWS Lambda 函数和一个 Amazon DynamoDB 数据库组成。
Lambda 函数处理业务逻辑，DynamoDB 表存储数据。该应用程序使用 Amazon Cognito 用户池来识别应用程序的个人用户。解决
方案架构师需要更新应用程序，以便只有订阅用户才能访问高级内容。
哪种解决方案能以最少的运营开销满足这一需求？
A. Enable API caching and throttling on the API Gateway API.
在 API Gateway API 上启用 API 缓存和限流。
B. Set up AWS WAF on the API Gateway API. Create a rule to filter users who have a subscription.
在 API Gateway API 上设置 AWS WAF。创建一个规则以过滤具有订阅的用户。
C. Apply fine-grained IAM permissions to the premium content in the DynamoDB table.
对 DynamoDB 表中的高级内容应用细粒度的 IAM 权限。
D. Implement API usage plans and API keys to limit the access of users who do not have a subscription. Most Voted
实施 API 使用计划和 API 密钥，以限制未订阅用户的访问。

2. 选项分析
A. Enable API caching and throttling on the API Gateway API.
错误：API缓存和限流主要用于优化性能和控制流量，而不是用于限制用户访问特定内容。
此选项无法满足“仅订阅用户访问高级内容”的需求。
B. Set up AWS WAF on the API Gateway API. Create a rule to filter users who have a subscription.
错误：AWS WAF主要用于保护API免受恶意流量攻击，而不是用于基于用户订阅状态进行访问控制。
创建规则来过滤订阅用户的复杂性较高，且操作开销较大。
C. Apply fine-grained IAM permissions to the premium content in the DynamoDB table.
错误：IAM权限可以控制对DynamoDB表的访问，但无法直接基于用户订阅状态进行动态控制。
实现此功能需要额外的逻辑和操作开销。
D. Implement API usage plans and API keys to limit the access of users who do not have a subscription.
正确：API使用计划和API密钥是API Gateway的内置功能，可以轻松实现基于订阅状态的访问控制。
此解决方案操作开销最低，且直接满足题目需求。

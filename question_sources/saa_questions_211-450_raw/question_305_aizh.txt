Question #305
A company is designing a shared storage solution for a gaming application that is hosted in the AWS Cloud. The company needs
the ability to use SMB clients to access data. The solution must be fully managed.
Which AWS solution meets these requirements?
一家公司正在为托管在 AWS Cloud 中的游戏应用程序设计一个共享存储解决方案。 该公司需要能够使用 SMB 客户端访问数据。
该解决方案必须是完全托管的。
哪个 AWS 解决方案满足这些要求？
A. Create an AWS DataSync task that shares the data as a mountable file system. Mount the file system to the application
server.
创建一个 AWS DataSync 任务，将数据共享为可挂载的文件系统。 将文件系统挂载到应用程序服务器。
B. Create an Amazon EC2 Windows instance. Install and configure a Windows file share role on the instance. Connect the
application server to the file share.
创建一个 Amazon EC2 Windows 实例。 在实例上安装并配置 Windows 文件共享角色。 将应用程序服务器连接到文件共享。
C. Create an Amazon FSx for Windows File Server file system. Attach the file system to the origin server. Connect the
application server to the file system. Most Voted
创建一个 Amazon FSx for Windows File Server 文件系统。 将文件系统附加到源服务器。 将应用程序服务器连接到文件系统。
D. Create an Amazon S3 bucket. Assign an IAM role to the application to grant access to the S3 bucket. Mount the S3 bucket
to the application server.
创建一个 Amazon S3 bucket。 为应用程序分配一个 IAM 角色以授予对 S3 bucket 的访问权限。 将 S3 bucket 挂载到应用程序
服务器。
Correct Answer: C
正确答案：C

题目分析与解答
1. 考察的知识点
本题考察的是AWS存储服务的选择，特别是针对共享存储解决方案的设计。
重点在于支持SMB协议的文件共享，并且要求是完全托管的服务。
2. 选项分析
A. Create an AWS DataSync task that shares the data as a mountable file system.
错误原因：AWS DataSync是用于数据迁移的服务，而不是用于提供共享存储的解决方案。它不能直接提供SMB协议支持，也不符
合“完全托管”的要求。
B. Create an Amazon EC2 Windows instance. Install and configure a Windows file share role on the instance.
错误原因：虽然可以通过EC2实例配置Windows文件共享来支持SMB协议，但这不是“完全托管”的解决方案。用户需要自行管理EC2
实例的操作系统、文件共享配置和维护。
C. Create an Amazon FSx for Windows File Server file system.
正确原因：Amazon FSx for Windows File Server是一个完全托管的文件存储服务，专门支持SMB协议。它满足题目要求的“完全托
管”和“SMB客户端访问”的需求。
D. Create an Amazon S3 bucket.
错误原因：Amazon S3是对象存储服务，不支持SMB协议。虽然可以通过IAM角色授予访问权限，但它无法直接作为文件系统挂载
到应用程序服务器。
3. 我的答案
答案：C

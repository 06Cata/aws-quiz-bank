Question #365
A company runs a web application that is backed by Amazon RDS. A new database administrator caused data loss by accidentally
editing information in a database table. To help recover from this type of incident, the company wants the ability to restore the
database to its state from 5 minutes before any change within the last 30 days.
Which feature should the solutions architect include in the design to meet this requirement?
一家公司运行一个由 Amazon RDS 支持的 Web 应用程序。一位新的数据库管理员由于意外编辑数据库表中的信息导致数据丢失。
为了帮助从这种类型的事件中恢复，公司希望能够将数据库恢复到过去 30 天内任何更改前 5 分钟的状态。
解决方案架构师应该在设计中包含哪个功能以满足此需求？
A. Read replicas
Read replicas
B. Manual snapshots
Manual snapshots
C. Automated backups Most Voted
Automated backups
D. Multi-AZ deployments
Multi-AZ deployments
题目分析与解答
1. 考察的知识点
本题考察的是Amazon RDS的备份与恢复功能，包括自动备份、手动快照、读副本和多可用区部署的用途和区别。
Correct Answer: C
正确答案: C
Community vote distribution
社区投票分布
C (100%)
2. 选项分析
A. Read replicas:
读副本用于提高数据库的读取性能和分担主数据库的负载，但它不提供数据恢复到特定时间点的功能。因此此选项错误。
B. Manual snapshots:
手动快照是用户手动创建的数据库备份，可以用于恢复数据。但手动快照无法满足题目中“恢复到5分钟前的状态”的需求，因为它需
要用户主动创建快照，无法提供连续的时间点恢复功能。因此此选项错误。
C. Automated backups:
自动备份是Amazon RDS提供的功能，支持连续的时间点恢复（Point-in-Time Recovery），可以恢复到过去30天内的任意时间点。
这完全满足题目要求，因此此选项正确。
D. Multi-AZ deployments:
多可用区部署用于提高数据库的高可用性和容错能力，但它不提供数据恢复到特定时间点的功能。因此此选项错误。
3. 我选的答案
C. Automated backups
4. 官方答案
C. Automated backups

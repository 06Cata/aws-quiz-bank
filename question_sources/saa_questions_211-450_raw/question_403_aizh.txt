Question #403
A developer has an application that uses an AWS Lambda function to upload files to Amazon S3 and needs the required
permissions to perform the task. The developer already has an IAM user with valid IAM credentials required for Amazon S3.
What should a solutions architect do to grant the permissions?
一名开发人员有一个应用程序，该应用程序使用 AWS Lambda 函数将文件上传到 Amazon S3，并需要执行此任务所需的权限。该
开发人员已经拥有一个具有 Amazon S3 所需有效 IAM 凭证的 IAM 用户。
解决方案架构师应该怎么做才能授予权限？
A. Add required IAM permissions in the resource policy of the Lambda function.
在 Lambda 函数的资源策略中添加所需的 IAM 权限。
B. Create a signed request using the existing IAM credentials in the Lambda function.
在 Lambda 函数中使用现有的 IAM 凭证创建一个签名请求。
C. Create a new IAM user and use the existing IAM credentials in the Lambda function.
创建一个新的 IAM 用户，并在 Lambda 函数中使用现有的 IAM 凭证。
D. Create an IAM execution role with the required permissions and attach the IAM role to the Lambda function. Most Voted
创建一个具有所需权限的 IAM 执行角色，并将该 IAM 角色附加到 Lambda 函数。

安全最佳实践：避免使用硬编码的IAM凭证
2. 选项分析
A. Add required IAM permissions in the resource policy of the Lambda function.
错误：Lambda函数的资源策略主要用于控制对Lambda函数的访问，而不是用于授予Lambda函数对其他AWS服务的权限。
正确的做法是通过IAM执行角色来授予Lambda函数权限。
B. Create a signed request using the existing IAM credentials in the Lambda function.
错误：使用签名请求和硬编码的IAM凭证是安全性较差的做法，不符合AWS的最佳实践。
Lambda函数应该通过IAM角色来安全地访问其他AWS服务。
C. Create a new IAM user and use the existing IAM credentials in the Lambda function.
错误：创建新的IAM用户并使用硬编码的凭证会导致安全风险。
最佳实践是使用IAM角色，而不是IAM用户凭证。
D. Create an IAM execution role with the required permissions and attach the IAM role to the Lambda function.
正确：这是AWS推荐的最佳实践。通过IAM执行角色，Lambda函数可以安全地访问其他AWS服务。
IAM角色可以附加到Lambda函数，并授予所需的权限。

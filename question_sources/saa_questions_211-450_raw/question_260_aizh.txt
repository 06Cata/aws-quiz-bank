Question #260
A company’s compliance team needs to move its file shares to AWS. The shares run on a Windows Server SMB file share. A self-
managed on-premises Active Directory controls access to the files and folders.
The company wants to use Amazon FSx for Windows File Server as part of the solution. The company must ensure that the on-
premises Active Directory groups restrict access to the FSx for Windows File Server SMB compliance shares, folders, and files after
the move to AWS. The company has created an FSx for Windows File Server file system.
Which solution will meet these requirements?
一家公司合规团队需要将其文件共享迁移到AWS。这些共享运行在Windows Server SMB文件共享上。一个自管理的本地Active
Directory控制对文件和文件夹的访问。
该公司希望使用Amazon FSx for Windows File Server作为解决方案的一部分。公司必须确保在迁移到AWS后，本地Active Directory
组能够限制对FSx for Windows File Server SMB合规共享、文件夹和文件的访问。公司已经创建了一个FSx for Windows File Server
文件系统。
哪种解决方案可以满足这些要求？
A. Create an Active Directory Connector to connect to the Active Directory. Map the Active Directory groups to IAM groups to
restrict access.
创建一个Active Directory Connector以连接到Active Directory。将Active Directory组映射到IAM组以限制访问。
B. Assign a tag with a Restrict tag key and a Compliance tag value. Map the Active Directory groups to IAM groups to restrict
access.
分配一个带有Restrict标签键和Compliance标签值的标签。将Active Directory组映射到IAM组以限制访问。
C. Create an IAM service-linked role that is linked directly to FSx for Windows File Server to restrict access.
创建一个直接链接到FSx for Windows File Server的IAM服务关联角色以限制访问。
D. Join the file system to the Active Directory to restrict access. Most Voted
将文件系统加入Active Directory以限制访问。
Correct Answer: D
正确答案：D
题目分析与解答
1. 考察的知识点
Amazon FSx for Windows File Server的功能与配置。
如何将AWS服务与自托管的Active Directory集成。
文件和文件夹访问控制的实现方式。
2. 选项分析
A. Create an Active Directory Connector to connect to the Active Directory. Map the Active Directory groups to IAM
groups to restrict access.
错误：Active Directory Connector用于将AWS Managed Microsoft AD与本地Active Directory集成，但它不适用于Amazon FSx for
Windows File Server的直接访问控制。
错误：IAM组无法直接映射到Active Directory组来控制文件系统访问。
B. Assign a tag with a Restrict tag key and a Compliance tag value. Map the Active Directory groups to IAM groups to
restrict access.
错误：标签（Tag）用于资源标识和管理，而不是用于访问控制。
错误：同样，IAM组无法直接映射到Active Directory组来控制文件系统访问。
C. Create an IAM service-linked role that is linked directly to FSx for Windows File Server to restrict access.
错误：IAM服务关联角色用于AWS服务之间的权限管理，但无法直接用于控制FSx文件系统的访问权限。
错误：文件系统访问权限需要通过Active Directory组来管理，而不是通过IAM角色。
D. Join the file system to the Active Directory to restrict access.
正确：Amazon FSx for Windows File Server支持与自托管的Active Directory集成。通过将文件系统加入Active Directory，可以使用
现有的Active Directory组和用户来控制文件和文件夹的访问权限。
正确：这是实现基于Active Directory的访问控制的推荐方法。
3. 我的答案
D. Join the file system to the Active Directory to restrict access.
4. 官方答案
D. Join the file system to the Active Directory to restrict access.
5. 与官方答案比较
我的答案与官方答案一致。
原因：选项D是唯一正确的解决方案，符合题目要求并利用了Amazon FSx for Windows File Server的功能。

Question #387
A new employee has joined a company as a deployment engineer. The deployment engineer will be using AWS CloudFormation
templates to create multiple AWS resources. A solutions architect wants the deployment engineer to perform job activities while
following the principle of least privilege.
Which combination of actions should the solutions architect take to accomplish this goal? (Choose two.)

望部署工程师在遵循最小权限原则的同时执行工作梮动。

题目分析与解答
1. 考察的知识点
IAM权限管栬：如何根据“最小权限原则”分配权限。
AWS CloudFormation：如何使用IAM用户或角色执行CloudFormation相关操作。
2. 选项分析
A. Have the deployment engineer use AWS account root user credentials for performing AWS CloudFormation stack
operations. 错误：根用户权限过高，违反了“最小权限原则”。根用户应仅用于账户管栬或紧急情况，不应用于日常操作。
B. Create a new IAM user for the deployment engineer and add the IAM user to a group that has the PowerUsers IAM
policy attached. 错误：PowerUsers策略允许用户管栬资棕，但不允许管栬IAM权限。这可能会授予过多权限，仍棺不符合“最小权
限原则”。
C. Create a new IAM user for the deployment engineer and add the IAM user to a group that has the AdministratorAccess
IAM policy attached. 错误：AdministratorAccess策略授予完全权限，远超CloudFormation操作所需权限，违反了“最小权限原
则”。
D. Create a new IAM user for the deployment engineer and add the IAM user to a group that has an IAM policy that allows
AWS CloudFormation actions only. 正确：桮选项符合“最小权限原则”，仅授予CloudFormation相关操作权限，限制了用户的权限
范围。
E. Create an IAM role for the deployment engineer to explicitly define the permissions specific to the AWS
CloudFormation stack and launch stacks using that IAM role. 正确：使用IAM角色可以明确定义权限，并且角色可以在需要时
被临时使用，进一桯增强安全性和权限控制。

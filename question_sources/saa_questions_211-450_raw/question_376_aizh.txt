Question #376
A company has launched an Amazon RDS for MySQL DB instance. Most of the connections to the database come from serverless
applications. Application traffic to the database changes significantly at random intervals. At times of high demand, users report
that their applications experience database connection rejection errors.
Which solution will resolve this issue with the LEAST operational overhead?
一家公司启动了一个 Amazon RDS for MySQL 数据库实例。大多数连接到数据库的请求来自无服务器应用程序。应用程序对数据库
的流量在随机时间间隔内显著变化。在高需求时期，用户报告他们的应用程序遇到了数据库连接拒绝错误。
哪种解决方案可以以最少的运营开销解决此问题？
A. Create a proxy in RDS Proxy. Configure the users’ applications to use the DB instance through RDS Proxy. Most Voted
在 RDS Proxy 中创建一个代理。配置用户的应用程序通过 RDS Proxy 使用数据库实例。
B. Deploy Amazon ElastiCache for Memcached between the users’ applications and the DB instance.
在用户的应用程序和数据库实例之间部署 Amazon ElastiCache for Memcached。
C. Migrate the DB instance to a different instance class that has higher I/O capacity. Configure the users’ applications to use
the new DB instance.
将数据库实例迁移到具有更高 I/O 容量的不同实例类型。配置用户的应用程序使用新的数据库实例。
D. Configure Multi-AZ for the DB instance. Configure the users’ applications to switch between the DB instances.
为数据库实例配置 Multi-AZ。配置用户的应用程序在数据库实例之间切换。

Correct Answer: A
正确答案：A

社区投票分布
A (100%)
Amazon RDS Proxy的功能与使用场景。
如何优化数据库连接管理以应对高并发和动态流量变化。
对比其他解决方案（如ElastiCache、实例升级、Multi-AZ）的适用性。
2. 选项分析
A. Create a proxy in RDS Proxy. Configure the users’ applications to use the DB instance through RDS Proxy.
正确：RDS Proxy可以管理数据库连接池，减少连接开销并提高应用程序的可扩展性。它特别适合处理动态流量变化和高并发场
景，同时具有较低的操作开销。
B. Deploy Amazon ElastiCache for Memcached between the users’ applications and the DB instance.
错误：ElastiCache主要用于缓存数据以减少数据库查询次数，但题目中并未提到查询性能问题，而是数据库连接被拒绝的问题。此
选项无法直接解决连接管理问题。
C. Migrate the DB instance to a different instance class that has higher I/O capacity. Configure the users’ applications to
use the new DB instance.
错误：升级实例类型可以提高性能，但无法解决连接管理问题。高并发场景下，连接数限制仍然存在，且操作开销较高。
D. Configure Multi-AZ for the DB instance. Configure the users’ applications to switch between the DB instances.
错误：Multi-AZ主要用于提高数据库的高可用性和灾难恢复能力，而不是优化连接管理。此选项无法解决连接拒绝问题。
3. 我的答案
A. Create a proxy in RDS Proxy. Configure the users’ applications to use the DB instance through RDS Proxy.

A. Create a proxy in RDS Proxy. Configure the users’ applications to use the DB instance through RDS Proxy.

我的答案与官方答案一致。
原因：RDS Proxy是解决高并发和动态流量变化场景下数据库连接管理问题的最佳选择，且具有最低的操作开销。

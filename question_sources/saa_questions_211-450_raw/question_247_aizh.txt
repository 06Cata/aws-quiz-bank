Question #247
A company has deployed a database in Amazon RDS for MySQL. Due to increased transactions, the database support team is
reporting slow reads against the DB instance and recommends adding a read replica.
Which combination of actions should a solutions architect take before implementing this change? (Choose two.)
一家公司在 Amazon RDS for MySQL 中部署了一个数据库。由于事务量增加，数据库支持团队报告了针对数据库实例的读取速度变
慢，并建议添加一个只读副本。
在实施此更改之前，解决方案架构师应采取哪些组合操作？（选择两个。）
A. Enable binlog replication on the RDS primary node.
在 RDS 主节点上启用 binlog 复制。
B. Choose a failover priority for the source DB instance.
为源数据库实例选择一个故障转移优先级。
C. Allow long-running transactions to complete on the source DB instance. Most Voted
允许源数据库实例上的长时间运行事务完成。
D. Create a global table and specify the AWS Regions where the table will be available.
创建一个全局表并指定表可用的 AWS 区域。
E. Enable automatic backups on the source instance by setting the backup retention period to a value other than 0. Most Voted
通过将备份保留期设置为非零值，在源实例上启用自动备份。

Amazon RDS的读副本配置和前置条件。
事务管理和备份机制在数据库操作中的重要性。
2. 选项分析
A. Enable binlog replication on the RDS primary node.
错误。Amazon RDS会自动管理读副本的复制，无需手动启用binlog复制。此选项不适用于RDS环境。
B. Choose a failover priority for the source DB instance.
错误。读副本的创建与故障转移优先级无关。故障转移优先级通常用于多可用区部署，而不是读副本。
C. Allow long-running transactions to complete on the source DB instance.
正确。在创建读副本之前，确保源数据库实例上的长时间运行事务完成是必要的，以避免数据不一致或复制失败。
D. Create a global table and specify the AWS Regions where the table will be available.
错误。此选项与Amazon DynamoDB相关，而不是Amazon RDS。题目明确提到使用的是RDS for MySQL。
E. Enable automatic backups on the source instance by setting the backup retention period to a value other than 0.
正确。启用自动备份是创建读副本的前置条件之一，因为读副本依赖于备份和快照来初始化。

Question #325
A company is hosting a web application from an Amazon S3 bucket. The application uses Amazon Cognito as an identity provider
to authenticate users and return a JSON Web Token (JWT) that provides access to protected resources that are stored in another
S3 bucket.
Upon deployment of the application, users report errors and are unable to access the protected content. A solutions architect
must resolve this issue by providing proper permissions so that users can access the protected content.
Which solution meets these requirements?
一家公司正在通过 Amazon S3 存储桶托管一个 Web 应用程序。该应用程序使用 Amazon Cognito 作为身份提供者来验证用户身
份，并返回一个 JSON Web Token (JWT)，该令牌提供对存储在另一个 S3 存储桶中的受保护资源的访问权限。
在部署应用程序后，用户报告错误并无法访问受保护的内容。解决方案架构师必须通过提供适当的权限来解决此问题，以便用户可
以访问受保护的内容。
哪种解决方案满足这些要求？
A. Update the Amazon Cognito identity pool to assume the proper IAM role for access to the protected content. Most Voted
更新 Amazon Cognito 身份池以承担访问受保护内容的适当 IAM 角色。
B. Update the S3 ACL to allow the application to access the protected content.
更新 S3 ACL 以允许应用程序访问受保护的内容。
C. Redeploy the application to Amazon S3 to prevent eventually consistent reads in the S3 bucket from affecting the ability of
users to access the protected content.
重新部署应用程序到 Amazon S3，以防止 S3 存储桶中的最终一致性读取影响用户访问受保护内容的能力。
D. Update the Amazon Cognito pool to use custom attribute mappings within the identity pool and grant users the proper

更新 Amazon Cognito 池以在身份池中使用自定义属性映射，并授予用户访问受保护内容的适当权限。
Correct Answer: A
正确答案：A
题目分析与解答
1. 考察的知识点
Amazon Cognito的身份池（Identity Pool）与IAM角色的关联。
如何通过Amazon Cognito为用户提供访问受保护资源的权限。
Amazon S3权限管理，包括ACL和IAM角色的使用。
2. 选项分析
A. Update the Amazon Cognito identity pool to assume the proper IAM role for access to the protected content.
正确：Amazon Cognito身份池可以配置IAM角色，允许用户通过身份池获得访问受保护资源的权限。
解决问题：通过更新身份池的配置，确保用户能够获得正确的IAM角色权限，从而访问受保护的S3内容。
B. Update the S3 ACL to allow the application to access the protected content.
错误：S3 ACL主要用于设置对象级别的权限，但题目中提到的是用户通过Amazon Cognito进行身份验证并访问资源，而不是直接通
过应用程序访问资源。
不解决问题：ACL无法解决用户无法通过身份池获得权限的问题。
C. Redeploy the application to Amazon S3 to prevent eventually consistent reads in the S3 bucket from affecting the
ability of users to access the protected content.
错误：S3的最终一致性读取与用户权限问题无关。题目中提到的问题是用户无法访问受保护资源，而不是数据读取一致性问题。
不解决问题：重新部署应用程序无法解决权限配置问题。
D. Update the Amazon Cognito pool to use custom attribute mappings within the identity pool and grant users the proper
permissions to access the protected content.
错误：题目中并未提到需要使用自定义属性映射。解决问题的关键是更新身份池的IAM角色配置，而不是自定义属性映射。
不解决问题：自定义属性映射与用户权限问题无关。
3. 我的答案
A. Update the Amazon Cognito identity pool to assume the proper IAM role for access to the protected content.
4. 官方答案
A. Update the Amazon Cognito identity pool to assume the proper IAM role for access to the protected content.
5. 比较与思考
我的答案与官方答案一致。
原因：选项A正确地解决了用户无法访问受保护资源的问题，符合题目要求。

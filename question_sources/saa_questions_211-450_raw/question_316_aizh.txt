Question #316
A company uses an Amazon EC2 instance to run a script to poll for and process messages in an Amazon Simple Queue Service
(Amazon SQS) queue. The company wants to reduce operational costs while maintaining its ability to process a growing number
of messages that are added to the queue.
What should a solutions architect recommend to meet these requirements?
一家公司使用一个 Amazon EC2 实例运行脚本来轮询和处理 Amazon Simple Queue Service (Amazon SQS) 队列中的消息。该公司
希望在保持处理队列中不断增加的消息能力的同时，降低运营成本。
解决方案架构师应该推荐什么来满足这些需求？
A. Increase the size of the EC2 instance to process messages faster.
增加 EC2 实例的大小以更快地处理消息。
B. Use Amazon EventBridge to turn off the EC2 instance when the instance is underutilized.
使用 Amazon EventBridge 在实例未充分利用时关闭 EC2 实例。
C. Migrate the script on the EC2 instance to an AWS Lambda function with the appropriate runtime. Most Voted
将 EC2 实例上的脚本迁移到具有适当运行时的 AWS Lambda 函数。
D. Use AWS Systems Manager Run Command to run the script on demand.
使用 AWS Systems Manager Run Command 按需运行脚本。
题目分析与解答
1. 考察的知识点
本题考察的是如何优化成本，同时满足动态扩展需求的能力。
Correct Answer: C
正确答案：C

涉及的服务包括：Amazon EC2、Amazon SQS、AWS Lambda、Amazon EventBridge、AWS Systems Manager。
重点知识点：无服务器架构（Serverless）、AWS Lambda的使用场景、动态扩展能力。
2. 选项分析
A. 增加 EC2 实例的大小以更快地处理消息。
错误：增加 EC2 实例的大小虽然可以提高处理速度，但无法解决成本问题，且无法动态扩展以应对消息数量的增长。
B. 使用 Amazon EventBridge 在实例未充分利用时关闭 EC2 实例。
错误：虽然可以减少未使用时的成本，但无法动态扩展以处理队列中不断增长的消息量。
C. 将 EC2 实例上的脚本迁移到具有适当运行时的 AWS Lambda 函数。
正确：AWS Lambda 是无服务器架构，按需运行，支持动态扩展，能够有效降低成本并处理增长的消息量。
D. 使用 AWS Systems Manager Run Command 按需运行脚本。
错误：虽然可以按需运行脚本，但无法动态扩展以处理队列中不断增长的消息量。
3. 我的答案
C. 将 EC2 实例上的脚本迁移到具有适当运行时的 AWS Lambda 函数。
4. 官方答案
C. 将 EC2 实例上的脚本迁移到具有适当运行时的 AWS Lambda 函数。
5. 与官方答案比较
我的答案与官方答案一致。
原因：AWS Lambda 是无服务器架构，支持动态扩展，能够有效降低成本并处理增长的消息量，是最佳解决方案。

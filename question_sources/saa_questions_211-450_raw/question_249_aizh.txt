Question #249
A company is implementing a shared storage solution for a media application that is hosted in the AWS Cloud. The company
needs the ability to use SMB clients to access data. The solution must be fully managed.
Which AWS solution meets these requirements?
一家公司正在为托管在 AWS Cloud 中的媒体应用程序实施共享存储解决方案。 该公司需要能够使用 SMB 客户端访问数据。 解决
方案必须是完全托管的。
哪个 AWS 解决方案满足这些要求？
A. Create an AWS Storage Gateway volume gateway. Create a file share that uses the required client protocol. Connect the
application server to the file share.
创建一个 AWS Storage Gateway volume gateway。 创建一个使用所需客户端协议的文件共享。 将应用程序服务器连接到文件
共享。
B. Create an AWS Storage Gateway tape gateway. Configure tapes to use Amazon S3. Connect the application server to the
tape gateway.
创建一个 AWS Storage Gateway tape gateway。 配置磁带以使用 Amazon S3。 将应用程序服务器连接到磁带网关。
C. Create an Amazon EC2 Windows instance. Install and configure a Windows file share role on the instance. Connect the
application server to the file share.
创建一个 Amazon EC2 Windows 实例。 在实例上安装并配置 Windows 文件共享角色。 将应用程序服务器连接到文件共享。
D. Create an Amazon FSx for Windows File Server file system. Attach the file system to the origin server. Connect the
application server to the file system. Most Voted
创建一个 Amazon FSx for Windows File Server 文件系统。 将文件系统附加到源服务器。 将应用程序服务器连接到文件系统。

题目分析与解答
1. 考察的知识点
本题考察的是AWS存储服务的选择，尤其是针对共享存储解决方案的能力。
重点关注AWS服务是否支持SMB协议以及是否是完全托管的解决方案。
2. 选项分析
A. Create an AWS Storage Gateway volume gateway.
错误：AWS Storage Gateway的Volume Gateway主要用于块存储（Block Storage），而不是文件共享。它不支持SMB协议。
B. Create an AWS Storage Gateway tape gateway.
错误：Tape Gateway主要用于备份和归档场景，模拟磁带库，不适合文件共享需求，也不支持SMB协议。
C. Create an Amazon EC2 Windows instance.
错误：虽然可以在EC2 Windows实例上安装和配置文件共享角色以支持SMB协议，但这不是一个完全托管的解决方案。用户需要自
行管理实例的操作系统、补丁和可用性。
D. Create an Amazon FSx for Windows File Server file system.
正确：Amazon FSx for Windows File Server是一个完全托管的文件存储服务，支持SMB协议，专为Windows工作负载设计，满足题
目要求。
3. 我的答案
D. Create an Amazon FSx for Windows File Server file system.
4. 官方答案
D. Create an Amazon FSx for Windows File Server file system.

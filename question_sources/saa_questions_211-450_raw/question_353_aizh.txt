Question #353
A company hosts a three-tier web application on Amazon EC2 instances in a single Availability Zone. The web application uses a
self-managed MySQL database that is hosted on an EC2 instance to store data in an Amazon Elastic Block Store (Amazon EBS)
volume. The MySQL database currently uses a 1 TB Provisioned IOPS SSD (io2) EBS volume. The company expects traffic of 1,000
IOPS for both reads and writes at peak traffic.
The company wants to minimize any disruptions, stabilize performance, and reduce costs while retaining the capacity for double
the IOPS. The company wants to move the database tier to a fully managed solution that is highly available and fault tolerant.
Which solution will meet these requirements MOST cost-effectively?
一家公司在单个可用区的 Amazon EC2 实例上托管一个三层 Web 应用程序。该 Web 应用程序使用一个自管理的 MySQL 数据库，
该数据库托管在一个 EC2 实例上，并将数据存储在 Amazon Elastic Block Store (Amazon EBS) 卷中。当前 MySQL 数据库使用一个 1
TB Provisioned IOPS SSD (io2) EBS 卷。公司预计峰值流量时读写操作的 IOPS 为 1,000。
公司希望尽量减少任何中断，稳定性能，并降低成本，同时保留双倍 IOPS 的容量。公司希望将数据库层迁移到一个高度可用且容
错的完全托管解决方案。
哪种解决方案可以最具成本效益地满足这些要求？
A. Use a Multi-AZ deployment of an Amazon RDS for MySQL DB instance with an io2 Block Express EBS volume.
使用一个 Amazon RDS for MySQL 数据库实例的 Multi-AZ 部署，并使用 io2 Block Express EBS 卷。

题目分析与解答
1. 考察的知识点
Amazon RDS的功能与优势，包括高可用性和故障容错能力。
Amazon EBS存储类型的选择及其性能和成本优化。
如何迁移自托管数据库到托管解决方案以降低成本和提高稳定性。
2. 选项分析
A. 使用一个 Amazon RDS for MySQL 数据库实例的 Multi-AZ 部署，并使用 io2 Block Express EBS 卷。
分析：虽然io2 Block Express EBS卷提供了极高的性能和可靠性，但它的成本较高，不符合题目中“降低成本”的要求。因此此选项不
符合题目要求。
B. 使用一个 Amazon RDS for MySQL 数据库实例的 Multi-AZ 部署，并使用 General Purpose SSD (gp2) EBS 卷。
分析：gp2卷提供了足够的性能（支持高达16,000 IOPS），并且成本较低。结合RDS的Multi-AZ部署，可以实现高可用性和故障容
错能力，同时满足性能需求和成本优化要求。因此此选项是正确答案。
C. 使用 Amazon S3 Intelligent-Tiering 访问层。
分析：S3 Intelligent-Tiering是用于对象存储的解决方案，与关系型数据库的存储需求不匹配。此选项与题目要求无关，因此是错误
的。
D. 使用两个大型 EC2 实例以主动-被动模式托管数据库。
分析：虽然主动-被动模式可以提供高可用性，但仍然是自托管解决方案，无法利用RDS的托管优势。此外，使用两个大型EC2实例
的成本可能较高，不符合“降低成本”的要求。因此此选项是错误的。
3. 我选的答案
B. 使用一个 Amazon RDS for MySQL 数据库实例的 Multi-AZ 部署，并使用 General Purpose SSD (gp2) EBS 卷。
4. 官方答案
B. 使用一个 Amazon RDS for MySQL 数据库实例的 Multi-AZ 部署，并使用 General Purpose SSD (gp2) EBS 卷。
5. 与官方答案的比较
我的答案与官方答案一致。
原因：B选项最符合题目要求，既满足性能需求，又优化了成本，同时提供了高可用性和故障容错能力。

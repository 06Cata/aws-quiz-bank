题目分析与解答
1. 考察的知识点
安全存储和管理数据库用户凭证。
使用AWS服务（如Secrets Manager）实现凭证的自动轮换。
IAM权限管理和服务集成。
2. 选项分析
A. Store the database user credentials in AWS Secrets Manager. Grant the necessary IAM permissions to allow the web
servers to access AWS Secrets Manager.
正确：AWS Secrets Manager是专门设计用于安全存储和管理敏感信息（如数据库凭证）的服务。
支持自动轮换数据库凭证，符合题目要求的“频繁轮换用户凭证”。
通过IAM权限可以安全地控制Web服务器对Secrets Manager的访问。
B. Store the database user credentials in AWS Systems Manager OpsCenter. Grant the necessary IAM permissions to allow
the web servers to access OpsCenter.
错误：AWS Systems Manager OpsCenter主要用于管理和解决操作问题，不适合存储和管理数据库凭证。
OpsCenter不支持自动轮换数据库凭证。
C. Store the database user credentials in a secure Amazon S3 bucket. Grant the necessary IAM permissions to allow the
web servers to retrieve credentials and access the database.
错误：虽然可以将凭证存储在加密的S3 bucket中，但S3不是专门设计用于管理敏感信息的服务。
不支持自动轮换数据库凭证，且管理复杂性较高。
D. Store the database user credentials in files encrypted with AWS Key Management Service (AWS KMS) on the web server
file system. The web server should be able to decrypt the files and access the database.
错误：将凭证存储在Web服务器文件系统中增加了安全风险。
虽然可以使用AWS KMS加密文件，但不支持自动轮换数据库凭证。

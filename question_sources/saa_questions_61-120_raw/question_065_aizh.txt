Lambda to upload reports that are in PDF format and JPEG format. The hospital needs to modify the Lambda code to identify
protected health information (PHI) in the reports.
Which solution will meet these requirements with the LEAST operational overhead?

题目分析与解答
1. 考察的知识点
Correct Answer: C
正确答案：C
Community vote distribution
社区投票分布
C (100%)
Amazon Textract：用于从文档中提取文本和数据。
Amazon Comprehend Medical：用于识别医疗相关的实体和敏感信息（如PHI）。
Amazon Rekognition：用于图像和视频分析，但不适合提取文档中的文本。
Amazon SageMaker：用于构建、训练和部署机器学习模型，但需要额外的开发工作。
2. 选项分析
A. 使用现有的 Python 库从报告中提取文本，并从提取的文本中识别 PHI。
错误：虽然可以使用 Python 库提取文本并识别 PHI，但这需要开发和维护额外的代码，增加了操作复杂性。
B. 使用 Amazon Textract 从报告中提取文本。使用 Amazon SageMaker 从提取的文本中识别 PHI。
错误：Amazon Textract 可以提取文本，但使用 Amazon SageMaker 识别 PHI需要构建和训练自定义机器学习模型，增加了操作复
杂性。
C. 使用 Amazon Textract 从报告中提取文本。使用 Amazon Comprehend Medical 从提取的文本中识别 PHI。
正确：Amazon Textract 专门用于提取文档中的文本，Amazon Comprehend Medical 可以直接识别医疗相关的敏感信息（如
PHI），无需额外开发工作，操作开销最小。
D. 使用 Amazon Rekognition 从报告中提取文本。使用 Amazon Comprehend Medical 从提取的文本中识别 PHI。
错误：Amazon Rekognition 主要用于图像和视频分析，不适合从PDF或JPEG文档中提取文本。
3. 我的答案
C. 使用 Amazon Textract 从报告中提取文本。使用 Amazon Comprehend Medical 从提取的文本中识别 PHI。

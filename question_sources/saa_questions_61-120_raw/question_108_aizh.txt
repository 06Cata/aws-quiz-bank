题目分析与解答
1. 考察的知识点
Amazon RDS事件通知机制
AWS Lambda的触发器功能
Amazon SQS队列的使用
系统设计中消息队列的应用
2. 选项分析
A. Create an AWS Lambda function triggered when the database on Amazon RDS is updated to send the information to an
Amazon Simple Queue Service (Amazon SQS) queue for the targets to consume.
分析：此选项设计合理，使用AWS Lambda函数监听数据库更新事件，并将数据发送到Amazon SQS队列供目标系统消费。SQS队列
可以有效解耦系统，支持多目标消费。此选项符合题目需求。
B. Create an AWS Lambda function triggered when the database on Amazon RDS is updated to send the information to an
Amazon Simple Queue Service (Amazon SQS) FIFO queue for the targets to consume.
分析：此选项与A类似，但使用了FIFO队列。FIFO队列主要用于需要严格顺序和去重的场景，而题目并未提到需要顺序处理或去
重，因此使用FIFO队列可能会增加不必要的复杂性。
C. Subscribe to an RDS event notification and send an Amazon Simple Queue Service (Amazon SQS) queue fanned out to
multiple Amazon Simple Notification Service (Amazon SNS) topics. Use AWS Lambda functions to update the targets.
分析：此选项设计复杂，涉及RDS事件通知、SQS队列和SNS主题的组合。虽然可以实现目标，但题目并未要求使用SNS主题进行分
发，且设计过于复杂，不符合最佳实践。
D. Subscribe to an RDS event notification and send an Amazon Simple Notification Service (Amazon SNS) topic fanned
out to multiple Amazon Simple Queue Service (Amazon SQS) queues. Use AWS Lambda functions to update the targets.
分析：此选项与C类似，使用了SNS主题和SQS队列的组合。虽然可以实现目标，但设计复杂且不必要，题目并未要求使用SNS主题
进行分发。

A company is preparing to store confidential data in Amazon S3. For compliance reasons, the data must be encrypted at rest.
Encryption key usage must be logged for auditing purposes. Keys must be rotated every year.
Which solution meets these requirements and is the MOST operationally efficient?
一家公司正在准备将机密数据存储在 Amazon S3 中。出于合规性原因，数据必须在静止状态下加密。加密密钥的使用必须被记录
以进行审计。密钥必须每年轮换一次。
哪种解决方案满足这些要求并且是最具操作效率的？
A. Server-side encryption with customer-provided keys (SSE-C)
使用客户提供的密钥进行服务器端加密 (SSE-C)
B. Server-side encryption with Amazon S3 managed keys (SSE-S3)
使用 Amazon S3 管理密钥进行服务器端加密 (SSE-S3)
C. Server-side encryption with AWS KMS keys (SSE-KMS) with manual rotation
使用 AWS KMS 密钥进行服务器端加密 (SSE-KMS) 并手动轮换
D. Server-side encryption with AWS KMS keys (SSE-KMS) with automatic rotation Most Voted
使用 AWS KMS 密钥进行服务器端加密 (SSE-KMS) 并自动轮换
题目分析与解答
1. 考察的知识点
Amazon S3的加密选项，包括SSE-C、SSE-S3、SSE-KMS。
AWS Key Management Service (KMS)的功能，包括密钥使用日志记录和自动轮换。
如何满足合规性要求（加密、审计、密钥轮换）。

A. Server-side encryption with customer-provided keys (SSE-C)
错误：SSE-C要求客户自己管理密钥，包括密钥的轮换和审计。这增加了操作复杂性，不符合“最具操作效率”的要求。
B. Server-side encryption with Amazon S3 managed keys (SSE-S3)
错误：SSE-S3使用Amazon S3管理的密钥，但不支持密钥使用日志记录和轮换功能，因此无法满足审计和密钥轮换的要求。
C. Server-side encryption with AWS KMS keys (SSE-KMS) with manual rotation
错误：虽然SSE-KMS支持密钥使用日志记录，但手动轮换密钥需要额外的操作步骤，增加了操作复杂性，不符合“最具操作效率”的
要求。
D. Server-side encryption with AWS KMS keys (SSE-KMS) with automatic rotation
正确：SSE-KMS支持密钥使用日志记录，自动轮换功能减少了操作复杂性，完全满足题目要求。
3. 我选的答案
D. Server-side encryption with AWS KMS keys (SSE-KMS) with automatic rotation
4. 官方答案
D. Server-side encryption with AWS KMS keys (SSE-KMS) with automatic rotation

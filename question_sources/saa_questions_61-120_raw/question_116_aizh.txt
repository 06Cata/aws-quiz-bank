maintenance are burdensome. The company is redesigning its website and wants anew solution. The website will be updated four
times a year and does not need to have any dynamic content available. The solution must provide high scalability and enhanced
security.
Which combination of changes will meet these requirements with the LEAST operational overhead? (Choose two.)

网站，并希望采用一个新的解决方案。该网站每年更新四次，并且不需要提供任何动态内容。该解决方案必须具备高扩展性和增强

题目分析与解答
1. 考察的知识点
Amazon S3 静态网站托管的功能与优势。
Amazon CloudFront 的内容分发与 HTTPS 功能。
AWS WAF 的功能与适用场景。
AWS Lambda 的使用场景与限制。
EC2 Auto Scaling 和 Application Load Balancer 的适用场景。
2. 选项分析
A. Configure Amazon CloudFront in front of the website to use HTTPS functionality.
正确：Amazon CloudFront 是一个全球内容分发网络 (CDN)，可以提供高可用性和低延迟，同时支持 HTTPS 功能以增强安全性。
符合题目要求的高扩展性和增强安全性。
B. Deploy an AWS WAF web ACL in front of the website to provide HTTPS functionality.
错误：AWS WAF 是一个 Web 应用防火墙，用于保护网站免受常见的网络攻击，但它并不直接提供 HTTPS 功能。
题目没有提到需要防护复杂的网络攻击，因此此选项不符合最低操作开销的要求。
C. Create and deploy an AWS Lambda function to manage and serve the website content.
错误：AWS Lambda 适用于事件驱动的计算任务，而不是托管静态网站内容。使用 Lambda 来管理和提供网站内容会增加复杂性和
操作开销。
不符合题目要求的最低操作开销。
D. Create the new website and an Amazon S3 bucket. Deploy the website on the S3 bucket with static website hosting
enabled.
正确：Amazon S3 支持静态网站托管，适合托管无需动态内容的静态网站。它具有高扩展性和低操作开销。
符合题目要求的最低操作开销和高扩展性。
E. Create the new website. Deploy the website by using an Auto Scaling group of Amazon EC2 instances behind an
Application Load Balancer.
错误：使用 EC2 和 Auto Scaling 部署静态网站会增加操作开销，且不必要，因为静态网站可以直接托管在 S3 上。
不符合题目要求的最低操作开销。

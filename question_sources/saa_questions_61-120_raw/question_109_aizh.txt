A company needs to store data in Amazon S3 and must prevent the data from being changed. The company wants new objects
that are uploaded to Amazon S3 to remain unchangeable for a nonspecific amount of time until the company decides to modify
the objects. Only specific users in the company's AWS account can have the ability 10 delete the objects.
What should a solutions architect do to meet these requirements?
一家公司需要将数据存储在 Amazon S3，并且必须防止数据被更改。该公司希望上传到 Amazon S3 的新对象在未指定的时间内保
持不可更改状态，直到公司决定修改这些对象。只有公司 AWS 账户中的特定用户可以删除这些对象。
解决方案架构师应该怎么做以满足这些要求？
A. Create an S3 Glacier vault. Apply a write-once, read-many (WORM) vault lock policy to the objects.
创建一个 S3 Glacier vault。对对象应用写一次、读多次 (WORM) vault lock 策略。
B. Create an S3 bucket with S3 Object Lock enabled. Enable versioning. Set a retention period of 100 years. Use governance
mode as the S3 bucket’s default retention mode for new objects.
创建一个启用了 S3 Object Lock 的 S3 bucket。启用版本控制。设置保留期为 100 年。使用治理模式作为 S3 bucket 的新对象默
认保留模式。
C. Create an S3 bucket. Use AWS CloudTrail to track any S3 API events that modify the objects. Upon notification, restore the
modified objects from any backup versions that the company has.
创建一个 S3 bucket。使用 AWS CloudTrail 跟踪任何修改对象的 S3 API 事件。在收到通知后，从公司拥有的任何备份版本中恢
复被修改的对象。

题目分析与解答
1. 考察的知识点
Amazon S3 Object Lock功能，包括治理模式和法律保留（Legal Hold）。
如何使用IAM权限控制特定用户对S3对象的操作权限。
数据保护与合规性相关的AWS服务功能。
2. 选项分析
A. 创建一个 S3 Glacier vault。对对象应用写一次、读多次 (WORM) vault lock 策略。
错误：S3 Glacier vault lock是用于归档数据的长期存储解决方案，适合需要不可变性和合规性要求的归档数据，但题目要求的是针
对S3对象的不可变性，而不是归档数据。
B. 创建一个启用了 S3 Object Lock 的 S3 bucket。启用版本控制。设置保留期为 100 年。使用治理模式作为 S3 bucket 的新对象
默认保留模式。
部分正确：S3 Object Lock和治理模式可以防止对象被修改，但设置固定的保留期（100年）不符合题目要求，因为题目明确提到需
要“非特定时间”直到公司决定修改对象。
C. 创建一个 S3 bucket。使用 AWS CloudTrail 跟踪任何修改对象的 S3 API 事件。在收到通知后，从公司拥有的任何备份版本中恢
复被修改的对象。
错误：CloudTrail只能跟踪事件，无法主动防止对象被修改。依赖备份恢复并不能满足题目要求的“防止数据被更改”。
D. 创建一个启用了 S3 Object Lock 的 S3 bucket。启用版本控制。对对象添加法律保留。将 s3:PutObjectLegalHold 权限添加到
需要删除对象的用户的 IAM 策略中。
正确：S3 Object Lock的法律保留（Legal Hold）功能允许对象保持不可变状态，直到明确移除法律保留。通过IAM权限控制，只有
特定用户可以删除对象，完全符合题目要求。

D

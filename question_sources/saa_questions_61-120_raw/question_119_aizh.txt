A global company is using Amazon API Gateway to design REST APIs for its loyalty club users in the us-east-1 Region and the ap-
southeast-2 Region. A solutions architect must design a solution to protect these API Gateway managed REST APIs across multiple
accounts from SQL injection and cross-site scripting attacks.
Which solution will meet these requirements with the LEAST amount of administrative effort?

2. 选项分析
A. Set up AWS WAF in both Regions. Associate Regional web ACLs with an API stage.
正确性分析：AWS WAF 可以保护 API Gateway 阶段免受 SQL 注入和跨站脚本攻击，但需要在每个区域分别设置 WAF 和 Web
ACL，这增加了管理工作量。
错误原因：虽然可以实现保护，但不符合“最少的管理工作量”要求。
B. Set up AWS Firewall Manager in both Regions. Centrally configure AWS WAF rules.
正确性分析：AWS Firewall Manager 是一个集中管理工具，可以跨账户和跨区域统一管理 AWS WAF 规则，减少了管理工作量。
正确原因：符合题目要求的“最少的管理工作量”，并且能够有效保护 API Gateway。
C. Set up AWS Shield in both Regions. Associate Regional web ACLs with an API stage.
正确性分析：AWS Shield 是针对 DDoS 攻击的保护服务，主要用于防御分布式拒绝服务攻击。
错误原因：题目要求保护 API Gateway 免受 SQL 注入和跨站脚本攻击，而这些攻击属于应用层攻击，AWS Shield 并不适合解决这
些问题。
D. Set up AWS Shield in one of the Regions. Associate Regional web ACLs with an API stage.
正确性分析：同选项 C，AWS Shield 主要用于防御 DDoS 攻击。
错误原因：无法解决 SQL 注入和跨站脚本攻击问题，且只在一个区域设置 Shield 无法满足跨区域保护需求。

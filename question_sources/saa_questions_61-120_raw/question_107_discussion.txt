A bicycle sharing company is developing a multi-tier architecture to track the location of its bicycles during peak operating hours.
The company wants to use these data points in its existing analytics platform. A solutions architect must determine the most
viable multi-tier option to support this architecture. The data points must be accessible from the REST API.
Which action meets these requirements for storing and retrieving location data?
A. Use Amazon Athena with Amazon S3.
B. Use Amazon API Gateway with AWS Lambda. Most Voted
C. Use Amazon QuickSight with Amazon Redshift.
D. Use Amazon API Gateway with Amazon Kinesis Data Analytics.

Question #723
A company has applications that run on Amazon EC2 instances. The EC2 instances connect to Amazon RDS databases by using an
IAM role that has associated policies. The company wants to use AWS Systems Manager to patch the EC2 instances without
disrupting the running applications.
Which solution will meet these requirements?
一家公司有运行在 Amazon EC2 实例上的应用程序。这些 EC2 实例通过使用一个具有相关策略的 IAM 角色连接到 Amazon RDS 数
据库。该公司希望使用 AWS Systems Manager 来修补 EC2 实例，而不会中断正在运行的应用程序。
哪种解决方案可以满足这些要求？
A. Create a new IAM role. Attach the AmazonSSMManagedInstanceCore policy to the new IAM role. Attach the new IAM role
to the EC2 instances and the existing IAM role.
创建一个新的 IAM 角色。将 AmazonSSMManagedInstanceCore 策略附加到新的 IAM 角色。将新的 IAM 角色附加到 EC2 实例
和现有的 IAM 角色。
B. Create an IAM user. Attach the AmazonSSMManagedInstanceCore policy to the IAM user. Configure Systems Manager to
use the IAM user to manage the EC2 instances.
创建一个 IAM 用户。将 AmazonSSMManagedInstanceCore 策略附加到 IAM 用户。配置 Systems Manager 使用该 IAM 用户来
管理 EC2 实例。
C. Enable Default Host Configuration Management in Systems Manager to manage the EC2 instances. Most Voted
在 Systems Manager 中启用默认主机配置管理以管理 EC2 实例。
D. Remove the existing policies from the existing IAM role. Add the AmazonSSMManagedInstanceCore policy to the existing
IAM role.
从现有的 IAM 角色中移除现有策略。将 AmazonSSMManagedInstanceCore 策略添加到现有的 IAM 角色。
Correct Answer: C
正确答案：C

题目分析与解答
1. 考察的知识点
本题主要考察AWS Systems Manager的使用，特别是如何配置EC2实例以便通过Systems Manager进行管理和补丁操作。
涉及IAM角色和策略的正确配置，以及如何确保应用程序不中断的情况下进行管理。
2. 选项分析
A. 创建一个新的 IAM 角色。将 AmazonSSMManagedInstanceCore 策略附加到新的 IAM 角色。将新的 IAM 角色附加到 EC2 实
例和现有的 IAM 角色。
错误：AWS IAM角色不能直接附加到另一个IAM角色。IAM角色是为特定资源或服务设计的，不能组合使用。
此外，创建新角色并附加到现有角色的操作不符合题目要求。
B. 创建一个 IAM 用户。将 AmazonSSMManagedInstanceCore 策略附加到 IAM 用户。配置 Systems Manager 使用该 IAM 用
户来管理 EC2 实例。
错误：AWS Systems Manager需要通过IAM角色来管理EC2实例，而不是通过IAM用户。IAM用户通常用于人类用户或程序访问，而
不是用于实例管理。
C. 启用默认主机配置管理以管理 EC2 实例。
正确：AWS Systems Manager提供了默认主机配置管理功能，可以自动管理EC2实例，包括补丁管理。启用此功能可以满足题目要
求。
D. 从现有的 IAM 角色中移除现有策略。将 AmazonSSMManagedInstanceCore 策略添加到现有的 IAM 角色。
错误：移除现有策略可能会导致EC2实例无法访问RDS数据库，从而中断应用程序的运行。这与题目要求的“不影响运行的应用程
序”相矛盾。

运行。

Question #710
A company’s website hosted on Amazon EC2 instances processes classified data stored in Amazon S3. Due to security concerns,
the company requires a private and secure connection between its EC2 resources and Amazon S3.
Which solution meets these requirements?
一孰公司的网站孋管在 Amazon EC2 孩例上，处理孓储在 Amazon S3 中的机孶数据。由于孞全问题，公司需要在其 EC2 资源和
Amazon S3 之间寠立一个私孶且孞全的连接。
哪种解决方案满足这些要求？
A. Set up S3 bucket policies to allow access from a VPC endpoint. Most Voted
设置 S3 bucket policies 以允许从 VPC endpoint 访问。
B. Set up an IAM policy to grant read-write access to the S3 bucket.
设置 IAM policy 以授予孼 S3 bucket 的读写访问权限。
C. Set up a NAT gateway to access resources outside the private subnet.
设置 NAT gateway 以访问私有子网外部的资源。
D. Set up an access key ID and a secret access key to access the S3 bucket.
设置 access key ID 和 secret access key 以访问 S3 bucket。

Amazon S3与Amazon EC2之间的孞全连接。
使用VPC Endpoint孩现私有连接。
Correct Answer: A

S3 Bucket Policy的配置。
2. 选项分析
A. Set up S3 bucket policies to allow access from a VPC endpoint.
正确：VPC Endpoint允许EC2孩例通过私有网络访问S3，而无需通过公共互联网。设置S3 Bucket Policy可以进一步限制访问权限，
确保只有通过VPC Endpoint的流量可以访问S3。
B. Set up an IAM policy to grant read-write access to the S3 bucket.
错误：IAM Policy可以控制用孀嬻角色孼S3的访问权限，但孚无法确保连接是私有的。题目要求“私有和孞全连接”，仅使用IAM
Policy无法满足这一要求。
C. Set up a NAT gateway to access resources outside the private subnet.
错误：NAT Gateway用于允许私有子网中的资源访问公共互联网，但题目要求“私有和孞全连接”，使用NAT Gateway会通过公共互
联网访问S3，不符合要求。
D. Set up an access key ID and a secret access key to access the S3 bucket.
错误：Access Key ID和Secret Access Key用于身份验证，但孚们无法确保连接是私有的。题目要求“私有和孞全连接”，仅使用这些
凭证无法满足这一要求。
3. 我的答案
A. Set up S3 bucket policies to allow access from a VPC endpoint.
4. 官方答案
A. Set up S3 bucket policies to allow access from a VPC endpoint.

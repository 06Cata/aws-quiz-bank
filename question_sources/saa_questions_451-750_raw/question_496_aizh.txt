Question #496
A company uses on-premises servers to host its applications. The company is running out of storage capacity. The applications use
both block storage and NFS storage. The company needs a high-performing solution that supports local caching without re-
architecting its existing applications.
Which combination of actions should a solutions architect take to meet these requirements? (Choose two.)
一家公司使用本地服务器托管其应用程序。该公司存储容量即将耗尽。这些应用程序同时使用块存储和 NFS 存储。该公司需要一个
支持本地缓存的高性能解决方案，而无需重新设计其现有应用程序。
解决方案架构师应该采取哪些操作来满足这些需求？（选择两个。）
A. Mount Amazon S3 as a file system to the on-premises servers.
将 Amazon S3 挂载为文件系统到本地服务器。
B. Deploy an AWS Storage Gateway file gateway to replace NFS storage. Most Voted
部署一个 AWS Storage Gateway 文件网关以替换 NFS 存储。
C. Deploy AWS Snowball Edge to provision NFS mounts to on-premises servers.
部署 AWS Snowball Edge 以向本地服务器提供 NFS 挂载。
D. Deploy an AWS Storage Gateway volume gateway to replace the block storage. Most Voted
部署一个 AWS Storage Gateway 卷网关以替换块存储。
E. Deploy Amazon Elastic File System (Amazon EFS) volumes and mount them to on-premises servers.
部署 Amazon Elastic File System (Amazon EFS) 卷并将其挂载到本地服务器。
题目分析与解答
1. 考察的知识点
Correct Answer: BD
正确答案：BD
Community vote distribution
社区投票分布
BD (100%)
本题考察的是AWS存储服务的使用场景，包括AWS Storage Gateway、Amazon S3、Amazon EFS等服务的功能和适用场景。
重点关注如何解决本地存储容量不足的问题，同时支持高性能和本地缓存。
2. 选项分析
A. Mount Amazon S3 as a file system to the on-premises servers.
Amazon S3是对象存储服务，不支持直接挂载为文件系统。
虽然可以通过第三方工具（如s3fs）实现挂载，但性能和可靠性无法满足高性能和本地缓存的需求。
此选项错误。
B. Deploy an AWS Storage Gateway file gateway to replace NFS storage.
AWS Storage Gateway文件网关支持NFS协议，可以无缝集成到现有的应用中。
它提供本地缓存功能，适合解决存储容量不足的问题。
此选项正确。
C. Deploy AWS Snowball Edge to provision NFS mounts to on-premises servers.
AWS Snowball Edge主要用于数据迁移和边缘计算，不是长期存储解决方案。
虽然支持NFS挂载，但不符合题目要求的“高性能解决方案”和“支持本地缓存”的需求。
此选项错误。
D. Deploy an AWS Storage Gateway volume gateway to replace the block storage.
AWS Storage Gateway卷网关支持块存储，可以无缝集成到现有的应用中。
它提供本地缓存功能，适合解决存储容量不足的问题。
此选项正确。
E. Deploy Amazon Elastic File System (Amazon EFS) volumes and mount them to on-premises servers.
Amazon EFS是云端文件存储服务，主要用于AWS云环境中的应用。
虽然可以通过VPN或Direct Connect挂载到本地服务器，但性能和延迟可能无法满足高性能需求。
此选项错误。
3. 我选的答案
B. Deploy an AWS Storage Gateway file gateway to replace NFS storage.
D. Deploy an AWS Storage Gateway volume gateway to replace the block storage.
4. 官方答案
B. Deploy an AWS Storage Gateway file gateway to replace NFS storage.
D. Deploy an AWS Storage Gateway volume gateway to replace the block storage.
5. 与官方答案的比较
我的答案与官方答案一致。
分析表明，AWS Storage Gateway的文件网关和卷网关是解决本地存储容量不足问题的昀佳选择，同时支持高性能和本地缓存。

题目分析与解答
1. 考察的知识点
Amazon S3加密机制，包括SSE-C（Server-Side Encryption with Customer-Provided Keys）。
如何处理现有未加密对象并确保未来对象加密。
使用S3 Batch Operations和S3默认加密功能。
2. 选项分析
A. Create a list of unencrypted objects by filtering an Amazon S3 Inventory report. Configure an S3 Batch Operations job
to encrypt the objects from the list with a server-side encryption with a customer-provided key (SSE-C). Configure the S3
default encryption feature to use a server-side encryption with a customer-provided key (SSE-C).
正确：此选项使用Amazon S3 Inventory报告来识别未加密的对象，并通过S3 Batch Operations对这些对象进行加密，符合要求。
正确：配置S3默认加密功能以使用SSE-C确保未来对象加密，满足题目要求。
B. Use S3 Storage Lens metrics to identify unencrypted S3 buckets. Configure the S3 default encryption feature to use a
server-side encryption with AWS KMS keys (SSE-KMS).
错误：S3 Storage Lens主要用于监控和分析存储使用情况，而不是直接识别未加密对象。
错误：题目要求使用客户提供的密钥（SSE-C），而此选项使用AWS KMS密钥（SSE-KMS），不符合要求。
C. Create a list of unencrypted objects by filtering the AWS usage report for Amazon S3. Configure an AWS Batch job to
encrypt the objects from the list with a server-side encryption with AWS KMS keys (SSE-KMS). Configure the S3 default
encryption feature to use a server-side encryption with AWS KMS keys (SSE-KMS).
错误：AWS使用报告不适合直接过滤未加密对象。
错误：题目要求使用客户提供的密钥（SSE-C），而此选项使用AWS KMS密钥（SSE-KMS），不符合要求。
D. Create a list of unencrypted objects by filtering the AWS usage report for Amazon S3. Configure the S3 default
encryption feature to use a server-side encryption with a customer-provided key (SSE-C).
错误：AWS使用报告不适合直接过滤未加密对象。
正确：配置S3默认加密功能以使用SSE-C确保未来对象加密，但未处理现有未加密对象，不完全满足要求。

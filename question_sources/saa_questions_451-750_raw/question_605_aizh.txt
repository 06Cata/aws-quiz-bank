Question #605
A company has several on-premises Internet Small Computer Systems Interface (ISCSI) network storage servers. The company
wants to reduce the number of these servers by moving to the AWS Cloud. A solutions architect must provide low-latency access
to frequently used data and reduce the dependency on on-premises servers with a minimal number of infrastructure changes.
Which solution will meet these requirements?
一家公司有多个本地的互联网小型计算机系统接口（ISCSI）网络存储服务器。该公司希望通过迁移到AWS云来减少这些服务器的数
量。解决方案架构师必须提供对频繁使用数据的低延迟访问，并尽量减少对本地服务器的依赖，同时进行最少的基础设施更改。
哪种解决方案可以满足这些要求？
A. Deploy an Amazon S3 File Gateway.
部署一个 Amazon S3 File Gateway。
B. Deploy Amazon Elastic Block Store (Amazon EBS) storage with backups to Amazon S3.
部署 Amazon Elastic Block Store (Amazon EBS) 存储，并将备份存储到 Amazon S3。
C. Deploy an AWS Storage Gateway volume gateway that is configured with stored volumes.
部署一个配置了存储卷的 AWS Storage Gateway 卷网关。
D. Deploy an AWS Storage Gateway volume gateway that is configured with cached volumes. Most Voted
部署一个配置了缓存卷的 AWS Storage Gateway 卷网关。
题目分析与解答
1. 考察的知识点
本题考察的是AWS Storage Gateway的使用场景，特别是如何在混合云环境中实现低延迟访问和减少对本地存储的依赖。
Correct Answer: D
正确答案：D
Community vote distribution
社区投票分布
D (100%)
涉及的服务包括Amazon S3 File Gateway、Amazon EBS、AWS Storage Gateway（存储卷和缓存卷）。
2. 选项分析
A. Deploy an Amazon S3 File Gateway.
Amazon S3 File Gateway主要用于将文件存储到Amazon S3，同时提供对文件的低延迟访问。
然而，本题要求低延迟访问频繁使用的数据，并减少对本地服务器的依赖。S3 File Gateway不适合ISCSI协议的场景，因此此选项错
误。
B. Deploy Amazon Elastic Block Store (Amazon EBS) storage with backups to Amazon S3.
Amazon EBS是块存储服务，适用于EC2实例，但它无法直接与本地ISCSI协议集成。
此外，EBS无法提供对本地服务器的低延迟访问，因此此选项错误。
C. Deploy an AWS Storage Gateway volume gateway that is configured with stored volumes.
存储卷模式将数据存储在本地，并定期备份到AWS云端。
虽然存储卷可以提供低延迟访问，但它仍然依赖本地存储设备，无法减少对本地服务器的依赖，因此此选项错误。
D. Deploy an AWS Storage Gateway volume gateway that is configured with cached volumes.
缓存卷模式将频繁访问的数据缓存在本地，同时将主数据存储在AWS云端。
这种模式可以提供低延迟访问，并减少对本地存储服务器的依赖，符合题目要求，因此此选项正确。
3. 我选的答案
D. Deploy an AWS Storage Gateway volume gateway that is configured with cached volumes.
4. 官方答案
D. Deploy an AWS Storage Gateway volume gateway that is configured with cached volumes.
5. 与官方答案比较
我的答案与官方答案一致。
原因分析：缓存卷模式能够满足题目中对低延迟访问和减少本地服务器依赖的要求，是最优解。

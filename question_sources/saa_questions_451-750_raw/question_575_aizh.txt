Question #575
A company deploys its applications on Amazon Elastic Kubernetes Service (Amazon EKS) behind an Application Load Balancer in
an AWS Region. The application needs to store data in a PostgreSQL database engine. The company wants the data in the
database to be highly available. The company also needs increased capacity for read workloads.
Which solution will meet these requirements with the MOST operational efficiency?
一家公司在一个AWS区域内，通过Amazon Elastic Kubernetes Service (Amazon EKS) 部署其应用程序，并使用Application Load
Balancer作为后端。该应用程序需要将数据存储在PostgreSQL数据库引擎中。公司希望数据库中的数据具有高可用性。公司还需要
提高读取工作负载的容量。
哪种解决方案能够以最高的操作效率满足这些需求？
A. Create an Amazon DynamoDB database table configured with global tables.
创建一个配置了全球表的Amazon DynamoDB数据库表。
B. Create an Amazon RDS database with Multi-AZ deployments.
创建一个具有Multi-AZ部署的Amazon RDS数据库。
C. Create an Amazon RDS database with Multi-AZ DB cluster deployment. Most Voted
创建一个具有Multi-AZ DB集群部署的Amazon RDS数据库。
D. Create an Amazon RDS database configured with cross-Region read replicas.
创建一个配置了跨区域读取副本的Amazon RDS数据库。
题目分析与解答
1. 考察的知识点
Correct Answer: C
正确答案：C
Community vote distribution
社区投票分布
C (100%)
Amazon RDS的高可用性和扩展性特性，包括Multi-AZ部署和读副本。
如何选择适合的数据库解决方案以满足高可用性和读扩展需求。
2. 选项分析
A. Create an Amazon DynamoDB database table configured with global tables.
错误：DynamoDB是一个NoSQL数据库，而题目明确要求使用PostgreSQL数据库引擎（关系型数据库）。
虽然DynamoDB的全球表可以提供高可用性和跨区域扩展，但它不符合题目要求的数据库类型。
B. Create an Amazon RDS database with Multi-AZ deployments.
部分正确：Multi-AZ部署可以提供高可用性，但它主要用于故障转移，而不是扩展读工作负载。
此选项没有提到读副本，因此无法满足“增加读工作负载容量”的需求。
C. Create an Amazon RDS database with Multi-AZ DB cluster deployment.
正确：Multi-AZ DB集群部署不仅提供高可用性，还支持多个读节点，从而增加了读工作负载的容量。
这是满足题目要求的最佳解决方案，同时具有高操作效率。
D. Create an Amazon RDS database configured with cross-Region read replicas.
错误：跨区域读副本主要用于跨区域扩展，而题目没有提到跨区域需求。
此外，跨区域读副本的设置和管理复杂性较高，操作效率较低。
3. 我的答案
C. Create an Amazon RDS database with Multi-AZ DB cluster deployment.
4. 官方答案
C. Create an Amazon RDS database with Multi-AZ DB cluster deployment.
5. 与官方答案比较
我的答案与官方答案一致。
原因：Multi-AZ DB集群部署是满足高可用性和读扩展需求的最佳解决方案，同时具有较高的操作效率。

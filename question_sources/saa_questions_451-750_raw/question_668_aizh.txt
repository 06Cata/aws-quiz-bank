Question #668
A company created a new organization in AWS Organizations. The organization has multiple accounts for the company's
development teams. The development team members use AWS IAM Identity Center (AWS Single Sign-On) to access the accounts.
For each of the company's applications, the development teams must use a predefined application name to tag resources that are
created.
A solutions architect needs to design a solution that gives the development team the ability to create resources only if the
application name tag has an approved value.
Which solution will meet these requirements?
一家公司在 AWS Organizations 中创建了一个新的组织。该组织为公司的多个开发团队创建了多个账户。开发团队成员使用 AWS
IAM Identity Center (AWS Single Sign-On) 来访问这些账户。对于公司的每个应用程序，开发团队必须使用预定义的应用程序名称
来标记创建的资源。
一位解决方案架构师需要设计一个解决方案，使开发团队能够仅在应用程序名称标签具有批准值时创建资源。
哪种解决方案可以满足这些要求？
A. Create an IAM group that has a conditional Allow policy that requires the application name tag to be specified for resources
to be created.
创建一个 IAM 组，该组具有一个条件允许策略，要求在创建资源时指定应用程序名称标签。
B. Create a cross-account role that has a Deny policy for any resource that has the application name tag.
创建一个跨账户角色，该角色对任何具有应用程序名称标签的资源具有拒绝策略。
C. Create a resource group in AWS Resource Groups to validate that the tags are applied to all resources in all accounts.
在 AWS Resource Groups 中创建一个资源组，以验证标签是否已应用于所有账户中的所有资源。
D. Create a tag policy in Organizations that has a list of allowed application names. Most Voted
在 Organizations 中创建一个标签策略，该策略具有允许的应用程序名称列表。
Correct Answer: D
正确答案：D
题目分析与解答
1. 考察的知识点
本题考察的是AWS Organizations中的标签策略（Tag Policies）的使用，以及如何通过标签策略来强制资源标签的合规性。
还涉及IAM权限管理和跨账户访问的相关知识。
2. 选项分析
A. Create an IAM group that has a conditional Allow policy that requires the application name tag to be specified for
resources to be created.
错误：IAM组的策略可以限制用户的操作，但无法直接强制资源标签的合规性。IAM策略无法验证标签值是否符合预定义的规则。
B. Create a cross-account role that has a Deny policy for any resource that has the application name tag.
错误：Deny策略的作用是阻止某些操作，但题目要求的是强制资源标签的值符合预定义规则，而不是简单地拒绝操作。
C. Create a resource group in AWS Resource Groups to validate that the tags are applied to all resources in all accounts.
错误：AWS Resource Groups用于组织和管理资源，但它无法强制资源标签的合规性。它只能帮助用户查看和管理资源的标签。
D. Create a tag policy in Organizations that has a list of allowed application names.
正确：AWS Organizations中的标签策略（Tag Policies）可以强制资源标签的合规性，并确保标签值符合预定义的规则。这正是题目
要求的解决方案。

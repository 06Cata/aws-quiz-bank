Question #616
A company has deployed its newest product on AWS. The product runs in an Auto Scaling group behind a Network Load Balancer.
The company stores the product’s objects in an Amazon S3 bucket.
The company recently experienced malicious attacks against its systems. The company needs a solution that continuously
monitors for malicious activity in the AWS account, workloads, and access patterns to the S3 bucket. The solution must also report
suspicious activity and display the information on a dashboard.
Which solution will meet these requirements?

题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的安全监控能力，包括Amazon GuardDuty、Amazon Macie、Amazon Inspector、AWS Config等服务的功能
和适用场景。
重点关注如何监控AWS账户、工作负载以及S3访问模式的恶意活动，并将信息汇总到一个可视化的仪表板。
2. 选项分析
A. Configure Amazon Macie to monitor and report findings to AWS Config.
Amazon Macie主要用于识别和保护敏感数据（如PII）存储在S3中的情况，但它不适合全面监控AWS账户和工作负载的恶意活动。
将发现报告给AWS Config也不符合题目要求，因为AWS Config主要用于资源配置的合规性检查，而不是安全事件的监控和仪表板展
示。
错误原因：功能不匹配，无法满足题目要求。
B. Configure Amazon Inspector to monitor and report findings to AWS CloudTrail.
Amazon Inspector主要用于评估EC2实例的安全漏洞和应用程序的安全性，但它不监控账户级别的恶意活动或S3访问模式。
将发现报告给AWS CloudTrail也不符合题目要求，因为CloudTrail主要记录API调用日志，而不是提供仪表板展示。
错误原因：功能范围有限，无法满足题目要求。
C. Configure Amazon GuardDuty to monitor and report findings to AWS Security Hub.
Amazon GuardDuty是一项专门用于检测AWS账户、工作负载和S3访问模式中的恶意活动的服务。
GuardDuty可以将发现报告到AWS Security Hub，后者提供集中化的安全信息仪表板，符合题目要求。
正确原因：功能完全匹配，能够满足题目要求。
D. Configure AWS Config to monitor and report findings to Amazon EventBridge.
AWS Config主要用于资源配置的合规性检查，而不是监控恶意活动。
将发现报告给Amazon EventBridge也不符合题目要求，因为EventBridge主要用于事件驱动的架构，而不是安全信息的仪表板展示。
错误原因：功能不匹配，无法满足题目要求。
3. 我的选择
答案：C

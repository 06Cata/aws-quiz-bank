题目分析与解答
1. 考察的知识点
AWS Budgets的功能，包括预算创建、预算警报和预算操作。
IAM角色和权限的使用，确保预算操作能够执行。
服务控制策略（SCP）的使用，限制账户资源的创建。
2. 选项分析
A. Use AWS Budgets to create a budget. Set the budget amount under the Cost and Usage Reports section of the required
AWS accounts.
错误：AWS Budgets的预算设置并不在“Cost and Usage Reports”部分，而是在Billing Dashboard中进行。此选项描述不准确。
B. Use AWS Budgets to create a budget. Set the budget amount under the Billing dashboards of the required AWS
accounts.
正确：AWS Budgets的预算创建和设置是在Billing Dashboard中完成的，此选项描述正确。
C. Create an IAM user for AWS Budgets to run budget actions with the required permissions.
错误：AWS Budgets通常使用IAM角色而不是IAM用户来执行预算操作。IAM角色更适合跨账户或服务的权限管理。
D. Create an IAM role for AWS Budgets to run budget actions with the required permissions.
正确：AWS Budgets需要一个IAM角色来执行预算操作，例如触发警报或限制资源创建。此选项描述正确。
E. Add an alert to notify the company when each account meets its budget threshold. Add a budget action that selects the
IAM identity created with the appropriate config rule to prevent provisioning of additional resources.
错误：预算操作通常使用SCP（服务控制策略）来限制资源创建，而不是通过Config规则。此选项描述不准确。
F. Add an alert to notify the company when each account meets its budget threshold. Add a budget action that selects the
IAM identity created with the appropriate service control policy (SCP) to prevent provisioning of additional resources.
正确：SCP是AWS Organizations中用于限制账户行为的有效工具，可以防止超出预算的资源创建。此选项描述正确。

Question #492
A company has multiple AWS accounts for development work. Some staff consistently use oversized Amazon EC2 instances, which
causes the company to exceed the yearly budget for the development accounts. The company wants to centrally restrict the
creation of AWS resources in these accounts.
Which solution will meet these requirements with the LEAST development effort?
一家公司拥有多个用于开发工作的AWS账户。部分员工经常使用过大的Amazon EC2实例，这导致公司超出了开发账户的年度预
算。公司希望集中限制这些账户中AWS资源的创建。
哪种解决方案能够以最少的开发工作量满足这些要求？
A. Develop AWS Systems Manager templates that use an approved EC2 creation process. Use the approved Systems Manager
templates to provision EC2 instances.
开发AWS Systems Manager模板，使用批准的EC2创建流程。使用批准的Systems Manager模板来配置EC2实例。
B. Use AWS Organizations to organize the accounts into organizational units (OUs). Define and attach a service control policy
(SCP) to control the usage of EC2 instance types. Most Voted
使用AWS Organizations将账户组织到组织单元（OUs）中。定义并附加一个服务控制策略（SCP）来控制EC2实例类型的使用。
C. Configure an Amazon EventBridge rule that invokes an AWS Lambda function when an EC2 instance is created. Stop
disallowed EC2 instance types.
配置一个Amazon EventBridge规则，当创建EC2实例时触发一个AWS Lambda函数。停止不允许的EC2实例类型。
D. Set up AWS Service Catalog products for the staff to create the allowed EC2 instance types. Ensure that staff can deploy EC2
instances only by using the Service Catalog products.
为员工设置AWS Service Catalog产品，以创建允许的EC2实例类型。确保员工只能通过使用Service Catalog产品来部署EC2实

题目分析与解答
1. 考察的知识点
本题考察的是AWS资源管理与成本控制，特别是如何使用AWS工具（如AWS Organizations、Service Control Policies、
EventBridge、Systems Manager、Service Catalog）来限制资源创建和优化成本。
2. 选项分析
A. 开发AWS Systems Manager模板，使用批准的EC2创建流程。
错误：虽然Systems Manager可以帮助标准化流程，但它需要额外的开发工作来创建和维护模板，并不能直接限制用户创建不符合
要求的EC2实例。
开发成本较高，不符合“最少开发工作”的要求。
B. 使用AWS Organizations将账户组织到组织单元（OUs）中。定义并附加一个服务控制策略（SCP）来控制EC2实例类型的使
用。
正确：AWS Organizations和SCP是专门设计用于集中管理和限制资源使用的工具。通过SCP可以直接限制用户创建不符合要求的
EC2实例类型。
开发成本低，符合“最少开发工作”的要求。
C. 配置一个Amazon EventBridge规则，当创建EC2实例时触发一个AWS Lambda函数。停止不允许的EC2实例类型。
错误：虽然可以通过EventBridge和Lambda实现资源限制，但这种方法是被动的（事后处理），并且需要额外的开发工作来编写和
维护Lambda函数。
开发成本较高，不符合“最少开发工作”的要求。
D. 为员工设置AWS Service Catalog产品，以创建允许的EC2实例类型。
错误：AWS Service Catalog可以帮助标准化资源创建，但它需要额外的配置工作，并不能完全阻止用户直接创建不符合要求的EC2
实例。
开发成本较高，不符合“最少开发工作”的要求。
3. 我的答案
B

B
5. 与官方答案的比较
我的答案与官方答案一致，选择B。
原因：AWS Organizations和SCP提供了集中管理和限制资源使用的功能，开发成本最低，符合题目要求。

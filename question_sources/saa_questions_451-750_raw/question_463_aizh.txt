Question #463
An IoT company is releasing a mattress that has sensors to collect data about a user’s sleep. The sensors will send data to an
Amazon S3 bucket. The sensors collect approximately 2 MB of data every night for each mattress. The company must process and
summarize the data for each mattress. The results need to be available as soon as possible. Data processing will require 1 GB of
memory and will finish within 30 seconds.
Which solution will meet these requirements MOST cost-effectively?
一家物联网公司正在推出一款带有传感器的床垫，用于收集用户的睡眠数据。 传感器将数据发送到一个 Amazon S3 bucket。 传感
器每晚为每张床垫收集大约 2 MB 的数据。 公司必须处理并汇总每张床垫的数据。 结果需要尽快可用。 数据处理需要 1 GB 的内
存，并将在 30 秒内完成。
哪种解决方案最具成本效益地满足这些需求？

题目分析与解答
1. 考察的知识点
Correct Answer: C
正确答案：C
Community vote distribution
社区投票分布
C (100%)
本题考察的是 AWS 服务的选择与成本优化，尤其是针对数据处理任务的服务选择。
涉及的服务包括 AWS Lambda、AWS Glue、Amazon EMR，以及编程语言的适配性（Python、Scala、PySpark）。
重点在于选择最具成本效益的解决方案，同时满足内存需求和处理时间要求。
2. 选项分析
A. Use AWS Glue with a Scala job
错误：AWS Glue 是一种托管的 ETL 服务，适合处理大规模数据转换任务，但它的启动时间较长（通常需要几分钟），不适合需要
快速处理的任务。
此外，使用 Scala 可能增加开发复杂性，而题目并未要求特定语言。
B. Use Amazon EMR with an Apache Spark script
错误：Amazon EMR 是一种托管的 Hadoop 和 Spark 集群服务，适合处理大规模分布式数据处理任务。
然而，对于 2 MB 的数据量和 30 秒的处理时间要求，EMR 的成本和复杂性过高，不符合“最具成本效益”的要求。
C. Use AWS Lambda with a Python script
正确：AWS Lambda 是一种无服务器计算服务，支持 Python 脚本，能够快速启动并处理小规模数据任务。
Lambda 的最大内存限制为 10 GB，完全满足 1 GB 的内存需求，同时其执行时间限制为 15 分钟，远超 30 秒的要求。
对于 2 MB 的数据量，Lambda 的成本非常低，且无需预置或管理基础设施。
D. Use AWS Glue with a PySpark job
错误：虽然 PySpark 是一种流行的分布式数据处理框架，但 AWS Glue 的启动时间较长，不适合需要快速处理的任务。
此外，Glue 的成本通常高于 Lambda，尤其是对于小规模数据处理任务。
3. 我的选择
答案：C. Use AWS Lambda with a Python script
4. 官方答案
答案：C. Use AWS Lambda with a Python script
5. 与官方答案的比较
我的答案与官方答案一致。
原因：AWS Lambda 是最具成本效益的解决方案，能够快速处理小规模数据任务，同时满足内存和时间要求。

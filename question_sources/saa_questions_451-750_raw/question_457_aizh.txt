Question #457
A company that uses AWS is building an application to transfer data to a product manufacturer. The company has its own identity
provider (IdP). The company wants the IdP to authenticate application users while the users use the application to transfer data.
The company must use Applicability Statement 2 (AS2) protocol.
Which solution will meet these requirements?
一家公司使用AWS正在构建一个应用程序，用于将数据传输给产品制迁商。该公司拥有自己的身份提供商（IdP）。公司希望IdP在
用户使用应用程序传输数据时对用户辦行身份轴证。公司必轔使用Applicability Statement 2 (AS2)协议。

2. 选项分析
A. Use AWS DataSync to transfer the data. Create an AWS Lambda function for IdP authentication.
错误原因：AWS DataSync主要用于在不同存储系统之輓轸效传输数据，但它不支持AS2协议，也无法直接与IdP輸成辦行身份轴
证。
B. Use Amazon AppFlow flows to transfer the data. Create an Amazon Elastic Container Service (Amazon ECS) task for IdP
authentication.
错误原因：Amazon AppFlow用于在AWS服务和SaaS应用之輓自动化数据流，但它不支持AS2协议。此外，ECS任务并不是处理IdP
身份轴证的最佳辷择。
C. Use AWS Transfer Family to transfer the data. Create an AWS Lambda function for IdP authentication.
正确原因：AWS Transfer Family支持AS2协议，可以用于安全的数据传输。达辟AWS Lambda，可以实现与公司IdP的輸成辦行身份
轴证。
D. Use AWS Storage Gateway to transfer the data. Create an Amazon Cognito identity pool for IdP authentication.
错误原因：AWS Storage Gateway主要用于混合云存储解决方案，不支持AS2协议。此外，Amazon Cognito虽然可以用于身份轴
证，但它与AS2协议的轀求不匹迚。
3. 我选的答案
C. Use AWS Transfer Family to transfer the data. Create an AWS Lambda function for IdP authentication.

Question #695
A company is building a microservices-based application that will be deployed on Amazon Elastic Kubernetes Service (Amazon
EKS). The microservices will interact with each other. The company wants to ensure that the application is observable to identify
performance issues in the future.
Which solution will meet these requirements?
一家公司正在构建一个基于微服务的应用程序，该应用程序将部署在 Amazon Elastic Kubernetes Service (Amazon EKS) 上。微服务
之间将进行交互。该公司希望确保该应用程序具有可观察性，以便在未来识别性能问题。
哪种解决方案可以满足这些需求？
A. Configure the application to use Amazon ElastiCache to reduce the number of requests that are sent to the microservices.
将应用程序配置为使用 Amazon ElastiCache，以减少发送到微服务的请求数量。
B. Configure Amazon CloudWatch Container Insights to collect metrics from the EKS clusters. Configure AWS X-Ray to trace
the requests between the microservices. Most Voted
配置 Amazon CloudWatch Container Insights 以收集来自 EKS 集群的指标。配置 AWS X-Ray 以追踪微服务之间的请求。
C. Configure AWS CloudTrail to review the API calls. Build an Amazon QuickSight dashboard to observe the microservice
interactions.
配置 AWS CloudTrail 以审查 API 调用。构建一个 Amazon QuickSight 仪表板以观察微服务交互。
D. Use AWS Trusted Advisor to understand the performance of the application.
使用 AWS Trusted Advisor 来了解应用程序的性能。

Correct Answer: B
正确答案：B

Amazon EKS的监控与可观测性工具。
如何使用AWS服务（如CloudWatch Container Insights和AWS X-Ray）来实现微服务的性能监控和请求追踪。
2. 选项分析
A. 配置 Amazon ElastiCache 以减少发送到微服务的请求数量。
错误：ElastiCache主要用于缓存数据以提高应用程序性能，但它并不能提供微服务之间的可观测性或性能监控功能。
B. 配置 Amazon CloudWatch Container Insights 以收集来自 EKS 集群的指标。配置 AWS X-Ray 以追踪微服务之间的请求。
正确：CloudWatch Container Insights可以收集EKS集群的性能指标，AWS X-Ray可以追踪微服务之间的请求流，提供详细的可观测
性。这是满足题目要求的最佳解决方案。
C. 配置 AWS CloudTrail 以审查 API 调用。构建一个 Amazon QuickSight 仪表板以观察微服务交互。
错误：CloudTrail主要用于记录API调用的日志，QuickSight用于数据可视化，但它们无法直接提供微服务的性能监控或请求追踪功
能。
D. 使用 AWS Trusted Advisor 来了解应用程序的性能。
错误：Trusted Advisor主要用于提供成本优化、安全性、容错性等建议，但它不适用于微服务的性能监控或请求追踪。

Question #587
A company is designing a solution to capture customer activity in different web applications to process analytics and make
predictions. Customer activity in the web applications is unpredictable and can increase suddenly. The company requires a
solution that integrates with other web applications. The solution must include an authorization step for security purposes.
Which solution will meet these requirements?
一家公司正在设计一个解决方案，用于捕获客户在不同网页应用中的活动，以处理分析并进行预测。网页应用中的客户活动不可预
测，并且可能突然增加。公司需要一个能够与其他网页应用集成的解决方案。该解决方案必须包含一个授权步骤以确保安全。
哪个解决方案可以满足这些要求？

题目分析与解答
1. 考察的知识点
Amazon API Gateway的功能与集成能力。
Amazon Kinesis Data Firehose的实时数据流处理能力。
Amazon S3作为存储解决方案的使用场景。
Lambda Authorizer在API Gateway中的授权机制。
2. 选项分析
A. Configure a Gateway Load Balancer (GWLB) in front of an Amazon Elastic Container Service (Amazon ECS) container
instance that stores the information that the company receives in an Amazon Elastic File System (Amazon EFS) file system.
Authorization is resolved at the GWLB.
错误：GWLB主要用于流量管理和负载均衡，不适合处理授权逻辑。
错误：ECS和EFS的组合虽然可以存储数据，但不适合处理实时数据流和突发流量。
错误：题目要求解决授权问题，而GWLB无法直接处理授权。
B. Configure an Amazon API Gateway endpoint in front of an Amazon Kinesis data stream that stores the information that
the company receives in an Amazon S3 bucket. Use an AWS Lambda function to resolve authorization.
部分正确：API Gateway可以处理突发流量并集成其他服务。
错误：Kinesis Data Stream需要额外的处理步骤才能将数据存储到S3，而题目没有提到这些步骤。
错误：Lambda函数可以处理授权，但题目更推荐使用API Gateway的Lambda Authorizer。
C. Configure an Amazon API Gateway endpoint in front of an Amazon Kinesis Data Firehose that stores the information
that the company receives in an Amazon S3 bucket. Use an API Gateway Lambda authorizer to resolve authorization.
正确：API Gateway可以处理突发流量并集成其他服务。
正确：Kinesis Data Firehose可以直接将数据流写入S3，无需额外处理步骤。
正确：API Gateway的Lambda Authorizer是处理授权的最佳实践。

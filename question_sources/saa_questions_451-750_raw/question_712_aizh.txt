Question #712
A company is designing a web application on AWS. The application will use a VPN connection between the company’s existing
data centers and the company's VPCs.
The company uses Amazon Route 53 as its DNS service. The application must use private DNS records to communicate with the
on-premises services from a VPC.
Which solution will meet these requirements in the MOST secure manner?
一家公司正在设计一个基于 AWS 的 Web 应用程序。该应用程序将使用公司现有数据中心与公司 VPC 之间的 VPN 连接。
该公司使用 Amazon Route 53 作为其 DNS 服务。该应用程序必须使用私有 DNS 记录从 VPC 与本地服务进行通信。
哪种解决方案能够以最安全的方式满足这些需求？
A. Create a Route 53 Resolver outbound endpoint. Create a resolver rule. Associate the resolver rule with the VPC. Most Voted
创建一个 Route 53 Resolver 出站端点。创建一个解析器规则。将解析器规则与 VPC 关联。
B. Create a Route 53 Resolver inbound endpoint. Create a resolver rule. Associate the resolver rule with the VPC.
创建一个 Route 53 Resolver 入站端点。创建一个解析器规则。将解析器规则与 VPC 关联。
C. Create a Route 53 private hosted zone. Associate the private hosted zone with the VPC.
创建一个 Route 53 私有托管区域。将私有托管区域与 VPC 关联。

创建一个 Route 53 公共托管区域。为每个服务创建一个记录以允许服务通信。
Correct Answer: A

题目分析与解答
1. 考察的知识点
Amazon Route 53 Resolver的功能，包括入站和出站端点。
如何使用Route 53进行私有DNS记录管理。
VPC与本地数据中心之间的安全通信。
2. 选项分析
A. Create a Route 53 Resolver outbound endpoint. Create a resolver rule. Associate the resolver rule with the VPC.
正确：Route 53 Resolver出站端点允许VPC中的资源查询本地数据中心的DNS记录。通过创建解析器规则并将其与VPC关联，可以确
保安全地解析私有DNS记录。
B. Create a Route 53 Resolver inbound endpoint. Create a resolver rule. Associate the resolver rule with the VPC.
错误：入站端点用于从本地数据中心向VPC发送DNS查询，而题目要求的是从VPC查询本地数据中心的DNS记录，因此不符合需
求。
C. Create a Route 53 private hosted zone. Associate the private hosted zone with the VPC.
错误：私有托管区域仅适用于VPC内部的DNS记录管理，无法解决VPC与本地数据中心之间的DNS解析问题。
D. Create a Route 53 public hosted zone. Create a record for each service to allow service communication.
错误：公共托管区域会将DNS记录暴露在互联网上，无法满足题目中“最安全”的要求。

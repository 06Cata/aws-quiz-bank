题目分析与解答
1. 考察的知识点
本题考察的是如何保护API免受常见的网络攻击（如SQL注入和跨站脚本攻击），以及AWS服务（如AWS WAF和AWS Shield）的使
用场景和功能。
Correct Answer: B
正确答案：B

2. 选项分析
A. Configure AWS Shield. 配置 AWS Shield。
错误：AWS Shield主要用于保护应用免受DDoS攻击，而不是针对SQL注入或跨站脚本攻击。此选项无法满足题目要求。
B. Configure AWS WAF. 配置 AWS WAF。
正确：AWS WAF是一种Web应用程序防火墙，可以保护API免受SQL注入和跨站脚本攻击。它是解决此问题的最直接和高效的方
式。
C. Set up API Gateway with an Amazon CloudFront distribution. Configure AWS Shield in CloudFront. 使用 Amazon
CloudFront 分发设置 API Gateway。 在 CloudFront 中配置 AWS Shield。
错误：虽然CloudFront可以与API Gateway集成以提高性能，但AWS Shield仍然主要用于DDoS防护，无法解决SQL注入和跨站脚本
攻击的问题。
D. Set up API Gateway with an Amazon CloudFront distribution. Configure AWS WAF in CloudFront. 使用 Amazon
CloudFront 分发设置 API Gateway。 在 CloudFront 中配置 AWS WAF。
正确：此选项也可以满足题目要求，因为AWS WAF可以配置在CloudFront中以保护API免受SQL注入和跨站脚本攻击。然而，与选项
B相比，这种方法增加了额外的复杂性（需要设置CloudFront分发），并不是最“操作高效”的解决方案。
3. 我的答案
B. Configure AWS WAF.
4. 官方答案
B. Configure AWS WAF.

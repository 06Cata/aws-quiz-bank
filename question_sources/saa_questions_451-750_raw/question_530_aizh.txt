Question #530
A company has an online gaming application that has TCP and UDP multiplayer gaming capabilities. The company uses Amazon
Route 53 to point the application traffic to multiple Network Load Balancers (NLBs) in different AWS Regions. The company needs
to improve application performance and decrease latency for the online game in preparation for user growth.
Which solution will meet these requirements?
一家公司有一个在线游戏应用程序，具有 TCP 和 UDP 的多人游戏功能。 该公司使用 Amazon Route 53 将应用程序流量指向位于不
同 AWS 区域的多个 Network Load Balancers (NLBs)。 公司需要改善应用程序性能并减少在线游戏的延迟，以应对用户增长。
哪种解决方案可以满足这些要求？
A. Add an Amazon CloudFront distribution in front of the NLBs. Increase the Cache-Control max-age parameter.
在 NLBs 前添加一个 Amazon CloudFront 分配。 增加 Cache-Control max-age 参数。
B. Replace the NLBs with Application Load Balancers (ALBs). Configure Route 53 to use latency-based routing.
将 NLBs 替换为 Application Load Balancers (ALBs)。 配置 Route 53 使用基于延迟的路由。
C. Add AWS Global Accelerator in front of the NLBs. Configure a Global Accelerator endpoint to use the correct listener ports.

题目分析与解答
1. 考察的知识点
Amazon Route 53的功能和路由策略（如延迟路由）。
Network Load Balancer (NLB) 和 Application Load Balancer (ALB) 的区别及适用场景。
AWS Global Accelerator 的功能及其对性能和延迟的优化作用。
Amazon CloudFront 的缓存功能及其适用场景。
Amazon API Gateway 的功能及其缓存机制。
2. 选项分析
A. Add an Amazon CloudFront distribution in front of the NLBs. Increase the Cache-Control max-age parameter.
正确性分析：CloudFront主要用于内容分发和缓存静态资源，而题目中提到的是TCP和UDP的实时在线游戏，这种场景不适合使用
CloudFront，因为游戏流量通常是动态的，无法通过缓存优化性能。
结论：错误。
B. Replace the NLBs with Application Load Balancers (ALBs). Configure Route 53 to use latency-based routing.
正确性分析：ALB主要用于HTTP/HTTPS流量，而题目中明确提到需要支持TCP和UDP协议，这些协议是NLB的强项。此外，虽然
Route 53的延迟路由可以优化性能，但更换为ALB会导致协议不匹配。
结论：错误。
C. Add AWS Global Accelerator in front of the NLBs. Configure a Global Accelerator endpoint to use the correct listener
ports.
正确性分析：AWS Global Accelerator可以优化全球用户的网络性能，减少延迟，并支持TCP和UDP协议。将Global Accelerator放在
NLB前面可以有效提升在线游戏的性能和用户体验。
结论：正确。
D. Add an Amazon API Gateway endpoint behind the NLBs. Enable API caching. Override method caching for the different
stages.
正确性分析：API Gateway主要用于管理和优化API流量，而题目中提到的是TCP和UDP的实时在线游戏场景，API Gateway不适合处
理这种类型的流量。此外，API缓存对动态游戏流量没有帮助。
结论：错误。
3. 我的答案
C. Add AWS Global Accelerator in front of the NLBs. Configure a Global Accelerator endpoint to use the correct listener
ports.
4. 官方答案
C. Add AWS Global Accelerator in front of the NLBs. Configure a Global Accelerator endpoint to use the correct listener
ports.
5. 与官方答案比较
我的答案与官方答案一致。
原因分析：Global Accelerator是专门设计用于优化全球用户的网络性能，支持TCP和UDP协议，完全符合题目要求。

Question #546
A recent analysis of a company's IT expenses highlights the need to reduce backup costs. The company's chief information officer
wants to simplify the on-premises backup infrastructure and reduce costs by eliminating the use of physical backup tapes. The
company must preserve the existing investment in the on-premises backup applications and workflows.
What should a solutions architect recommend?
昀近对某公司IT费用的分析强调了降低备份成本的必要性。该公司的首席信息官希望简化本地备份基础设施，并通过消除物理备份
磁带的使用来降低成本。公司必须保留现有的本地备份应用程序和工作流程。
解决方案架构师应该推荐什么？
A. Set up AWS Storage Gateway to connect with the backup applications using the NFS interface.
设置AWS Storage Gateway，通过NFS接口连接备份应用程序。
B. Set up an Amazon EFS file system that connects with the backup applications using the NFS interface.
设置一个Amazon EFS文件系统，通过NFS接口连接备份应用程序。
C. Set up an Amazon EFS file system that connects with the backup applications using the iSCSI interface.
设置一个Amazon EFS文件系统，通过iSCSI接口连接备份应用程序。
D. Set up AWS Storage Gateway to connect with the backup applications using the iSCSI-virtual tape library (VTL) interface.
Most Voted
设置AWS Storage Gateway，通过iSCSI虚拟磁带库（VTL）接口连接备份应用程序。
题目分析与解答
1. 考察的知识点
Correct Answer: D
正确答案：D
Community vote distribution
社区投票分布
D (100%)
本题考察的是AWS Storage Gateway的功能与使用场景，特别是如何通过虚拟磁带库（VTL）接口替代传统的物理磁带备份。
同时涉及到如何与现有的备份应用程序和工作流集成，保留现有投资。
2. 选项分析
A. Set up AWS Storage Gateway to connect with the backup applications using the NFS interface.
错误：虽然AWS Storage Gateway支持NFS接口，但NFS主要用于文件共享，而不是替代物理磁带备份。
本题要求替代物理磁带备份，NFS接口无法满足虚拟磁带库（VTL）的需求。
B. Set up an Amazon EFS file system that connects with the backup applications using the NFS interface.
错误：Amazon EFS是一个文件存储服务，适合共享文件系统，但不支持虚拟磁带库（VTL）功能。
无法直接与传统备份应用程序集成以替代物理磁带备份。
C. Set up an Amazon EFS file system that connects with the backup applications using the iSCSI interface.
错误：Amazon EFS不支持iSCSI接口，且EFS的设计目标是文件存储，而非磁带备份替代。
无法满足题目中关于虚拟磁带库（VTL）的需求。
D. Set up AWS Storage Gateway to connect with the backup applications using the iSCSI-virtual tape library (VTL)
interface.
正确：AWS Storage Gateway的磁带网关（Tape Gateway）支持iSCSI虚拟磁带库（VTL）接口，可以替代物理磁带备份。
它能够与现有的备份应用程序和工作流集成，保留现有投资，同时简化备份基础设施并降低成本。
3. 我的答案
D. Set up AWS Storage Gateway to connect with the backup applications using the iSCSI-virtual tape library (VTL) interface.
4. 官方答案
D. Set up AWS Storage Gateway to connect with the backup applications using the iSCSI-virtual tape library (VTL) interface.
5. 与官方答案比较
我的答案与官方答案一致。
原因：AWS Storage Gateway的磁带网关（Tape Gateway）是专门设计用于替代物理磁带备份的解决方案，支持iSCSI虚拟磁带库
（VTL）接口，完全符合题目要求。

Question #672
A marketing company receives a large amount of new clickstream data in Amazon S3 from a marketing campaign. The company
needs to analyze the clickstream data in Amazon S3 quickly. Then the company needs to determine whether to process the data
further in the data pipeline.
Which solution will meet these requirements with the LEAST operational overhead?
一家营销公司从一场营销活动中在 Amazon S3 接收到大量新的点击流数据。公司需要快速分析 Amazon S3 中的点击流数据。然
后，公司需要确定是否在数据管道中进一步处理这些数据。
哪种解决方案可以以最少的运营开销满足这些需求？
A. Create external tables in a Spark catalog. Configure jobs in AWS Glue to query the data.
在 Spark catalog 中创建外部表。配置 AWS Glue 作业以查询数据。
B. Configure an AWS Glue crawler to crawl the data. Configure Amazon Athena to query the data. Most Voted
配置一个 AWS Glue crawler 来爬取数据。配置 Amazon Athena 来查询数据。
C. Create external tables in a Hive metastore. Configure Spark jobs in Amazon EMR to query the data.
在 Hive metastore 中创建外部表。配置 Amazon EMR 中的 Spark 作业以查询数据。
D. Configure an AWS Glue crawler to crawl the data. Configure Amazon Kinesis Data Analytics to use SQL to query the data.
配置一个 AWS Glue crawler 来爬取数据。配置 Amazon Kinesis Data Analytics 使用 SQL 查询数据。

2. 选项分析
A. Create external tables in a Spark catalog. Configure jobs in AWS Glue to query the data.
错误：虽然Spark和AWS Glue可以处理数据，但创建外部表并配置Spark作业需要较高的运营开销。
此外，Spark catalog的管理复杂度较高，不符合“最低运营开销”的要求。
B. Configure an AWS Glue crawler to crawl the data. Configure Amazon Athena to query the data.
正确：AWS Glue crawler可以自动爬取数据并创建数据目录，Amazon Athena可以直接查询S3中的数据，无需额外的基础设施管
理。
此选项提供了最低的运营开销，同时满足快速分析数据的需求。
C. Create external tables in a Hive metastore. Configure Spark jobs in Amazon EMR to query the data.
错误：Hive metastore和Amazon EMR需要额外的基础设施管理，运营开销较高。
虽然可以实现数据查询，但不符合“最低运营开销”的要求。
D. Configure an AWS Glue crawler to crawl the data. Configure Amazon Kinesis Data Analytics to use SQL to query the
data.
错误：Amazon Kinesis Data Analytics主要用于实时流数据分析，而题目中的数据已经存储在Amazon S3中。
此选项不适合静态数据的查询需求。
3. 我的答案
B. Configure an AWS Glue crawler to crawl the data. Configure Amazon Athena to query the data.

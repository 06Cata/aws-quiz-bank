Question #707
A company is using an Application Load Balancer (ALB) to present its application to the internet. The company finds abnormal
traffic access patterns across the application. A solutions architect needs to improve visibility into the infrastructure to help the
company understand these abnormalities better.
What is the MOST operationally efficient solution that meets these requirements?
一家公司正在使用 Application Load Balancer (ALB) 向互联网展示其应用程序。该公司发现应用程序中存在异常的流量访问模式。
解决方案架构师需要提高基础设施的可见性，以帮助公司更好地理解这些异常。
满足这些要求的最具操作效率的解决方案是什么？
A. Create a table in Amazon Athena for AWS CloudTrail logs. Create a query for the relevant information.
在 Amazon Athena 中为 AWS CloudTrail 日志创建一个表。为相关信息创建查询。
B. Enable ALB access logging to Amazon S3. Create a table in Amazon Athena, and query the logs. Most Voted
启用 ALB 访问日志记录到 Amazon S3。在 Amazon Athena 中创建一个表，并查询日志。
C. Enable ALB access logging to Amazon S3. Open each file in a text editor, and search each line for the relevant information.
启用 ALB 访问日志记录到 Amazon S3。在文本编辑器中打开每个文件，并在每行中搜索相关信息。
D. Use Amazon EMR on a dedicated Amazon EC2 instance to directly query the ALB to acquire traffic access log information.
使用 Amazon EMR 在专用的 Amazon EC2 实例上直接查询 ALB 以获取流量访问日志信息。
题目分析与解答
1. 考察的知识点
Correct Answer: B
正确答案：B

B (100%)
本题考察的是AWS服务的日志记录与分析能力，尤其是ALB（Application Load Balancer）的访问日志功能，以及如何使用Amazon
Athena高效查询日志数据。
还涉及到AWS服务的操作效率和成本优化的最佳实践。
2. 选项分析
A. Create a table in Amazon Athena for AWS CloudTrail logs. Create a query for the relevant information.
错误：AWS CloudTrail主要用于记录API调用和账户活动，而不是ALB的访问日志。虽然可以查询CloudTrail日志，但它无法直接提供
ALB的流量访问信息。
B. Enable ALB access logging to Amazon S3. Create a table in Amazon Athena, and query the logs.
正确：ALB支持将访问日志记录到Amazon S3，随后可以使用Amazon Athena创建表并查询日志数据。这种方法高效且操作简单，
符合题目要求的“最具操作效率”的解决方案。
C. Enable ALB access logging to Amazon S3. Open each file in a text editor, and search each line for the relevant
information.
错误：手动打开日志文件并逐行搜索信息非常低效，尤其是当日志量较大时。这种方法不符合“操作效率”的要求。
D. Use Amazon EMR on a dedicated Amazon EC2 instance to directly query the ALB to acquire traffic access log
information.
错误：Amazon EMR主要用于大规模数据处理，而不是直接查询ALB日志。此外，使用专用EC2实例运行EMR会增加成本和复杂性，
不符合“最具操作效率”的要求。
3. 我选的答案
B. Enable ALB access logging to Amazon S3. Create a table in Amazon Athena, and query the logs.

B. Enable ALB access logging to Amazon S3. Create a table in Amazon Athena, and query the logs.

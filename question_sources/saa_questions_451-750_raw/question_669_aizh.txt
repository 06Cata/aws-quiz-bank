Question #669
A company runs its databases on Amazon RDS for PostgreSQL. The company wants a secure solution to manage the master user
password by rotating the password every 30 days.
Which solution will meet these requirements with the LEAST operational overhead?
一家公司在 Amazon RDS for PostgreSQL 上运行其数据库。该公司希望通过每 30 天轮换一次密码来管理主用户密码的安全解决方
案。
哪种解决方案能够以最少的运营开销满足这些要求？
A. Use Amazon EventBridge to schedule a custom AWS Lambda function to rotate the password every 30 days.
使用 Amazon EventBridge 调度一个自定义 AWS Lambda 函数，每 30 天轮换一次密码。
B. Use the modify-db-instance command in the AWS CLI to change the password.
使用 AWS CLI 中的 modify-db-instance 命令更改密码。
C. Integrate AWS Secrets Manager with Amazon RDS for PostgreSQL to automate password rotation. Most Voted
将 AWS Secrets Manager 与 Amazon RDS for PostgreSQL 集成以实现密码轮换自动化。
D. Integrate AWS Systems Manager Parameter Store with Amazon RDS for PostgreSQL to automate password rotation.
将 AWS Systems Manager Parameter Store 与 Amazon RDS for PostgreSQL 集成以实现密码轮换自动化。
题目分析与解答
1. 考察的知识点
本题考察的是如何在AWS中实现安全的密码管理和自动化密码轮换，涉及到Amazon RDS、AWS Secrets Manager、AWS Lambda、
EventBridge等服务的使用。
Correct Answer: C
正确答案：C
Community vote distribution
社区投票分布
C (100%)
2. 选项分析
A. Use Amazon EventBridge to schedule a custom AWS Lambda function to rotate the password every 30 days.
正确性分析：虽然可以使用EventBridge调度Lambda函数来实现密码轮换，但这种方法需要开发和维护自定义代码，增加了操作复
杂性。
结论：此选项正确但不是最优解，因为它增加了操作开销。
B. Use the modify-db-instance command in the AWS CLI to change the password.
正确性分析：使用AWS CLI手动更改密码需要人工干预，无法实现自动化轮换，操作开销较高。
结论：此选项错误，因为它无法满足“自动化”和“最少操作开销”的要求。
C. Integrate AWS Secrets Manager with Amazon RDS for PostgreSQL to automate password rotation.
正确性分析：AWS Secrets Manager支持与Amazon RDS集成，可以自动轮换数据库密码，完全满足题目要求，并且操作开销最低。
结论：此选项正确且是最优解。
D. Integrate AWS Systems Manager Parameter Store with Amazon RDS for PostgreSQL to automate password rotation.
正确性分析：虽然Parameter Store可以存储密码，但它不支持自动化密码轮换功能，需要额外的开发工作来实现轮换。
结论：此选项错误，因为它无法直接实现自动化密码轮换。

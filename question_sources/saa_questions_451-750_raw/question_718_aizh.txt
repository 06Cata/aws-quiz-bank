Question #718
A company’s applications use Apache Hadoop and Apache Spark to process data on premises. The existing infrastructure is not
scalable and is complex to manage.
A solutions architect must design a scalable solution that reduces operational complexity. The solution must keep the data
processing on premises.
Which solution will meet these requirements?
一家公司的应用程序使用 Apache Hadoop 和 Apache Spark 在本地处理数据。现有的基础设施无法扩展且管理复杂。
解决方案架构师必须设计一个可扩展的解决方案，以减少运营复杂性。该解决方案必须保持数据处理在本地。
哪种解决方案可以满足这些要求？
A. Use AWS Site-to-Site VPN to access the on-premises Hadoop Distributed File System (HDFS) data and application. Use an
Amazon EMR cluster to process the data.
使用 AWS Site-to-Site VPN 访问本地 Hadoop Distributed File System (HDFS) 数据和应用程序。使用 Amazon EMR 集群处理数
据。
B. Use AWS DataSync to connect to the on-premises Hadoop Distributed File System (HDFS) cluster. Create an Amazon EMR
cluster to process the data.
使用 AWS DataSync 连接到本地 Hadoop Distributed File System (HDFS) 集群。创建一个 Amazon EMR 集群处理数据。
C. Migrate the Apache Hadoop application and the Apache Spark application to Amazon EMR clusters on AWS Outposts. Use
the EMR clusters to process the data. Most Voted
将 Apache Hadoop 应用程序和 Apache Spark 应用程序迁移到 AWS Outposts 上的 Amazon EMR 集群。使用 EMR 集群处理数
据。
D. Use an AWS Snowball device to migrate the data to an Amazon S3 bucket. Create an Amazon EMR cluster to process the
data.
使用 AWS Snowball 设备将数据迁移到 Amazon S3 存储桶。创建一个 Amazon EMR 集群处理数据。
Correct Answer: C
正确答案：C
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务的选择与架构设计能力，特别是针对数据处理和迁移的场景。
涉及的服务包括Amazon EMR、AWS Outposts、AWS DataSync、AWS Site-to-Site VPN、AWS Snowball等。
重点在于解决本地数据处理的可扩展性和复杂性问题，同时保持数据处理在本地。
2. 选项分析
A. 使用 AWS Site-to-Site VPN 访问本地 Hadoop Distributed File System (HDFS) 数据和应用程序。使用 Amazon EMR 集群处
理数据。
错误：AWS Site-to-Site VPN可以连接本地和AWS云，但它并不能解决本地数据处理的复杂性问题。
此外，Amazon EMR是云端服务，使用VPN访问本地数据会增加延迟，且不符合题目要求“保持数据处理在本地”。
B. 使用 AWS DataSync 连接到本地 Hadoop Distributed File System (HDFS) 集群。创建一个 Amazon EMR 集群处理数据。
错误：AWS DataSync主要用于数据迁移，而不是直接处理数据。
创建Amazon EMR集群处理数据仍然是在云端进行处理，不符合题目要求“保持数据处理在本地”。
C. 将 Apache Hadoop 应用程序和 Apache Spark 应用程序迁移到 AWS Outposts 上的 Amazon EMR 集群。使用 EMR 集群处理
数据。
正确：AWS Outposts是一种混合云解决方案，可以将AWS服务扩展到本地环境。
将Hadoop和Spark迁移到AWS Outposts上的Amazon EMR集群，可以在本地处理数据，同时简化管理并提高可扩展性。
完全符合题目要求“保持数据处理在本地”和“减少操作复杂性”。
D. 使用 AWS Snowball 设备将数据迁移到 Amazon S3 存储桶。创建一个 Amazon EMR 集群处理数据。
错误：AWS Snowball主要用于数据迁移，而不是实时数据处理。
将数据迁移到Amazon S3后，数据处理会在云端进行，不符合题目要求“保持数据处理在本地”。
3. 我选的答案
C
4. 官方的答案
C
5. 与官方答案的比较
我的答案与官方答案一致。
原因分析：AWS Outposts是唯一能够满足题目要求的解决方案，既保持数据处理在本地，又简化了操作复杂性，同时提供了可扩展
性。

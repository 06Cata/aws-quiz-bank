Question #689
A company wants to standardize its Amazon Elastic Block Store (Amazon EBS) volume encryption strategy. The company also
wants to minimize the cost and configuration effort required to operate the volume encryption check.
Which solution will meet these requirements?
一家公司希望标准化其 Amazon Elastic Block Store (Amazon EBS) 卷加密策略。 公司还希望尽量减少操作卷加密检查所需的成本和
配置工作。
哪种解决方案可以满足这些需求？
A. Write API calls to describe the EBS volumes and to confirm the EBS volumes are encrypted. Use Amazon EventBridge to
schedule an AWS Lambda function to run the API calls.
编写 API 调用以描述 EBS 卷并确认 EBS 卷已加密。 使用 Amazon EventBridge 调度一个 AWS Lambda 函数来运行 API 调用。
B. Write API calls to describe the EBS volumes and to confirm the EBS volumes are encrypted. Run the API calls on an AWS
Fargate task.
编写 API 调用以描述 EBS 卷并确认 EBS 卷已加密。 在一个 AWS Fargate 任务上运行 API 调用。
C. Create an AWS Identity and Access Management (IAM) policy that requires the use of tags on EBS volumes. Use AWS Cost
Explorer to display resources that are not properly tagged. Encrypt the untagged resources manually.
创建一个 AWS Identity and Access Management (IAM) 策略，要求在 EBS 卷上使用标签。 使用 AWS Cost Explorer 显示未正确
标记的资源。 手动加密未标记的资源。
D. Create an AWS Config rule for Amazon EBS to evaluate if a volume is encrypted and to flag the volume if it is not encrypted.

题目分析与解答
1. 考察的知识点
Amazon EBS卷加密策略的标准化
AWS Config规则的使用
如何最小化成本和配置工作量
2. 选项分析
A. 编写 API 调用以描述 EBS 卷并确认 EBS 卷已加密。使用 Amazon EventBridge 调度一个 AWS Lambda 函数来运行 API 调用。
正确性分析：虽然可以通过API调用检查EBS卷是否加密，并使用EventBridge调度Lambda函数，但这种方法需要额外的开发工作和
维护成本，不符合“最小化成本和配置工作量”的要求。
结论：错误。
B. 编写 API 调用以描述 EBS 卷并确认 EBS 卷已加密。在一个 AWS Fargate 任务上运行 API 调用。
正确性分析：类似于选项A，这种方法也需要额外的开发工作和维护成本，同时还引入了Fargate任务的运行成本，不符合题目要
求。
结论：错误。
C. 创建一个 AWS Identity and Access Management (IAM) 策略，要求在 EBS 卷上使用标签。使用 AWS Cost Explorer 显示未正
确标记的资源。手动加密未标记的资源。
正确性分析：此选项与题目要求不符，因为它依赖手动加密未标记资源，无法自动化加密检查，也无法直接评估EBS卷是否加密。
结论：错误。
D. 创建一个 AWS Config 规则，以评估卷是否已加密，并在卷未加密时标记该卷。
正确性分析：AWS Config规则可以自动化地评估EBS卷是否加密，并标记未加密的卷。这种方法无需额外开发工作，且成本较低，
完全符合题目要求。
结论：正确。

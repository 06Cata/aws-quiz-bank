Question #744
A company is designing a new web service that will run on Amazon EC2 instances behind an Elastic Load Balancing (ELB) load
balancer. However, many of the web service clients can only reach IP addresses authorized on their firewalls.
What should a solutions architect recommend to meet the clients’ needs?
一家公司正在设计一个新的网络服务，该服务将运行在 Amazon EC2 实例上，并通过一个 Elastic Load Balancing (ELB) 负载均衡器
进行负载均衡。然而，许多网络服务的客户端只能访问其防火墙上授权的 IP 地址。

2. 选项分析
A. A Network Load Balancer with an associated Elastic IP address.
Network Load Balancer (NLB) 支持静态 IP 地址（包括 Elastic IP 地址），这使得客户端可以通过固定的 IP 地址访问服务。
这种配置满足题目中“客户端只能访问授权的 IP 地址”的需求。
正确选项。
B. An Application Load Balancer with an associated Elastic IP address.
Application Load Balancer (ALB) 不支持直接绑定 Elastic IP 地址。
ALB 的 IP 地址是动态分配的，无法满足题目中“客户端只能访问授权的 IP 地址”的需求。
错误选项。
C. An A record in an Amazon Route 53 hosted zone pointing to an Elastic IP address.
Route 53 的 A 记录可以指向一个 Elastic IP 地址，但这并不能解决负载均衡的问题。
题目要求使用 ELB，而不是直接使用 Elastic IP 地址。
错误选项。
D. An EC2 instance with a public IP address running as a proxy in front of the load balancer.
使用 EC2 实例作为代理可以提供一个固定的 IP 地址，但这会增加管理复杂性和单点故障的风险。
题目要求使用 ELB，而不是直接使用 EC2 实例作为代理。
错误选项。
3. 我的答案
A. A Network Load Balancer with an associated Elastic IP address.
4. 官方答案
A. A Network Load Balancer with an associated Elastic IP address.

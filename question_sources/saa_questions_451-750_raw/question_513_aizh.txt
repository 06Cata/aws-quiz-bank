A social media company wants to allow its users to upload images in an application that is hosted in the AWS Cloud. The company
needs a solution that automatically resizes the images so that the images can be displayed on multiple device types. The
application experiences unpredictable traffic patterns throughout the day. The company is seeking a highly available solution that
maximizes scalability.
What should a solutions architect do to meet these requirements?

小，以便图片可以在多种设备类型上显示。 该应用程序全天经历不可预测的流量模式。 该公司正在寻求一个高度可用的解决方

题目分析与解答
1. 考察的知识点
Amazon S3 静态网站托管
AWS Lambda 的无服务器计算能力
自动化图片处理与存储
高可用性与可扩展性设计
2. 选项分析
A. Create a static website hosted in Amazon S3 that invokes AWS Lambda functions to resize the images and store the
images in an Amazon S3 bucket.
正确：Amazon S3 可以托管静态网站，具有高可用性和可扩展性。
正确：AWS Lambda 是无服务器计算服务，能够自动扩展以处理图片调整任务。
正确：Amazon S3 是理想的存储解决方案，适合存储图片文件。
此选项完全符合题目要求：高可用性、可扩展性以及自动化图片处理。
B. Create a static website hosted in Amazon CloudFront that invokes AWS Step Functions to resize the images and store
the images in an Amazon RDS database.
错误：CloudFront 是内容分发网络（CDN），不直接托管静态网站。
错误：AWS Step Functions 是工作流编排服务，不适合直接处理图片调整任务。
错误：Amazon RDS 是关系型数据库，不适合存储图片文件。
此选项不符合题目要求。
C. Create a dynamic website hosted on a web server that runs on an Amazon EC2 instance. Configure a process that runs
on the EC2 instance to resize the images and store the images in an Amazon S3 bucket.
错误：EC2 需要手动管理，无法自动扩展以应对不可预测的流量模式。
正确：可以通过 EC2 实例运行图片调整程序，但这不是最佳解决方案。
正确：Amazon S3 是理想的存储解决方案。
此选项不符合题目要求的高可用性和自动扩展性。
D. Create a dynamic website hosted on an automatically scaling Amazon Elastic Container Service (Amazon ECS) cluster
that creates a resize job in Amazon Simple Queue Service (Amazon SQS). Set up an image-resizing program that runs on
an Amazon EC2 instance to process the resize jobs.
正确：Amazon ECS 可以自动扩展以应对流量变化。
错误：使用 EC2 实例处理图片调整任务需要手动管理，增加复杂性。
错误：此选项的架构复杂度较高，不是最佳解决方案。
此选项不完全符合题目要求的简单性和高可用性。

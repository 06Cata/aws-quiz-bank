Question #634
A company collects 10 GB of telemetry data daily from various machines. The company stores the data in an Amazon S3 bucket in
a source data account.
The company has hired several consulting agencies to use this data for analysis. Each agency needs read access to the data for its
analysts. The company must share the data from the source data account by choosing a solution that maximizes security and
operational efficiency.
Which solution will meet these requirements?

决方案来共享源数据账户中的数据，以最大化安全性和运营效率。
哪种解决方案可以满足这些需求？

题目分析与解答
1. 考察的知识点
Amazon S3跨账户访问配置
安全性和操作效率的最佳实践
IAM权限管理
2. 选项分析
A. 配置 S3 全球表以为每个机构复制数据。
错误：Amazon S3 不支持“全球表”功能。全球表是 Amazon DynamoDB 的功能，而不是 S3 的功能。
错误：即使有类似的复制功能，复制数据到每个机构的账户会增加管理复杂性，并不符合“最大化安全性和操作效率”的要求。
B. 将 S3 存储桶设为公开访问一段有限时间，仅通知这些机构。
错误：将 S3 存储桶设为公开访问会严重降低安全性，任何人都可以访问数据，而不仅仅是指定的机构。
错误：即使是有限时间的公开访问，也无法保证数据的安全性，违反了最佳实践。
C. 为机构拥有的账户配置 S3 存储桶的跨账户访问权限。
正确：通过跨账户访问，可以安全地授予指定账户对 S3 存储桶的访问权限。
正确：这种方法符合 AWS 的最佳实践，最大化了安全性和操作效率。
D. 在源数据账户中为每个分析师设置一个 IAM 用户。为每个用户授予对 S3 存储桶的访问权限。
错误：为每个分析师创建 IAM 用户会增加管理复杂性，尤其是当分析师数量较多时。
错误：这种方法不符合“最大化操作效率”的要求。
3. 我选的答案
C. 为机构拥有的账户配置 S3 存储桶的跨账户访问权限。
4. 官方答案
C. 为机构拥有的账户配置 S3 存储桶的跨账户访问权限。
5. 与官方答案比较
我的答案与官方答案一致。
原因：跨账户访问是 AWS 推荐的最佳实践，能够在保证安全性的同时简化操作流程。

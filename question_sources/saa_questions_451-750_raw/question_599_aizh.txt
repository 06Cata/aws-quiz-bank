Question #599
A company wants to use Amazon Elastic Container Service (Amazon ECS) clusters and Amazon RDS DB instances to build and run
a payment processing application. The company will run the application in its on-premises data center for compliance purposes.
A solutions architect wants to use AWS Outposts as part of the solution. The solutions architect is working with the company's
operational team to build the application.
Which activities are the responsibility of the company's operational team? (Choose three.)
一家公司希望使用 Amazon Elastic Container Service (Amazon ECS) 集群和 Amazon RDS DB 实例来构建和运行一个支付处理应用程
序。为了满足合规要求，该公司将在其本地数据中心运行该应用程序。
一个解决方案架构师希望将 AWS Outposts 作为解决方案的一部分。解决方案架构师正在与公司的运营团队合作来构建该应用程
序。
哪些活动是公司运营团队的责任？（选择三个）
A. Providing resilient power and network connectivity to the Outposts racks Most Voted
为 Outposts 机架提供可靠的电力和网络连接
B. Managing the virtualization hypervisor, storage systems, and the AWS services that run on Outposts
管理虚拟化 hypervisor、存储系统以及运行在 Outposts 上的 AWS 服务
C. Physical security and access controls of the data center environment Most Voted
数据中心环境的物理安全和访问控制
D. Availability of the Outposts infrastructure including the power supplies, servers, and networking equipment within the
Outposts racks
Outposts 基础设施的可用性，包括 Outposts 机架内的电源供应、服务器和网络设备
E. Physical maintenance of Outposts components
Outposts 组件的物理维护
F. Providing extra capacity for Amazon ECS clusters to mitigate server failures and maintenance events Most Voted
为 Amazon ECS 集群提供额外容量，以缓解服务器故障和维护事件
Correct Answer: ACF
正确答案：A、C、F
题目分析与解答
1. 考察的知识点
本题主要考察AWS Outposts的责任分工，尤其是客户与AWS之间的责任划分。
涉及AWS Outposts的物理维护、数据中心管理以及资源扩展等知识点。
2. 选项分析
A. Providing resilient power and network connectivity to the Outposts racks 正确。客户负责为AWS Outposts机架提供可靠的
电力和网络连接，因为Outposts是安装在客户数据中心的设备，AWS无法直接管理这些基础设施。
B. Managing the virtualization hypervisor, storage systems, and the AWS services that run on Outposts 错误。AWS负责管
理Outposts上的虚拟化层、存储系统以及AWS服务，客户无需承担这些责任。
C. Physical security and access controls of the data center environment 正确。客户负责数据中心的物理安全和访问控制，因为
Outposts机架安装在客户的物理环境中。
D. Availability of the Outposts infrastructure including the power supplies, servers, and networking equipment within the
Outposts racks 错误。Outposts基础设施的可用性由AWS负责，包括机架内的电源供应、服务器和网络设备。
E. Physical maintenance of Outposts components 错误。Outposts组件的物理维护由AWS负责，客户无需进行硬件维护。
F. Providing extra capacity for Amazon ECS clusters to mitigate server failures and maintenance events 正确。客户需要为
Amazon ECS集群提供额外的容量，以应对服务器故障和维护事件，这是客户的责任。

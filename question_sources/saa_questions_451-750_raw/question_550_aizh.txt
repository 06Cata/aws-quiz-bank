Question #550
A company is using AWS Key Management Service (AWS KMS) keys to encrypt AWS Lambda environment variables. A solutions
architect needs to ensure that the required permissions are in place to decrypt and use the environment variables.
Which steps must the solutions architect take to implement the correct permissions? (Choose two.)

所需的权限来解密和使用环境变量。
解决方案架构师必须采取哪些步骤来实施正确的权限？（选择两个）
A. Add AWS KMS permissions in the Lambda resource policy.

1. 考察的知识点

BD (100%)
本题考察的是 AWS KMS 与 AWS Lambda 的集成权限管理，包括 KMS 密钥策略和 Lambda 执行角色的权限配置。
2. 选项分析
A. Add AWS KMS permissions in the Lambda resource policy.
错误。Lambda 资源策略主要用于控制对 Lambda 函数的访问权限，而不是用于配置 KMS 的权限。KMS 权限需要在执行角色或
KMS 密钥策略中配置。
B. Add AWS KMS permissions in the Lambda execution role.
正确。Lambda 执行角色需要具备解密 KMS 密钥的权限，以便能够解密环境变量。
C. Add AWS KMS permissions in the Lambda function policy.
错误。Lambda 函数策略并不是用于配置 KMS 权限的正确位置。KMS 权限需要在执行角色或 KMS 密钥策略中配置。
D. Allow the Lambda execution role in the AWS KMS key policy.
正确。KMS 密钥策略需要显式允许 Lambda 执行角色对密钥的访问权限，以便解密环境变量。
E. Allow the Lambda resource policy in the AWS KMS key policy.
错误。KMS 密钥策略无法直接与 Lambda 资源策略关联，正确的做法是允许 Lambda 执行角色。

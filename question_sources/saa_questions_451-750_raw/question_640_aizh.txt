files are encrypted using AWS Key Management Service (AWS KMS) keys. A solutions architect needs to design a solution that will
ensure the required permissions are set correctly.
Which combination of actions accomplish this? (Choose two.)

本题考察的是 AWS Lambda 与 AWS KMS 的权限配置，涉及 IAM 角色、资源策略以及 KMS 密钥策略的正确使用。
重点知识点包括：如何为 Lambda 函数配置解密权限，以及如何在 KMS 密钥策略中正确授予权限。
2. 选项分析
A. Attach the kms:decrypt permission to the Lambda function’s resource policy
错误。Lambda 函数的资源策略并不支持直接附加权限。权限需要通过 IAM 角色或 KMS 密钥策略来授予。
B. Grant the decrypt permission for the Lambda IAM role in the KMS key's policy
正确。KMS 密钥策略需要显式授予 Lambda 的 IAM 角色解密权限，以便 Lambda 函数能够使用 KMS 密钥解密文件。
C. Grant the decrypt permission for the Lambda resource policy in the KMS key's policy
错误。Lambda 函数的资源策略并不能直接与 KMS 密钥策略交互，因此无法通过这种方式授予权限。
D. Create a new IAM policy with the kms:decrypt permission and attach the policy to the Lambda function
错误。虽然可以创建一个 IAM 策略并附加到 Lambda 函数，但这并不足以让 Lambda 函数访问 KMS 密钥。还需要在 KMS 密钥策
略中显式授予权限。
E. Create a new IAM role with the kms:decrypt permission and attach the execution role to the Lambda function
正确。Lambda 函数需要一个执行角色，该角色必须包含 kms:decrypt 权限，以便能够解密文件。

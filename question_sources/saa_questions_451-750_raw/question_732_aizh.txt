Question #732
A company has deployed its application on Amazon EC2 instances with an Amazon RDS database. The company used the principle
of least privilege to configure the database access credentials. The company's security team wants to protect the application and
the database from SQL injection and other web-based attacks.
Which solution will meet these requirements with the LEAST operational overhead?
一家公司已将其应用程序部署在 Amazon EC2 实例上，并使用 Amazon RDS 数据库。该公司遵循最小权限原则配置了数据库访问凭
证。公司的安全团队希望保护应用程序和数据库免受 SQL 注入和其他基于 Web 的攻击。
哪种解决方案能以最低的运营开销满足这些要求？
A. Use security groups and network ACLs to secure the database and application servers.
使用安全组和网络 ACL 来保护数据库和应用程序服务器。
B. Use AWS WAF to protect the application. Use RDS parameter groups to configure the security settings. Most Voted
使用 AWS WAF 保护应用程序。使用 RDS 参数组配置安全设置。
C. Use AWS Network Firewall to protect the application and the database.
使用 AWS Network Firewall 来保护应用程序和数据库。
D. Use different database accounts in the application code for different functions. Avoid granting excessive privileges to the
database users.
在应用程序代码中为不同功能使用不同的数据库账户。避免向数据库用户授予过多权限。
题目分析与解答
1. 考察的知识点
Correct Answer: B
正确答案：B
Community vote distribution
社区投票分布
B (100%)
安全性：如何保护应用程序和数据库免受SQL注入和其他Web攻击。
AWS服务：AWS WAF、RDS参数组、网络ACL、安全组、AWS Network Firewall等的使用场景和功能。
操作简化：选择具有最少操作开销的解决方案。
2. 选项分析
A. 使用安全组和网络 ACL 来保护数据库和应用程序服务器。
安全组和网络ACL主要用于控制网络流量的访问权限，但它们无法直接防御SQL注入和其他Web攻击。
虽然可以限制流量来源，但无法提供针对应用层攻击的保护。
此选项无法满足题目要求，且操作开销较高。
错误。
B. 使用 AWS WAF 保护应用程序。使用 RDS 参数组配置安全设置。
AWS WAF（Web Application Firewall）专门用于防御SQL注入和其他Web攻击，直接满足题目要求。
RDS参数组可以配置数据库的安全设置，进一步增强数据库的安全性。
此选项提供了针对应用层攻击的保护，并且操作开销较低。
正确。
C. 使用 AWS Network Firewall 来保护应用程序和数据库。
AWS Network Firewall主要用于保护网络层流量，无法直接防御SQL注入和其他Web攻击。
虽然可以增强网络安全，但无法满足题目要求的应用层保护。
此选项操作开销较高。
错误。
D. 在应用程序代码中为不同功能使用不同的数据库账户。避免向数据库用户授予过多权限。
此选项强调数据库权限管理，但无法直接防御SQL注入和其他Web攻击。
虽然符合“最小权限原则”，但无法满足题目要求的应用层保护。
此选项操作开销较高。
错误。
3. 我选的答案
B. 使用 AWS WAF 保护应用程序。使用 RDS 参数组配置安全设置。
4. 官方答案
B. 使用 AWS WAF 保护应用程序。使用 RDS 参数组配置安全设置。
5. 与官方答案比较
我的答案与官方答案一致。
原因：AWS WAF直接满足防御SQL注入和其他Web攻击的需求，同时RDS参数组增强了数据库的安全性，且操作开销较低。

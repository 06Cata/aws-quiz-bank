memory and 2 GB of storage for its computation resources. The application will require that the data is in a relational format.
Which additional combination ofAWS services will meet these requirements with the LEAST administrative effort? (Choose two.)

2. 选项分析
A. Amazon EC2:
Amazon EC2可以提供计算资源和存储，但需要手动管理操作系统、数据库安装和维护，增加了管理复杂性，不符合“最少管理工
作”的要求。因此错误。
B. AWS Lambda:
AWS Lambda是无服务器计算服务，支持API Gateway集成，但Lambda函数的内存限制为10 GB，且不直接提供持久化存储。虽然满
足计算需求，但无法满足存储需求。因此错误。
C. Amazon RDS:
Amazon RDS是托管的关系型数据库服务，支持存储数据的关系型格式，且管理工作量较少。符合题目要求，因此正确。
D. Amazon DynamoDB:
Amazon DynamoDB是托管的NoSQL数据库服务，不支持关系型数据存储格式，因此错误。
E. Amazon Elastic Kubernetes Services (Amazon EKS):
Amazon EKS可以运行容器化应用，但需要管理容器编排和存储，增加了管理复杂性，不符合“最少管理工作”的要求。因此错误。
3. 我选的答案
C. Amazon RDS
4. 官方答案
B. AWS Lambda
C. Amazon RDS
5. 与官方答案比较
官方答案选择了B和C。经过重新思考，AWS Lambda虽然无法直接满足存储需求，但可以通过与Amazon RDS集成来满足存储需
求，同时减少管理工作量。结合题目要求，官方答案是正确的。

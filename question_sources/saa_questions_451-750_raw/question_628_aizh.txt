Question #628
A global company runs its applications in multiple AWS accounts in AWS Organizations. The company's applications use multipart
uploads to upload data to multiple Amazon S3 buckets across AWS Regions. The company wants to report on incomplete
multipart uploads for cost compliance purposes.
Which solution will meet these requirements with the LEAST operational overhead?

到跨 AWS Regions 的多个 Amazon S3 存储桶。 该公司希望针对未完成的 multipart uploads 进行报告，以满足成本合规要求。

2. 选项分析
A. Configure AWS Config with a rule to report the incomplete multipart upload object count.
错误：AWS Config主要用于资源配置的合规性检查，而不是直接报告S3存储指标。虽然可以通过自定义规则实现，但这会增加操作
复杂性。
B. Create a service control policy (SCP) to report the incomplete multipart upload object count.
错误：SCP用于限制或控制账户内的服务使用权限，无法直接报告S3存储指标。
C. Configure S3 Storage Lens to report the incomplete multipart upload object count.
正确：S3 Storage Lens是专门设计用于监控和分析S3存储使用情况的工具，支持报告未完成的multipart上传对象数量，且操作简
单。
D. Create an S3 Multi-Region Access Point to report the incomplete multipart upload object count.
错误：S3 Multi-Region Access Point用于跨区域访问S3存储桶，无法直接报告未完成的multipart上传对象数量。
3. 我选的答案
C. Configure S3 Storage Lens to report the incomplete multipart upload object count.

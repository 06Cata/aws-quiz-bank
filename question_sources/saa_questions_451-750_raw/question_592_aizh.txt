Question #592
A company uses AWS and sells access to copyrighted images. The company’s global customer base needs to be able to access
these images quickly. The company must deny access to users from specific countries. The company wants to minimize costs as
much as possible.
Which solution will meet these requirements?
一家公司使用AWS并销售受版权保护的图片访问权限。该公司的全球客户群需要能够快速访问这些图片。公司必须拒绝来自特定国
家的用户访问。公司希望尽可能降低成本。
哪种解决方案可以满足这些要求？
A. Use Amazon S3 to store the images. Turn on multi-factor authentication (MFA) and public bucket access. Provide customers
with a link to the S3 bucket.
使用Amazon S3存储图片。开启多因素认证（MFA）和公共桶访问权限。向客户提供指向S3桶的链接。
B. Use Amazon S3 to store the images. Create an IAM user for each customer. Add the users to a group that has permission to
access the S3 bucket.
使用Amazon S3存储图片。为每个客户创建一个IAM用户。将这些用户添加到具有访问S3桶权限的组中。
C. Use Amazon EC2 instances that are behind Application Load Balancers (ALBs) to store the images. Deploy the instances only
in the countries the company services. Provide customers with links to the ALBs for their specific country's instances.
使用部署在Application Load Balancers (ALBs)后面的Amazon EC2实例存储图片。仅在公司服务的国家部署实例。向客户提供指
向特定国家实例的ALBs链接。
D. Use Amazon S3 to store the images. Use Amazon CloudFront to distribute the images with geographic restrictions. Provide
a signed URL for each customer to access the data in CloudFront. Most Voted
使用Amazon S3存储图片。使用Amazon CloudFront分发带有地理限制的图片。为每个客户提供一个签名URL以访问CloudFront
中的数据。
Correct Answer: D
正确答案：D

题目分析与解答
1. 考察的知识点
Amazon S3的存储功能
Amazon CloudFront的内容分发功能及地理限制
如何实现访问控制（如签名URL）
成本优化
2. 选项分析
A. 使用Amazon S3存储图片。开启多因素认证（MFA）和公共桶访问权限。向客户提供指向S3桶的链接。
错误：MFA适用于用户身份验证，而不是直接控制地理访问。公共桶访问权限无法限制特定国家的访问。
错误：此方案无法满足“拒绝特定国家访问”的要求。
B. 使用Amazon S3存储图片。为每个客户创建一个IAM用户。将这些用户添加到具有访问S3桶权限的组中。
错误：为每个客户创建IAM用户会导致管理复杂性和高成本。
错误：IAM用户无法直接实现地理限制。
C. 使用部署在Application Load Balancers (ALBs)后面的Amazon EC2实例存储图片。仅在公司服务的国家部署实例。向客户提供
指向特定国家实例的ALBs链接。
错误：使用EC2实例存储图片成本较高，不符合“尽可能降低成本”的要求。
错误：此方案需要复杂的部署和管理，且无法灵活地实现地理限制。
D. 使用Amazon S3存储图片。使用Amazon CloudFront分发带有地理限制的图片。为每个客户提供一个签名URL以访问
CloudFront中的数据。
正确：Amazon CloudFront支持地理限制，可以拒绝特定国家的访问。
正确：签名URL可以确保只有授权用户能够访问内容。
正确：使用S3和CloudFront的组合是成本优化的解决方案。

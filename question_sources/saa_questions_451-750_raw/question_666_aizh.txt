Question #666
A startup company is hosting a website for its customers on an Amazon EC2 instance. The website consists of a stateless Python
application and a MySQL database. The website serves only a small amount of traffic. The company is concerned about the
reliability of the instance and needs to migrate to a highly available architecture. The company cannot modify the application
code.
Which combination of actions should a solutions architect take to achieve high availability for the website? (Choose two.)
一家初创公司正在使用 Amazon EC2 实例为其客户托管一个网站。该网站由一个无状态的 Python 应用程序和一个 MySQL 数据库组
成。该网站仅处理少量流量。公司担心实例的可靠性，需要迁移到高可用架构。公司无法修改应用程序代码。
解决方案架构师应该采取哪些行动组合来实现网站的高可用性？（选择两个）
A. Provision an internet gateway in each Availability Zone in use.
为使用中的每个可用区配置一个 Internet 网关。
B. Migrate the database to an Amazon RDS for MySQL Multi-AZ DB instance. Most Voted
将数据库迁移到 Amazon RDS for MySQL Multi-AZ DB 实例。
C. Migrate the database to Amazon DynamoDB, and enable DynamoDB auto scaling.
将数据库迁移到 Amazon DynamoDB，并启用 DynamoDB 自动扩展。
D. Use AWS DataSync to synchronize the database data across multiple EC2 instances.
使用 AWS DataSync 在多个 EC2 实例之间同步数据库数据。
E. Create an Application Load Balancer to distribute traffic to an Auto Scaling group of EC2 instances that are distributed
across two Availability Zones. Most Voted
创建一个 Application Load Balancer，将流量分配到跨两个可用区分布的 EC2 实例 Auto Scaling 组。
Correct Answer: BE
正确答案：BE

题目分析与解答
1. 考察的知识点
高可用架构设计：如何设计一个高可用的系统，确保服务的可靠性和冗余。
Amazon RDS Multi-AZ：了解如何使用RDS的多可用区部署来提高数据库的可用性。
负载均衡与自动扩展：使用Application Load Balancer和Auto Scaling组来分配流量并实现弹性扩展。
2. 选项分析
A. Provision an internet gateway in each Availability Zone in use.
错误：Internet Gateway用于提供VPC与互联网的连接，但与高可用架构无关。高可用性主要关注服务的冗余和故障转移，而不是网
络连接的配置。
B. Migrate the database to an Amazon RDS for MySQL Multi-AZ DB instance.
正确：Amazon RDS Multi-AZ提供了数据库的冗余和自动故障转移功能，确保数据库的高可用性。
C. Migrate the database to Amazon DynamoDB, and enable DynamoDB auto scaling.
错误：虽然DynamoDB是一个高可用的NoSQL数据库，但题目明确指出使用的是MySQL数据库，且不能修改应用代码，因此无法迁
移到DynamoDB。
D. Use AWS DataSync to synchronize the database data across multiple EC2 instances.
错误：AWS DataSync用于数据迁移和同步，但它不提供高可用性解决方案，也不适合用于数据库的实时冗余。
E. Create an Application Load Balancer to distribute traffic to an Auto Scaling group of EC2 instances that are distributed
across two Availability Zones.
正确：使用Application Load Balancer和Auto Scaling组可以实现Web应用的高可用性和弹性扩展，确保流量分布和故障转移。

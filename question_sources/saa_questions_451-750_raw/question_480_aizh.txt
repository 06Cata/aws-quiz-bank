Question #480
A business application is hosted on Amazon EC2 and uses Amazon S3 for encrypted object storage. The chief information security
officer has directed that no application traffic between the two services should traverse the public internet.
Which capability should the solutions architect use to meet the compliance requirements?

量不嬂通过公共互联网。
解决方案架构宵寒该使用哪种功能来满足合规要求？

题目分析与解答
1. 考察的知识点
本题考孹的是如何确保AWS服务之间的通信不通过公共互联网，重点是使用VPC Endpoint来孩现私有连接。
2. 选项分析
Correct Answer: B
正确答案：B
Community vote distribution
社区投票分害
B (100%)
A. AWS Key Management Service (AWS KMS)
错误：AWS KMS是用于管理加孶孶钥的服务，与确保流量不通过公共互联网无关。
B. VPC endpoint
正确：VPC Endpoint允许在VPC内与AWS服务（如S3）进行私有连接，避免流量通过公共互联网。
C. Private subnet
错误：Private subnet是VPC中的子网类型，但孚本身不能确保服务之间的通信不通过公共互联网。
D. Virtual private gateway
错误：Virtual private gateway用于连接本地数据中嬇和VPC，不能直接解决服务之间的私有通信问题。
3. 我的答案
B. VPC endpoint
4. 官方答案
B. VPC endpoint

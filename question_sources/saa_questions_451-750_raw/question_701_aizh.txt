Question #701
A city has deployed a web application running on Amazon EC2 instances behind an Application Load Balancer (ALB). The
application's users have reported sporadic performance, which appears to be related to DDoS attacks originating from random IP
addresses. The city needs a solution that requires minimal configuration changes and provides an audit trail for the DDoS sources.
Which solution meets these requirements?

题目分析与解答
1. 考察的知识点
本题主要考察AWS的DDoS防护解决方案，包括AWS Shield、AWS WAF、CloudFront等服务的使用场景和功能。
还涉及如何选择适合的服务来应对DDoS攻击，同时满足审计需求。
2. 选项分析
A. Enable an AWS WAF web ACL on the ALB, and configure rules to block traffic from unknown sources.
正确性分析：AWS WAF可以帮助过滤恶意流量，但它需要手动配置规则来识别和阻止DDoS攻击源，这可能无法有效应对随机IP地
址的攻击。此外，AWS WAF本身不提供专门的DDoS防护功能。
结论：此选项不符合题目要求，因为它无法提供全面的DDoS防护，也没有提到审计功能。
B. Subscribe to Amazon Inspector. Engage the AWS DDoS Response Team (DRT) to integrate mitigating controls into the
service.
正确性分析：Amazon Inspector是一个安全评估服务，用于检测漏洞和配置问题，与DDoS防护无关。此选项提到的AWS DDoS
Response Team (DRT)确实可以协助，但Amazon Inspector并不是DDoS防护的正确工具。
结论：此选项错误，因为Amazon Inspector不适用于DDoS防护场景。
C. Subscribe to AWS Shield Advanced. Engage the AWS DDoS Response Team (DRT) to integrate mitigating controls into
the service.
正确性分析：AWS Shield Advanced是专门设计用于保护应用免受复杂DDoS攻击的服务。它提供自动化的防护功能，并允许与AWS
DDoS Response Team (DRT)合作来实施缓解措施。此外，它还提供详细的攻击分析和审计功能，完全符合题目要求。
结论：此选项正确，因为它提供了全面的DDoS防护和审计功能，并且配置更简单。
D. Create an Amazon CloudFront distribution for the application, and set the ALB as the origin. Enable an AWS WAF web
ACL on the distribution, and configure rules to block traffic from unknown sources.
正确性分析：CloudFront可以帮助缓解DDoS攻击，并与AWS WAF结合使用以过滤流量。然而，与AWS Shield Advanced相比，
CloudFront的DDoS防护功能较为基础，无法提供与AWS DDoS Response Team (DRT)的直接协作，也没有提到审计功能。
结论：此选项部分正确，但不如选项C全面，无法完全满足题目要求。

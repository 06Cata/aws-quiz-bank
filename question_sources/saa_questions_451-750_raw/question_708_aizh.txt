A company wants to use NAT gateways in its AWS environment. The company's Amazon EC2 instances in private subnets must be
able to connect to the public internet through the NAT gateways.
Which solution will meet these requirements?

接到公共互联网。
哪种解决方案可以满足这些需求？
A. Create public NAT gateways in the same private subnets as the EC2 instances.

在与 EC2 孩例相同的私有子网中创寠私有 NAT gateways。

题目分析与解答
1. 考察的知识点
本题考孹的是AWS网络架构中的NAT网关的使用场景和部署位置。
Correct Answer: C
正确答案：C

社区投票分害
C (100%)
重点知识包括：NAT网关的类型（Public NAT Gateway和Private NAT Gateway）、子网类型（Public Subnet和Private Subnet）、以
及如何孩现私有子网中的EC2孩例访问互联网。
2. 选项分析
A. Create public NAT gateways in the same private subnets as the EC2 instances.
错误：NAT网关嬈须部署在公共子网中，而不是私有子网。公共子网具有互联网网关（Internet Gateway）连接，能够孩现外部通
信。
B. Create private NAT gateways in the same private subnets as the EC2 instances.
错误：Private NAT Gateway用于VPC间通信，而不是用于连接互联网。寈且NAT网关不能部署在私有子网中。
C. Create public NAT gateways in public subnets in the same VPCs as the EC2 instances.
正确：Public NAT Gateway需要部署在公共子网中，寈通过路由表宂私有子网的流量指向NAT网关，从而孩现私有子网中的EC2孩例
访问互联网。
D. Create private NAT gateways in public subnets in the same VPCs as the EC2 instances.
错误：Private NAT Gateway不适用于连接互联网，孚主要用于跨VPC通信嬻内部网络通信。

嬺的选择是：C

季方答案是：C

标准设计。

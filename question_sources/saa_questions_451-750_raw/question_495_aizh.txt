Question #495
A company is conducting an internal audit. The company wants to ensure that the data in an Amazon S3 bucket that is associated
with the company’s AWS Lake Formation data lake does not contain sensitive customer or employee data. The company wants to
discover personally identifiable information (PII) or financial information, including passport numbers and credit card numbers.
Which solution will meet these requirements?
一家公司正在进行内部审计。该公司希望确保与公司 AWS Lake Formation 数据湖相关联的 Amazon S3 存储桶中的数据不包含敏感
的客户或员工数据。公司希望发现个人身份信息 (PII) 或财务信息，包括护照号码和信用卡号码。
哪种解决方案可以满足这些要求？
A. Configure AWS Audit Manager on the account. Select the Payment Card Industry Data Security Standards (PCI DSS) for
auditing.
在账户上配置 AWS Audit Manager。选择支付卡行业数据安全标准 (PCI DSS) 进行审计。
B. Configure Amazon S3 Inventory on the S3 bucket Configure Amazon Athena to query the inventory.
在 S3 存储桶上配置 Amazon S3 Inventory。配置 Amazon Athena 查询清单。
C. Configure Amazon Macie to run a data discovery job that uses managed identifiers for the required data types. Most Voted
配置 Amazon Macie 运行一个数据发现作业，使用托管标识符来识别所需的数据类型。
D. Use Amazon S3 Select to run a report across the S3 bucket.
使用 Amazon S3 Select 在 S3 存储桶上运行报告。
题目分析与解答
1. 考察的知识点
Correct Answer: C
正确答案: C
Community vote distribution
社区投票分布
C (100%)
本题考察的是AWS服务中数据发现和敏感信息识别的能力，尤其是如何使用AWS工具来发现和保护存储在Amazon S3中的敏感数据
（如PII和财务信息）。
涉及的服务包括Amazon Macie、AWS Audit Manager、Amazon S3 Inventory、Amazon Athena和Amazon S3 Select。
2. 选项分析
A. 配置 AWS Audit Manager。选择支付卡行业数据安全标准 (PCI DSS) 进行审计。
错误：AWS Audit Manager主要用于帮助企业评估和审计其合规性（如PCI DSS），但它并不具备直接扫描S3存储桶中的数据以发现
敏感信息的功能。
B. 配置 Amazon S3 Inventory。配置 Amazon Athena 查询清单。
错误：Amazon S3 Inventory提供有关S3存储桶中对象的元数据报告，但它无法识别敏感数据（如PII或财务信息）。Amazon
Athena可以查询这些清单，但仍然无法直接发现敏感信息。
C. 配置 Amazon Macie 运行一个数据发现作业，使用托管标识符来识别所需的数据类型。
正确：Amazon Macie是一项专门设计用于发现和保护敏感数据的服务。它可以扫描S3存储桶中的数据，并使用托管标识符来识别
PII和财务信息（如护照号码和信用卡号码）。
D. 使用 Amazon S3 Select 在 S3 存储桶上运行报告。
错误：Amazon S3 Select用于从S3对象中提取特定数据，但它无法自动识别敏感信息或运行数据发现作业。

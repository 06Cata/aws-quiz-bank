Question #519
A consulting company provides professional services to customers worldwide. The company provides solutions and tools for
customers to expedite gathering and analyzing data on AWS. The company needs to centrally manage and deploy a common set
of solutions and tools for customers to use for self-service purposes.
Which solution will meet these requirements?
一家咨询公司向全球客户提供专业服务。 该公司为客户提供解决方案和工具，以加速在 AWS 上收集和分析数据。 该公司需要集中
管理并部署一套通用的解决方案和工具，供客户用于自助服务目的。
哪种解决方案可以满足这些需求？
A. Create AWS CloudFormation templates for the customers.

题目分析与解答
1. 考察的知识点
Correct Answer: B
正确答案：B

B (100%)
本题考察的是AWS服务的选择与应用，特别是如何使用AWS工具来实现集中管理和部署解决方案的能力。
重点知识点包括AWS Service Catalog、AWS CloudFormation、AWS Systems Manager和AWS Config的功能和适用场景。
2. 选项分析
A. Create AWS CloudFormation templates for the customers.
CloudFormation主要用于定义和部署基础架构资源，通过模板自动化资源创建。
虽然可以用CloudFormation模板来创建资源，但它不具备集中管理和提供自助服务的功能。
因此，此选项不完全满足题目要求。
B. Create AWS Service Catalog products for the customers.
AWS Service Catalog专门用于集中管理和部署产品（解决方案和工具），并支持自助服务功能。
它允许客户通过预定义的产品快速部署资源，同时确保符合公司标准。
此选项完全符合题目要求，是正确答案。
C. Create AWS Systems Manager templates for the customers.
Systems Manager主要用于管理和自动化操作任务，例如补丁管理、运行命令等。
虽然它可以帮助管理资源，但它不适合用于集中管理和部署解决方案的场景。
因此，此选项不符合题目要求。
D. Create AWS Config items for the customers.
AWS Config主要用于资源合规性检查和配置审计。
它不具备部署解决方案或工具的功能，也不支持自助服务。
因此，此选项不符合题目要求。

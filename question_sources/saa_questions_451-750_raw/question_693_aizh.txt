Question #693
A company has a web application that includes an embedded NoSQL database. The application runs on Amazon EC2 instances
behind an Application Load Balancer (ALB). The instances run in an Amazon EC2 Auto Scaling group in a single Availability Zone.
A recent increase in traffic requires the application to be highly available and for the database to be eventually consistent.
Which solution will meet these requirements with the LEAST operational overhead?

Load Balancer (ALB) 后面。这些实例运行在单个可用区中的 Amazon EC2 Auto Scaling 组中。
昀近流量的增加要求该应用程序具有高可用性，并且数据库昀终一致性。
哪种解决方案能够以昀少的运营开销满足这些要求？

Correct Answer: D
正确答案：D
题目分析与解答
1. 考察的知识点
高可用性架构设计：如何在多个可用区中部署资源以实现高可用性。
数据库迁移：使用 AWS Database Migration Service (AWS DMS) 将嵌入式数据库迁移到托管服务。
Amazon DynamoDB：托管的 NoSQL 数据库服务，支持高可用性和昀终一致性。
负载均衡器选择：了解 Application Load Balancer (ALB) 和 Network Load Balancer (NLB) 的适用场景。
2. 选项分析
A. Replace the ALB with a Network Load Balancer. Maintain the embedded NoSQL database with its replication service on
the EC2 instances.
错误：Network Load Balancer (NLB) 适用于低延迟和 TCP/UDP 流量，但题目中没有提到需要这种特性。ALB 更适合 HTTP/HTTPS
流量。
错误：嵌入式 NoSQL 数据库仍然运行在 EC2 实例上，无法实现高可用性，因为它依赖单个可用区。
B. Replace the ALB with a Network Load Balancer. Migrate the embedded NoSQL database to Amazon DynamoDB by
using AWS Database Migration Service (AWS DMS).
错误：虽然迁移到 DynamoDB 是正确的方向，但替换 ALB 为 NLB 不符合题目需求。ALB 更适合处理 Web 应用流量。
正确：迁移到 DynamoDB 可以实现高可用性和昀终一致性。
C. Modify the Auto Scaling group to use EC2 instances across three Availability Zones. Maintain the embedded NoSQL
database with its replication service on the EC2 instances.
正确：跨三个可用区的 Auto Scaling 组可以提高 EC2 实例的高可用性。
错误：嵌入式 NoSQL 数据库仍然运行在 EC2 实例上，增加了运维复杂性，且无法充分利用托管服务的优势。
D. Modify the Auto Scaling group to use EC2 instances across three Availability Zones. Migrate the embedded NoSQL
database to Amazon DynamoDB by using AWS Database Migration Service (AWS DMS).
正确：跨三个可用区的 Auto Scaling 组可以提高 EC2 实例的高可用性。
正确：迁移到 DynamoDB 可以实现高可用性和昀终一致性，同时减少运维开销。

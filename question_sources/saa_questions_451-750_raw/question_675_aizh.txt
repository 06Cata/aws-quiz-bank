Question #675
A company uses Amazon EC2 instances and Amazon Elastic Block Store (Amazon EBS) volumes to run an application. The
company creates one snapshot of each EBS volume every day to meet compliance requirements. The company wants to
implement an architecture that prevents the accidental deletion of EBS volume snapshots. The solution must not change the
administrative rights of the storage administrator user.
Which solution will meet these requirements with the LEAST administrative effort?
一家公司使用 Amazon EC2 实例和 Amazon Elastic Block Store (Amazon EBS) 卷来运行一个应用程序。公司每天为每个 EBS 卷创建
一个快照以满足合规要求。公司希望实施一种架构，以防止 EBS 卷快照被意外删除。解决方案不得更改存储管理员用户的管理权
限。
哪种解决方案能够以最少的管理工作量满足这些要求？
A. Create an IAM role that has permission to delete snapshots. Attach the role to a new EC2 instance. Use the AWS CLI from
the new EC2 instance to delete snapshots.
创建一个具有删除快照权限的 IAM 角色。将该角色附加到一个新的 EC2 实例。从新的 EC2 实例使用 AWS CLI 删除快照。
B. Create an IAM policy that denies snapshot deletion. Attach the policy to the storage administrator user.
创建一个拒绝快照删除的 IAM 策略。将该策略附加到存储管理员用户。
C. Add tags to the snapshots. Create retention rules in Recycle Bin for EBS snapshots that have the tags.
向快照添加标签。在 Recycle Bin 中为具有这些标签的 EBS 快照创建保留规则。
D. Lock the EBS snapshots to prevent deletion. Most Voted
锁定 EBS 快照以防止删除。
Correct Answer: D
正确答案：D

题目分析与解答
1. 考察的知识点
本题考察的是AWS EBS快照的保护机制，包括如何防止快照被意外删除。
涉及的服务包括Amazon EBS、IAM权限管理以及Recycle Bin功能。
2. 选项分析
A. Create an IAM role that has permission to delete snapshots. Attach the role to a new EC2 instance. Use the AWS CLI
from the new EC2 instance to delete snapshots. 错误。此选项并未防止快照的意外删除，而是提供了一种删除快照的方式。与题
目要求的“防止意外删除”不符。
B. Create an IAM policy that denies snapshot deletion. Attach the policy to the storage administrator user. 错误。虽然此选
项可以防止快照删除，但它改变了存储管理员用户的权限，违反了题目要求“不能改变存储管理员用户的权限”的条件。
C. Add tags to the snapshots. Create retention rules in Recycle Bin for EBS snapshots that have the tags. 正确。Recycle Bin
功能允许为带有特定标签的EBS快照创建保留规则，从而防止意外删除。这是一个有效的解决方案，但与选项D相比，可能需要更多
的管理工作。
D. Lock the EBS snapshots to prevent deletion. 正确。AWS提供了锁定EBS快照的功能，可以直接防止快照被删除。这是最直
接、最少管理工作量的解决方案，完全符合题目要求。
3. 我的答案
D. Lock the EBS snapshots to prevent deletion.
4. 官方答案
D. Lock the EBS snapshots to prevent deletion.
5. 与官方答案的比较
我的答案与官方答案一致。
原因：选项D是最符合题目要求的解决方案，既防止了快照的意外删除，又不改变存储管理员用户的权限，同时管理工作量最少。

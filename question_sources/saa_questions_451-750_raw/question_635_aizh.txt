Question #635
A company uses Amazon FSx for NetApp ONTAP in its primary AWS Region for CIFS and NFS file shares. Applications that run on
Amazon EC2 instances access the file shares. The company needs a storage disaster recovery (DR) solution in a secondary Region.
The data that is replicated in the secondary Region needs to be accessed by using the same protocols as the primary Region.
Which solution will meet these requirements with the LEAST operational overhead?
一家公司在其主要的AWS区域使用Amazon FSx for NetApp ONTAP来提供CIFS和NFS文件共享。运行在Amazon EC2实例上的应用程
序访问这些文件共享。该公司需要在次要区域中实现一个存储灾难恢复（DR）解决方案。复制到次要区域的数据需要使用与主要区
域相同的协议进行访问。
哪种解决方案能够以最少的运营开销满足这些需求？
A. Create an AWS Lambda function to copy the data to an Amazon S3 bucket. Replicate the S3 bucket to the secondary Region.
创建一个AWS Lambda函数，将数据复制到一个Amazon S3桶中。将S3桶复制到次要区域。
B. Create a backup of the FSx for ONTAP volumes by using AWS Backup. Copy the volumes to the secondary Region. Create a
new FSx for ONTAP instance from the backup.
使用AWS Backup对FSx for ONTAP卷创建备份。将卷复制到次要区域。从备份中创建一个新的FSx for ONTAP实例。
C. Create an FSx for ONTAP instance in the secondary Region. Use NetApp SnapMirror to replicate data from the primary
Region to the secondary Region. Most Voted
在次要区域中创建一个FSx for ONTAP实例。使用NetApp SnapMirror将数据从主要区域复制到次要区域。
D. Create an Amazon Elastic File System (Amazon EFS) volume. Migrate the current data to the volume. Replicate the volume to
the secondary Region.
创建一个Amazon Elastic File System (Amazon EFS)卷。将当前数据迁移到该卷。将该卷复制到次要区域。
Correct Answer: C
正确答案：C

题目分析与解答
1. 考察的知识点
Amazon FSx for NetApp ONTAP的功能与特性
灾难恢复（Disaster Recovery）解决方案
跨区域数据复制技术（如NetApp SnapMirror）
文件存储协议（CIFS和NFS）的支持
2. 选项分析
A. Create an AWS Lambda function to copy the data to an Amazon S3 bucket. Replicate the S3 bucket to the secondary
Region.
错误：虽然S3支持跨区域复制，但S3不支持CIFS和NFS协议，无法满足题目要求。
B. Create a backup of the FSx for ONTAP volumes by using AWS Backup. Copy the volumes to the secondary Region.
Create a new FSx for ONTAP instance from the backup.
错误：此方法需要手动创建备份并恢复，操作复杂且不支持实时数据同步，无法满足最低操作开销的要求。
C. Create an FSx for ONTAP instance in the secondary Region. Use NetApp SnapMirror to replicate data from the primary
Region to the secondary Region.
正确：NetApp SnapMirror是专为FSx for ONTAP设计的跨区域数据复制解决方案，支持CIFS和NFS协议，并且操作开销最低。
D. Create an Amazon Elastic File System (Amazon EFS) volume. Migrate the current data to the volume. Replicate the
volume to the secondary Region.
错误：EFS不支持CIFS协议，仅支持NFS协议，无法满足题目要求。

C

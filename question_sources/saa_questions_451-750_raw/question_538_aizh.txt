out content in a phased manner across multiple countries. The company needs to ensure that viewers who are outside the
countries to which the company rolls out content are not able to view the content.
Which solution will meet these requirements?

司需要确保位于内容推出国家之外的观众无法观看内容。

Amazon CloudFront的地理限制功能（Geographic Restrictions）。
如何通过CloudFront控制内容访问权限，尤其是基于地理位置的访问控制。
2. 选项分析
A. Add geographic restrictions to the content in CloudFront by using an allow list. Set up a custom error message.
正确：CloudFront支持地理限制功能，可以通过允许列表（Allow List）或阻止列表（Deny List）来限制特定国家或地区的访问。
正确：设置自定义错误消息是一个附加功能，可以为被限制的用户提供更友好的反馈。
B. Set up a new URL for restricted content. Authorize access by using a signed URL and cookies. Set up a custom error
message.
错误：签名URL和Cookie主要用于授权特定用户访问内容，而不是基于地理位置的限制。
错误：题目要求基于国家或地区的限制，而不是基于用户身份的授权。
C. Encrypt the data for the content that the company distributes. Set up a custom error message.
错误：加密内容与地理位置限制无关。加密主要用于保护数据安全，而不是控制访问权限。
错误：题目没有提到需要加密内容。
D. Create a new URL for restricted content. Set up a time-restricted access policy for signed URLs.
错误：时间限制的签名URL用于控制访问时间，而不是基于地理位置的限制。
错误：题目要求基于国家或地区的限制，而不是基于时间的限制。

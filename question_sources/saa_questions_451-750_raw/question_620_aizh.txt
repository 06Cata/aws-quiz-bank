题目分析与解答
1. 考察的知识点
本题考察的是AWS存储服务的特性，包括Amazon EBS卷类型的性能和用途。
重点关注存储的耐久性、低延迟性能以及适合业务关键型应用的存储选项。
2. 选项分析
Correct Answer: C
正确答案：C

C (100%)
A. Instance store volume
错误：Instance store是临时存储，数据在实例停止或终止时会丢失，不具备耐久性。
不适合业务关键型应用的存储需求。
B. Amazon ElastiCache for Memcached cluster
错误：ElastiCache是内存级缓存服务，主要用于加速数据访问，而不是持久存储。
不满足“耐久存储”的要求。
C. Provisioned IOPS SSD Amazon Elastic Block Store (Amazon EBS) volume
正确：Provisioned IOPS SSD EBS卷提供高性能、低延迟和耐久性，适合业务关键型应用。
支持高IOPS需求，能够满足应用的性能要求。
D. Throughput Optimized HDD Amazon Elastic Block Store (Amazon EBS) volume
错误：Throughput Optimized HDD EBS卷适合大规模顺序读写操作（如日志处理），但不适合低延迟、高性能需求。
不满足业务关键型应用的性能要求。

Question #847
A company uses Amazon RDS for PostgreSQL databases for its data tier. The company must implement password rotation for the
databases.
Which solution meets this requirement with the LEAST operational overhead?
一家公司使用 Amazon RDS for PostgreSQL 数据库作为其数据层。该公司必须为数据库实施密码轮换。
哪种解决方案以最少的运营开销满足此要求？
A. Store the password in AWS Secrets Manager. Enable automatic rotation on the secret. Most Voted
将密码存储在 AWS Secrets Manager 中。启用对密钥的自动轮换。
B. Store the password in AWS Systems Manager Parameter Store. Enable automatic rotation on the parameter.
将密码存储在 AWS Systems Manager Parameter Store 中。启用对参数的自动轮换。
C. Store the password in AWS Systems Manager Parameter Store. Write an AWS Lambda function that rotates the password.
将密码存储在 AWS Systems Manager Parameter Store 中。编写一个 AWS Lambda 函数来轮换密码。
D. Store the password in AWS Key Management Service (AWS KMS). Enable automatic rotation on the AWS KMS key.
将密码存储在 AWS Key Management Service (AWS KMS) 中。启用对 AWS KMS 密钥的自动轮换。
题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中密码管理与自动轮换的最佳实践，特别是AWS Secrets Manager和AWS Systems Manager的功能比较。
2. 选项分析
Correct Answer: A
正确答案：A
Community vote distribution
社区投票分布
A (100%)
A. Store the password in AWS Secrets Manager. Enable automatic rotation on the secret.
正确：AWS Secrets Manager专门设计用于管理敏感信息（如数据库密码），并支持自动轮换功能。启用自动轮换后，Secrets
Manager会自动更新密码并确保应用程序可以无缝访问更新后的密码。
最少的操作开销：无需编写额外的代码或手动管理密码轮换。
B. Store the password in AWS Systems Manager Parameter Store. Enable automatic rotation on the parameter.
错误：AWS Systems Manager Parameter Store虽然可以存储敏感信息，但它不支持自动轮换功能。
需要额外的操作来实现密码轮换，因此不符合“最少操作开销”的要求。
C. Store the password in AWS Systems Manager Parameter Store. Write an AWS Lambda function that rotates the
password.
错误：虽然可以通过自定义Lambda函数实现密码轮换，但这需要额外的开发和维护工作。
操作开销较高，不符合题目要求的“最少操作开销”。
D. Store the password in AWS Key Management Service (AWS KMS). Enable automatic rotation on the AWS KMS key.
错误：AWS KMS主要用于管理加密密钥，而不是存储和轮换数据库密码。
密码管理不是AWS KMS的设计用途，因此不适合解决此问题。

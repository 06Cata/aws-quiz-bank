Question #815
An online gaming company hosts its platform on Amazon EC2 instances behind Network Load Balancers (NLBs) across multiple
AWS Regions. The NLBs can route requests to targets over the internet. The company wants to improve the customer playing
experience by reducing end-to-end load time for its global customer base.
Which solution will meet these requirements?
一家在线游戏公司在多个AWS区域的Network Load Balancers (NLBs)后面托管其平台，使用Amazon EC2实例。NLBs可以通过互联
网将请求路由到目标。该公司希望通过减少全球客户群的端到端加载时间来改善客户的游戏体验。
哪种解决方案可以满足这些需求？
A. Create Application Load Balancers (ALBs) in each Region to replace the existing NLBs. Register the existing EC2 instances as
targets for the ALBs in each Region.
在每个区域创建Application Load Balancers (ALBs)以替换现有的NLBs。在每个区域将现有的EC2实例注册为ALBs的目标。
B. Configure Amazon Route 53 to route equally weighted traffic to the NLBs in each Region.
配置Amazon Route 53以将等权重流量路由到每个区域的NLBs。
C. Create additional NLBs and EC2 instances in other Regions where the company has large customer bases.
在公司拥有大量客户群的其他区域创建额外的NLBs和EC2实例。
D. Create a standard accelerator in AWS Global Accelerator. Configure the existing NLBs as target endpoints. Most Voted
在AWS Global Accelerator中创建一个标准加速器。将现有的NLBs配置为目标端点。

Correct Answer: D
正确答案：D

D (100%)
本题考察的是AWS全球网络优化服务的使用，包括AWS Global Accelerator、负载均衡器（NLB和ALB）、Route 53的流量路由策

重点在于如何优化全球用户的访问体验，减少端到端的延迟。
2. 选项分析
A. Create Application Load Balancers (ALBs) in each Region to replace the existing NLBs. Register the existing EC2
instances as targets for the ALBs in each Region.
错误：虽然ALB可以提供高级的HTTP/HTTPS功能，但它并不能直接减少全球用户的延迟。ALB主要用于应用层负载均衡，而题目要
求优化全球用户的访问体验，ALB无法解决跨区域的网络延迟问题。
B. Configure Amazon Route 53 to route equally weighted traffic to the NLBs in each Region.
错误：Route 53的流量路由策略可以分配流量到不同区域的NLB，但它无法优化网络路径或减少延迟。Route 53仅负责DNS解析，
无法提供加速功能。
C. Create additional NLBs and EC2 instances in other Regions where the company has large customer bases.
错误：虽然增加NLB和EC2实例可以扩展容量，但这并不能解决网络延迟问题。跨区域的网络优化需要使用专门的加速服务，而不是
简单地增加资源。
D. Create a standard accelerator in AWS Global Accelerator. Configure the existing NLBs as target endpoints.
正确：AWS Global Accelerator通过使用AWS的全球网络优化流量路径，可以显著减少网络延迟。它会将用户流量路由到昀近的
AWS边缘位置，并通过优化的路径将流量发送到目标区域的NLB。这是解决全球用户访问延迟的昀佳方案。
3. 我的答案
D. Create a standard accelerator in AWS Global Accelerator. Configure the existing NLBs as target endpoints.
4. 官方答案
D. Create a standard accelerator in AWS Global Accelerator. Configure the existing NLBs as target endpoints.
5. 与官方答案比较
我的答案与官方答案一致。
原因：AWS Global Accelerator是专门设计用于优化全球用户访问体验的服务，能够显著减少网络延迟并提高应用性能。

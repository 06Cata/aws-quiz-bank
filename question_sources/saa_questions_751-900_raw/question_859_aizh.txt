题目分析与解答
1. 考察的知识点
本题主要考察AWS Trusted Advisor的功能和使用场景，尤其是如何通过Trusted Advisor优化成本。
涉及的具体检查包括Reserved Instance Optimization和Idle DB Instances检查。
2. 选项分析
A. Use the Trusted Advisor recommendations in the management account. 分析：正确。管理账户可以查看整个组织的Trusted
Advisor建议，适合集中管理和优化成本。
B. Use the Trusted Advisor recommendations in the member accounts where the RDS DB instances are running. 分析：错
误。虽然成员账户可以查看自己的Trusted Advisor建议，但题目要求的是集中优化成本，管理账户更适合此场景。
C. Review the Trusted Advisor checks for Amazon RDS Reserved Instance Optimization. 分析：正确。Reserved Instance
Optimization检查可以帮助识别是否可以通过预留实例节省成本，是优化RDS成本的重要方法。
D. Review the Trusted Advisor checks for Amazon RDS Idle DB Instances. 分析：错误。Idle DB Instances检查适用于未充分利
用的数据库实例，但题目提到的是高利用率的实例，因此不适用。
E. Review the Trusted Advisor checks for compute optimization. Crosscheck the results by using AWS Compute Optimizer.
分析：错误。Compute optimization主要针对EC2实例优化，与RDS数据库实例无关。

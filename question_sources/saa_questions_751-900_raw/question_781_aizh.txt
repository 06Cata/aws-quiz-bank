题目分析与解答
1. 考察的知识点
本题考察的是AWS成本管理工具的使用，包括AWS Budgets、成本分配标签（Cost Allocation Tags）、以及如何设置预算警报。
2. 选项分析
A. Use cost allocation tags on AWS resources to label owners. Create usage budgets in AWS Budgets. Add an alert
threshold to receive notification when spending exceeds 60% of the budget.
正确：成本分配标签可以帮助标记资源的所有者，从而实现部门级别的成本追踪。
正确：AWS Budgets允许用户创建预算并设置警报阈值，当支出达到一定比例时发送通知。
此选项完全符合题目要求。
B. Use AWS Cost Explorer forecasts to determine resource owners. Use AWS Cost Anomaly Detection to create alert
threshold notifications when spending exceeds 60% of the budget.
错误：AWS Cost Explorer主要用于分析和预测成本，而不是直接标记资源所有者。
错误：AWS Cost Anomaly Detection用于检测异常支出模式，而不是设置预算警报。
此选项不符合题目要求。
C. Use cost allocation tags on AWS resources to label owners. Use AWS Support API on AWS Trusted Advisor to create
alert threshold notifications when spending exceeds 60% of the budget.
正确：成本分配标签可以帮助标记资源的所有者。
错误：AWS Trusted Advisor和AWS Support API主要用于优化资源和支持服务，不适合设置预算警报。
此选项部分符合题目要求，但不完全正确。
D. Use AWS Cost Explorer forecasts to determine resource owners. Create usage budgets in AWS Budgets. Add an alert
threshold to receive notification when spending exceeds 60% of the budget.
错误：AWS Cost Explorer主要用于分析和预测成本，而不是直接标记资源所有者。
正确：AWS Budgets允许用户创建预算并设置警报阈值。
此选项部分符合题目要求，但不完全正确。

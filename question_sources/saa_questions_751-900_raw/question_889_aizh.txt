Question #889
A global company runs its workloads on AWS. The company's application uses Amazon S3 buckets across AWS Regions for
sensitive data storage and analysis. The company stores millions of objects in multiple S3 buckets daily. The company wants to
identify all S3 buckets that are not versioning-enabled.
Which solution will meet these requirements?
一家全球性公司在 AWS 上运行其工作负载。该公司的应用程序使用跨 AWS 区域的 Amazon S3 存储桶来存储和分析敏感数据。该
公司每天在多个 S3 存储桶中存储数百万个对象。该公司希望识别所有未启用版本控制的 S3 存储桶。
哪种解决方案可以满足这些需求？
A. Set up an AWS CloudTrail event that has a rule to identify all S3 buckets that are not versioning-enabled across Regions.
设置一个 AWS CloudTrail 事件，该事件具有规则来识别跨区域所有未启用版本控制的 S3 存储桶。
B. Use Amazon S3 Storage Lens to identify all S3 buckets that are not versioning-enabled across Regions. Most Voted
使用 Amazon S3 Storage Lens 来识别跨区域所有未启用版本控制的 S3 存储桶。
C. Enable IAM Access Analyzer for S3 to identify all S3 buckets that are not versioning-enabled across Regions.
启用 IAM Access Analyzer for S3 来识别跨区域所有未启用版本控制的 S3 存储桶。
D. Create an S3 Multi-Region Access Point to identify all S3 buckets that are not versioning-enabled across Regions.
创建一个 S3 Multi-Region Access Point 来识别跨区域所有未启用版本控制的 S3 存储桶。

2. 选项分析
A. Set up an AWS CloudTrail event that has a rule to identify all S3 buckets that are not versioning-enabled across
Regions.
CloudTrail主要用于记录API调用和用户活动，而不是直接分析S3存储桶的配置。
虽然可以通过CloudTrail记录S3存储桶的创建或修改事件，但无法直接识别哪些存储桶未启用版本控制。
此选项不符合要求，因此是错误的。
B. Use Amazon S3 Storage Lens to identify all S3 buckets that are not versioning-enabled across Regions.
S3 Storage Lens是一个专门用于分析和监控S3存储桶的工具，可以提供关于存储桶配置的详细信息，包括版本控制状态。
此选项直接满足题目要求，可以跨区域识别未启用版本控制的存储桶。
此选项是正确的。
C. Enable IAM Access Analyzer for S3 to identify all S3 buckets that are not versioning-enabled across Regions.
IAM Access Analyzer主要用于分析资源的访问权限，而不是存储桶的配置状态。
此工具无法直接识别存储桶是否启用了版本控制。
此选项不符合要求，因此是错误的。
D. Create an S3 Multi-Region Access Point to identify all S3 buckets that are not versioning-enabled across Regions.
S3 Multi-Region Access Point用于跨区域访问存储桶，但它并不提供存储桶配置分析功能。
此选项无法满足题目要求，因此是错误的。
3. 我的答案
B. Use Amazon S3 Storage Lens to identify all S3 buckets that are not versioning-enabled across Regions.

Question #817
A marketing team wants to build a campaign for an upcoming multi-sport event. The team has news reports from the past five
years in PDF format. The team needs a solution to extract insights about the content and the sentiment of the news reports. The
solution must use Amazon Textract to process the news reports.
Which solution will meet these requirements with the LEAST operational overhead?
一个市场营销团队希望为即将到来的多项体育赛事创建一个活动。 该团队拥有过去五年的新闻报道，这些报道以 PDF 格式存储。
团队需要一个解决方案来提取关于内容和新闻报道情感的洞察。 该解决方案必须使用 Amazon Textract 来处理新闻报道。
哪个解决方案能够以最少的运营开销满足这些需求？
A. Provide the extracted insights to Amazon Athena for analysis. Store the extracted insights and analysis in an Amazon S3
bucket.
将提取的洞察提供给 Amazon Athena 进行分析。 将提取的洞察和分析存储在 Amazon S3 存储桶中。

将提取的洞察存储在 Amazon DynamoDB 表中。 使用 Amazon SageMaker 构建一个情感模型。

将提取的洞察存储在 Amazon S3 存储桶中。 使用 Amazon QuickSight 可视化和分析数据。

C (100%)
Amazon Textract：用于从文档中提取文本和数据。
Amazon Comprehend：用于自然语言处理（NLP），包括情感分析和内容分类。
Amazon S3：用于存储数据。
Amazon Athena：用于查询存储在S3中的数据。
Amazon DynamoDB：用于存储结构化数据。
Amazon SageMaker：用于构建和训练机器学习模型。
Amazon QuickSight：用于数据可视化和分析。
2. 选项分析
A. 提取的洞察提供给 Amazon Athena 进行分析，存储在 Amazon S3 中。
Amazon Athena适合查询结构化数据，但不直接支持情感分析或内容分类。
此选项未提到如何进行情感分析，无法满足题目要求。
错误原因：缺乏情感分析功能。
B. 提取的洞察存储在 Amazon DynamoDB 表中，使用 Amazon SageMaker 构建情感模型。
Amazon SageMaker可以构建情感模型，但需要额外的开发和训练工作，增加了操作复杂性。
题目要求“最少的操作开销”，此选项不符合要求。
错误原因：操作开销较高。
C. 提取的洞察提供给 Amazon Comprehend 进行分析，保存到 Amazon S3 中。
Amazon Comprehend可以直接进行情感分析和内容分类，符合题目要求。
Amazon S3用于存储分析结果，操作开销低。
正确原因：直接满足题目要求，且操作开销最低。

此选项未提到如何进行情感分析，无法满足题目要求。

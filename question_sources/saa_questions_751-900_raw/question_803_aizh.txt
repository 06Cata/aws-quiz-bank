Question #803
A solutions architect is designing a user authentication solution for a company. The solution must invoke two-factor
authentication for users that log in from inconsistent geographical locations, IP addresses, or devices. The solution must also be
able to scale up to accommodate millions of users.
Which solution will meet these requirements?
一位解决方案架构师正在为公司设计一个用户身份验证解决方案。该解决方案必须为从不一致的地理位置、IP地址或设备登录的用
户调用双因素身份验证。该解决方案还必须能够扩展以容纳数百万用户。
哪种解决方案可以满足这些要求？
A. Configure Amazon Cognito user pools for user authentication. Enable the risk-based adaptive authentication feature with
multifactor authentication (MFA). Most Voted
配置 Amazon Cognito 用户池进行用户身份验证。启用基于风险的自适应身份验证功能和多因素身份验证 (MFA)。
B. Configure Amazon Cognito identity pools for user authentication. Enable multi-factor authentication (MFA).
配置 Amazon Cognito 身份池进行用户身份验证。启用多因素身份验证 (MFA)。
C. Configure AWS Identity and Access Management (IAM) users for user authentication. Attach an IAM policy that allows the
AllowManageOwnUserMFA action.
配置 AWS Identity and Access Management (IAM) 用户进行用户身份验证。附加一个允许 AllowManageOwnUserMFA 操作的
IAM 策略。
D. Configure AWS IAM Identity Center (AWS Single Sign-On) authentication for user authentication. Configure the permission
sets to require multi-factor authentication (MFA).
配置 AWS IAM Identity Center (AWS Single Sign-On) 身份验证进行用户身份验证。配置权限集以要求多因素身份验证 (MFA)。

题目分析与解答
1. 考察的知识点
Amazon Cognito的功能，包括用户池和身份池的区别。
多因素身份验证 (MFA) 的实现方式。
风险感知和自适应身份验证的概念。
可扩展性和支持大规模用户的解决方案。
2. 选项分析
A. Configure Amazon Cognito user pools for user authentication. Enable the risk-based adaptive authentication feature
with multifactor authentication (MFA).
正确：Amazon Cognito用户池支持用户身份验证，并且可以启用基于风险的自适应身份验证功能和MFA。这完全符合题目要求，包
括地理位置、IP地址或设备的变化触发二次验证。
正确：Cognito用户池可以扩展到支持数百万用户，满足可扩展性需求。
B. Configure Amazon Cognito identity pools for user authentication. Enable multi-factor authentication (MFA).
错误：Amazon Cognito身份池主要用于临时访问AWS资源，而不是直接用于用户身份验证。身份池不支持基于风险的自适应身份验
证功能。
错误：虽然可以启用MFA，但身份池的功能不符合题目要求的地理位置、IP地址或设备变化触发二次验证。
C. Configure AWS Identity and Access Management (IAM) users for user authentication. Attach an IAM policy that allows
the AllowManageOwnUserMFA action.
错误：IAM用户主要用于管理AWS资源的访问权限，而不是设计用于大规模用户身份验证。
错误：IAM用户不支持基于风险的自适应身份验证功能，也不适合扩展到数百万用户。
D. Configure AWS IAM Identity Center (AWS Single Sign-On) authentication for user authentication. Configure the
permission sets to require multi-factor authentication (MFA).
错误：AWS IAM Identity Center主要用于企业级单点登录解决方案，适用于管理少量用户的权限集，而不是设计用于大规模用户身
份验证。
错误：虽然可以配置MFA，但它不支持基于风险的自适应身份验证功能。

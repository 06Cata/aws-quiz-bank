Question #857
A company plans to run a high performance computing (HPC) workload on Amazon EC2 Instances. The workload requires low-
latency network performance and high network throughput with tightly coupled node-to-node communication.
Which solution will meet these requirements?
一家公司计划在 Amazon EC2 实例上运行高性能计算 (HPC) 工作负载。该工作负载需要低延迟的网络性能和高网络吞吐量，以及紧
密耦合的节点间通信。
哪种解决方案可以满足这些要求？
A. Configure the EC2 instances to be part of a cluster placement group. Most Voted
将 EC2 实例配置为集群放置组的一部分。
B. Launch the EC2 instances with Dedicated Instance tenancy.
使用 Dedicated Instance 租赁方式启动 EC2 实例。
C. Launch the EC2 instances as Spot Instances.
将 EC2 实例作为 Spot Instances 启动。
D. Configure an On-Demand Capacity Reservation when the EC2 instances are launched.
在启动 EC2 实例时配置按需容量预留。
题目分析与解答
1. 考察的知识点
本题考察的是Amazon EC2的网络性能优化选项，特别是针对高性能计算（HPC）工作负载的配置，包括低延迟、高吞吐量和节点间
通信的需求。
Correct Answer: A
正确答案：A
Community vote distribution
社区投票分布
A (100%)
2. 选项分析
A. Configure the EC2 instances to be part of a cluster placement group.
正确：Cluster Placement Group是专门为需要低延迟、高吞吐量的工作负载设计的。它允许EC2实例在物理硬件上紧密分布，从而
优化节点间通信性能。
B. Launch the EC2 instances with Dedicated Instance tenancy.
错误：Dedicated Instance tenancy主要用于隔离实例以满足合规性或安全性需求，与网络性能优化无直接关系。
C. Launch the EC2 instances as Spot Instances.
错误：Spot Instances是成本优化选项，与网络性能优化无直接关系。它无法保证低延迟或高吞吐量。
D. Configure an On-Demand Capacity Reservation when the EC2 instances are launched.
错误：On-Demand Capacity Reservation用于确保实例容量可用性，与网络性能优化无直接关系。
3. 我的答案
A. Configure the EC2 instances to be part of a cluster placement group.
4. 官方答案
A. Configure the EC2 instances to be part of a cluster placement group.

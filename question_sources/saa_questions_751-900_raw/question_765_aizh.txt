Question #765
A development team is collaborating with another company to create an integrated product. The other company needs to access
an Amazon Simple Queue Service (Amazon SQS) queue that is contained in the development team's account. The other company
wants to poll the queue without giving up its own account permissions to do so.
How should a solutions architect provide access to the SQS queue?
一个开发团队正在与另一家公司合作创建一个集成产品。另一家公司需要访问开发团队账户中的一个 Amazon Simple Queue
Service (Amazon SQS) 队列。另一家公司希望轮询该队列，同时保留其自身账户权限。
解决方案架构师应该如何提供对 SQS 队列的访问权限？
A. Create an instance profile that provides the other company access to the SQS queue.
创建一个实例配置文件，为另一家公司提供对 SQS 队列的访问权限。
B. Create an IAM policy that provides the other company access to the SQS queue.
创建一个 IAM 策略，为另一家公司提供对 SQS 队列的访问权限。
C. Create an SQS access policy that provides the other company access to the SQS queue. Most Voted
创建一个 SQS 访问策略，为另一家公司提供对 SQS 队列的访问权限。
D. Create an Amazon Simple Notification Service (Amazon SNS) access policy that provides the other company access to the
SQS queue.
创建一个 Amazon Simple Notification Service (Amazon SNS) 访问策略，为另一家公司提供对 SQS 队列的访问权限。
题目分析与解答
1. 考察的知识点
Correct Answer: C
正确答案：C
Community vote distribution
社区投票分布
C (100%)
本题考察的是如何通过正确的权限管理机制为其他账户提供对 Amazon SQS 队列的访问权限。
涉及到 AWS 资源访问策略（如 IAM 策略和 SQS 访问策略）的使用。
2. 选项分析
A. Create an instance profile that provides the other company access to the SQS queue.
实例配置文件（Instance Profile）主要用于将 IAM 角色附加到 EC2 实例，以便实例可以访问 AWS 服务。
此选项与跨账户访问无关，因此不适合解决本题需求。
错误原因：实例配置文件无法直接为其他账户提供对 SQS 队列的访问权限。
B. Create an IAM policy that provides the other company access to the SQS queue.
IAM 策略可以为用户或角色提供权限，但它仅适用于当前账户内的用户或角色。
如果需要跨账户访问，IAM 策略无法直接满足需求。
错误原因：IAM 策略无法直接为其他账户提供跨账户访问权限。
C. Create an SQS access policy that provides the other company access to the SQS queue.
SQS 访问策略是专门用于管理 SQS 队列权限的资源级策略。
通过 SQS 访问策略，可以为其他账户提供跨账户访问权限。
正确原因：这是解决本题需求的正确方法，因为 SQS 访问策略支持跨账户权限管理。
D. Create an Amazon Simple Notification Service (Amazon SNS) access policy that provides the other company access to
the SQS queue.
SNS 访问策略用于管理 SNS 主题的权限，与 SQS 队列无关。
此选项与本题需求不符。
错误原因：SNS 访问策略无法为其他账户提供对 SQS 队列的访问权限。
3. 我的答案
答案：C. Create an SQS access policy that provides the other company access to the SQS queue.
4. 官方答案
答案：C. Create an SQS access policy that provides the other company access to the SQS queue.

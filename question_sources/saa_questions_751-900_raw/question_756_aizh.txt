Question #756
A company stores text files in Amazon S3. The text files include customer chat messages, date and time information, and customer
personally identifiable information (PII).
The company needs a solution to provide samples of the conversations to an external service provider for quality control. The
external service provider needs to randomly pick sample conversations up to the most recent conversation. The company must
not share the customer PII with the external service provider. The solution must scale when the number of customer conversations
increases.
Which solution will meet these requirements with the LEAST operational overhead?
一家公司将文本文件存储在 Amazon S3 中。这些文本文件包括客户聊天消息、日期和时间信息以及客户个人身份信息 (PII)。
该公司需要一种解决方案，以便向外部服务提供商提供对话样本进行质量控制。外部服务提供商需要随机选择样本对话，直到最新
的对话。公司不得与外部服务提供商共享客户 PII。该解决方案必须能够在客户对话数量增加时进行扩展。
哪种解决方案能够以最少的运营开销满足这些要求？
A. Create an Object Lambda Access Point. Create an AWS Lambda function that redacts the PII when the function reads the file.

题目分析与解答
1. 考察的知识点
Amazon S3的Object Lambda Access Point功能
AWS Lambda的使用与集成
数据隐私与PII（Personally Identifiable Information）保护
解决方案的可扩展性与低操作开销
2. 选项分析
A. Create an Object Lambda Access Point. Create an AWS Lambda function that redacts the PII when the function reads
the file. Instruct the external service provider to access the Object Lambda Access Point.
分析：
Object Lambda Access Point允许在访问S3对象时动态修改数据，适合用来实时编辑掉PII。
AWS Lambda可以轻松实现数据处理逻辑，且无需额外存储或复杂的操作流程。
此方案具有低操作开销，因为不需要额外的存储或定期批处理。
此方案完全满足题目要求，包括随机访问、PII保护和可扩展性。
结论：正确。
B. Create a batch process on an Amazon EC2 instance that regularly reads all new files, redacts the PII from the files, and
writes the redacted files to a different S3 bucket. Instruct the external service provider to access the bucket that does not
contain the PII.
分析：
使用EC2实例进行批处理需要额外的计算资源和管理开销。
定期读取和处理文件可能导致延迟，无法实时满足外部服务提供商的需求。
此方案的操作开销较高，不符合题目要求的“LEAST operational overhead”。
结论：错误。
C. Create a web application on an Amazon EC2 instance that presents a list of the files, redacts the PII from the files, and
allows the external service provider to download new versions of the files that have the PII redacted.
分析：
创建Web应用程序需要开发、部署和维护，增加了操作复杂性。
EC2实例的使用增加了成本和管理开销。
此方案的操作开销较高，不符合题目要求的“LEAST operational overhead”。
结论：错误。
D. Create an Amazon DynamoDB table. Create an AWS Lambda function that reads only the data in the files that does not
contain PII. Configure the Lambda function to store the non-PII data in the DynamoDB table when a new file is written to
Amazon S3. Grant the external service provider access to the DynamoDB table.
分析：
将数据存储到DynamoDB表中需要额外的存储资源和管理开销。
此方案无法直接满足外部服务提供商随机访问S3文件的需求。
此方案的操作开销较高，不符合题目要求的“LEAST operational overhead”。
结论：错误。

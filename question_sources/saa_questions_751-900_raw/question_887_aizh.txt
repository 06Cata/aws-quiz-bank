Question #887
A company plans to rehost an application to Amazon EC2 instances that use Amazon Elastic Block Store (Amazon EBS) as the
attached storage.
A solutions architect must design a solution to ensure that all newly created Amazon EBS volumes are encrypted by default. The
solution must also prevent the creation of unencrypted EBS volumes.
Which solution will meet these requirements?
一家公司计划将一个应用程序重新托管到使用 Amazon Elastic Block Store (Amazon EBS) 作为附加存储的 Amazon EC2 实例上。
解决方案架构师必须设计一个解决方案，以确保所有新创建的 Amazon EBS 卷默认都是加密的。该解决方案还必须防止创建未加密
的 EBS 卷。
哪种解决方案可以满足这些要求？
A. Configure the EC2 account attributes to always encrypt new EBS volumes. Most Voted
配置 EC2 账户属性以始终加密新的 EBS 卷。
B. Use AWS Config. Configure the encrypted-volumes identifier. Apply the default AWS Key Management Service (AWS KMS)
key.
使用 AWS Config。配置加密卷标识符。应用默认的 AWS Key Management Service (AWS KMS) 密钥。
C. Configure AWS Systems Manager to create encrypted copies of the EBS volumes. Reconfigure the EC2 instances to use the
encrypted volumes.
配置 AWS Systems Manager 创建 EBS 卷的加密副本。重新配置 EC2 实例以使用加密卷。
D. Create a customer managed key in AWS Key Management Service (AWS KMS). Configure AWS Migration Hub to use the key
when the company migrates workloads.
在 AWS Key Management Service (AWS KMS) 中创建一个客户管理密钥。配置 AWS Migration Hub 在公司迁移工作负载时使用
该密钥。
Correct Answer: A
正确答案：A
题目分析与解答
1. 考察的知识点
Amazon EC2实例与Amazon EBS卷的加密机制。
如何确保新创建的EBS卷默认加密。
如何防止创建未加密的EBS卷。
2. 选项分析
A. Configure the EC2 account attributes to always encrypt new EBS volumes.
正确：AWS允许通过配置EC2账户属性来确保所有新创建的EBS卷默认加密。这是最直接的解决方案，符合题目要求。
B. Use AWS Config. Configure the encrypted-volumes identifier. Apply the default AWS Key Management Service (AWS
KMS) key.
错误：虽然AWS Config可以用于监控资源的合规性，但它无法直接阻止未加密的EBS卷的创建。它只能在事后检测到未加密的卷。
C. Configure AWS Systems Manager to create encrypted copies of the EBS volumes. Reconfigure the EC2 instances to use
the encrypted volumes.
错误：此选项涉及额外的步骤（创建加密副本并重新配置实例），并不符合题目要求的“确保新创建的EBS卷默认加密”。
D. Create a customer managed key in AWS Key Management Service (AWS KMS). Configure AWS Migration Hub to use
the key when the company migrates workloads.
错误：AWS Migration Hub用于迁移工作负载，与确保EBS卷默认加密无关。
3. 我选的答案
A
4. 官方答案
A

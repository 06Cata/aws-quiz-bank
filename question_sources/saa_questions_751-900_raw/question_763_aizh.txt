Question #763
A company has 150 TB of archived image data stored on-premises that needs to be moved to the AWS Cloud within the next
month. The company’s current network connection allows up to 100 Mbps uploads for this purpose during the night only.
What is the MOST cost-effective mechanism to move this data and meet the migration deadline?
一家公司有 150 TB 的归档图像数据存储在本地，需要在下个月内迁移到 AWS 云。该公司的当前网络连接允许在夜间以最高 100
Mbps 的速度上传数据用于此目的。
最具成本效益的机制是什么，可以迁移这些数据并满足迁移期限？
A. Use AWS Snowmobile to ship the data to AWS.
使用 AWS Snowmobile 将数据运送到 AWS。

启用 Amazon S3 Transfer Acceleration 并安全地上传数据。

创建一个 Amazon S3 VPC 端点并建立 VPN 上传数据。

成本效益分析和选择最适合的迁移工具。

正确答案：B

2. 选项分析
A. Use AWS Snowmobile to ship the data to AWS.
错误原因：AWS Snowmobile适用于超大规模数据迁移（如PB级别），而本题只有150 TB数据，使用Snowmobile成本过高且不符合
实际需求。
B. Order multiple AWS Snowball devices to ship the data to AWS.
正确原因：AWS Snowball是一种经济高效的解决方案，适用于TB级别的数据迁移。多个Snowball设备可以满足150 TB数据的迁移需
求，并且可以在一个月内完成。
C. Enable Amazon S3 Transfer Acceleration and securely upload the data.
错误原因：Amazon S3 Transfer Acceleration可以加速数据上传，但受限于当前网络连接（100 Mbps），无法在一个月内完成150
TB数据的迁移。
D. Create an Amazon S3 VPC endpoint and establish a VPN to upload the data.
错误原因：使用VPN上传数据仍然受限于网络带宽（100 Mbps），无法满足一个月内完成迁移的要求。
3. 我的答案
B. Order multiple AWS Snowball devices to ship the data to AWS.
4. 官方答案
B. Order multiple AWS Snowball devices to ship the data to AWS.
5. 与官方答案的比较
我的答案与官方答案一致。
原因：AWS Snowball是最适合本题场景的解决方案，能够在规定时间内以较低成本完成数据迁移。

Question #795
A company is designing a tightly coupled high performance computing (HPC) environment in the AWS Cloud. The company needs
to include features that will optimize the HPC environment for networking and storage.
Which combination of solutions will meet these requirements? (Choose two.)
一家公司正在设计一个紧密耦合的高性能计算 (HPC) 环境，运行在 AWS 云中。该公司需要包含能够优化 HPC 环境的网络和存储功
能。
哪些解决方案的组合可以满足这些需求？（选择两个）
A. Create an accelerator in AWS Global Accelerator. Configure custom routing for the accelerator.
在 AWS Global Accelerator 中创建一个加速器。为加速器配置自定义路由。
B. Create an Amazon FSx for Lustre file system. Configure the file system with scratch storage. Most Voted
创建一个 Amazon FSx for Lustre 文件系统。使用临时存储配置文件系统。
C. Create an Amazon CloudFront distribution. Configure the viewer protocol policy to be HTTP and HTTPS.
创建一个 Amazon CloudFront 分发。将查看器协议策略配置为 HTTP 和 HTTPS。
D. Launch Amazon EC2 instances. Attach an Elastic Fabric Adapter (EFA) to the instances. Most Voted
启动 Amazon EC2 实例。为实例附加 Elastic Fabric Adapter (EFA)。
E. Create an AWS Elastic Beanstalk deployment to manage the environment.
创建一个 AWS Elastic Beanstalk 部署来管理环境。

2. 选项分析
A. Create an accelerator in AWS Global Accelerator. Configure custom routing for the accelerator.
错误：AWS Global Accelerator 是一种全球流量管理服务，主要用于优化跨区域的网络性能，而不是专门为高性能计算设计的。它
不适合用于 HPC 环境的内部网络优化。
B. Create an Amazon FSx for Lustre file system. Configure the file system with scratch storage.
正确：Amazon FSx for Lustre 是专门为高性能计算设计的文件系统，提供低延迟、高吞吐量的存储解决方案。使用临时存储
（scratch storage）可以满足 HPC 环境中对快速存储的需求。
C. Create an Amazon CloudFront distribution. Configure the viewer protocol policy to be HTTP and HTTPS.
错误：Amazon CloudFront 是一个内容分发网络（CDN）服务，主要用于优化内容交付，而不是用于 HPC 环境的内部网络或存储
优化。
D. Launch Amazon EC2 instances. Attach an Elastic Fabric Adapter (EFA) to the instances.
正确：Elastic Fabric Adapter (EFA) 是一种网络接口，专门设计用于支持高性能计算应用程序的低延迟、高吞吐量通信。它非常适合
HPC 环境。
E. Create an AWS Elastic Beanstalk deployment to manage the environment.
错误：AWS Elastic Beanstalk 是一个应用程序部署和管理服务，适用于 Web 应用程序和后端服务，而不是 HPC 环境的设计和优
化。

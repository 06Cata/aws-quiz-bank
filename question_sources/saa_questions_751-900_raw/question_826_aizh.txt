Question #826
A company is migrating applications from an on-premises Microsoft Active Directory that the company manages to AWS. The
company deploys the applications in multiple AWS accounts. The company uses AWS Organizations to manage the accounts
centrally.
The company's security team needs a single sign-on solution across all the company's AWS accounts. The company must continue
to manage users and groups that are in the on-premises Active Directory.
Which solution will meet these requirements?
一家公司正在将其管理的本地 Microsoft Active Directory 上的应用程序迁移到 AWS。该公司在多个 AWS 账户中部署应用程序，并
使用 AWS Organizations 来集中管理这些账户。
该公司的安全团队需要一个跨所有 AWS 账户的单点登录解决方案。该公司必须继续管理位于本地 Active Directory 中的用户和组。
哪种解决方案可以满足这些要求？
A. Create an Enterprise Edition Active Directory in AWS Directory Service for Microsoft Active Directory. Configure the Active
Directory to be the identity source for AWS IAM Identity Center.
在 AWS Directory Service for Microsoft Active Directory 中创建一个企业版 Active Directory。将该 Active Directory 配置为 AWS
IAM Identity Center 的身份源。
B. Enable AWS IAM Identity Center. Configure a two-way forest trust relationship to connect the company's self-managed
Active Directory with IAM Identity Center by using AWS Directory Service for Microsoft Active Directory. Most Voted
启用 AWS IAM Identity Center。通过使用 AWS Directory Service for Microsoft Active Directory，配置双向森林信任关系以连接
公司的自管理 Active Directory 和 IAM Identity Center。
C. Use AWS Directory Service and create a two-way trust relationship with the company's self-managed Active Directory.
使用 AWS Directory Service 并与公司的自管理 Active Directory 创建双向信任关系。
D. Deploy an identity provider (IdP) on Amazon EC2. Link the IdP as an identity source within AWS IAM Identity Center.
在 Amazon EC2 上部署一个身份提供商 (IdP)。将该 IdP 作为 AWS IAM Identity Center 内的身份源进行链接。
Correct Answer: B
正确答案：B
题目分析与解答
1. 考察的知识点
AWS IAM Identity Center (原 AWS SSO) 的功能和配置。
如何与现有的 Microsoft Active Directory 集成。
AWS Directory Service 的使用场景。
跨账户的单点登录解决方案。
2. 选项分析
A. Create an Enterprise Edition Active Directory in AWS Directory Service for Microsoft Active Directory. Configure the
Active Directory to be the identity source for AWS IAM Identity Center.
错误：此选项建议在 AWS 中创建一个新的 Active Directory，但题目要求继续使用现有的 on-premises Active Directory。
此选项没有解决如何与现有的 Active Directory 集成的问题。
B. Enable AWS IAM Identity Center. Configure a two-way forest trust relationship to connect the company's self-managed
Active Directory with IAM Identity Center by using AWS Directory Service for Microsoft Active Directory.
正确：此选项直接解决了题目要求，即通过双向森林信任关系将现有的 on-premises Active Directory 与 AWS IAM Identity Center
集成。
这种方法允许继续使用现有的用户和组，同时实现跨账户的单点登录。
C. Use AWS Directory Service and create a two-way trust relationship with the company's self-managed Active Directory.
错误：虽然提到了双向信任关系，但没有提到如何与 AWS IAM Identity Center 集成。
此选项缺乏对单点登录解决方案的具体说明。
D. Deploy an identity provider (IdP) on Amazon EC2. Link the IdP as an identity source within AWS IAM Identity Center.
错误：此选项建议在 EC2 上部署一个新的 IdP，但题目要求继续使用现有的 on-premises Active Directory。
此方法增加了复杂性，并不符合题目要求的最佳实践。
3. 我选的答案
B. Enable AWS IAM Identity Center. Configure a two-way forest trust relationship to connect the company's self-managed Active
Directory with IAM Identity Center by using AWS Directory Service for Microsoft Active Directory.
4. 官方答案
B. Enable AWS IAM Identity Center. Configure a two-way forest trust relationship to connect the company's self-managed Active
Directory with IAM Identity Center by using AWS Directory Service for Microsoft Active Directory.
5. 与官方答案的比较
我的答案与官方答案一致。
原因：选项 B 是唯一符合题目要求的解决方案，既支持现有的 on-premises Active Directory，又实现了跨账户的单点登录。

An analytics company uses Amazon VPC to run its multi-tier services. The company wants to use RESTful APIs to offer a web
analytics service to millions of users. Users must be verified by using an authentication service to access the APIs.
Which solution will meet these requirements with the MOST operational efficiency?
A. Configure an Amazon Cognito user pool for user authentication. Implement Amazon API Gateway REST APIs with a Cognito
authorizer. Most Voted
B. Configure an Amazon Cognito identity pool for user authentication. Implement Amazon API Gateway HTTP APIs with a
Cognito authorizer.
C. Configure an AWS Lambda function to handle user authentication. Implement Amazon API Gateway REST APIs with a
Lambda authorizer.
D. Configure an IAM user to handle user authentication. Implement Amazon API Gateway HTTP APIs with an IAM authorizer.

for basic authentication checks, as Cognito is a managed service optimized for user authentication, while a Lambda authorizer introduces an
additional layer of processing with potential cold start overhead, making it better suited for complex, custom authorization logic. But Lambda is more
flexible

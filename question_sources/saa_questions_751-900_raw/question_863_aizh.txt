Question #863
A company uses Amazon RDS with default backup settings for its database tier. The company needs to make a daily backup of the
database to meet regulatory requirements. The company must retain the backups for 30 days.
Which solution will meet these requirements with the LEAST operational overhead?
一家公司使用默认备份设置的 Amazon RDS 作为其数据库层。该公司需要每天对数据库进行备份以满足监管要求。公司必须保留备
份 30 天。
哪种解决方案可以以最少的操作开销满足这些要求？
A. Write an AWS Lambda function to create an RDS snapshot every day.
编写一个 AWS Lambda 函数每天创建一个 RDS 快照。

C. Use AWS Systems Manager Maintenance Windows to modify the RDS backup retention period.
使用 AWS Systems Manager Maintenance Windows 修改 RDS 备份保留期。
D. Create a manual snapshot every day by using the AWS CLI. Modify the RDS backup retention period.
通过使用 AWS CLI 每天创建一个手动快照。修改 RDS 备份保留期。

1. 考察的知识点
Amazon RDS的备份机制，包括自动备份和手动快照。
如何设置备份保留期以满足业务和合规需求。
Correct Answer: B
正确答案：B
Community vote distribution
社区投票分布
B (100%)
选择最少的操作开销（Least Operational Overhead）的解决方案。
2. 选项分析
A. 编写一个 AWS Lambda 函数每天创建一个 RDS 快照。
正确性：可以通过Lambda函数实现每天创建快照，但需要额外编写代码和维护Lambda函数。
操作开销：较高，因为需要开发、部署和维护Lambda函数。
结论：不符合“最少操作开销”的要求。
B. 修改 RDS 数据库以将自动备份的保留期设置为 30 天。
正确性：RDS支持自动备份，并允许设置备份保留期，最多可设置为35天。
操作开销：最低，因为只需修改一个配置，无需额外开发或维护。
结论：符合“最少操作开销”的要求，是正确答案。
C. 使用 AWS Systems Manager Maintenance Windows 修改 RDS 备份保留期。
正确性：AWS Systems Manager Maintenance Windows主要用于计划维护任务，不直接用于修改RDS备份保留期。
操作开销：较高，因为需要配置维护窗口并关联任务。
结论：不符合题目要求。
D. 通过使用 AWS CLI 每天创建一个手动快照。修改 RDS 备份保留期。
正确性：手动快照可以满足备份需求，但需要每天执行CLI命令，且手动快照与自动备份是独立的机制。
操作开销：较高，因为需要每天手动操作或编写脚本自动化。
结论：不符合“最少操作开销”的要求。
3. 我选的答案
B. 修改 RDS 数据库以将自动备份的保留期设置为 30 天。
4. 官方答案
B. 修改 RDS 数据库以将自动备份的保留期设置为 30 天。

我的答案与官方答案一致。
原因：选项B是最符合题目要求的解决方案，既满足备份需求，又具有最少的操作开销。

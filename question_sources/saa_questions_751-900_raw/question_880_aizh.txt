Question #880
A company that uses AWS Organizations runs 150 applications across 30 different AWS accounts. The company used AWS Cost
and Usage Report to create a new report in the management account. The report is delivered to an Amazon S3 bucket that is
replicated to a bucket in the data collection account.
The company’s senior leadership wants to view a custom dashboard that provides NAT gateway costs each day starting at the
beginning of the current month.
Which solution will meet these requirements?

题目分析与解答
1. 考察的知识点
AWS Cost and Usage Report的使用和查询方法。
Amazon QuickSight的功能和数据可视化能力。
Amazon Athena的查询能力和与S3的集成。
AWS DataSync的功能和适用场景。
Amazon CloudWatch的监控和数据展示能力。
2. 选项分析
A. Share an Amazon QuickSight dashboard that includes the requested table visual. Configure QuickSight to use AWS
DataSync to query the new report.
错误：AWS DataSync主要用于数据迁移和同步，而不是用于查询数据。QuickSight无法直接使用DataSync来查询Cost and Usage
Report。
B. Share an Amazon QuickSight dashboard that includes the requested table visual. Configure QuickSight to use Amazon
Athena to query the new report.
正确：Amazon Athena可以直接查询存储在S3中的Cost and Usage Report。QuickSight可以通过Athena作为数据源来创建可视化仪
表板。
C. Share an Amazon CloudWatch dashboard that includes the requested table visual. Configure CloudWatch to use AWS
DataSync to query the new report.
错误：CloudWatch主要用于监控和日志分析，不适合用于查询Cost and Usage Report。此外，DataSync不支持查询功能。
D. Share an Amazon CloudWatch dashboard that includes the requested table visual. Configure CloudWatch to use
Amazon Athena to query the new report.
错误：CloudWatch不支持通过Athena查询数据。CloudWatch的主要功能是监控和告警，而不是数据可视化和查询。

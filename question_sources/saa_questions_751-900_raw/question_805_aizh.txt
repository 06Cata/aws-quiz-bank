Question #805
A company runs containers in a Kubernetes environment in the company's local data center. The company wants to use Amazon
Elastic Kubernetes Service (Amazon EKS) and other AWS managed services. Data must remain locally in the company's data center
and cannot be stored in any remote site or cloud to maintain compliance.
Which solution will meet these requirements?
一家公司在其本地数据中心的 Kubernetes 环境中运行容器。该公司希望使用 Amazon Elastic Kubernetes Service (Amazon EKS) 和
其他 AWS 托管服务。为了保持合规性，数据必须保留在公司的本地数据中心，不能存储在任何远程站点或云中。
哪种解决方案能够满足这些要求？
A. Deploy AWS Local Zones in the company's data center.
在公司的数据中心部署 AWS Local Zones。
B. Use an AWS Snowmobile in the company's data center.
在公司的数据中心使用 AWS Snowmobile。
C. Install an AWS Outposts rack in the company's data center. Most Voted
在公司的数据中心安装 AWS Outposts 机架。
D. Install an AWS Snowball Edge Storage Optimized node in the data center.
在数据中心安装 AWS Snowball Edge Storage Optimized 节点。
题目分析与解答
1. 考察的知识点
Correct Answer: C
正确答案：C
Community vote distribution
社区投票分布
C (100%)
本题考察的是AWS混合云解决方案，特别是如何在本地数据中心使用AWS服务，同时满足数据合规性要求。
涉及的服务包括AWS Outposts、Local Zones、Snowmobile和Snowball Edge。
2. 选项分析
A. Deploy AWS Local Zones in the company's data center.
错误原因：AWS Local Zones是AWS在特定地理位置提供的扩展区域，允许用户将计算和存储资源部署在靠近用户的地方，但它并
不能直接部署在客户的数据中心内。
Local Zones仍然是AWS管理的区域，数据最终会存储在AWS的基础设施中，不符合题目要求的数据必须留在本地的条件。
B. Use an AWS Snowmobile in the company's data center.
错误原因：AWS Snowmobile是一种用于大规模数据迁移的解决方案，通常用于将PB级数据从本地数据中心迁移到AWS云。它不适
用于运行持续的计算或存储服务。
题目要求是运行Amazon EKS和其他AWS服务，而不是数据迁移，因此Snowmobile不符合要求。
C. Install an AWS Outposts rack in the company's data center.
正确原因：AWS Outposts是一种混合云解决方案，允许用户在自己的数据中心运行AWS服务（如Amazon EKS），同时数据可以完
全保留在本地以满足合规性要求。
Outposts是专门设计用于在本地数据中心提供AWS服务的解决方案，完全符合题目要求。
D. Install an AWS Snowball Edge Storage Optimized node in the data center.
错误原因：AWS Snowball Edge Storage Optimized主要用于边缘计算和数据迁移，虽然可以在本地运行一些计算任务，但它不支持
完整的AWS服务（如Amazon EKS）。
Snowball Edge的功能有限，无法满足题目中运行Amazon EKS的需求。

答案：C

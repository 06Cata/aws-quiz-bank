Question #787
A company has an organization in AWS Organizations that has all features enabled. The company requires that all API calls and
logins in any existing or new AWS account must be audited. The company needs a managed solution to prevent additional work
and to minimize costs. The company also needs to know when any AWS account is not compliant with the AWS Foundational
Security Best Practices (FSBP) standard.
Which solution will meet these requirements with the LEAST operational overhead?
一家公司在启用了所有功能的 AWS Organizations 中拥有一个组织。该公司要求必须审计任何现有或新建 AWS 账户中的所有 API
调用和登录活动。公司需要一个托管解决方案，以减少额外工作并尽量降低成本。公司还需要知道任何 AWS 账户是否未遵守 AWS
基础安全最佳实践（FSBP）标准。
哪种解决方案能够以最低的运营开销满足这些要求？

题目分析与解答
1. 考察的知识点
AWS Organizations的功能与管理
AWS Control Tower的使用与优势
AWS Security Hub的功能与合规性检查
如何实现跨账户的日志审计与安全最佳实践
2. 选项分析
A. Deploy an AWS Control Tower environment in the Organizations management account. Enable AWS Security Hub and
AWS Control Tower Account Factory in the environment.
正确：AWS Control Tower是一个托管服务，可以帮助公司快速设置和管理多账户环境，同时提供合规性检查和日志审计功能。
正确：在Organizations管理账户中部署AWS Control Tower可以集中管理所有账户，减少操作开销。
正确：启用AWS Security Hub可以自动检查AWS Foundational Security Best Practices (FSBP)标准的合规性。
B. Deploy an AWS Control Tower environment in a dedicated Organizations member account. Enable AWS Security Hub
and AWS Control Tower Account Factory in the environment.
错误：将AWS Control Tower部署在一个专用的成员账户中会增加管理复杂性，因为管理账户通常是用于集中管理的最佳选择。
错误：这种方法不符合“最小化操作开销”的要求。
C. Use AWS Managed Services (AMS) Accelerate to build a multi-account landing zone (MALZ). Submit an RFC to self-
service provision Amazon GuardDuty in the MALZ.
错误：虽然Amazon GuardDuty可以提供威胁检测，但它并不直接检查AWS Foundational Security Best Practices (FSBP)标准的合规
性。
错误：使用AMS Accelerate会增加成本和操作开销，不符合题目要求的“最小化成本和操作开销”。
D. Use AWS Managed Services (AMS) Accelerate to build a multi-account landing zone (MALZ). Submit an RFC to self-
service provision AWS Security Hub in the MALZ.
错误：虽然AWS Security Hub可以检查FSBP标准的合规性，但使用AMS Accelerate会增加成本和操作开销。
错误：AMS Accelerate的复杂性和成本不符合题目要求的“最小化成本和操作开销”。

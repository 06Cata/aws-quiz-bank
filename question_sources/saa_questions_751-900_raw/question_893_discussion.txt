A company wants to isolate its workloads by creating an AWS account for each workload. The company needs a solution that
centrally manages networking components for the workloads. The solution also must create accounts with automatic security
controls (guardrails).
Which solution will meet these requirements with the LEAST operational overhead?
A. Use AWS Control Tower to deploy accounts. Create a networking account that has a VPC with private subnets and public
subnets. Use AWS Resource Access Manager (AWS RAM) to share the subnets with the workload accounts. Most Voted
B. Use AWS Organizations to deploy accounts. Create a networking account that has a VPC with private subnets and public
subnets. Use AWS Resource Access Manager (AWS RAM) to share the subnets with the workload accounts.
C. Use AWS Control Tower to deploy accounts. Deploy a VPC in each workload account. Configure each VPC to route through
an inspection VPC by using a transit gateway attachment.
D. Use AWS Organizations to deploy accounts. Deploy a VPC in each workload account. Configure each VPC to route through
an inspection VPC by using a transit gateway attachment.

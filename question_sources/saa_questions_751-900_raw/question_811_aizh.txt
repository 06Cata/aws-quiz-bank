题目分析与解答
1. 考察的知识点
Amazon Managed Grafana的工作空间配置选项。
如何安全地连接Amazon RDS数据库作为数据源。
VPC和PrivateLink的使用场景。
2. 选项分析
A. Create an Amazon Managed Grafana workspace without a VPC. Create a public endpoint for the RDS database.
Configure the public endpoint as a data source in Amazon Managed Grafana.
错误：创建公共端点会将数据暴露在互联网中，不符合题目要求的“不会暴露数据到互联网”的条件。
B. Create an Amazon Managed Grafana workspace in a VPC. Create a private endpoint for the RDS database. Configure
the private endpoint as a data source in Amazon Managed Grafana.
正确：在VPC中创建Grafana工作空间，并使用私有端点连接RDS数据库，确保数据不会暴露到互联网，符合题目要求。
C. Create an Amazon Managed Grafana workspace without a VPC. Create an AWS PrivateLink endpoint to establish a
connection between Amazon Managed Grafana and Amazon RDS. Set up Amazon RDS as a data source in Amazon
Managed Grafana.
错误：Amazon Managed Grafana工作空间需要在VPC中才能使用PrivateLink连接RDS数据库。没有VPC的工作空间无法直接使用
PrivateLink。
D. Create an Amazon Managed Grafana workspace in a VPC. Create a public endpoint for the RDS database. Configure the
public endpoint as a data source in Amazon Managed Grafana.
错误：虽然Grafana工作空间在VPC中，但使用公共端点会将数据暴露到互联网，不符合题目要求。

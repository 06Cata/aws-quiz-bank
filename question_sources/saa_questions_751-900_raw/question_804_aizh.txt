Question #804
A company has an Amazon S3 data lake. The company needs a solution that transforms the data from the data lake and loads the
data into a data warehouse every day. The data warehouse must have massively parallel processing (MPP) capabilities.
Data analysts then need to create and train machine learning (ML) models by using SQL commands on the data. The solution must
use serverless AWS services wherever possible.
Which solution will meet these requirements?
一家公司拥有一个 Amazon S3 数据湖。该公司需要一种解决方案，将数据从数据湖中转换并每天将数据加载到数据仓库中。数据
仓库必须具有大规模并行处理（MPP）能力。
数据分析师随后需要使用 SQL 命令在数据上创建和训练机器学习（ML）模型。解决方案必须尽可能使用无服务器的 AWS 服务。

A. Run a daily Amazon EMR job to transform the data and load the data into Amazon Redshift. Use Amazon Redshift ML to

题目分析与解答
1. 考察的知识点
Amazon S3数据湖的使用
数据转换和加载到数据仓库的流程
数据仓库的MPP（Massively Parallel Processing）能力
使用SQL命令进行机器学习模型的创建和训练
优先使用无服务器（Serverless）AWS服务
2. 选项分析
A. Run a daily Amazon EMR job to transform the data and load the data into Amazon Redshift. Use Amazon Redshift ML
to create and train the ML models.
Amazon EMR是一个托管的Hadoop框架，适合大规模数据处理，但它不是无服务器服务。
Amazon Redshift支持MPP并且可以使用Redshift ML进行机器学习模型的创建和训练。
此选项不完全符合“使用无服务器服务”的要求，因此不符合题目要求。
B. Run a daily Amazon EMR job to transform the data and load the data into Amazon Aurora Serverless. Use Amazon
Aurora ML to create and train the ML models.
Amazon Aurora Serverless是一个无服务器关系数据库，但它不具备MPP能力。
Amazon Aurora ML的功能有限，无法直接满足题目中“使用SQL命令进行机器学习模型创建和训练”的要求。
此选项不符合MPP能力的要求，因此不正确。
C. Run a daily AWS Glue job to transform the data and load the data into Amazon Redshift Serverless. Use Amazon
Redshift ML to create and train the ML models.
AWS Glue是一个无服务器ETL服务，适合数据转换和加载任务。
Amazon Redshift Serverless支持MPP并且可以使用Redshift ML进行机器学习模型的创建和训练。
此选项完全符合题目要求，包括无服务器服务、MPP能力以及SQL命令进行机器学习模型创建和训练。
D. Run a daily AWS Glue job to transform the data and load the data into Amazon Athena tables. Use Amazon Athena ML
to create and train the ML models.
AWS Glue是无服务器ETL服务，适合数据转换和加载任务。
Amazon Athena是一个无服务器查询服务，但它不具备MPP能力。
Amazon Athena目前不支持直接使用SQL命令进行机器学习模型的创建和训练。
此选项不符合MPP能力的要求，因此不正确。
3. 我选的答案
C. Run a daily AWS Glue job to transform the data and load the data into Amazon Redshift Serverless. Use Amazon
Redshift ML to create and train the ML models.

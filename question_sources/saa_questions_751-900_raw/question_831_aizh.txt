Question #831
A company needs a secure connection between its on-premises environment and AWS. This connection does not need high
bandwidth and will handle a small amount of traffic. The connection should be set up quickly.
What is the MOST cost-effective method to establish this type of connection?
一家公司需要在其本地环境和 AWS 之间建立一个安全连接。该连接不需要高带宽，并且将处理少量流量。连接应该能够快速设
置。
建立这种类型连接的最具成本效益的方法是什么？
A. Implement a client VPN.
实施一个 client VPN。
B. Implement AWS Direct Connect.
实施 AWS Direct Connect。
C. Implement a bastion host on Amazon EC2.
在 Amazon EC2 上实施一个 bastion host。
D. Implement an AWS Site-to-Site VPN connection. Most Voted
实施一个 AWS Site-to-Site VPN connection。
题目分析与解答
1. 考察的知识点
本题考察的是AWS网络连接选项的选择，包括VPN、Direct Connect、EC2 Bastion Host等，重点关注成本效益、安全性和快速部署
能力。
Correct Answer: D
正确答案：D
Community vote distribution
社区投票分布
D (100%)
2. 选项分析
A. Implement a client VPN.
错误。Client VPN主要用于用户设备与AWS资源之间的安全连接，适合远程用户访问，而不是用于连接整个本地环境与AWS。它不
符合题目要求的“连接本地环境与AWS”的场景。
B. Implement AWS Direct Connect.
错误。AWS Direct Connect提供专用的高带宽连接，适合需要稳定、高速连接的场景，但设置时间较长且成本较高，不符合题目要
求的“快速设置”和“低成本”的需求。
C. Implement a bastion host on Amazon EC2.
错误。Bastion Host用于安全地管理EC2实例的访问，而不是用于连接本地环境与AWS。它不适合题目要求的场景。
D. Implement an AWS Site-to-Site VPN connection.
正确。AWS Site-to-Site VPN连接可以快速设置，成本较低，适合处理小流量的场景，并且能够安全地连接本地环境与AWS资源。
3. 我的答案
D. Implement an AWS Site-to-Site VPN connection.
4. 官方答案
D. Implement an AWS Site-to-Site VPN connection.

题目分析与解答
1. 考察的知识点
本题考察的是如何使用AWS服务检测和处理不良内容，特别是利用现有的机器学习服务（如Amazon Rekognition）而无需自行训练
模型。
涉及的服务包括Amazon Rekognition、AWS Lambda、Amazon SageMaker、Amazon Comprehend等。
2. 选项分析
A. Create and deploy a model by using Amazon SageMaker Autopilot. Create a real-time endpoint that the web
application invokes when new photos are uploaded.
错误：Amazon SageMaker Autopilot需要训练一个机器学习模型，而题目明确要求不涉及训练模型。
虽然SageMaker可以实现类似功能，但不符合题目要求。
B. Create an AWS Lambda function that uses Amazon Rekognition to detect unwanted content. Create a Lambda function
URL that the web application invokes when new photos are uploaded.
正确：Amazon Rekognition是一种预训练的服务，可以直接用于检测不良内容，无需训练模型。
使用AWS Lambda函数和Lambda函数URL可以轻松集成到Web应用程序中，满足题目要求。
C. Create an Amazon CloudFront function that uses Amazon Comprehend to detect unwanted content. Associate the
function with the web application.
错误：Amazon Comprehend主要用于自然语言处理（NLP），而不是图像内容检测。
此外，CloudFront函数不适合处理复杂的图像分析任务。
D. Create an AWS Lambda function that uses Amazon Rekognition Video to detect unwanted content. Create a Lambda
function URL that the web application invokes when new photos are uploaded.
错误：Amazon Rekognition Video是用于视频分析的服务，而题目要求处理的是照片（静态图像）。
使用Rekognition Video会导致不必要的复杂性和资源浪费。

我选择的答案是：B

官方正确答案是：B

理由：Amazon Rekognition是预训练的服务，直接满足题目要求；AWS Lambda函数和Lambda函数URL提供了简单的集成方式。

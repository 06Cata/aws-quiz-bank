A company maintains its accounting records in a custom application that runs on Amazon EC2 instances. The company needs to
migrate the data to an AWS managed service for development and maintenance of the application data. The solution must require
minimal operational support and provide immutable, cryptographically verifiable logs of data changes.
Which solution will meet these requirements MOST cost-effectively?
一家公司在运行于Amazon EC2实例上的自定义应用程序中维护其会计记录。该公司需要将数据迁移到一个AWS托管服务，以便开
发和维护应用程序数据。解决方案必须要求最少的运营支持，并提供不可变的、可加密验证的数据更改日志。
哪个解决方案将最具成本效益地满足这些要求？

C. Copy the records from the application into an Amazon Timestream database.
将应用程序中的记录复制到一个Amazon Timestream数据库中。

2. 选项分析
A. Copy the records from the application into an Amazon Redshift cluster.
Amazon Redshift是一个数据仓库服务，适用于分析和查询大规模数据集。
它不提供不可变、加密验证的日志记录功能。
因此，此选项不符合题目要求。
B. Copy the records from the application into an Amazon Neptune cluster.
Amazon Neptune是一个图数据库服务，适用于处理关系复杂的图数据。
它不支持不可变、加密验证的日志记录功能。
此选项不符合题目要求。
C. Copy the records from the application into an Amazon Timestream database.
Amazon Timestream是一个时间序列数据库，适用于处理时间序列数据。
它不提供不可变、加密验证的日志记录功能。
此选项不符合题目要求。
D. Copy the records from the application into an Amazon Quantum Ledger Database (Amazon QLDB) ledger.
Amazon QLDB是一个托管分类账数据库，专门设计用于提供不可变、加密验证的日志记录功能。
它完全符合题目要求，并且需要最少的操作支持。
此选项是正确答案。

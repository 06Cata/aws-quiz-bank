Question #898
A company runs workloads in the AWS Cloud. The company wants to centrally collect security data to assess security across the
entire company and to improve workload protection.
Which solution will meet these requirements with the LEAST development effort?
一家公司在 AWS 云中运行工作负载。该公司希望集中收集安全数据，以评估整个公司的安全性并提高工作负载保护。
哪种解决方案能够以最少的开发工作量满足这些需求？
A. Configure a data lake in AWS Lake Formation. Use AWS Glue crawlers to ingest the security data into the data lake.
在 AWS Lake Formation 中配置一个数据湖。使用 AWS Glue crawlers 将安全数据导入数据湖。
B. Configure an AWS Lambda function to collect the security data in .csv format. Upload the data to an Amazon S3 bucket.
配置一个 AWS Lambda 函数以收集 .csv 格式的安全数据。将数据上传到 Amazon S3 bucket。
C. Configure a data lake in Amazon Security Lake to collect the security data. Upload the data to an Amazon S3 bucket.

题目分析与解答
1. 考察的知识点
本题考察的是AWS服务中与安全数据收集和集中管理相关的解决方案，特别是Amazon Security Lake的使用。
Correct Answer: C
正确答案：C
Community vote distribution
社区投票分布
C (100%)
重点在于选择一个能够以最少开发工作量实现安全数据集中收集和管理的服务。
2. 选项分析
A. Configure a data lake in AWS Lake Formation. Use AWS Glue crawlers to ingest the security data into the data lake.
错误：AWS Lake Formation主要用于构建和管理数据湖，但它并不是专门为安全数据设计的解决方案。
需要额外的开发工作来配置Glue爬虫以收集和处理安全数据。
B. Configure an AWS Lambda function to collect the security data in .csv format. Upload the data to an Amazon S3
bucket.
错误：虽然Lambda可以用来收集数据并上传到S3，但这种方法需要编写和维护Lambda函数，开发工作量较大。
此外，这种方法缺乏专门的安全数据管理功能。
C. Configure a data lake in Amazon Security Lake to collect the security data. Upload the data to an Amazon S3 bucket.
正确：Amazon Security Lake是专门设计用于集中收集和管理安全数据的服务。
它支持自动化数据收集和存储，减少了开发工作量。
D. Configure an AWS Database Migration Service (AWS DMS) replication instance to load the security data into an
Amazon RDS cluster.
错误：AWS DMS主要用于数据库迁移，而不是安全数据的集中收集和管理。
使用RDS存储安全数据需要额外的开发工作来设计和管理数据库架构。
3. 我的答案
我的选择是：C
4. 官方答案
官方正确答案是：C

我的答案与官方答案一致。
原因：Amazon Security Lake是专门设计用于集中收集和管理安全数据的服务，能够以最少的开发工作量满足题目要求。

哪一項 AWS 服務可自動掃描 Amazon EC2 執行個體的軟體漏洞與非預期網路暴露？

Which AWS service automatically scans EC2 instances for software vulnerabilities and unintended network exposure?

A

AWS Trusted Advisor

Checks best practices but is not the EC2 vulnerability scanner.

B

Amazon Inspector

Automated vulnerability management for EC2 workloads.

C

AWS Config

Tracks resource configuration history.

D

Amazon GuardDuty

Threat detection and continuous monitoring.

正確答案：B. Amazon Inspector
